# 11.21 更多阅读材料

## 本节讲什么

**Further reading** — 离开 **Expert C ch11** 的 **入门桥** 后，往 **现代 C++**、**ABI**、**内核 C 风格** 继续深化的 **书单与方向**。本节并做 **全书收束**：**ch1–ch10 奠定 C 地基（声明、链接、内存、数组、指针、函数表）→ ch11 打开 C++ 大门** — 读者应 **带着底层视角** 读 **Stroustrup / Meyers / Sutter**，而非 **把 C++ 当第二门 Java**。

## 要点

### 推荐延续阅读

| 方向 | 材料 | 与本书衔接 |
|------|------|------------|
| **C++ 语言创立** | Stroustrup *The C++ Programming Language* | **11.1–11.4** OOP 原意 |
| **实践准则** | Meyers *Effective C++* / *Effective Modern C++* | **11.17** 陷阱清单 |
| **异常安全/RAII** | Sutter *Exceptional C++* | **11.4 ctor/dtor** |
| **C 深度** | 本书 ch1–10 + K&R | **地基** |
| **内核多态** | LDD *Linux Device Drivers* | **11.15 file_operations** |
| **ABI/对象布局** | Itanium C++ ABI 文档 | **11.14–11.16 vptr** |
| **现代特性** | cppreference.com C++11/14/17/20 | **`constexpr`、move、智能指针** |

### 全书路径图（Expert C 主线）

```text
ch01–02  历史与预处理     → 宏的代价 → 11.17 constexpr
ch03     声明分析           → 读 C++ 声明延伸
ch04–05  数组≠指针、链接    → extern "C"、ODR
ch06–07  运行时、内存       → new/delete、RAII
ch08     软件规则之难       → C++ 规则更多层
ch09–10  数组与指针收束     → 函数表 → vtable
ch11     C++ for C 程序员   → 桥梁，非终点
```

**一句话**：**没有 ch1–10，ch11 的 virtual 只是魔法；有了 ch1–10，virtual 就是 ops 表。**

### 学习顺序建议

1. **复盘 ch10** — 函数指针、**`void*`**、动态内存
2. **精读 ch11.13–11.17** — 多态 + C++ 严格性
3. **做 demo01–04** — 链接、引用、多态、内存
4. **对照 checklist** — 改一个小 C 模块为 C++ 子集
5. **选 Effective Modern C++** — 按项规则巩固

### 不必急于掌握

- **模板元编程** — 非系统编程首要
- **多继承细节** — **11.9** 已警告
- **完整 STL 算法库** — 用时再查

**内核/DPDK 读者**：**11.15 + checklist §7** 足够支撑 **读源码**；C++  **用户态工具链** 再深化。

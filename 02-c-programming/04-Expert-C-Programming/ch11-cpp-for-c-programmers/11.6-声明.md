# 11.6 声明

## 本节讲什么

C++ **继承 C 的声明语法** 并扩展 **类、引用、命名空间、函数重载** 等规则。对 C 老手，**ch03 声明分析** 仍是基本功；本节聚焦 **C++ 新增/变严** 的部分：**成员函数声明**、**作用域解析 `::`**、**引用类型 `T&`**、**默认参数**、**inline**、以及 **「声明即可用」的类内定义**。读懂 C++ 声明 = **ch03 优先级规则** + **类作用域**。

## 要点

### 类内声明 vs 类外定义

```cpp
class Timer {
public:
    void start();                    /* 声明 */
    void stop() { running_ = false; } /* 类内定义 → 隐式 inline 候选 */
private:
    bool running_ = false;
};

void Timer::start() { running_ = true; }  /* 类外定义，作用域解析 */
```

**`Timer::start`** 中的 **`::`** 表示 **成员属于 `Timer`** — 不是 C 的全局命名空间。

### 引用声明（对比 C 指针）

| C | C++ |
|---|-----|
| `void f(int *p)` | `void f(int &r)` |
| 可能 **`NULL`** | **必须绑定已有对象**，无空引用 |
| `(*p)++` | `r++` — 语法像值 |

```cpp
void bump(int &x) { ++x; }
int n = 5;
bump(n);   /* n 变为 6 — 非拷贝 */
```

**`const T&`**：绑定临时量、大对象只读传参 — 见 **11.17**, **demo02_ref_vs_ptr**。

### 默认参数

```cpp
void log(const char *msg, int level = 1);
log("warn");           /* level = 1 */
log("err", 3);
```

**规则**：默认参数 **从右向左** 连续；**声明处** 指定（头文件），定义处可省略。

### `inline` 函数

```cpp
inline int sq(int x) { return x * x; }
```

**意图**：建议编译器 **内联展开**，消除调用开销 — 类似 **无宏副作用的 `#define`**（**ch02**）。现代编译器 **自行决定** 是否真内联；**`inline`** 还影响 **ODR / 头文件多重定义** 规则。

### C++ 声明严格性（相对 C）

| 点 | C | C++ |
|----|---|-----|
| **`void*` → `T*`** | 隐式 | **显式 cast** |
| **函数声明** | 旧式 `f()` 可能隐式 int | **必须类型** |
| **重载** | 无 | **同名不同参**（**11.10**） |
| **`const` 全局** | 常外部链接 | 默认 **内部链接**（**11.17**） |

### 与 ch03 的衔接

读 **`int (*pf)(int)`** 仍用 **ch03 螺旋法则**；读 **`void (C::*pmf)()`**（成员函数指针）多一层 **类作用域** — 本章只需知道 **存在**，日常用 **虚函数** 替代（**11.13**）。

```cpp
/* 成员函数指针 — 少见，了解即可 */
void (String::*p)() = &String::length;
```

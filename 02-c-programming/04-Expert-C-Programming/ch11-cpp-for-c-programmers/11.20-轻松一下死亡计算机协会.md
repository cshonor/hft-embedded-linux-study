# 11.20 轻松一下——死亡计算机协会

## 本节讲什么

**Light relief** — 作者借 **「Death Computer Society（死亡计算机协会）」** 等 **行业内部笑话**，调侃 **程序员文化、语言圣战、过度优化** 与 **计算机行业的「黑色幽默」**。无新语法；**收束张力** 的一章调味，类似 **ch10.8 程序检验的限制** — 提醒：**再精通 C/C++，也要对 **工具与自身的局限**** 保持幽默与清醒。

## 要点

### 死亡计算机协会（梗概）

书中虚构/引用的 **「协会」** 戏谑：

- **把 bug 当特性** 的 legacy 系统
- **「在我机器上能跑」** 作为发布标准
- **优化到 **既 unreadable 又不 faster**** 的代码
- **语言战争**：C vs C++ vs 「明天的新语言」

**寓意**：**技术严肃，人不必永远严肃** — 读 **Expert C** 是为了 **少成为笑话主角**。

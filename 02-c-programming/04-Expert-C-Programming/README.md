# 《C 专家编程》

**Expert C Programming: Deep C Secrets** — Peter van der Linden

## 定位

阶段 1 · C 语言进阶。链接器行为、数组/指针深层规则、C 与 C++ 差异、历史包袱与「为什么 C 长这样」。

## 阅读建议

K&R +《C 和指针》之后阅读；帮助从「会写 C」到「懂 C 为什么这么设计」。

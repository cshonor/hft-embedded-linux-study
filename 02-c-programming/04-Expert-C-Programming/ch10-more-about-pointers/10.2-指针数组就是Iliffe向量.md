# 10.2 指针数组就是 Iliffe 向量

## 本节讲什么

**Iliffe 向量**（以 David Iliffe 命名）指 **指针数组**：元素是指针，各指针指向 **独立的数据块**。C 里 **`char *str_list[]`**、**`main` 的 `argv`** 都是典型。本节把 **10.1 的分散布局** 与 **9.5 的 `char *str[10]` vs `char str[10][20]`** 串起来，明确：**语法上的「二维下标」可以来自指针数组，但内存并非 `int[M][N]` 那种连续块**。

## 要点

### Iliffe 向量定义

```c
char *str_list[] = {
    "hello",
    "world",
    "expert c",
};
```

**内存拓扑**：

```text
str_list → [ ptr0 ][ ptr1 ][ ptr2 ]   ← 指针数组（连续）
              ↓       ↓       ↓
           "hello" "world" "expert c"  ← rodata 中各字符串（通常不相邻）
```

- **表**（vector of pointers）连续；**数据** 分散
- **行宽可变**：各行长度不必相同（与固定列宽 **`char[N][M]`** 对比，**9.5**）

### `argv`：`char **argv`

```c
int main(int argc, char *argv[]) { ... }
/* 完全等价 */
int main(int argc, char **argv) { ... }
```

| 维度 | 说明 |
|------|------|
| **`argv`** | **`char *` 数组** 的首元素地址，decay 为 **`char **` |
| **`argv[0]`** | 程序名（可写性因平台而异，勿依赖修改） |
| **`argv[i]`** | **`char*`**，通常指向 **只读 rodata** 中的参数字符串 |
| **与 `char argv[][N]`** | 后者是 **内嵌字符块**；前者是 **指针表** |

```c
for (int i = 0; i < argc; i++)
    printf("argv[%d] = %s\n", i, argv[i]);
```

**链接与启动**：操作系统构造 **`argv`** 表；详见 **[ch05 链接](../ch05-thinking-of-linking/)**、**[ch06 运行时](../ch06-runtime-data-structures/)** 中程序启动环境。

### `char *str_list[]` vs `char str_list[][N]`

| 维度 | **`char *str_list[]`** Iliffe | **`char str_list[][20]`** 二维数组 |
|------|-------------------------------|-------------------------------------|
| **类型** | `char*[n]` | `char[n][20]` |
| **字符串位置** | 指针指 **字面量 / 堆 / 别处** | **拷贝进** 每行 20 字节槽 |
| **可写 `str[i][0]`** | 若指字面量 → **SIGSEGV** | ✅ 栈数组可写 |
| **行宽** | 各行不等长 | 固定 20（含 `\0`） |
| **传参** | `char **` | `char (*)[20]` |
| **典型** | **`argv`**、字符串表 | 固定列宽缓冲 |

```c
char *ptr_row[] = {"hi", "bye"};       /* 只读字面量 */
char buf_row[2][8] = {"hi", "bye"};    /* 可写副本 */

buf_row[0][0] = 'H';   /* OK */
/* ptr_row[0][0] = 'H';  ← 未定义行为 / 段错误 */
```

见 **ch09 demo02_string_arrays**、**9.5** 对照表。

### 与 `int **` 锯齿数组的关系

- **`char *str[]`**：指针表 + 各字符串（**只读 rodata 或堆**）
- **`int **matrix`**：指针表 + 各行 **`malloc`** 的 int 块（**10.3**）

二者 **都是 Iliffe 结构**（指针数组 + 间接数据），但 **数据来源与生命周期** 不同。

### 声明螺旋（ch03）

```c
char *p[10];    /* 指针数组：10 个 char* — Iliffe 表 */
char (*p)[10];  /* 数组指针：指向 char[10] — 连续行 */
```

# 10.7 使用指针创建和使用动态数组

## 本节讲什么

本节是全章 **技术密度最高** 的一节：用 **`malloc`/`free`** 创建动态数组；**`void*`** 的规则与 **`malloc` 桥接**；**多级指针** 出参（**`alloc_buf(char**)`**）；**函数指针** 声明、**`typedef Callback`**、**函数指针数组** 与 **`signal` 原型**；**指针算术** **`p+1`**、**`p2-p1`** 仅限 **同一数组对象** 内。函数指针工程模板见 **[function-pointer-typedef-templates.md](./function-pointer-typedef-templates.md)**。

## 要点

### 动态一维数组

```c
size_t n = 100;
int *p = malloc(n * sizeof *p);   /* sizeof *p 防错 */
if (!p) { /* 处理 OOM */ }

/* 使用 */
for (size_t i = 0; i < n; i++)
    p[i] = 0;

free(p);
p = NULL;   /* 防悬垂 — 10.8 */
```

**原则**：**`malloc(n * sizeof *ptr)`**；**配对 `free`**；**不要对非堆指针 `free`**（**ch07**）。

### `void*`：通用指针桥接

| 规则 | 说明 |
|------|------|
| **任意 `T*` → `void*`** | 隐式转换 ✅ |
| **`void*` → `T*`** | 需 **显式 cast**（C 中 `malloc` 赋给 `T*` 可省略 cast） |
| **解引用 `*vp`** | ❌ **`void*` 不能解引用** |
| **算术 `vp+1`** | ❌ **不知元素大小，不能偏移** |
| **比较** | 仅当同属 **同一分配块** 时有意义 |

```c
int x = 42;
void *vp = &x;
int *ip = vp;              /* C 允许 void* 隐式转 int*（C++ 不行） */
printf("%d\n", *ip);

/* 必须 cast 后解引用 */
printf("%d\n", *(int *)vp);

/* malloc 返回 void* */
int *arr = malloc(10 * sizeof *arr);
free(arr);
```

见 **demo04_void_ptr**。

### 多级指针：`char **` 出参

**模式**：在函数内 **`malloc`**，通过 **`T**`** 写回调用方指针。

```c
static void alloc_str(char **out, const char *src)
{
    size_t n = strlen(src) + 1;
    *out = malloc(n);
    if (*out)
        memcpy(*out, src, n);
}

/* 调用 */
char *s = NULL;
alloc_str(&s, "hello");
free(s);
```

**类比**：**`int **` 锯齿行分配**（**10.3**）、**`realloc` 改 `char**`**。见 **demo03_double_ptr**。

### 函数指针基础

```c
int add(int a, int b) { return a + b; }

int (*fp)(int, int) = add;   /* fp 是指向函数的指针 */
fp(3, 5);                    /* 8 */
(*fp)(3, 5);                  /* 等价，冗余括号 */
add(3, 5);                   /* 直接调用 */
```

**声明读法**（**ch03 螺旋**）：**`fp` 先与 `*` 结合 → 指针；`[]` 无；后面 **`(int,int)` 函数原型**。

### `typedef` 简化 — Callback 模板

```c
typedef int (*MathFunc)(int, int);
typedef void (*Callback)(void);
typedef void (*CallbackInt)(int code);

MathFunc fp = add;
Callback cb = my_init;
if (cb) cb();
```

**工程模板**（命令表、ISR 向量、`signal`）：**[function-pointer-typedef-templates.md](./function-pointer-typedef-templates.md)**

| 场景 | typedef 示例 |
|------|--------------|
| **无参回调** | `typedef void (*Callback)(void);` |
| **带 int 码** | `typedef void (*CallbackInt)(int code);` |
| **内核 IRQ** | `typedef int (*IrqHandler)(void *dev_id);` |
| **命令分发** | `typedef int (*CmdFunc)(int argc, char **argv);` |

### 函数指针数组（ops / 向量表）

```c
typedef int (*MathFunc)(int, int);

static int add(int a, int b) { return a + b; }
static int sub(int a, int b) { return a - b; }

static MathFunc ops[] = { add, sub };

ops[0](10, 3);   /* 13 */
ops[1](10, 3);   /* 7 */
```

**用途**：命令表、状态机、**中断向量**（**function-pointer-typedef-templates.md §4**）。见 **demo02_func_ptr**。

### `signal` 原型：返回函数指针的函数

```c
/* 原始 K&R 风格 */
void (*signal(int sig, void (*handler)(int)))(int);

/* typedef 后 */
typedef void (*SigHandler)(int);
SigHandler signal(int sig, SigHandler handler);
```

**读法**：**`signal(...)` 的返回值** 是 **`void (*)(int)`** — 旧 handler 地址。

### 指针算术规则

```c
int a[10];
int *p = &a[2];
int *q = &a[5];

p + 1;      /* 指向 a[3]，步长 sizeof(int) */
q - p;      /* 3 — 同一数组内元素个数 */
/* q - &some_other_array[0];  ← UB */
```

| 运算 | 条件 |
|------|------|
| **`p + n` / `p - n`** | **`p`** 指向数组某元素或 **尾后** |
| **`p2 - p1`** | **`p1`、`p2`** 指向 **同一数组对象** |
| **`p1 < p2`** | 同上 — 否则 **UB** |
| **`&a[i] - &a[j]`** | ✅ 同一 **`a`** |

**二维**：**`arr`（`int (*)[N]`）+1** 跳一行；**`arr[0]`（`int*`）+1** 跳一列（**10.1**, **9.6**）。

### 动态多维（复习）

```c
/* 锯齿 — 10.3 */
int **m = malloc(rows * sizeof *m);
for (int r = 0; r < rows; r++)
    m[r] = malloc(cols * sizeof **m);

/* 单块 + 索引 */
int *block = malloc(rows * cols * sizeof *block);
/* block[r*cols+c] */
```

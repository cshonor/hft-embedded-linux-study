# 第 8 章 为什么程序员无法分清万圣节和圣诞节

**Why Programmers Can't Tell Halloween from Christmas** — Peter van der Linden, *Expert C Programming*

## 本章目标

从 **八进制 `031`（像 10/31）= 十进制 `25`（12/25）** 的双关出发，建立 **运算符优先级、求值顺序、副作用、序列点/sequenced-before** 的完整图景。掌握 **Christmas 问题** `i = i++ + ++i`（UB）、**Halloween 问题** `if (a=b==0)` vs `if ((a=b)==0)`、**`val & mask == 1`** 位测陷阱；区分 **`&&`/`||` 短路** 与 **`&`/`|` 按位**、**逗号运算符** 与 **函数实参逗号**；理解 **整型提升/usual arithmetic conversions**（**8.3**）、**K&R vs ANSI 原型**（**8.4–8.5**）、**cast 与优先级**（**8.9**）。能写 **FSM**（**8.7**）、配置 **termios 即时输入**（**8.6**）；遵守 **Yoda 条件、一行一副作用** 工程规则。衔接 **ch02 UB 框架**、**ch03 声明**、**ch07 内存/别名**、**ch09/ch10 数组指针**。

## 章节双关（8.1 核心）

```c
031 == 25   /* true：八进制 031 = 3×8+1 = 25 */
```

| 视角 | 含义 |
|------|------|
| **万圣节 Oct 31** | 字面量 **`031`** 像日期 |
| **圣诞节 Dec 25** | **`031` 的数值 = 25** |
| **程序员笑话** | 分不清的是 **进制规则**，不是日历 |

### 经典陷阱一行对照

| 写法 | 解析 / 后果 |
|------|-------------|
| **`val & mask == 1`** | `val & (mask==1)` ❌ |
| **`(val & mask) == 1`** | 测位 ✅ |
| **`if (a = b == 0)`** | `a = (b==0)` → **Halloween** ❌ |
| **`if ((a = b) == 0)`** | 赋值再比较 ✅ |
| **`if (0 == a)`** | **Yoda** — 防 `0 = a` |
| **`i = i++ + ++i`** | **Christmas — UB** ❌ |
| **`f(i++, ++i)`** | 实参顺序未指定 — **UB** ❌ |
| **`ptr && ptr->x`** | 短路 ✅ |
| **`ptr & ptr->x`** | 可能空指针崩溃 ❌ |

## 求值顺序与序列点（8.8 精华）

| 概念 | 说明 |
|------|------|
| **UB** | 同一标量无 sequenced 的多次修改，如 **`i=i++ + ++i`** |
| **Sequence point（口语）** | **`&&`/`||`/`?:` 首操作数后**、**`;` 前**、**逗号左侧后** |
| **C11 sequenced-before** | 精确定义 **A 的副作用对 B 可见**；取代「序列点」教学 |
| **Indeterminately sequenced** | **函数实参** 之间 — 故 **`f(i++,++i)` UB** |
| **Unsequenced** | 读写同一对象无顺序 → **UB** |

**工程规则**：**一行一副作用**；**位测/比较加括号**；**`-Wsequence-point`**。

## 高频考点 / 面试题

1. **为什么 `031 == 25`？程序员「分不清万圣节与圣诞节」指什么？** → **八进制 `031` = 25**；双关 **字面量进制** vs 日期；**勿用前导 0 写十进制**（**8.1**）

2. **`i = i++ + ++i` 结果多少？** → **未定义行为（UB）**；**同一表达式多次修改 `i` 且无 sequenced-before**；**禁止写**；拆语句（**8.8**, **demo01_christmas**）

3. **`if (a = b == 0)` 与 `if ((a = b) == 0)` 区别？** → 前者 **`a = (b==0)`**，a 为 0/1；后者 **赋值 b 给 a 再与 0 比**；**Yoda：`if (0==a)`**（**8.8**, **demo02_halloween**）

4. **`val & mask == 1` 错在哪？`&&` 与 `&` 何时用？** → **`==` 高于 `&`**；应 **`(val & mask) == 1`**；**逻辑条件/空指针用 `&&` 短路**；**位操作用 `&`**（**8.2, 8.8**, **demo03/demo04**）

5. **函数实参里的逗号与逗号运算符有何不同？`f(i++, ++i)` 为何 UB？** → 实参逗号 **非逗号运算符**，各实参 **indeterminately sequenced**；**`i++`/`++i` 同次调用修改同一对象 → UB**；逗号运算符 **`a, b` 左 sequenced before 右**（**8.8**）

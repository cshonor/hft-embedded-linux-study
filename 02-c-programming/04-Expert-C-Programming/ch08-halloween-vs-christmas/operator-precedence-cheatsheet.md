# C 运算符优先级速查表

**Expert C Programming ch08** — 完整表；**易混淆项** 用 **⚠️ 粗体** 标出。建议打印或 pin 在 IDE 旁。

## 优先级表（从高到低）

| 优先级 | 运算符 | 结合性 | 说明 |
|:------:|--------|:------:|------|
| 1 | **`()` `[` `->` `.`** | 左→右 | 调用、下标、成员。**⚠️ `.` 高于 `*`**：`p->x` 等价 `(*p).x` |
| 2 | **`!` `~` `++` `--` `+` `-` `*` `&` `(type)` `sizeof`** | **右→左** | 一元。**⚠️ `&` 取址 vs 二元按位与**；**⚠️ `++` 后缀绑定紧** |
| 3 | **`*` `/` `%`** | 左→右 | 乘除模 |
| 4 | **`+` `-`** | 左→右 | 二元加减 |
| 5 | **`<<` `>>`** | 左→右 | 移位 |
| 6 | **`<` `<=` `>` `>=`** | 左→右 | 关系 |
| 7 | **`==` `!=`** | 左→右 | **⚠️ `==` 高于 `&`** → `val & mask == 1` 错 |
| 8 | **`&`** | 左→右 | 按位与。**⚠️ `&` 低于 `==`，高于 `^`** |
| 9 | **`^`** | 左→右 | 按位异或 |
| 10 | **`|`** | 左→右 | 按位或 |
| 11 | **`&&`** | 左→右 | 逻辑与，**短路**。**⚠️ `&&` 远低于 `&`** |
| 12 | **`||`** | 左→右 | 逻辑或，**短路** |
| 13 | **`?:`** | **右→左** | 条件；仅求值一个分支 |
| 14 | **`=` `+=` `-=` …** | **右→左** | 赋值。**⚠️ `=` 低于 `==`** → Halloween 陷阱 |
| 15 | **`,`** | 左→右 | **逗号运算符**（非实参逗号） |

## 易混淆对照

| 表达式 | 实际解析 | 常见意图 |
|--------|----------|----------|
| **`val & mask == 1`** | `val & (mask == 1)` | **`(val & mask) == 1`** |
| **`a = b == 0`** | `a = (b == 0)` | **`(a = b) == 0`** 或分两句 |
| **`ptr && ptr->x`** | 短路，空安全 | **`ptr & ptr->x`** ❌ |
| **`p++`** 与 **`*p++`** | `*(p++)` | 先取用旧 p，再 p++ |
| **`f(a++, b++)`** | 实参顺序 **未指定** | 勿共享可变状态 |
| **`i = i++ + ++i`** | **UB** | 拆成多语句 |

## 记忆口诀（简版）

> **一元 → 算术 → 移位 → 关系 → 相等 → 位与 → 位异或 → 位或 → 逻辑 → 条件 → 赋值 → 逗号**

**相等（7）在按位与（8）之前** —  Halloween 位测坑的根源。

## 工程规则

1. **怀疑就加括号**
2. **比较用 Yoda**：`if (0 == x)`
3. **逻辑用 `&&`/`||`，掩码用 `&`/`|`
4. **一行一副作用**
5. **GCC**：`-Wall -Wsequence-point -Wparentheses`

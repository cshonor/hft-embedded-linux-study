# 8.3 在等待时类型发生了变化

## 本节讲什么

表达式求值时，操作数 **在「你等待结果」的过程中已被隐式转换**——这就是书名 **「types changed while waiting」** 的含义。本节讲 **整型提升（integer promotion）** 与 **寻常算术转换（usual arithmetic conversions）**： **`char` 变 `int`**、 **`unsigned` 与 `signed` 混合** 时 **符号位被 reinterpret**，导致比较、移位、位运算结果 **与纸面推算不符**。这是 **8.8 求值顺序** 之外的第二维陷阱：**类型在变，值域也在变**。

## 要点

### 整型提升（Integer Promotion）

C 标准：在 **int 能表示原类型所有值** 时， **`char`/`short`/`unsigned char`/`unsigned short`** 等 **小于 `int` 的类型** 在大多数表达式中 **先提升为 `int`**（若 `int` 不够则 **`unsigned int`**）。

```c
char c1 = 0xFF, c2 = 0xFF;
if (c1 == c2) ...   /* 两操作数都提升为 int 0xFF，比较为真 */

unsigned char u = 0xFF;
int i = u;           /* u 提升为 int 255，赋值正确 */
```

** surprises**：

```c
char c = -1;
unsigned u = 1;
if (c < u) ...       /* c 提升为 int(-1)；与 u 比较时 c 再转为 unsigned → 很大的数 → 假 */
```

| 场景 | 机制 | 结果 |
|------|------|------|
| **`signed` vs `unsigned` 比较** | **`signed` → `unsigned`** | 负数变超大 **unsigned** |
| **混合运算** | **usual arithmetic conversions** | 两操作数变 **同一类型** |
| **printf `%d` 传 `char`** | 可变参数 **default argument promotions** | 必须 **显式 `(int)c`** 或匹配格式 |

### 寻常算术转换（Usual Arithmetic Conversions）

对二元算术/比较运算，编译器将操作数转为 **「共同类型」**：

1. 若类型 **等级不同**，较低者 **提升** 到较高（ **`long double` > `double` > `float` > 整型链** ）
2. 若 **signed / unsigned 同级**， **`signed` 变 `unsigned`**（当 `unsigned` 能表示 signed 全范围时；否则两者都变 **更大 signed**——规则细节见标准，面试记 **「unsigned 常赢 signed」** 即可）

```c
unsigned short a = 65535;
int b = -1;
if (a + b > 0) ...  /* a 提升为 int；65535 + (-1) = 65534 > 0 */
```

### 「等待时类型变化」与副作用

类型转换本身 **不是** 序列点，但会 **改变副作用可见时的位宽**：

```c
char buf[4];
/* 假设某平台 int 为 4 字节；*(int*)buf 读写可能 UB（对齐 ch07） */
```

**与 8.8 联动**：即使 **求值顺序** 合法， **转换后的值** 也可能与程序员 mental model 不一致—— **`i = i++ + ++i`** 在 **UB** 之外，还有大量 **「合法但反直觉」** 的 **`unsigned` 环绕** 行为（**实现定义/模运算**）。

### 整型常量与 `031`（8.1）

```c
int x = 031;        /* 八进制 25，类型 int */
long y = 031L;      /* long */
unsigned z = 031u;  /* unsigned */
```

字面量 **后缀 `L/u/ll`** 与 **前导 `0` 八进制** 叠加时，review 要 **双重核对**。

### 工程规则

- 比较 **有符号与无符号** 时 **显式 cast** 到同一宽度的 **`int64_t`/`uint64_t`**（ `<stdint.h>` ）
- **位运算** 操作数用 **`unsigned`** 或 **`uint32_t`**，避免符号扩展
- **不要** 假设 **`char` 是有符号还是无符号**——由实现定义（**`-funsigned-char`**）

# 6.6 auto和static关键字

## 本节讲什么

函数内的 **`auto` 局部变量**（默认）活在 **栈帧** 里；**`static` 局部变量** 活在 **`.data` / `.bss`**，**不在栈上**，程序运行期间 **只初始化一次**，地址固定。**demo02_static_local** 对比 **`auto_cnt` 每次为 1** 与 **`static_cnt` 累加**。

## 要点

### 存储位置对比

```c
void counter(void)
{
    int auto_cnt = 0;        /* 栈：每次进入函数重新分配逻辑上的「新对象」 */
    static int static_cnt;   /* .bss：程序级存储，首次为零，之后保持 */

    auto_cnt++;
    static_cnt++;
}
```

| 关键字 | 默认 | 存储区域 | 初始化 | 生命周期 |
|--------|------|----------|--------|----------|
| **`auto`**（可省略） | 是 | **栈** | 每次进入（若写 `=0`） | 函数调用期间 |
| **`static`**（块内） | — | **`.data`（有初值）或 `.bss`** | **一次** | **整个进程** |
| **`static`**（文件作用域） | — | **`.data/.bss`** | 同上 | 进程；**内部链接**（**ch02**） |

```bash
cd demo/demo02_static_local && make && ./demo02
# auto_cnt=1,1,1  static_cnt=1,2,3
```

### demo02 与 demo01 地址

**demo01** 中 **`static_init` / `static_uninit`** 地址与 **`global_*`** 同属 **数据段** 簇，远离 **`stack_var`**。

**demo02** 可用 **`nm`** 见 **`static_cnt`** 在 **bss**：

```bash
nm demo02 | grep static_cnt
# b ... static_cnt  （B/b = bss）
```

### 生命周期对比表（高频）

| 声明形式 | 存储 | 作用域 | 链接 | 初值未写时 |
|----------|------|--------|------|------------|
| 函数内 **`int x;`** | 栈 | 块 | — | **不确定**（勿依赖） |
| 函数内 **`static int x;`** | .bss/.data | 块 | **内部** | **0** |
| 文件内 **`static int x;`** | .bss/.data | 文件 | **内部** | **0** |
| **`int g;`**（全局） | .bss | 文件 / extern | **外部** | **0** |
| **`int g = 1;`** | .data | 同上 | 外部 | **1** |

### static 局部的语义

```c
void foo(void) {
    static int n = 5;   /* 仅第一次执行到此处时 = 5，等价 crt 前 .data 初值 */
    n++;
}
```

- **不是**「每次调用重新赋 5」
- **线程安全**：C11 前 **不** 保证 **`static` 局部初始化** 的并发安全（多线程首次进入可能竞态；**6.7**）

### 与 ch02 static 语义

**ch02 2.4**、**demo04_static_dual**：

| 语境 | **`static` 含义 |
|------|-----------------|
| 函数内 | **持久存储** + **块作用域** |
| 文件内 | **内部链接**：符号 **不** 导出给其他 `.o`（**5.1** **`nm` 的 t/T**） |

勿与 C++ **`static` 成员** 混淆（本书 C 语境）。

### 常见 bug

| 错误 | 原因 |
|------|------|
| 返回 **`&local_auto`** | 栈帧销毁后 **悬垂** |
| 以为 **`static` 局部在栈上** | 地址与堆栈无关，**跨调用保留** |
| 多线程无锁改 **`static` 局部** | 数据竞争 |

### 与 5.1 BSS 规则

**`static int x;`**（无初值）→ **`.bss`** → **载入时清零**（**5.1 NOBITS**, **6.3**）。

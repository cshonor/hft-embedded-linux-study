# 4.6 for 循环

## 本节讲什么

**计数与遍历**首选：`for (初始化; 条件; 更新)`。三部分均可省略。

## 要点

```c
for (int i = 0; i < pkt_cnt; i++)
    decode_mbuf(buf[i]);

for (;;) {          /* 无限循环：DPDK 数据面主循环 */
    rx_process();
    tx_process();
}
```

### 等价 while 展开

```c
for (A; B; C) { D; }
/* 等价于 */
A;
while (B) {
    D;
    C;
}
```

- **A** 只执行一次；**C** 在 **D** 之后执行
- **continue** 在 for 中仍执行 **C**（再判断 B）

### 省略形式

| 形式 | 含义 |
|------|------|
| `for (;;)` | 恒真 |
| `for (; i < n; )` | 无初始化 |
| 逗号表达式 | `for (i=0,j=n; i<j; i++,j--)` |

### break / continue 示例

```c
for (int i = 0; i < cnt; i++) {
    if (buf[i]->type == PKT_HEARTBEAT)
        continue;
    parse_order(buf[i]);
}
```

## 实操

DPDK 风格 `for(;;)` + **continue** 过滤心跳包（模拟）。

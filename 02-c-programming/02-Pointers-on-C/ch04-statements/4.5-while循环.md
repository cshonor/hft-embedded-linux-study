# 4.5 while 循环

## 本节讲什么

**先判断、后执行**；条件为假时**零次**循环。DPDK 批量收包、条件驱动处理的核心结构。

## 要点

```c
while ((n = rte_rx_burst(port, q, bufs, 32)) > 0) {
    handle_batch(n, bufs);
}
```

### 执行流程

1. 求值条件  
2. 非 0 → 执行循环体 → 回到 1  
3. 0 → 退出  

### 赋值陷阱

```c
while (ret = 0)   /* 赋 0，条件假，不执行 */
while (ret = 1)   /* 恒真，死循环 —— 线上致命 */
```

- 意图比较时写 **`==`**；故意赋值时加括号并注释

### break / continue

```c
while (1) {
    if (bad_pkt)
        continue;   /* 跳过本次，回到 while 条件 */
    if (shutdown)
        break;      /* 跳出 while */
    process();
}
```

| 语句 | 作用 |
|------|------|
| **break** | 跳出**最内层** enclosing 循环（或 switch） |
| **continue** | 跳过剩余体，回到**条件判断** |

- 多层嵌套 **break** 只出一层；外层需 flag 或 **goto**（4.9）

## 与后续衔接

**4.6 for** 等价形式；**ch06** 指针遍历 `while (*p)`。

# 4.1 空语句

## 本节讲什么

**空语句**仅由一个分号 **`;`** 构成，语法合法、运行时无操作。与 **if 后误加分号** 陷阱形成对比。

## 要点

```c
;   /* 合法空语句 */
```

### 合法场景

```c
while (getchar() != '\n')
    ;   /* 快速清空输入行，循环体为空 */

for (; *p; p++)
    ;   /* 仅做指针步进 */
```

### 与陷阱对比

```c
if (flag);          /* 空语句：if 不控制下面 */
    do_something(); /* 永远执行 */
```

- 量化、驱动代码**最常见隐蔽 bug** 之一
- 根治：if/else 一律 **`{}`**；`-Wempty-body` 开警告

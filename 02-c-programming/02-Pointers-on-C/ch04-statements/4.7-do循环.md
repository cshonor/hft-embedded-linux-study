# 4.7 do 循环

## 本节讲什么

**do-while**：**先执行、后判断**；至少执行**一次**。硬件重试、发送阻塞重发适用。

## 要点

```c
do {
    ret = send_order(order);
} while (ret == EAGAIN);
```

- 末尾分号 **不能少**
- 条件为真继续，为假退出

### vs while

| | while | do-while |
|---|--------|----------|
| 判断时机 | 前 | 后 |
| 最少次数 | 0 | 1 |
| 场景 | 可能零包 | 至少读/发一次 |

### 寄存器重试示例

```c
do {
    val = read_reg(REG_STATUS);
} while (!(val & READY_MASK));
```

## 与后续衔接

**K&R 3.6**；菜单/交互至少读一次输入。

## 实操

do-while 模拟订单发送 **EAGAIN** 重试。

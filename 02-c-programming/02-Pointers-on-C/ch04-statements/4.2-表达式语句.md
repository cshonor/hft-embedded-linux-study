# 4.2 表达式语句

## 本节讲什么

**表达式 + 分号** 构成语句；分号是**语句终止符**，不属于表达式本身。

## 要点

```c
a = 10;                        /* 赋值 */
count++;                       /* 自增 */
rte_ring_enqueue(ring, mbuf);  /* 函数调用 */
(void)func();                  /* 显式丢弃返回值 */
```

- 表达式求值后**丢弃**结果（除非有副作用）
- 函数调用语句：忽略返回值时需确认是否允许（错误码常需检查）

### return（函数返回）

```c
return 0;      /* 带值返回 */
return;        /* void 函数 */
```

- 执行 **return** 后函数立即结束，**栈局部变量销毁**
- 返回值经 **ABI** 约定寄存器传递（如 x86 **rax**）—— **ch18** 展开
- 表达式自动转换为函数返回类型

## 陷阱

- 把 **`if (a = b)`** 当比较 —— 赋值表达式语句为真值

## 与后续衔接

**ch05** 表达式优先级；**ch07** 函数返回与错误码。

# 4.8 switch 语句

## 本节讲什么

**整型多路分支**：报文类型、错误码、设备状态枚举的标准写法。

## 要点

```c
switch (pkt->msg_type) {
case MSG_QUOTE:
    process_quote(pkt);
    break;
case MSG_ORDER:
    process_order(pkt);
    break;
default:
    log_err("unknown");
    break;
}
```

### 规则

| 规则 | 说明 |
|------|------|
| 控制表达式 | **整型**（char/short/int/enum）；**无 float/指针** |
| case 标签 | **编译期整型常量** |
| 匹配后 | **顺序执行**直至 **break** / **return** / **goto** / 结尾 |
| default | 建议保留兜底 |

### 穿透（fall-through）

```c
case MSG_CANCEL:
case MSG_MODIFY:
    handle_transaction(pkt);
    break;
```

- 刻意合并分支：加注释说明
- **漏 break** → 不同 case 逻辑串联 → 线上偶发 bug

### 踩坑

- case 块内**定义变量**需加 `{}`（C 语法）
- 未知类型无 **default** → 静默无处理

## 实操

一组 case 共享处理函数，主动利用穿透。

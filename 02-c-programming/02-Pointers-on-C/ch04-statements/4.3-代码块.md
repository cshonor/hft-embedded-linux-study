# 4.3 代码块

## 本节讲什么

**复合语句（代码块）** `{ ... }` 将多条语句合成**一条**；if/for/while/函数体必须使用。

## 要点

```c
{
    struct rte_mbuf *pkt = rx_one();
    if (pkt != NULL) {
        parse_quote(pkt);
        rte_pktmbuf_free(pkt);
    }
}   /* 块结束，块内 auto 变量失效 */
```

| 特性 | 说明 |
|------|------|
| 块作用域 | 块内声明仅 `{}` 内可见（**ch02/ch03**） |
| 遮蔽 | 内层同名覆盖外层 |
| 栈 | 块内 **auto** 变量栈分配，块结束销毁 |

### 工程规范

- **所有 if/else/循环分支强制 `{}`** —— 内核/DPDK 规范
- 避免与全局同名局部变量（内核常禁止）

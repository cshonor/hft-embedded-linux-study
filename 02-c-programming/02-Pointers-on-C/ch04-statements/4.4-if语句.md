# 4.4 if 语句

## 本节讲什么

**条件分支**：表达式非 0 为真，0 为假。交易风控、报文校验、驱动错误判断的基础。

## 要点

```c
if (expr)
    stmt;
else if (expr2)
    stmt2;
else
    stmt3;
```

### 惯用写法（HFT/驱动）

```c
if (mbuf)           /* 等价 mbuf != NULL */
if (!err)           /* 等价 err == 0 */
```

- 不必写 `== 1`；指针、整数均可作条件

### 悬垂 else（《C陷阱与缺陷》ch02）

**`else` 匹配最近未配对的 `if`**，缩进无效。

```c
if (a > 0)
    if (b > 0)
        puts("a>0,b>0");
else
    puts("???");   /* 绑定内层 if，非外层 */
```

**根治**：每层 **`{}`**；或多层改 **`else if`** 扁平化。

### 嵌套规范

- 报文类型、错误码：优先 **else if** 链，避免深层嵌套

## 实操

制造无 `{}` 的悬垂 else，打印验证绑定关系。

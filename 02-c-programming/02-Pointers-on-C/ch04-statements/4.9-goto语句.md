# 4.9 goto 语句

## 本节讲什么

**无条件跳转**到同函数内 **label**。应用层慎用；**Linux 内核/驱动** 允许规范用于错误清理与多层循环跳出。

## 要点

### 内核式错误释放

```c
int func(void)
{
    void *buf1 = malloc(1024);
    if (!buf1)
        goto err1;
    void *buf2 = malloc(2048);
    if (!buf2)
        goto err2;

    /* 正常路径 */
    free(buf2);
    free(buf1);
    return 0;

err2:
    free(buf1);
err1:
    return -1;
}
```

### 合法场景

- 多层循环**一次跳出**
- **统一 err:** 资源释放（驱动 probe/remove）

### 禁止场景

- 交易引擎业务分支、普通逻辑替代 if/for
- 跨函数跳转（不可能）
- 跳进块跳过变量初始化（C 限制）

## 与 break 对比

- **break**：结构化退出单层循环/switch  
- **goto**：多层退出 + 集中 cleanup  

## 与后续衔接

**K&R 3.8**；读内核 `goto out_free` 模式。

## 实操

仿写 goto 多分支 malloc 失败路径。

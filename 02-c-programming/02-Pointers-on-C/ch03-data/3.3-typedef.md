# 3.3 typedef

## 本节讲什么

**typedef** 为已有类型起别名；**不创建新类型**。DPDK/内核跨平台定长的基石。

## 要点

```c
typedef unsigned char  uint8_t;
typedef unsigned long long uint64_t;
typedef struct rte_mbuf rte_mbuf_t;
```

| 价值 | 说明 |
|------|------|
| 定长 | 避免 `long` 在 32/64 位宽度不一 |
| 可读 | `MBUF` 代替长 struct 名 |
| 复杂声明 | 函数指针、数组指针简化 |

- 与 **`#define`** 不同：typedef 受类型检查
- **enum** 也可 typedef：`typedef enum { ... } pkt_type_t;`

## 与 `<stdint.h>`

- 优先标准定宽类型，而非手写 `typedef unsigned int uint32_t`（除非裸机无 stdint）

## 与后续衔接

读 DPDK `rte_*`、`linux/types.h` 全依赖 typedef。

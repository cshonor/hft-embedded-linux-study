# 3.7 存储类型

## 本节讲什么

**存储期（storage duration）**：对象**存活多久**；与 **ELF 段** 一一对应。衔接 ch02、ch18。

## 要点

| 存储期 | 关键字 | 位置 | 初始化 |
|--------|--------|------|--------|
| **自动** | auto（默认局部） | **栈** | 不保证 |
| **静态** | static（局部/文件） | **.data/.bss** | 0 或显式 |
| **线程** | _Thread_local（C11） | TLS | 视情况 |
| **动态** | malloc | **堆** | ch11 |

### .data vs .bss

| 段 | 内容 |
|----|------|
| **.data** | 有初值的全局/static（占镜像空间） |
| **.bss** | 无初值全局/static（加载时**清零**，省体积） |
| **.rodata** | 字符串常量、const 只读数据 |
| **.text** | 代码 |

- 嵌入式/DPDK 大配置数组放 bss 利用清零
- **栈 auto 不清零** → 报文解析必须初始化

### 动态内存

- **heap**：`malloc`/`free`（**ch11**）

## 实操

`readelf -S` 区分 `.rodata/.data/.bss`；全局变量分别放三处观察。

# 3.2 基本声明

## 本节讲什么

如何**声明**变量、数组、指针；**初始化**规则；**限定符** const/volatile/restrict 在声明中的位置。

## 要点

- 声明格式：**存储类 + 类型 + 限定符 + 名字 + 初始化**
- 一条声明可多个名字：`int a, b, c;`
- **定义**同时分配存储（ch02）；文件域重复定义链接报错

## 限定符（底层三宝）

| 限定符 | 作用 |
|--------|------|
| **const** | 只读；配置、报文缓冲入参 |
| **volatile** | 禁止寄存器缓存；**MMIO 寄存器**、中断标志 |
| **restrict** | C99；唯一访问路径，利于优化（DPDK 拷贝） |

⚠️ 热路径勿滥用 **volatile**（破坏优化、增延迟）。

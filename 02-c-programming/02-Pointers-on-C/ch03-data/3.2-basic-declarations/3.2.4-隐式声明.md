# 3.2.4 隐式声明

## 本节讲什么

旧 C 允许函数**未声明先调用**，编译器假定返回 **`int`** —— 现代工程**禁止**。

## 要点

- C99 起：调用前无原型 → **错误**（或警告升级）
- 隐式声明假设：返回 `int`，参数类型不检查 → 栈/ABI 错乱
- 规范：**头文件原型** + `#include`

```c
/* 错误习惯 */
printf(3.14);   /* 有原型则报错；旧式隐式 int 灾难 */
```

## 工程规范

- `-Wall -Werror=implicit-function-declaration`
- 每个 `.c` 包含对应 `.h`

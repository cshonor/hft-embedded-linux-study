# 3.2.2 声明简单数组

## 本节讲什么

**数组声明**：连续同类型元素；长度在定义时确定（VLA 另论）。

## 要点

```c
int buf[1024];
char msg[64];
uint8_t pkt[1500];   /* MTU 级缓冲 */
```

- **`sizeof(buf)`** = 元素数 × 元素大小
- 数组名多数情况下退化为**首元素指针**（ch06/ch08）
- 作参数传**指针 + 长度**，不拷贝整数组
- 栈上大数组 → 溢出风险；DPDK 用 **hugepage / mempool**

# 3.2.1 初始化

## 本节讲什么

声明时赋**初值**；未初始化行为因**存储期**而异 —— 栈脏数据是报文 bug 温床。

## 要点

| 对象 | 未显式初始化 |
|------|--------------|
| 全局 / 静态 | **0**（`.bss` 或 `.data`） |
| 局部 **auto** | **不确定**（栈垃圾） |
| 堆（malloc） | **不确定**（除非 `calloc`） |

```c
int a = 10;              /* .data */
int b;                   /* .bss → 0 */
void f(void) {
    int x;               /* 必须赋值再用 */
    static int y;        /* 仅首次 0 */
}
```

- 数组：`int a[5] = {1,2,3};` 其余为 0
- 指针：未初始化指针 → 野指针

## 与后续衔接

**ch11** 动态分配初始化；局部大数组改 static/堆。

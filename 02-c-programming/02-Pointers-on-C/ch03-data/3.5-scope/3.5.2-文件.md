# 3.5.2 文件

## 本节讲什么

**文件作用域**：函数外声明的变量、函数名；从声明点到文件末尾可见。

## 要点

```c
int g_count;              /* 文件作用域，外部链接 */
static int s_hidden;      /* 文件作用域，内部链接 */
static void helper(void); /* 仅本 .c */
```

| 修饰 | 链接 | 可见性 |
|------|------|--------|
| 无 static | 外部 | 其他 .c 可 extern |
| static | 内部 | 仅本文件 |

- 全局行情配置、设备列表：extern 共享
- DPDK 驱动私有：static 隔离

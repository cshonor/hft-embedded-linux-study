# 3.5.1 代码块

## 本节讲什么

**块作用域**：在 `{ }` 内声明的标识符，仅块内可见；含函数体、if/for 子块。

## 要点

```c
void f(void) {
    int x = 1;
    if (x) {
        int y = 2;    /* 仅 if 块内 */
        /* x, y 可见 */
    }
    /* y 不可见；x 可见 */
}
```

- 块结束：**auto** 局部销毁；**static** 局部仍存活（**3.8**）
- C99 起可在块内任意位置声明（非必须在块首）

## 风险

- 大对象栈分配 → 栈溢出
- 未初始化 auto → 脏数据

## 与后续衔接

**ch04** 控制语句块；**ch11** 大块改堆。

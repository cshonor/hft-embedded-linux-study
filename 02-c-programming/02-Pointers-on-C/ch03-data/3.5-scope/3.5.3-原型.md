# 3.5.3 原型

## 本节讲什么

**函数原型作用域**：参数名仅在**声明**中有效，与函数体无关。

## 要点

```c
void swap(int a, int b);   /* a,b 仅原型内 */

void swap(int x, int y)   /* 定义处 x,y 是新作用域 */
{
    /* 使用 x, y */
}
```

- 原型可省略参数名：`void swap(int, int);`
- 文档化时保留参数名：`void parse(const uint8_t *buf, size_t len);`

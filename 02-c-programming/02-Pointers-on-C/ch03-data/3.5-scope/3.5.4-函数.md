# 3.5.4 函数

## 本节讲什么

**函数作用域**仅用于 **label**（`goto` 目标）；标签名不与块内变量共享命名空间规则的特殊性。

## 要点

```c
void f(void) {
    if (error)
        goto cleanup;
    /* ... */
cleanup:
    /* 清理 */
}
```

- label **仅本函数**可见
- 内核错误路径、驱动 rollback 少量使用
- 勿滥用；优先结构化 return + **ch04** 控制流

## 与后续衔接

**K&R 3.8 goto**；Linux `goto out` 模式。

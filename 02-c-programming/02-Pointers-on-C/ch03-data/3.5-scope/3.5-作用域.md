# 3.5 作用域

## 本节讲什么

**作用域**决定标识符**可见范围**；与 **存储期**、**链接** 配合决定生命周期与符号能否跨文件。ch02 已入门，本章从**数据**角度细化。

## 要点

| 作用域 | 见 |
|--------|-----|
| 块（代码块） | 3.5.1 |
| 文件 | 3.5.2 |
| 函数原型 | 3.5.3 |
| 函数（label） | 3.5.4 |

- **内层遮蔽外层**同名变量 —— 内核规范常禁止
- 作用域 ≠ 存储期：块内 **static** 块作用域但静态存储期（**3.7**）

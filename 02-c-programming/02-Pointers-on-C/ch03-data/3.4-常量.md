# 3.4 常量

## 本节讲什么

**字面量**与 **const** 对象；**enum** 命名整型常量。内核/DPDK 宏与枚举的基础。

## 要点

### 整型常量

- 十进制 `123`、八进制 `0123`（易错）、十六进制 **`0x1F`**（寄存器/掩码）
- 后缀 `U` `L` `LL`

### 字符与转义

```c
'A'    '\n'    '\0'    '\x0F'
```

- **`'x'`** → `int` 字符值；**`"x"`** → 字符串
- **`\xHH`** 硬编码报文字节

### 字符串常量

```c
"abc"                    /* .rodata，只读 */
"hello" "world"          /* 拼接 → "helloworld" */
sizeof("abc") == 4       /* 含 '\0' */
strlen("abc") == 3       /* 不含 '\0' */
```

- 修改 `"..."` 内容 → **段错误**
- `char buf[] = "abc";` 可改（栈/静态数组）

### 浮点常量

`3.14`、`1e-6`

### enum

```c
enum pkt_type {
    PKT_ORDER = 0,
    PKT_QUOTE,
    PKT_HEARTBEAT
};
```

- 编译期整型；默认可递增
- 比纯 `#define` 有类型检查；驱动/协议类型标准写法

### const 对象

```c
const int max_retry = 3;
```

- 可放 **.rodata**；运行期不可改

## 实操

`const char *` 改字符串常量 → 观察 SIGSEGV。

## 与后续衔接

**ch14** `#define` 宏常量；**ch09** 字符串细节。

# 3.9 作用域、存储类型示例

## 本节讲什么

**综合示例**：同一程序中块/文件作用域、auto/static、extern、链接与段的对应关系。建议对照源码 + `readelf` + gdb `print &var`。

## 要点

### 典型布局（概念）

```c
#include <stdio.h>

int g_init = 42;           /* .data */
int g_zero;                /* .bss */
static int s_file = 1;     /* .data, 内部链接 */
const char *msg = "hi";    /* ptr .data, "hi" .rodata */

void demo(void) {
    auto int x;            /* 栈，未初始化 */
    static int call = 0;   /* .bss/data, 仅 demo 可见 */
    call++;
    printf("call=%d x=%d g=%d\n", call, x, g_init);
}

extern void other(void);   /* 声明在别 .c 定义 */
```

### 观察清单

| 变量 | 作用域 | 存储期 | 段 |
|------|--------|--------|-----|
| `g_init` | 文件 | 静态 | .data |
| `g_zero` | 文件 | 静态 | .bss |
| `s_file` | 文件 | 静态 | .data |
| `x` | 块 | 自动 | 栈 |
| `call` | 块 | 静态 | .data/.bss |

### 实验

1. 多文件 `extern` / `static` 链接实验  
2. 无符号 char 与 `-1` 比较（**3.1.1**）  
3. 自写 **htonl** 理解大小端（**3.1**）

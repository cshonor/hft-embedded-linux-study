# 3.6 链接属性

## 本节讲什么

**链接（linkage）**：标识符能否在**多个翻译单元**间指同一对象。

## 要点

| 链接 | 条件 | 例子 |
|------|------|------|
| **外部** | 文件域，无 static | `int g_cfg;` |
| **内部** | 文件域 **static** | `static int s;` |
| **无** | 块域 auto、typedef 名 | 局部变量 |

```c
/* a.c */
int shared = 1;

/* b.c */
extern int shared;
```

- **extern 声明**不分配存储
- **定义**全局唯一 → 否则 `multiple definition`

## 与 HFT/DPDK

- 多 `.o` 链成网关：符号表与 **3.6** 直接相关
- 头文件：`extern`；单 `.c`：`定义`

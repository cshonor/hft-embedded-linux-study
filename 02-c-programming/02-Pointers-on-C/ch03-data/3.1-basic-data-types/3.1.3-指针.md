# 3.1.3 指针

## 本节讲什么

**指针**作为一种**数据类型**：存地址，有指向类型。本章仅类型层面预告；**ch06** 完整展开。

## 要点

```c
int *p;           /* 指向 int */
const char *s;    /* 指向只读 char */
void *generic;    /* 泛型地址，使用前需 cast */
```

- **`sizeof(指针)`** 与指向类型无关，64 位通常 8 字节
- 指针类型决定：**解引用宽度**、**指针算术步长**
- **NULL**：空指针，解引用 UB

### 限定符与指针（预告）

| 写法 | 含义 |
|------|------|
| `const int *p` | 指向内容只读 |
| `int *const p` | 指针本身不可改 |
| `const int *const p` | 二者都只读 |

详见 **3.2.3**、**ch06**。

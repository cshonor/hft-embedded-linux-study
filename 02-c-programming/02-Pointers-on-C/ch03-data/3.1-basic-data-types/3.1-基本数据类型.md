# 3.1 基本数据类型

## 本节讲什么

C **内置类型**体系：整型、浮点、指针；配合 **限定符**（const/volatile/restrict）、**字节/ endian**。本章是 DPDK mbuf、内核 struct、交易二进制报文、寄存器读写的**数据地基**。

## 要点

- 类型决定：**位宽**、**取值范围**、**内存布局**、**运算规则**
- 工程实践：优先 **`<stdint.h>`** 定长类型（`uint8_t`…），DPDK/内核风格
- **整数提升**、**有符号/无符号混合** → 隐蔽 bug（详见 3.1.1）
- **1 字节 = 8 bit**；协议与硬件按字节/bit 组织
- **大小端**：x86 小端；**网络序大端** → `htonl`/`ntohl`（HFT 网关必备）

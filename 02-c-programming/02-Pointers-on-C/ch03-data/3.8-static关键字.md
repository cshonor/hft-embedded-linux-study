# 3.8 static 关键字

## 本节讲什么

**static** 在数据章节的两种语义：**改变存储期/链接**，DPDK 单例池与符号隔离的核心。

## 要点

### 1. 块内 static（静态存储期）

```c
void init_once(void) {
    static void *pool = NULL;
    if (!pool)
        pool = alloc_hugepage();
}
```

- 存 **.data/.bss**，非栈
- **程序全程**存活；**仅首次**执行到声明时初始化
- 作用域仍限块内

### 2. 文件域 static（内部链接）

```c
static int helper(void) { return 0; }
static rte_spinlock_t lock;
```

- 符号**不导出**；其他 `.c` 不可链接
- 驱动/DPDK 模块私有 API

### 对比表

| 写法 | 存储 | 链接 | 可见 |
|------|------|------|------|
| 局部 auto | 栈 | 无 | 块 |
| 局部 static | .data/.bss | 无 | 块 |
| 全局 | .data/.bss | 外部 | 全工程 |
| 全局 static | .data/.bss | 内部 | 本文件 |

## 实操

循环调用含 static 局部的函数，验证只初始化一次。

# 15.14 流错误函数

## 本节讲什么

**`feof` / `ferror` / `clearerr`** —— 正确判断读循环结束 vs 出错。

## 要点

```c
Quote pkt;
while (fread(&pkt, sizeof pkt, 1, fp) == 1)
    process_quote(&pkt);

if (ferror(fp))
    perror("read quote file");
/* feof 在正常读完时为真，ferror 区分磁盘错误 */
```

| 函数 | 含义 |
|------|------|
| **feof(fp)** | 到达 EOF（**读后**才可靠） |
| **ferror(fp)** | 发生 IO 错误 |
| **clearerr(fp)** | 清除 EOF/错误标志 |

### 陷阱

- **不要用 feof 作 while 前置条件** → 多读一次或漏判错误

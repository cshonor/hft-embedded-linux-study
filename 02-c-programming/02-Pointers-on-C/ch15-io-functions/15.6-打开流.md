# 15.6 打开流

## 本节讲什么

**`fopen(path, mode)`** 打开模式 —— 文本/二进制、读/写/追加。

## 要点

| 模式 | 作用 |
|------|------|
| **`r`** | 只读，须存在 |
| **`w`** | 只写，截断或创建 |
| **`a`** | 追加写 |
| **`r+`** | 读写，须存在 |
| **`rb`/`wb`/`ab`** | 二进制（**推荐**数据文件） |

```c
FILE *fp = fopen("quote.dat", "rb");
if (!fp) {
    perror("fopen");
    return -1;
}
```

- 跨平台统一加 **`b`**，防 Windows 换行污染二进制

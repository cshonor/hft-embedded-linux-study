# 15.8 字符 I/O

## 本节讲什么

**`fgetc` / `fputc`**、**`getchar` / `putchar`** —— 逐字节读写。

## 要点

```c
int c = fgetc(fp);
if (c == EOF) { /* 结束或错误 */ }
else { /* c 为 0..255 */ }
```

### 陷阱

- 返回值必须用 **`int`**，不能 **`char`**
- **EOF = -1**，`signed char` 会误判

```c
fputc('A', fp);
```

- 二进制字节流可逐字节解析；大批量用 **fread** 更高效

## 与后续衔接

**ch09** 单字节；pcap 头可 fgetc 或 fread。

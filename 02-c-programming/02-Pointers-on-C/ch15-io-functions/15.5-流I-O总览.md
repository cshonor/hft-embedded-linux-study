# 15.5 流 I/O 总览

## 本节讲什么

stdio 按**粒度**分层：字符 → 行 → 格式化 → **二进制块** —— 选型一览。

## 要点

```
字符 fgetc/fputc
  ↓
行   fgets/fputs
  ↓
格式化 fprintf/snprintf
  ↓
二进制 fread/fwrite  ← 报文/struct 持久化
```

| 数据类型 | 接口 |
|----------|------|
| 文本配置 | **fgets** |
| 日志 | **fprintf** + **fflush** |
| 二进制行情 | **fread/fwrite** |
| 禁止 | sprintf 无界；gets |

## 与后续衔接

**ch09** 二进制 vs 字符串；**ch10** packed struct。

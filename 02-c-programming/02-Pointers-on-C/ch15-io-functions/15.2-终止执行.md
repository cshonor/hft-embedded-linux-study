# 15.2 终止执行

## 本节讲什么

**`exit` / `abort`** —— 致命 IO/配置错误时终止进程；与 **fflush** 日志留存配合。

## 要点

```c
#include <stdlib.h>

exit(1);    /* 正常清理：flush 缓冲、atexit 回调 */
abort();    /* 异常终止，可能不 flush */
```

| 函数 | 行为 |
|------|------|
| **`exit(status)`** | 关闭 stdio 流、调用 atexit |
| **`abort()`** | SIGABRT，调试 core |

- 网关启动 fopen 失败可 **exit**；数据面热路径避免 exit

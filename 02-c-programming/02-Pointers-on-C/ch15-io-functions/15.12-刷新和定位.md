# 15.12 刷新和定位

## 本节讲什么

**`fflush`** 强制刷缓冲；**`fseek` / `ftell` / `rewind`** 随机偏移。

## 要点

### fflush

```c
fflush(stdout);   /* 不等 \n */
fflush(log_fp);   /* 崩溃前落盘 */
fflush(stderr);
```

- **`fflush(stdin)`** → 未定义，**禁止**

### 定位

```c
rewind(fp);                          /* 回到开头 */
long pos = ftell(fp);
fseek(fp, offset, SEEK_SET);         /* 绝对 */
fseek(fp, delta, SEEK_CUR);          /* 相对 */
fseek(fp, -8, SEEK_END);             /* 从尾 */
```

- stdio 缓冲与 fseek 交互复杂；**大文件随机读**考虑 **lseek+read**
- 历史行情按 seq 跳转索引文件 + fseek

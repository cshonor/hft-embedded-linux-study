# 15.16 文件操纵函数

## 本节讲什么

**`remove` / `rename`** 等文件级操作 —— 日志轮转、临时文件清理。

## 要点

```c
remove("old_quote.dat");           /* 删除文件 */
rename("quote.tmp", "quote.dat");  /* 原子替换（同 FS） */
```

- 滚动日志：rename 当前 → 归档，再 fopen 新文件
- 与 **fopen/fclose** 配合；打开中的文件 remove 行为实现相关

### stdio vs syscall

| 层 | 接口 |
|----|------|
| stdio | fopen, fread, fprintf |
| POSIX | open, read, write, lseek |

- 极致吞吐见 K&R UNIX IO

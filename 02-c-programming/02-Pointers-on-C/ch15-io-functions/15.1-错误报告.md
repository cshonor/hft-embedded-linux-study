# 15.1 错误报告

## 本节讲什么

IO 失败时用 **`perror`**、**`strerror(errno)`** 输出可读错误 —— 网关 fopen/fread 容错标配。

## 要点

```c
FILE *fp = fopen("quote.dat", "rb");
if (!fp) {
    perror("fopen quote.dat");
    return -1;
}
```

- 配合 **`errno`**（**ch09** strerror）
- **`stderr`** 无缓冲，故障日志立即输出

# 15.11 二进制 I/O

## 本节讲什么

**`fread` / `fwrite`** —— 原始内存块读写，**DPDK/HFT 持久化核心**。

## 要点

```c
size_t fread(void *buf, size_t size, size_t nmemb, FILE *fp);
size_t fwrite(const void *buf, size_t size, size_t nmemb, FILE *fp);
```

- 返回成功读写的**元素个数**；`< nmemb` → EOF 或错误

```c
typedef struct __attribute__((packed)) Quote {
    uint64_t seq;
    uint32_t bid, ask;
} Quote;

Quote pkt;
while (fread(&pkt, sizeof pkt, 1, fp) == 1)
    process_quote(&pkt);

Quote batch[32];
fwrite(batch, sizeof batch[0], 32, fp);
```

### 规范

- 二进制**只用 fread/fwrite**，不用 fprintf/fputs
- 注意 **packed**、字节序、跨平台（**ch10**）
- pcap、行情 `.dat` 同模型

# 15.13 改变缓冲方式

## 本节讲什么

**`setvbuf` / `setbuf`** 调整全缓冲、行缓冲、无缓冲 —— HFT 日志抖动调优。

## 要点

```c
setvbuf(fp, NULL, _IONBF, 0);   /* 无缓冲 */
setvbuf(fp, NULL, _IOLBF, 0);   /* 行缓冲 */
setvbuf(fp, buf, _IOFBF, size); /* 全缓冲 + 自定义 buf */
```

| 模式 | 行为 |
|------|------|
| **_IONBF** | 类似 stderr，立即写 |
| **_IOLBF** | 遇 `\n` 刷新 |
| **_IOFBF** | 满缓冲才 syscall |

- 高频 **fprintf** 到 stdout 易抖动 → 改 _IONBF 或写 stderr/文件
- 数据面仍建议 bypass stdio

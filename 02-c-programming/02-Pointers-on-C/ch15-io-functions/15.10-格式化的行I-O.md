# 15.10 格式化的行 I/O

## 本节讲什么

**printf 族**：**fprintf** 落盘、**snprintf** 安全拼装 —— 日志核心。

## 要点

| 函数 | 输出目标 |
|------|----------|
| **printf** | stdout |
| **fprintf(fp, ...)** | 指定流 |
| **sprintf** | 数组，**无界，禁用** |
| **snprintf(buf, size, ...)** | **限长，生产强制** |

```c
snprintf(log_line, sizeof log_line,
         "seq=%llu bid=%u ask=%u", seq, bid, ask);
fprintf(log_fp, "%s\n", log_line);
```

### 常用占位符

| 符 | 用途 |
|----|------|
| `%u`/`%llu` | 订单 ID、时间戳 |
| `%x`/`%X` | 标志位、寄存器 |
| `%p` | 指针调试 |
| `%s` | **仅 `\0` 终止文本** |

- 底层 **vfprintf** + **stdarg**（**ch07**）
- 崩溃后 **`fflush(stderr)`**（**15.12**）

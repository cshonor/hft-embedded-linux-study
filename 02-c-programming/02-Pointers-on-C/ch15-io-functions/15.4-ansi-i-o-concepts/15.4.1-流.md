# 15.4.1 流

## 本节讲什么

**`FILE *` 流**：封装 fd、用户缓冲、读写位置、错误/EOF 标志。

## 要点

### 预定义标准流

```c
FILE *stdin;   /* fd=0 输入 */
FILE *stdout;  /* fd=1 输出 */
FILE *stderr;  /* fd=2 错误 */
```

| 流 | 缓冲 |
|----|------|
| **stdout** | 行缓冲（遇 `\n` 或满） |
| **stderr** | **无缓冲**（日志/崩溃首选） |
| **stdin** | 通常行缓冲 |

### 本质

- 读写先经**用户态缓冲区**，减少 syscall → 吞吐↑、延迟抖动↑
- DPDK 数据面常绕过此层

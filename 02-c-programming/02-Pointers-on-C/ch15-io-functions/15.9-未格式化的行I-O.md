# 15.9 未格式化的行 I/O

## 本节讲什么

**`fgets` / `fputs`** —— 安全读行、写字符串（无自动 `\n` on fputs）。

## 要点

### fgets（替代已移除的 gets）

```c
char buf[256];
if (fgets(buf, sizeof buf, fp) != NULL) {
    /* 含 \n 或 EOF 前最多 sizeof-1 字符，末尾 \0 */
}
```

- 配置文件、文本日志解析标准接口
- 缓冲区满时截断，仍带 `\0`

### fputs

```c
fputs("[TRACE] new quote\n", fp);   /* 不自动追加换行 */
```

- 二进制 buf **禁止**当 `%s` 打印；文本行用 fgets

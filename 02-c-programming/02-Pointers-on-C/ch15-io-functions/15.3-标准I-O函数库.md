# 15.3 标准 I/O 函数库

## 本节讲什么

**`<stdio.h>`** 标准 IO 库总览 —— 带缓冲的上层封装，对应底层 **open/read/write**。

## 要点

| 类别 | 代表函数 |
|------|----------|
| 打开/关闭 | fopen, fclose |
| 字符 | fgetc, fputc |
| 行 | fgets, fputs |
| 格式化 | fprintf, snprintf |
| 二进制 | **fread, fwrite** |
| 定位/刷新 | fseek, fflush |

### HFT 取舍

- **控制面/日志/持久化** → stdio  
- **数据面极致延迟** → 绕过 stdio，原生 syscall

# 15.15 临时文件

## 本节讲什么

**`tmpfile`**、**`fmemopen`**（POSIX）— 临时中转、内存流，减磁盘 IO。

## 要点

### tmpfile

```c
FILE *tf = tmpfile();   /* 二进制临时文件，fclose 后自动删除 */
```

- 报文中转、单测；程序异常退出可能残留（实现相关）

### fmemopen（Linux）

```c
char membuf[4096];
FILE *mf = fmemopen(membuf, sizeof membuf, "w+");
fprintf(mf, "seq=%u", seq);
fflush(mf);
/* membuf 内已有格式化结果，无需落盘 */
```

- 低延迟网关内存内日志格式化

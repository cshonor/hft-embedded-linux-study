# 《C 和指针》

**Pointers on C** — Kenneth Reek

## 定位

阶段 1 · 内存与指针专精。主攻结构体内存布局、联合体、指针、数组——读懂内核结构体、DPDK 结构、栈帧、ABI 的基础。

## 阅读建议

与 K&R 配合：K&R 建立 C 语法习惯，本书深挖指针与内存模型。

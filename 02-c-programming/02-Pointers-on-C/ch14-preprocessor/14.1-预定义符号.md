# 14.1 预定义符号

## 本节讲什么

编译器**自动提供**的内置宏 —— 崩溃定位、平台判定、版本信息，无需 `#define`。

## 要点

### 源码位置

```c
#define ERR_LOG() \
    fprintf(stderr, "at %s:%d %s\n", __FILE__, __LINE__, __func__)
```

| 宏 | 含义 |
|----|------|
| **`__FILE__`** | 当前源文件名 |
| **`__LINE__`** | 当前行号 |
| **`__func__`** | 当前函数名（C99） |
| **`__DATE__` / `__TIME__`** | 编译时间戳 |

### 平台 / 架构

```c
#if defined(__x86_64__)
#elif defined(__aarch64__)
#endif

#if defined(__linux__)
#endif
```

- DPDK/内核跨 x86/ARM 条件编译依赖此类宏

### 编译器

- **`__GNUC__`**、**`__clang__`**：GCC/Clang 版本判定

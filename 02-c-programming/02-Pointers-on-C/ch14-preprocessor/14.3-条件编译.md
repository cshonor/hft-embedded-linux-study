# 14.3 条件编译

## 本节讲什么

**`#if` / `#ifdef` / `#elif` / `#else` / `#endif`** —— 调试开关、跨平台、特性裁剪。

## 要点

### 调试开关（HFT 线上关日志）

```c
#define DEBUG 1
#if DEBUG
#define DBG_LOG(...) LOG(__VA_ARGS__)
#else
#define DBG_LOG(...) do {} while (0)
#endif
```

- 关闭时 **零开销**，汇编中无调用

### 跨平台

```c
#if defined(__x86_64__)
    /* ... */
#elif defined(__aarch64__)
    /* ... */
#endif
```

### 特性开关

```c
#ifdef ENABLE_TCP_FAST
    /* 快速路径 */
#else
    /* 兼容路径 */
#endif
```

### 指令集

| 指令 | 作用 |
|------|------|
| `#ifdef NAME` | 已定义则编译 |
| `#ifndef NAME` | 未定义则编译 |
| `#if EXPR` | 表达式非 0 |

## 与后续衔接

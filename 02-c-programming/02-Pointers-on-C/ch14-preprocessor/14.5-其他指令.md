# 14.5 其他指令

## 本节讲什么

**`#error`、 `#pragma`、 `#line`** 等补充指令。

## 要点

### `#error` — 编译期阻断

```c
#if BURST_SIZE > 64
#error "BURST_SIZE cannot exceed 64"
#endif
```

- 非法配置**主动失败**，优于运行期崩溃

### `#pragma` — 编译器扩展

| 用法 | 作用 |
|------|------|
| **`#pragma once`** | 头文件保护 |
| **`#pragma pack(1)`** | 结构体对齐（**ch10** packed 替代） |
| **`#pragma GCC diagnostic`** | 局部屏蔽告警 |

```c
#pragma pack(push, 1)
struct proto_hdr { /* ... */ };
#pragma pack(pop)
```

### `#line`

- 重写 **`__LINE__` / `__FILE__`**；代码生成器极少用

### `#undef`

```c
#undef DEBUG
```

- 取消宏，避免泄漏到后续代码

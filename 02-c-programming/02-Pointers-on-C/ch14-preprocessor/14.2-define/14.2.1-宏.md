# 14.2.1 宏

## 本节讲什么

**对象宏**：常量、标志位、掩码 —— 编译期纯文本替换。

## 要点

```c
#define BURST_SIZE 32
#define PKT_FLAG_ORDER (1U << 2)
```

### 括号陷阱

```c
#define MUL(a,b) a * b
MUL(1+2, 3+4)   /* → 1+2*3+4 = 11，非 21 */

#define MUL(a,b) ((a) * (b))   /* 正确 */
```

- **表达式宏整体加括号**；参数各自加括号

### 特点

| 项 | 说明 |
|----|------|
| 类型 | 无 |
| 作用域 | 从 `#define` 到 `#undef` 或文件末 |
| 调试 | gdb 难断点（已展开） |

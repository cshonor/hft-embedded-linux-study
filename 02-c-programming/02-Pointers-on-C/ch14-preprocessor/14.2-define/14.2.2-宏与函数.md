# 14.2.2 宏与函数

## 本节讲什么

**宏 vs `static inline` 函数** —— 底层工程取舍标准。

## 要点

| 特性 | `#define` 函数宏 | **`static inline`** |
|------|------------------|---------------------|
| 参数求值 | 可能**多次** | **一次** |
| 类型检查 | 无 | 有 |
| 作用域 | 全局污染 | 文件内 static |
| 调试 | 难断点 | 可断点 |
| 适用 | 常量、日志、`#`/`##` | 计算、MIN/MAX |

```c
static inline int min_int(int a, int b)
{
    return a < b ? a : b;
}
```

### 硬性准则

- **复杂计算** → inline  
- **日志、常量、符号拼接** → 宏

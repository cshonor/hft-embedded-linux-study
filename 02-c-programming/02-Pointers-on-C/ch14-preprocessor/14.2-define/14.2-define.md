# 14.2 #define

## 本节讲什么

**预处理器**编译第一阶段：纯**文本 token 替换**，无类型、无作用域检查。

## 要点

### 预处理流程（概要）

1. 分 token、删注释  
2. **`#include`** 嵌入头文件  
3. **`#define`** 宏展开  
4. **`#if` / `#ifdef`** 条件编译  
5. **`#error` / `#pragma`** 等  

### 两大宏类型

| 类型 | 示例 |
|------|------|
| **对象宏** | `#define BURST_SIZE 32` |
| **函数宏** | `#define MIN(x,y) ...` |

- DPDK 常量掩码、**rte_log** 可变参数宏均基于此

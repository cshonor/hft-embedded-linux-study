# 14.2.3 带参数的宏

## 本节讲什么

**函数宏**、**`#` 字符串化**、**`##` 拼接**、**可变参数 `__VA_ARGS__`**。

## 要点

### 函数宏

```c
#define MIN(x,y) ((x) < (y) ? (x) : (y))
```

### 字符串化 `#`

```c
#define LOG_VAR(name) printf(#name " = %d\n", name)
LOG_VAR(seq);   /* printf("seq" " = %d\n", seq); */
```

### 标记拼接 `##`

```c
#define PKT_TYPE(name) PKT_##name
uint32_t t = PKT_TYPE(QUOTE);   /* PKT_QUOTE */
```

### 可变参数宏

```c
#define LOG(fmt, ...) printf("[TRACE] " fmt, __VA_ARGS__)

/* GCC：无参时去掉多余逗号 */
#define LOG(fmt, ...) printf(fmt, ##__VA_ARGS__)
LOG("ok");
```

### 多语句宏

```c
#define DBG(x) do { \
    fprintf(stderr, "%s\n", (x)); \
} while (0)
```

- **`do {} while(0)`** 保证 if/for 后语法完整（**14.3**）

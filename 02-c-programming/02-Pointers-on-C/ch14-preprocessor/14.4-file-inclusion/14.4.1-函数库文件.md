# 14.4.1 函数库文件

## 本节讲什么

**`#include <header.h>`** —— 系统 / 标准库头，从系统 include 路径搜索。

## 要点

```c
#include <stdint.h>
#include <string.h>
#include <rte_mbuf.h>   /* DPDK 安装路径 */
```

| 格式 | 搜索 |
|------|------|
| **`<...>`** | 系统标准路径 + `-I` 追加 |
| libc、内核 UAPI | 工程外依赖 |

### 与链接

- 头文件 = **声明**；实现由库链接阶段提供（**ch18**）

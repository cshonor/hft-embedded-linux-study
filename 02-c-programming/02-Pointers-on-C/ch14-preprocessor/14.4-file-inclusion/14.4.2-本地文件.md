# 14.4.2 本地文件

## 本节讲什么

**`#include "local.h"`** 与**头文件保护** —— 防重复定义编译错误。

## 要点

```c
#include "quote_proto.h"   /* 先搜当前目录，再 -I */
```

### Include guard（强制）

```c
#ifndef RTE_MBUF_H
#define RTE_MBUF_H
/* 声明 */
#endif
```

### `#pragma once`

```c
#pragma once
/* 等价一次性保护，主流编译器支持 */
```

- 多重包含同一头 → **重复定义**报错；无保护时必现

### 工程实践

- 声明放 `.h`，实现放 `.c`  
- 减少头文件互相 include，用前置声明  

## 实操

去掉 guard 制造重复定义，再加 guard 修复。

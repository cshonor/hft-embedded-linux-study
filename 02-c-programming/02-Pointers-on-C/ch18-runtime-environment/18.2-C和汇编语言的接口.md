# 18.2 C 和汇编语言的接口

## 本节讲什么

**ABI 调用约定**、**静态/动态链接** —— C 与机器码、`.so` 的边界。

## 要点

### 调用约定（System V x86-64）

| 项 | 规则 |
|----|------|
| 整数/指针参数 | rdi, rsi, rdx, rcx, r8, r9，再压栈 |
| 返回值 | rax（及 rdx 128 位） |
| 被调保存 | rbx, rbp, r12–r15 |
| 栈对齐 | call 前 rsp 16 字节对齐 |

- 内核模块、DPDK PMD 与 C 互调须遵守同一 ABI

### 静态 vs 动态链接

| 方式 | 特点 |
|------|------|
| **`-static`** | 库打进 ELF，无 .so 依赖，体积大；**DPDK 数据面常用** |
| **动态（默认）** | 小体积，**ld-linux.so** 运行时解析；库版本须一致 |

```bash
gcc -static -o app app.c
ldd ./app          /* 动态依赖 */
nm ./app | head    /* 符号表 */
```

### 符号

- 未定义符号 → 链接失败 **`undefined reference`**  
- 动态库缺符号 → 运行 **`undefined symbol`**

# 18.3 运行时效率

## 本节讲什么

**brk/mmap 堆**、**缓存行/大页**、**信号与调试** —— DPDK/HFT 性能与稳定性收官。

## 要点

### 堆底层

| 机制 | 用途 |
|------|------|
| **brk/sbrk** | 扩展堆顶，小块连续分配 |
| **mmap(MAP_ANONYMOUS)** | 大块、**DPDK 大页**、独立映射 |

- **malloc**：小块多 brk；大块 mmap  
- **free** 后 libc 未必立刻 munmap  
- **rte_malloc** / 大页：**mmap** 物理连续，避 brk 碎片与全局锁

### 对齐与缓存

- **自然对齐**；ARM 非对齐 → **SIGBUS**  
- **Cache Line ~64B**：伪共享 → 多线程改同线不同变量 → 延迟抖动  
- **对策**：`posix_memalign`、结构体 **padding** 隔离（DPDK per-lcore 变量）  
- **大页 2MB/1GB**：减 TLB miss，DPDK 强制 `-huge`

### 信号

| 信号 | 常见原因 |
|------|----------|
| **SIGSEGV** | 空指针、栈/堆越界、写 .rodata |
| **SIGABRT** | double free、abort |
| **SIGBUS** | 非对齐（ARM） |
| **SIGKILL** | OOM killer |

### 工具

- **gdb**：ELF 符号 + 栈帧 backtrace  
- **valgrind**：拦截 malloc，查泄漏/越界/悬垂  

### 内核对照

| 用户态 | 内核 |
|--------|------|
| brk 堆 | **kmalloc** |
| mmap | **vmalloc** / 大映射 |

# 18.1.1 测试程序

## 本节讲什么

用**打印地址**、`readelf` 观察各段与 **crt0→main** 启动链。

## 要点

```c
int g_init = 42;
int g_zero;
static int s_init = 1;

int main(void)
{
    int local = 0;
    void *heap = malloc(16);
    printf("text/main  %p\n", (void *)main);
    printf("rodata     %p\n", (void *)"quote");
    printf("data g     %p\n", (void *)&g_init);
    printf("bss g_zero %p\n", (void *)&g_zero);
    printf("stack local%p\n", (void *)&local);
    printf("heap       %p\n", heap);
    free(heap);
}
```

```bash
readelf -S ./a.out   /* .text .rodata .data .bss */
```

### 启动流程（非直接进 main）

```
内核 execve 加载 ELF
  → _start (crt0)
  → 初始化 .data/.bss、stdio、environ
  → 加载动态库（若动态链接）
  → main(argc, argv, environ)
```

### 退出

- **exit/return**：atexit、fflush stdio、内核回收 VMA  
- **abort/SIGSEGV**：常跳过 atexit，日志可能丢失

## 实操

对比栈地址 > 堆地址（x86 Linux 典型）。

# 18.1.4 寄存器变量

## 本节讲什么

**register** 提示与 **ABI 寄存器传参** —— 现代编译器自动寄存器分配为主。

## 要点

```c
register int hot_counter;   /* C 标准：仅 hint，可能被忽略 */
```

- x86-64：**call** 约定已用寄存器传参（**18.2**）  
- 热变量靠 **`-O2` 优化** 进寄存器，少手写 register  

### 与 HFT

- 微基准勿过度依赖 register 关键字  
- 真正优化：**inline**、减间接调用、数据面少函数层  

## 与后续衔接

**18.2** 调用约定；DPDK **likely/unlikely** 分支提示（编译器扩展）。

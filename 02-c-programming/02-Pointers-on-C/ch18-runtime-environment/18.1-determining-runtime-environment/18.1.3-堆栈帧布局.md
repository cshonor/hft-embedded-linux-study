# 18.1.3 堆栈帧布局

## 本节讲什么

**栈帧 stack frame** —— call/ret、rbp/rsp；HFT 禁递归、禁大栈数组、禁返回栈指针。

## 要点

### x86-64 ABI 概要

1. 前 6 个整型/指针参：**rdi rsi rdx rcx r8 r9**  
2. **call**：压返回地址  
3. 被调：**push rbp**；**mov rbp,rsp**；sub rsp 开局部空间  
4. 返回：**leave**；**ret**；结果常放 **rax**

```c
/* gdb: info frame, x/10gx $rsp */
```

### 工程结论

| 禁止 | 原因 |
|------|------|
| 深递归 | 栈 ~8MB 耗尽 → **SIGSEGV** |
| `char buf[1024*1024]` 局部 | 栈溢出 |
| **返回局部变量地址** | 悬垂指针 UB |
| 写 `"literal"[0]='x'` | .rodata → SIGSEGV |

### 优化

- **`static inline`** 小函数：消除 call/ret（**ch07/ch14**）

## 实操

gdb 观察 rbp/rsp；超大局部数组复现 SIGSEGV。

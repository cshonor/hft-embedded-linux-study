# 18.1 判断运行时环境

## 本节讲什么

**Linux 进程虚拟地址空间**六大段 —— 把 ch02 变量存储、ch11 堆、栈帧映射到 ELF 加载后的真实布局。

## 要点

| 段 | 内容 | 权限 |
|----|------|------|
| **`.text`** | 机器码 | 读+执行 |
| **`.rodata`** | 字面量、const 全局 | 只读 |
| **`.data`** | 已初始化全局/static | 读写 |
| **`.bss`** | 未初始化全局（加载时清零） | 读写 |
| **Heap** | malloc/brk/mmap | 读写，↑ 增长 |
| **Stack** | 局部变量、栈帧 | 读写，↓ 增长 |

- **虚拟地址 ≠ 物理地址**；缺页时才映射物理页  
- DPDK **MAP_SHARED** 多进程共享大页  
- 栈默认 ~8MB；堆可达 GB 级

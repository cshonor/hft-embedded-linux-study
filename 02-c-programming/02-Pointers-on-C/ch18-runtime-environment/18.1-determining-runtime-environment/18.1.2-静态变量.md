# 18.1.2 静态变量

## 本节讲什么

**全局/static** 与 **链接属性**；跨文件初始化顺序陷阱。

## 要点

```c
uint64_t seq = 100;    /* .data */
uint64_t cnt;            /* .bss，启动时清零 */
static int helper_only;  /* 内部链接，仅本 .o 可见 */
```

| 关键字 | 链接 |
|--------|------|
| **static**（文件内） | 内部链接，不导出符号 |
| 全局无 static | 外部链接，extern 可引用 |
| **extern** | 声明，定义在他处 |

### 初始化顺序陷阱

- **不同 `.c` 的全局初始化顺序未定义**  
- 跨文件依赖 → 可能读到 BSS 的 0  
- **根治**：复杂初始化放 **main 开头**

### 动态库符号

```c
__attribute__((visibility("hidden"))) void priv_fn(void);
```

- 缩小符号表，减动态链接查找（HFT 优化）

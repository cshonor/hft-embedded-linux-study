# 7.3 函数的参数

## 本节讲什么

C **只有值传递**：形参是实参的**副本**；通过**指针副本**间接改外部内存。

## 要点

```c
void set_val(int x)   { x = 100; }      /* 不改外部 */
void set_ptr(int *x)  { *x = 100; }     /* 改外部 */
```

| 实参 | 拷贝内容 | 改外部？ |
|------|----------|----------|
| `int` | 整数值 | 否 |
| `int *` | 地址值 | 是（通过 `*`） |
| `struct rte_mbuf *` | 指针 | 是（改 mbuf 内容） |
| 大 `struct` 值 | 整块结构 | 否（且昂贵） |

### 工程选型（DPDK/HFT）

- **大结构体 / mbuf**：传 **指针**（`const` 若只读）
- **小标量**：传值
- **输出多个结果**：指针参数或 struct 指针

```c
int decode(const uint8_t *pkt, size_t len, struct quote *out);
```

### 改指针本身

要改调用者的指针变量（如 `alloc` 输出），需 **`T **`（**ch06 6.10**）。

### 函数指针铺垫

```c
typedef int (*handler_t)(struct pkt *);
handler_t h = process_order;
h(pkt);
```

**ch13** 回调、中断、报文分发器。

## 实操

对比值传递与指针传递对外部变量的影响。

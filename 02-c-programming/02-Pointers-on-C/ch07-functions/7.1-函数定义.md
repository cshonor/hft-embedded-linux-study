# 7.1 函数定义

## 本节讲什么

**函数定义** = 签名 + 函数体；产生 **.text** 机器指令，外部链接符号**全局唯一**（`static` 除外）。DPDK API、内核驱动的基础单元。

## 要点

```c
int rte_mbuf_refcnt_add(struct rte_mbuf *m, int delta)
{
    m->refcnt += delta;
    return m->refcnt;
}
```

- 函数体 **`{}`** 为块作用域；局部 **auto** 在**栈帧**分配
- **`static`** 修饰函数 → **内部链接**，仅本 `.c` 可见（DPDK/内核私有 helper）

### return 与返回值

| 类型 | 规则 |
|------|------|
| 非 void | **`return expr;`**；值经 **ABI** 寄存器（x86-64 **rax**）返回 |
| void | **`return;`** 或落到末尾 |
| 转换 | 返回值隐式转为声明类型；溢出可能 **UB** |

### 禁止：返回局部栈指针

```c
char *bad(void) {
    char buf[64] = "hello";
    return buf;   /* 悬垂指针 */
}
```

合法：**malloc** 堆内存、**static** 缓冲、调用者提供的缓冲区。

### inline（C99）

```c
static inline uint32_t get_flags(const struct rte_mbuf *m)
{
    return m->ol_flags;
}
```

- 建议编译期展开，减少 **call/ret**
- 高频小函数放 **头文件**；过大函数勿 inline（I-cache 膨胀）
- DPDK/内核：**`static inline`** 标配

### 栈调用预览（**ch18** 详述）

`call` → 压返回地址 → 建栈帧 → 执行 → `ret` 销毁帧

### 函数指针铺垫

函数名在表达式中退化为**函数地址**（**ch13** 完整展开）。

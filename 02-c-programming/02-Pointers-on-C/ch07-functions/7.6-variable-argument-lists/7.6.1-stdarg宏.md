# 7.6.1 stdarg 宏

## 本节讲什么

**va_list / va_start / va_arg / va_end**（C99 另有 **va_copy**）实现可变参迭代。

## 要点

```c
#include <stdarg.h>

void log_msg(const char *fmt, ...)
{
    va_list ap;
    va_start(ap, fmt);
    vprintf(fmt, ap);
    va_end(ap);
}
```

| 宏 | 作用 |
|----|------|
| **`va_list ap`** | 遍历状态 |
| **`va_start(ap, last)`** | `last` 为 `...` 前最后一个固定参数 |
| **`va_arg(ap, type)`** | 取下一参数，**type 必须正确** |
| **`va_end(ap)`** | 结束 |

### 包装 printf

```c
void err(const char *fmt, ...)
{
    va_list ap;
    va_start(ap, fmt);
    vfprintf(stderr, fmt, ap);
    va_end(ap);
}
```

## 与后续衔接

自定义日志层；**不要**对可变参做类型泛化魔法。

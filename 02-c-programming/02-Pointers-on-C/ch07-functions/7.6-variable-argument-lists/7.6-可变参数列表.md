# 7.6 可变参数列表

## 本节讲什么

**`...` 可变参数**函数：`printf`、`rte_log` 等。实现依赖 **`<stdarg.h>`**（**ch05 7.3** 已接触 `vfprintf`）。

## 要点

- 至少一个**固定参数**（常为 format 或 count）
- 编译器**不检查**可变部分类型 → 传错 → UB
- 底层优先**固定签名** API；日志场景再用可变参

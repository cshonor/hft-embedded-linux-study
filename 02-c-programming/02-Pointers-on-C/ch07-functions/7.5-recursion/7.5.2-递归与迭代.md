# 7.5.2 递归与迭代

## 本节讲什么

同一问题**递归** vs **循环**实现的权衡；HFT 与内核热路径偏好迭代。

## 要点

| | 递归 | 迭代 |
|---|------|------|
| 代码 | 简洁、贴近定义 | 常多几行状态变量 |
| 栈 | O(深度) | O(1) |
| 性能 | call/ret 开销 | 常更快 |
| HFT | **避免** | **首选** |

### 链表遍历迭代版

```c
void print_list(struct node *p)
{
    for (; p != NULL; p = p->next)
        printf("%d\n", p->data);
}
```

### 何时仍用递归

- 深度有界且小（AST、配置树）
- 非热路径工具代码

## 实操

同一链表递归 vs 循环；`ulimit -s` 观察深递归崩溃。

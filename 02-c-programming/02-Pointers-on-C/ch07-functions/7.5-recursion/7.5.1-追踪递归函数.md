# 7.5.1 追踪递归函数

## 本节讲什么

手工/trace **递归调用栈**：每层参数、局部变量、返回点。理解栈帧如何叠加。

## 要点

以链表打印为例：

```c
void print_list(struct node *p)
{
    if (p == NULL)
        return;
    printf("%d\n", p->data);
    print_list(p->next);   /* 先序；后序则先 next 再 printf */
}
```

### 追踪表（概念）

| 层 | p 指向 | 待返回后执行 |
|----|--------|--------------|
| 1 | node1 | print node2 链 |
| 2 | node2 | print node3 链 |
| … | … | … |

- **gdb**：`bt` 看递归深度；`frame N` 切换层

## 与后续衔接

# 7.5 递归

## 本节讲什么

函数**调用自身**；每层调用占一层**栈帧**。适用树/链表；HFT 数据面**禁止**深递归。

## 要点

- 组成：**基线条件** + 向基线推进
- 每层：局部变量、返回地址保存在栈上
- 深度过大 → **栈溢出** SIGSEGV

### 适用

- 内核树遍历、目录深度、排序（教学）
- 链表递归 walk（深度有限时）

### 不适用（HFT 热路径）

- 用 **while/for** 迭代替代
- 显式栈数组模拟递归

# 7.2.2 函数的缺省认定

## 本节讲什么

**`func()`** vs **`func(void)`**；旧式 C 的「缺省认定」与现代规范。

## 要点

| 写法 | 含义 |
|------|------|
| **`f(void)`** | **无参数**；调用 `f(1)` 应报错 |
| **`f()`** | 旧式：参数**未声明**（K&R）；C++ 非法；现代 C 仍可见遗留代码 |
| 无原型调用 | C89 隐式假定返回 **int**，参数不检查 → **灾难** |

### 工程强制

- 公共 API 一律 **`func(void)`** 表示无参
- 禁止无原型调用；**`-Wall`** 全开

### 旧式定义（了解即可）

```c
int f(a, b) int a; float b; { }  /* K&R */
```

- 读内核老代码可能遇到；新代码用 ANSI 原型

## 与后续衔接

**ch01** `int main(void)`；链接符号名 **name mangling** 无（C  plain）。

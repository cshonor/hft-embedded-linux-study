# 7.2.1 原型

## 本节讲什么

**函数原型（prototype）**：返回类型 + 函数名 + **参数类型列表**（参数名可选）。

## 要点

```c
int parse(const uint8_t *pkt, size_t len, struct quote *out);
void log_err(const char *fmt, ...);   /* 可变参见 7.6 */
```

- 调用前可见原型 → 类型/个数检查
- 头文件放原型；实现文件 `#include` 自己的 `.h`
- **`extern`** 可省略：`extern int foo(int);` ≡ `int foo(int);`

### 与定义一致

- 声明与定义的返回类型、参数类型必须兼容
- **`const`**、指针层级属于类型一部分

## 工程规范

- 每个 `.c` 对应 `.h` 导出公共 API
- 私有函数 **`static`** 放 `.c` 内，不进 `.h`

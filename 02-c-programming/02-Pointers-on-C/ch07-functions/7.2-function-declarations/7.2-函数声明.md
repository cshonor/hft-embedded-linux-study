# 7.2 函数声明

## 本节讲什么

**声明（原型）** vs **定义**：声明告知签名，可重复；定义分配代码，外部符号唯一。

## 要点

| | 声明 | 定义 |
|---|------|------|
| 位置 | `.h` 或 `.c` 顶部 | 某一个 `.c` |
| 函数体 | 无 | 有 `{ }` |
| 链接 | `extern` 隐含 | 产生符号 |

```c
/* header.h */
int rte_mbuf_refcnt_add(struct rte_mbuf *m, int delta);

/* mbuf.c */
int rte_mbuf_refcnt_add(struct rte_mbuf *m, int delta) { /* ... */ }
```

- 原型作用：**编译期**校验参数个数与类型
- 无原型 → 旧式隐式 **int** 规则（**7.2.2** 高危）

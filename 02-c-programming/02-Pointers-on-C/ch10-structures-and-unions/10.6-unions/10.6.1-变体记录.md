# 10.6.1 变体记录

## 本节讲什么

**变体记录（variant record）**：同一消息头后，载荷类型不同 —— 用 **union + 类型判别字段** 建模。

## 要点

```c
struct msg {
    uint32_t magic;
    uint8_t  type;   /* PKT_ORDER / PKT_QUOTE */
    union {
        struct { uint64_t order_id; } order;
        struct { uint32_t seq; } quote;
    } body;
};

if (m->type == PKT_ORDER)
    use(m->body.order.order_id);
```

### 匿名 struct/union（C11）

```c
struct msg {
    uint32_t magic;
    union {
        struct { uint64_t order_id; } order;
        struct { uint32_t seq; } quote;
    };   /* 直接 msg.order.order_id */
};
```

### 安全

- **必须**用 `type` / `magic` 等字段记录当前有效成员
- 禁止无判别地读 union 另一分支

## 与后续衔接

FIX/SBE 等协议变体；**ch13** 回调携带不同 payload 类型。

# 10.6.2 联合的初始化

## 本节讲什么

**union 初始化**：C99 指定**第一个成员**或具名成员；与 struct 指定初始化类似。

## 要点

```c
union raw_data u = { .word = 0x12345678 };

struct msg m = {
    .magic = MAGIC,
    .type  = PKT_QUOTE,
    .body  = { .quote = { .seq = 1001 } },
};
```

- 未写入的成员与「上次有效成员」共享存储 → 切换类型前可 **`memset`**

```c
memset(&u, 0, sizeof u);
u.word = value;
```

# 10.6 联合

## 本节讲什么

**union**：成员**共享同一块内存**；大小 = **最大成员** —— 字节拆分、寄存器多视图、变体载荷。

## 要点

```c
union raw_data {
    uint32_t word;
    uint8_t  bytes[4];
};

union raw_data u;
u.word = htobe32(0x12345678);
uint8_t b0 = u.bytes[0];
```

| 特性 | 说明 |
|------|------|
| 存储 | 同一时刻仅一个成员「有效」 |
| 大小 | max(各成员) |
| 风险 | 无类型标记 → 混读得脏数据 |

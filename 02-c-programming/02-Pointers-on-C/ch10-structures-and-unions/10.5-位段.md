# 10.5 位段

## 本节讲什么

**Bit-field**：在 struct 内按 **bit** 分配成员 —— 映射**本地硬件寄存器**，不用于网络协议。

## 要点

```c
struct dev_status {
    uint32_t ready : 1;
    uint32_t err   : 2;
    uint32_t mode  : 3;
    uint32_t rsv   : 26;
};
```

### 规则与限制

| 项 | 说明 |
|----|------|
| 类型 | 整型（`int` / `uint32_t` 等） |
| 顺序 | **编译器/平台相关** → 禁止用于线格式 |
| 取址 | **`&pkt->ready` 非法** |
| 跨平台 | bit 分配顺序不保证 |

### 适用

- PCI/MMIO 状态寄存器、驱动控制位
- **禁止**用于交易所报文、DPDK 线协议

## 与后续衔接

**ch05** 位运算；**ch09** 单字节掩码；网络序用 **htonl** + 移位。

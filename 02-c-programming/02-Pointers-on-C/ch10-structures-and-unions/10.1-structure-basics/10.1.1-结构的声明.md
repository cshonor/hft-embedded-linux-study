# 10.1.1 结构的声明

## 本节讲什么

**声明 struct 类型**、定义变量、**typedef** 简化 —— DPDK/内核标准写法。

## 要点

```c
/* 1. 先声明类型 */
struct quote_hdr {
    uint32_t seq;
    uint16_t price;
    uint8_t  pkt_type;
};

/* 2. 再定义变量 */
struct quote_hdr pkt;

/* 3. typedef（推荐） */
typedef struct quote_hdr QuoteHdr;
QuoteHdr pkt;
```

| 写法 | 说明 |
|------|------|
| `struct tag { ... };` | 标准声明 |
| `typedef struct tag Name;` | 省略每次写 struct |
| 旧式 `struct tag { ... } pkt;` | 可读性差，少用 |

### 定长无符号类型

协议字段用 **`uint8_t` / `uint16_t` / `uint32_t`**（**ch03**），避免 `int` 宽度随平台变化。

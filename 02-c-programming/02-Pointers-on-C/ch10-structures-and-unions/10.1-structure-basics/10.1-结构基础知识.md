# 10.1 结构基础知识

## 本节讲什么

**struct** 把多个字段绑成**二进制数据模型** —— 协议头、DPDK 元数据、内核对象的基础单元。

## 要点

```c
struct quote_hdr {
    uint32_t seq;
    uint16_t price;
    uint8_t  pkt_type;
};

typedef struct quote_hdr QuoteHdr;
QuoteHdr pkt;
```

### 嵌套结构体

```c
struct pkt_head {
    uint32_t magic;
    struct quote_hdr quote;
    uint8_t payload[64];
};

struct pkt_head *p = ...;
p->quote.seq = 1001;
```

- DPDK **`rte_mbuf`**、内核 **`task_struct`** 均为多层嵌套 struct

### 整体赋值

```c
QuoteHdr a = {0}, b;
b = a;   /* 逐成员拷贝 */
```

- 大 struct **按值拷贝**开销大 → HFT 传 **指针**（**10.4**）

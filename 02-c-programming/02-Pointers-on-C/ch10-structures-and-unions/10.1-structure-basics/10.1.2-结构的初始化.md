# 10.1.2 结构的初始化

## 本节讲什么

结构体**初始化**三种常用方式；DPDK mbuf / 协议头清零标配 **`{0}`**。

## 要点

### 顺序初始化

```c
QuoteHdr pkt = {1001, 9999, PKT_QUOTE};
```

- 顺序须与成员声明一致

### 指定成员初始化（C99，推荐）

```c
QuoteHdr pkt = {
    .seq      = 1001,
    .pkt_type = PKT_QUOTE,
    .price    = 9999,
};
```

- 顺序无关、可读性强

### 零初始化

```c
QuoteHdr pkt = {0};   /* 全部成员置 0 */
memset(&pkt, 0, sizeof pkt);
```

- 未列出的成员自动为 0

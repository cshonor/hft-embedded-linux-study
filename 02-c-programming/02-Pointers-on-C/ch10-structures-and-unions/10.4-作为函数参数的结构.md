# 10.4 作为函数参数的结构

## 本节讲什么

结构体**按值传递** vs **传指针** —— HFT 延迟与拷贝开销的关键选择。

## 要点

```c
void process_by_value(QuoteHdr pkt);      /* 完整拷贝 */
void process_by_ptr(const QuoteHdr *p);   /* 仅传地址 */
```

| 方式 | 开销 | 适用 |
|------|------|------|
| **按值** | 拷贝全部成员 | 小 struct（几个 int） |
| **指针** | 一个地址 | 大 struct、热路径、DPDK/内核默认 |
| **`const T *`** | 只读，防误改 | 解析报文头 |

### 返回 struct

```c
QuoteHdr make_hdr(void);   /* 可能按值返回，大对象避免 */
```

- 大对象：返回指针或写入调用者提供的 buffer

### 与 memcpy

- 协议解析：`memcpy(&hdr, buf, sizeof hdr)` 等价于「从字节流载入 struct」

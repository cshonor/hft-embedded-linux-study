# 10.2.3 结构的自引用

## 本节讲什么

结构体含**自身类型的指针**（非嵌套实体）→ 链表、树 —— **ch12/ch17** 前置。

## 要点

```c
typedef struct node {
    uint64_t order_id;
    struct node *next;   /* 正确：指针 */
    /* struct node next; 错误：大小无限 */
} Node;
```

### 单向链表插入

```c
Node *head = NULL;

void push(Node **head, uint64_t id) {
    Node *n = malloc(sizeof *n);
    n->order_id = id;
    n->next = *head;
    *head = n;
}
```

- 内核进程链表、DPDK 空闲块链表同构

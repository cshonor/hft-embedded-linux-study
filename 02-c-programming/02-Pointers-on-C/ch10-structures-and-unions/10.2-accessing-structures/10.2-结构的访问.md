# 10.2 结构的访问

## 本节讲什么

**`.`** 与 **`->`** 两种成员访问；嵌套与自引用指针 —— DPDK/内核最常用 **`->`**。

## 要点

| 运算符 | 对象 |
|--------|------|
| **`pkt.seq`** | 结构体变量 |
| **`p->seq`** | 结构体指针（等价 `(*p).seq`） |

```c
struct pkt_head *p = mbuf_payload;
p->quote.seq = 1001;
```

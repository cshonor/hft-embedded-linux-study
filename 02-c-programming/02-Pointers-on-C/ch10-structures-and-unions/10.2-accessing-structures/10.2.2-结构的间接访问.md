# 10.2.2 结构的间接访问

## 本节讲什么

**结构体指针**用 **`->`** 访问成员 —— DPDK mbuf、内核 `kmalloc` 对象标准模式。

## 要点

```c
QuoteHdr *p = &pkt;
p->seq = 10001;
/* 等价 (*p).seq */
```

### 常见来源

```c
QuoteHdr *hdr = (QuoteHdr *)rte_pktmbuf_mtod(mbuf, void *);
hdr->seq = rte_be_to_cpu_32(raw_seq);
```

### 空指针

- **`p->seq`** 在 `p == NULL` 时 → **SIGSEGV**
- 解引用前检查或使用内核/DPDK 已有断言宏

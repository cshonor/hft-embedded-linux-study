# 13.5 字符串常量

## 本节讲什么

**字符串字面量**与 **`char *` / `const char *`** —— 只读段、指针数组、与 **argv** 对比。

## 要点

```c
const char *proto = "heartbeat";   /* .rodata 只读 */
char *bad = "text";                /* 旧代码；写 → SIGSEGV */

const char *msgs[] = {
    "QUOTE",
    "ORDER",
    "CANCEL",
};
```

| 类型 | 存储 | 可写 |
|------|------|------|
| 字面量 | **.rodata** | 否 |
| `char buf[] = "x"` | 栈数组 | 是 |

### 指针数组 of 常量

```c
const char **table;   /* 元素指向只读串 */
```

- 与 **`char **`** 动态 strdup 数组区分（**13.1**）

### 工程规范

- 协议名、日志模板：**`const char *`**
- 勿对字面量 **`strcpy` 入栈缓冲以外目标** 写入

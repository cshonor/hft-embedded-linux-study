# 13.2 高级声明

## 本节讲什么

**复杂指针声明解读**：从内向外、先右后左 —— 底层 C / 内核 / 面试高频。

## 要点

口诀：**先找标识符 → 括号优先 → 先右后左**

| 声明 | 解读 |
|------|------|
| `int *arr[10];` | **指针数组**：10 个 `int *` |
| `int (*arr)[10];` | **数组指针**：指向含 10 个 int 的数组 |
| `int (*func)(int);` | **函数指针**：入参 int，返回 int |
| `int (*table[16])(int);` | **函数指针数组** |
| `char ***msg;` | 三级 char 指针 |

### 与 DPDK/内核

```c
struct rte_mbuf *pkts[32];           /* 指针数组 */
int (*cmp)(const void *, const void *); /* qsort 式比较函数 */
```

### typedef 简化

```c
typedef int (*cmp_fn_t)(const void *, const void *);
cmp_fn_t cmp = my_compare;
```

- 复杂回调**必须 typedef**（DPDK/内核规范）

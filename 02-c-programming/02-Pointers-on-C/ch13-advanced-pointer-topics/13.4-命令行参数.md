# 13.4 命令行参数

## 本节讲什么

**`main(int argc, char **argv)`** —— `char **` 的标准应用：字符串指针数组。

## 要点

```c
int main(int argc, char **argv)
{
    /* argv[0] 程序名；argv[1..] 参数；argv[argc] == NULL */
    for (int i = 0; i < argc; i++)
        printf("%s\n", argv[i]);
}
```

| 符号 | 含义 |
|------|------|
| `argc` | 参数个数（含程序名） |
| `argv` | 指向 `char *` 数组的指针 |
| `argv[i]` | 第 i 个参数字符串（只读） |

### 与 DPDK/HFT

```c
/* EAL: ./app -l 0-3 -n 4 -- -p 0x1 */
/* 解析 -l、-n 等 = 遍历 argv */
```

- **getopt** / 自研解析：对 `argv` 下标遍历（**ch09** 文本）

### 等价写法

```c
int main(int argc, char *argv[]);  /* 同 char **argv */
```

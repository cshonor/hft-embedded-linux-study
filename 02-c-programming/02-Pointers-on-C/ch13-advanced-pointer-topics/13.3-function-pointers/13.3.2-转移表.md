# 13.3.2 转移表

## 本节讲什么

**转移表（jump table / dispatch table）**：函数指针数组按索引/类型跳转 —— 报文分发、状态机、PMD 操作表。

## 要点

```c
typedef void (*pkt_handler_t)(struct rte_mbuf *pkt);

static pkt_handler_t dispatch[] = {
    [0] = handle_heartbeat,
    [1] = handle_quote,
    [2] = handle_order,
    [3] = handle_cancel,
};

void on_packet(struct rte_mbuf *mbuf, uint8_t type)
{
    if (type < sizeof dispatch / sizeof dispatch[0])
        dispatch[type](mbuf);
}
```

### vs if-else 链

| 转移表 | if-else |
|--------|---------|
| O(1) 间接跳转 | 多级比较 |
| 表驱动扩展 | 改代码加分支 |
| HFT 数据面常用 | 少量分支可接受 |

### 内核/驱动

- **file_operations**、sys_call_table、PMD **eth_dev_ops**
- 指定初始化 `.read = dev_read`（C99）

### 二级函数指针（少见）

```c
pkt_handler_t *table;   /* 指向函数指针的指针 */
```

- 插件热切换回调表；日常用一级数组即可

## 实操

实现 type→handler 表；对比 if-else 结构。

## 与后续衔接

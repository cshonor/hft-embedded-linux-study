# 13.3.1 回调函数

## 本节讲什么

**回调三要素**：typedef 函数指针、**void \*priv** 私有数据、注册/触发 —— DPDK、内核、事件引擎通用范式。

## 要点

```c
typedef void (*event_cb_t)(uint32_t event_type, void *priv);

struct event_mgr {
    event_cb_t cb;
    void      *priv_data;
};

void event_register(struct event_mgr *mgr, event_cb_t cb, void *priv)
{
    mgr->cb = cb;
    mgr->priv_data = priv;
}

void event_trigger(struct event_mgr *mgr, uint32_t type)
{
    if (mgr->cb)
        mgr->cb(type, mgr->priv_data);
}
```

### void * + 释放回调

```c
typedef void (*free_cb_t)(void *obj);

void pool_destroy(void **list, int cnt, free_cb_t free_func)
{
    for (int i = 0; i < cnt; i++)
        free_func(list[i]);
}
```

| 设计 | 作用 |
|------|------|
| **void \*priv** | 多类型数据无强耦合 |
| **判空再调** | `if (cb) cb(...)` |
| **强转匹配** | priv 类型须与回调约定一致 |

### rte_mempool 类比

- 构造/释放钩子 = 函数指针 + void *

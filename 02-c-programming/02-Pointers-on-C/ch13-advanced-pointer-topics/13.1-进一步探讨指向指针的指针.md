# 13.1 进一步探讨指向指针的指针

## 本节讲什么

**二级及以上指针**（`T **`、`T ***`）：动态字符串数组、批量改指针数组、与 **argv** 同构。

## 要点

```c
char **msg_table = calloc(32, sizeof(char *));
msg_table[0] = strdup("QUOTE");
msg_table[1] = strdup("ORDER");
```

| 层级 | 存什么 |
|------|--------|
| `char *` | 字符串首地址 |
| `char **` | 指针数组元素地址 |
| `char ***` | 改外部 `char **` 头（极少用） |

### 函数内改外部指针数组

```c
void add_msg(char ***table, size_t *cnt, const char *s);
```

### 释放顺序

```c
for (size_t i = 0; i < n; i++)
    free(table[i]);
free(table);
```

- 逐层释放，防泄漏

### 工程约束

- 工业代码尽量 **≤ 三级**；层级越多空指针风险越大

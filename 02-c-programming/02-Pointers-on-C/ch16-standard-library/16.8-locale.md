# 16.8 locale

## 本节讲什么

**`<locale.h>`** 区域与多字节字符 —— 本地化文本；**二进制/DPDK/内核几乎不用**。

## 要点

```c
setlocale(LC_ALL, "C");   /* 默认 C locale，可预期行为 */
```

- **wchar_t**、**mbtowc** 等宽字符 API：UI/本地化
- 报文、寄存器、mbuf：**UTF-8 字节流当 binary 处理**，不依赖 locale

### HFT/DPDK

- 日志 ASCII 为主；避免 locale 改变 **printf** 小数点格式
- 性能测试前固定 **`LC_NUMERIC=C`**

## 与后续衔接

**模块10** 用户大纲：宽字符可跳过；专注 mem* 与 strtoll。

# 16.4 非本地跳转

## 本节讲什么

**`setjmp` / `longjmp`** —— 跨函数栈回溯；工业代码极少用，了解即可。

## 要点

```c
#include <setjmp.h>

jmp_buf env;
if (setjmp(env) == 0) {
    /* 正常路径 */
    if (fatal) longjmp(env, 1);
} else {
    /* longjmp 跳回此处 */
}
```

| 项 | 说明 |
|----|------|
| 用途 | 解析器错误恢复、旧式异常模拟 |
| 风险 | 跳过局部变量析构/free；与 C++ 混用危险 |
| HFT | **热路径禁用**；资源清理用 goto cleanup 或显式错误码 |

## 与后续衔接

**ch15** goto cleanup；现代代码 prefer 错误码返回。

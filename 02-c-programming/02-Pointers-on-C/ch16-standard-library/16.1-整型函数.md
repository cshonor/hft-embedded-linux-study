# 16.1 整型函数

## 本节讲什么

**数值转换**、**随机数**、**qsort/bsearch** —— 配置解析、订单排序检索；HFT 禁用不安全 atoi。

## 要点

### 字符串 → 整数

```c
/* 生产禁用：失败静默返回 0 */
int n = atoi("abc");

/* 推荐 */
char *end;
long long v = strtoll(s, &end, 10);
if (end == s || *end != '\0') { /* 非法 */ }
```

| 函数 | 说明 |
|------|------|
| **atoi/atol/atoll** | 无错误检测 |
| **strtol/strtoll** | 进制、尾指针、ERANGE |

### 随机数

```c
srand((unsigned)time(NULL));
int r = rand();   /* [0, RAND_MAX]，全局状态，多线程不安全 */
```

- 压测用 **`rand_r`** 或 DPDK **rte_rand**；热路径避免 rand 锁

### qsort / bsearch

```c
int cmp_order(const void *a, const void *b)
{
    uint64_t o1 = *(const uint64_t *)a;
    uint64_t o2 = *(const uint64_t *)b;
    return (o1 > o2) - (o1 < o2);
}

qsort(order_ids, cnt, sizeof order_ids[0], cmp_order);
```

- **void \*** + 比较回调（**ch13**）
- 海量订单 HFT 可手写专用排序，少 void* 开销

### 其他

- **abs/labs/llabs**、**div** 等整型辅助

# 16.7 执行环境

## 本节讲什么

**进程控制**、**环境变量**、**堆分配回顾** —— exit/atexit、禁用 system、对齐分配。

## 要点

### 终止与回调

```c
atexit(save_quote_snapshot);   /* 退出时落盘 */
exit(0);                       /* flush stdio + atexit */
abort();                       /* SIGABRT，不跑 atexit */
```

```c
system("cmd");   /* 创建 shell 子进程 — HFT 热路径禁用 */
```

### 环境

```c
char *path = getenv("LD_LIBRARY_PATH");
```

### 内存（ch11 拓展）

```c
void *p = malloc(n);
void *a;
posix_memalign(&a, 64, size);   /* DMA/缓存行对齐 */
free(a);
```

- malloc 全局锁 + 碎片 → DPDK **rte_mempool**
- 分配判空、realloc 临时指针、free 后置 NULL

### string/ctype 索引

| 头文件 | 要点 |
|--------|------|
| **string.h** | mem* 二进制；str* 文本（**ch09**） |
| **ctype.h** | `(unsigned char)c`（**ch09**） |

## 与后续衔接

**ch11** 堆；**ch15** exit 刷缓冲；**ch18** brk/mmap。

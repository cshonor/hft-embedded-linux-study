# 16.6 打印可变参数列表

## 本节讲什么

**`<stdarg.h>`** **`va_list`** —— printf/snprintf、自定义日志宏的底层。

## 要点

```c
#include <stdarg.h>
#include <stdio.h>

void log_msg(const char *fmt, ...)
{
    va_list ap;
    va_start(ap, fmt);
    vfprintf(stderr, fmt, ap);
    va_end(ap);
}
```

| 宏/函数 | 作用 |
|---------|------|
| **va_start** | 绑定最后一个固定参数 |
| **va_arg** | 取下一参数 |
| **va_end** | 结束 |
| **vfprintf/vsnprintf** | 转发可变参 |

- **ch14** `LOG(fmt,...)` 宏底层即此
- **ch15** fprintf/snprintf 族

### 封装

```c
void rte_log(int level, const char *fmt, ...);
/* DPDK 风格：级别 + vfprintf */
```

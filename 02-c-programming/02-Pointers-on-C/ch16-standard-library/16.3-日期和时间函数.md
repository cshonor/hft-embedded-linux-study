# 16.3 日期和时间函数

## 本节讲什么

**`<time.h>`** 秒级时间戳、**clock** CPU 计时；HFT 纳秒延迟用 **clock_gettime**。

## 要点

```c
time_t now = time(NULL);

clock_t t0 = clock();
parse_packet(pkt);
clock_t t1 = clock();
double ms = (double)(t1 - t0) / CLOCKS_PER_SEC * 1000;
```

```c
struct tm *tm_info = gmtime(&now);   /* UTC */
struct tm *local = localtime(&now);
```

### 高精度（POSIX，HFT 标配）

```c
struct timespec ts;
clock_gettime(CLOCK_MONOTONIC, &ts);
/* 纳秒单调时钟，不受 NTP 回拨 */
```

| API | 精度 | 用途 |
|-----|------|------|
| **time** | 秒 | 日志墙钟 |
| **clock** | CPU tick | 函数微基准 |
| **CLOCK_MONOTONIC** | ns | 延迟统计 |

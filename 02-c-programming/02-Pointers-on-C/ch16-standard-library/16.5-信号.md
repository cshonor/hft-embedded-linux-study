# 16.5 信号

## 本节讲什么

**`<signal.h>`** 异步信号 —— SIGSEGV/SIGABRT 与网关优雅退出。

## 要点

```c
#include <signal.h>

void on_sigint(int sig) { /* 设置退出标志，勿在 handler 里复杂 IO */ }
signal(SIGINT, on_sigint);
```

| 信号 | 常见原因 |
|------|----------|
| **SIGSEGV** | 野指针、堆损坏 |
| **SIGABRT** | abort、double free |
| **SIGINT/SIGTERM** | 运维停止网关 |

### 规范

- handler 内仅置 **volatile sig_atomic_t** 标志
- 复杂清理在主循环或 **atexit**（**16.7**）
- DPDK 进程常自定义 signal 处理 + 资源回收

## 与后续衔接

**ch11** double free → SIGABRT；**ch18** 信号与进程。

# 16.2 浮点型函数

## 本节讲什么

**`<math.h>`** 浮点运算 —— 回测/风控统计；**交易价格禁止 float 存价**。

## 要点

```c
#include <math.h>

double r = sqrt(x);
double a = fabs(pnl);
double u = floor(price);
double c = ceil(qty);
```

| 函数类 | 示例 |
|--------|------|
| 幂/对数 | pow, log, exp |
| 取整 | floor, ceil, round |
| 三角 | sin, cos（少用） |

### HFT 规范

- 价量用 **整型固定小数位**（分/厘），不用 double 存订单价
- 浮点仅统计、回测；多线程高频浮点注意缓存与可重复性

## 与后续衔接

**ch03** 浮点精度；交易所整数价字段。

# 11.3 calloc 和 realloc

## 本节讲什么

**`calloc`** 分配并清零；**`realloc`** 扩缩容 —— 后者有**丢指针泄漏**高危写法。

## 要点

### calloc

```c
struct rte_mbuf *pkts = calloc(32, sizeof *pkts);
/* 等价 malloc(32*sizeof) + memset(0) */
```

- 新数组、结构体池首选，避免脏数据污染报文

### realloc

```c
void *tmp = realloc(pkt_buf, new_len);
if (tmp != NULL)
    pkt_buf = tmp;
else {
    /* 失败：pkt_buf 仍有效，业务降级 */
}
```

| 入参 | 行为 |
|------|------|
| `ptr=NULL` | 等价 **malloc** |
| 更大 | 新块 + 拷贝旧数据 + 释放旧块 |
| 更小 | 截断（实现相关） |

### 禁止写法

```c
pkt_buf = realloc(pkt_buf, new_len);  /* 失败→NULL，原指针丢失→泄漏 */
```

## 实操

malloc vs calloc 脏数据对比；安全/不安全 realloc。

# 11.4 使用动态分配的内存

## 本节讲什么

堆内存**完整生命周期**工业规范：分配判空、记录长度、用完 free、置 NULL。

## 要点

```c
struct quote_hdr *pkt = malloc(sizeof *pkt);
if (!pkt)
    return -ENOMEM;

/* 使用：仅通过 pkt 原始指针访问 */
pkt->seq = 1001;

free(pkt);
pkt = NULL;
```

### 四条硬性规则

1. **分配后判 NULL**
2. **free 只用原始返回地址**
3. **每条成功路径都要 free**（含异常 return）
4. **free 后立即 `ptr = NULL`**

### 记录分配长度

```c
size_t cap = 1024;
uint8_t *buf = malloc(cap);
/* 读写前：offset < cap */
```

- 堆越界会破坏分配器元数据 → 延迟崩溃（**11.5**）

### 封装模式

```c
if (p) { free(p); p = NULL; }
```

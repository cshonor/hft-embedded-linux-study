# 11.6 内存分配实例

## 本节讲什么

**碎片、对齐、多线程 malloc、DPDK mempool** 与标准堆的对照；链表释放示例；调试工具。

## 要点

### 内存碎片

| 类型 | 说明 |
|------|------|
| **外部** | 离散小块，无法拼成大块 |
| **内部** | 对齐向上取整浪费 |

**对策**：预分配内存池、**统一块大小**（DPDK **rte_mempool**）、少频繁小块 malloc

### 对齐分配

```c
void *p;
posix_memalign(&p, 64, size);   /* 64B 缓存行对齐 */
free(p);
```

- DMA / 大页 / 缓存行 → DPDK 大页 mmap 同源思想

### 多线程 malloc

- libc malloc 带**全局锁** → 延迟抖动
- HFT：**线程本地池**；DPDK **每队列独立 mempool** 无锁

### 简易链表（防泄漏模板）

```c
Node *head = NULL;
/* ... push 若干 malloc 节点 ... */
for (Node *n = head; n; ) {
    Node *next = n->next;
    free(n);
    n = next;
}
head = NULL;
```

### 调试

- **valgrind** `--leak-check=full`：泄漏、越界
- **gdb** core：double free、SIGABRT 栈

### 内核对照

| 用户态 | 内核 |
|--------|------|
| malloc | **kmalloc** |
| 碎片 | **slab** |
| 大映射 | **vmalloc** / mmap |

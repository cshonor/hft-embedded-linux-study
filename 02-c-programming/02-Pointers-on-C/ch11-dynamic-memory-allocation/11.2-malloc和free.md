# 11.2 malloc 和 free

## 本节讲什么

**`<stdlib.h>`** 最基础堆 API：**`malloc`** 分配原始字节，**`free`** 归还。

## 要点

### malloc

```c
struct quote_hdr *pkt = malloc(sizeof *pkt);
if (!pkt) {
    /* 内存耗尽容错 */
}
```

| 项 | 说明 |
|----|------|
| 返回 | **`void *`** → 转为目标类型 |
| 清零 | **否**，残留脏数据 |
| 失败 | **`NULL`** |

### free

```c
free(pkt);
pkt = NULL;
```

| 规则 | 说明 |
|------|------|
| 指针 | 必须是 **malloc/calloc/realloc 返回的原始地址** |
| 次数 | **只能 free 一次** |
| `free(NULL)` | 安全无操作 |

- 禁止 free **偏移后**的子地址
- free 后 **立即置 NULL**（**11.4**）

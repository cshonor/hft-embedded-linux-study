# 11.5 常见错误

## 本节讲什么

四大堆故障：**泄漏、悬垂指针、双重释放、堆越界** —— HFT/网关线上崩溃主因。

## 要点

### 1. 内存泄漏 Memory Leak

- malloc 后未 free，或指针丢失
- 长跑进程 OOM 被 kill
- 高发：异常分支提前 return、循环内只分配不释放

### 2. 悬垂指针 Dangling Pointer

```c
free(p);
*p = 1;   /* UB：已释放 */
```

- 根治：**free 后 `p = NULL`**，解引用前判空

### 3. 双重释放 Double Free

```c
free(p);
free(p);  /* 堆损坏 → SIGABRT */
```

- 防御：`if (p) { free(p); p = NULL; }`

### 4. 堆越界 Heap Out-of-Bounds

- 读写超过分配 size
- 覆盖 malloc 元数据 → 随机崩溃、乱码
- 防御：保存 **cap**，校验 **offset < cap**

## 实操

双重 free 观察 SIGABRT；越界写观察延迟崩溃。

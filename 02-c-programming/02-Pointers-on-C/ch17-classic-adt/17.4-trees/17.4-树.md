# 17.4 树

## 本节讲什么

**二叉搜索树 BST** —— 有序 ADT；插入/删除/查找/遍历；对比 **有序数组+bsearch**、**哈希表**。

## 要点

| 结构 | 查找 | 适用 |
|------|------|------|
| **BST** | O(log n) 均摊 | 动态有序集 |
| **有序数组+bsearch** | O(log n) | 静态配置、少插入 |
| **哈希表** | O(1) 均摊 | 合约/会话映射（**17.5.1**） |

- 节点：`void *key` 或内嵌键 + **cmp 回调**

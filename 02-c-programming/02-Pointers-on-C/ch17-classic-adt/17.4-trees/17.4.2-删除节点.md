# 17.4.2 删除节点

## 本节讲什么

BST **删除**：叶子 / 单子 / 双子（中序后继替换）。

## 要点

- **0 子**：直接 free  
- **1 子**：用子节点顶替  
- **2 子**：右子树最小节点（或左子树最大）换键后删叶子  

```c
/* 伪代码骨架 */
TreeNode *tree_delete(TreeNode *root, const void *key, cmp_fn cmp);
```

- 删除后须 **free 节点**；载荷由 **free_cb** 或调用者管理  
- 错误 cmp → 删错节点

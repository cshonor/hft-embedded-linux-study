# 17.4.1 插入节点

## 本节讲什么

BST **按 cmp 递归插入**，保持左小右大（或自定义序）。

## 要点

```c
struct TreeNode {
    void *key;
    struct TreeNode *left, *right;
};

TreeNode *tree_insert(TreeNode *root, void *key,
                      int (*cmp)(const void *, const void *))
{
    if (!root) {
        TreeNode *n = calloc(1, sizeof *n);
        n->key = key;
        return n;
    }
    int c = cmp(key, root->key);
    if (c < 0)
        root->left  = tree_insert(root->left, key, cmp);
    else if (c > 0)
        root->right = tree_insert(root->right, key, cmp);
    return root;
}
```

- 重复键策略：忽略或放右子树（文档约定）  
- 退化链表 → O(n)；平衡树工业扩展（AVL/RB）

## 与后续衔接

有序动态数组 + **qsort** 为静态替代。

# 17.4.3 查找特定值的节点

## 本节讲什么

BST **查找**与 **bsearch** 有序表对比。

## 要点

### 树查找

```c
TreeNode *tree_find(TreeNode *root, const void *key, cmp_fn cmp)
{
    while (root) {
        int c = cmp(key, root->key);
        if (c == 0) return root;
        root = c < 0 ? root->left : root->right;
    }
    return NULL;
}
```

### 有序数组二分

```c
uint64_t key = target_id;
uint64_t *found = bsearch(&key, order_ids, cnt, sizeof key, cmp_order);
```

- 静态合约表、少更新 → **bsearch + realloc 扩容**  
- 动态订单集 → BST 或 **哈希**（**17.5.1**）

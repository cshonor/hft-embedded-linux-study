# 17.4.4 树的遍历

## 本节讲什么

BST **中序/前序/后序**遍历 + **foreach 回调**批量处理。

## 要点

```c
void tree_inorder(TreeNode *root, void (*handler)(void *))
{
    if (!root) return;
    tree_inorder(root->left, handler);
    handler(root->key);
    tree_inorder(root->right, handler);
}
```

| 遍历 | 顺序 | 用途 |
|------|------|------|
| **中序** | 有序键序 | 导出排序列表 |
| **前序** | 根左右 | 复制树 |
| **后序** | 左右根 | **destroy 先子后根** |

```c
void tree_destroy(TreeNode **root, void (*free_cb)(void *))
{
    /* 后序：destroy 左、右，free_cb(key); free(node) */
}
```

## 与后续衔接

**list_foreach**；内核 **list_for_each**。

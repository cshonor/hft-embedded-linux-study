# 17.3 队列

## 本节讲什么

**队列 ADT（FIFO）** —— HFT/DPDK **价值最高**；链表版 vs **环形数组**（rte_ring 原型）。

## 要点

| 实现 | 特点 |
|------|------|
| **链表** head/tail | O(1) 入出队，动态 malloc |
| **环形数组** | 预分配、无碎片、SPSC 无锁 |

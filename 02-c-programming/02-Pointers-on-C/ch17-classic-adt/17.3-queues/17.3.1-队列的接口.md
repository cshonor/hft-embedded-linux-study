# 17.3.1 队列的接口

## 本节讲什么

队列 **标准 API** —— 业务仅依赖接口，底层链表/Ring 可切换。

## 要点

```c
typedef struct Queue Queue;

Queue *queue_create(size_t cap);   /* cap>0 可预分配 Ring */
int    queue_enqueue(Queue *q, void *data);
void  *queue_dequeue(Queue *q);
size_t queue_len(const Queue *q);
void   queue_destroy(Queue **q, void (*free_cb)(void *));
```

- **enqueue** 尾入、**dequeue** 头出（FIFO）  
- 同一解析业务换 **17.3.2** / **17.3.3** 实现，上层不变

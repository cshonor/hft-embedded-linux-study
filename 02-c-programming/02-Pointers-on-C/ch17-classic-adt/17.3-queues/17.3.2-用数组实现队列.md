# 17.3.2 用数组实现队列

## 本节讲什么

**环形缓冲区 Ring Buffer** —— DPDK **`rte_ring`** 思想原型；SPSC 无锁、零堆分配。

## 要点

```c
struct Ring {
    void **buf;
    size_t cap;
    size_t r_idx, w_idx;
    size_t len;
};
```

```c
int ring_enqueue(Ring *r, void *d)
{
    if (r->len >= r->cap) return -1;
    r->buf[r->w_idx] = d;
    r->w_idx = (r->w_idx + 1) % r->cap;   /* 必须取模 */
    r->len++;
    return 0;
}

void *ring_dequeue(Ring *r)
{
    if (r->len == 0) return NULL;
    void *d = r->buf[r->r_idx];
    r->r_idx = (r->r_idx + 1) % r->cap;
    r->len--;
    return d;
}
```

### 陷阱

- **忘记 `% cap`** → 数组越界  
- 单生产者单消费者可用无锁；多线程需 **rte_ring** 原子/内存序  

### HFT 用途

行情分发、订单下发、端口收发队列

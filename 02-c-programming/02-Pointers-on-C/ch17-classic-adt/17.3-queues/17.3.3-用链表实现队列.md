# 17.3.3 用链表实现队列

## 本节讲什么

**链表队列**：**head 出队、tail 入队**，O(1) 双端操作。

## 要点

```c
struct Queue {
    Node *head;
    Node *tail;
    size_t len;
};

int queue_enqueue(Queue *q, void *d)
{
    Node *n = calloc(1, sizeof *n);
    if (!n) return -1;
    n->payload = d;
    if (!q->tail)
        q->head = q->tail = n;
    else {
        q->tail->next = n;
        q->tail = n;
    }
    q->len++;
    return 0;
}
```

- 变长、无容量上限；频繁 malloc → 长跑注意碎片  
- **禁止**业务直接改 `q->head`（破坏 ADT 封装）

# 17.5.1 拥有超过一种的类型

## 本节讲什么

**void\* 通用载荷** + **三大回调** + **通用链表/哈希** —— 一套容器管行情/订单/设备。

## 要点

### 三大回调

| 回调 | 签名 | 用途 |
|------|------|------|
| **cmp** | `int(const void*,const void*)` | 查找、删除、排序 |
| **handler** | `void(void*)` | 遍历处理 |
| **free_cb** | `void(void*)` | destroy 释放载荷 |

### 通用单向链表 API（回顾）

```c
List *list_create(void);
int list_push_front(List *l, void *data);
void *list_pop_front(List *l);
int list_remove(List *l, const void *key, cmp_fn cmp);
void list_foreach(List *l, handler_fn h);
void list_destroy(List **l, free_cb_fn free_cb);
```

### 链地址法哈希（概要）

```c
struct Hash {
    List **buckets;
    size_t nbuckets;
    hash_fn hash;
    cmp_fn  cmp;
};
/* hash_insert / hash_search / hash_delete / hash_destroy */
```

- 合约代码→信息、订单 ID、TCP 会话表  
- **cmp/hash 强转错误** → 查找错乱  

### 内核对比

- **list_head** 嵌入 struct，非 void* 包裹  
- **private_data** = void* 载荷思想  

### 条件编译切换底层

```c
#ifdef USE_RING_QUEUE
#include "queue_ring.h"
#else
#include "queue_list.h"
#endif
```

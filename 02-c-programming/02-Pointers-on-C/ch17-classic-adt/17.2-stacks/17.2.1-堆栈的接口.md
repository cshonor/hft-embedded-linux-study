# 17.2.1 堆栈的接口

## 本节讲什么

栈 **对外 API** —— 不透明 `Stack *`，隐藏数组/链表实现。

## 要点

```c
typedef struct Stack Stack;

Stack *stack_create(void);
int    stack_push(Stack *s, void *data);
void  *stack_pop(Stack *s);
void  *stack_peek(const Stack *s);
void   stack_destroy(Stack **s, void (*free_cb)(void *));
```

| 操作 | 语义 |
|------|------|
| **push** | 入栈 |
| **pop** | 出栈并返回载荷 |
| **peek** | 读栈顶不弹出 |
| **destroy** | 释放结构 + **free_cb(payload)** |

- 空栈 pop → 返回 NULL 或错误码

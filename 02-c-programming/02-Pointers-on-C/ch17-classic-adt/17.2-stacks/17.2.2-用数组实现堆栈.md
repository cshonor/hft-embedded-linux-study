# 17.2.2 用数组实现堆栈

## 本节讲什么

**固定/动态数组栈** —— 无 per-push malloc，数据面友好。

## 要点

```c
struct Stack {
    void **buf;
    size_t cap, top;   /* top 指向下一个空位 */
};
```

```c
int stack_push(Stack *s, void *d)
{
    if (s->top >= s->cap) return -1;   /* 或 realloc 扩容 */
    s->buf[s->top++] = d;
    return 0;
}

void *stack_pop(Stack *s)
{
    if (s->top == 0) return NULL;
    return s->buf[--s->top];
}
```

- **O(1)** push/pop；缓存友好  
- 容量固定 → 预分配，HFT 热路径首选

# 17.2 堆栈

## 本节讲什么

**栈 ADT（LIFO）** —— 报文递归解析、错误资源回滚栈；数组或链表实现。

## 要点

- **push/pop/peek** 仅栈顶  
- 链表版 = **头插/头删**（**17.2.3**）  
- 数组版 = 动态或固定 **`top` 索引**（**17.2.2**）

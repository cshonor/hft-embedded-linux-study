# 17.2.3 用链表实现堆栈

## 本节讲什么

**链表栈** = 单向链表 **头插/头删**。

## 要点

```c
struct Node { void *payload; struct Node *next; };

int stack_push(Stack *s, void *d)
{
    Node *n = calloc(1, sizeof *n);
    if (!n) return -1;
    n->payload = d;
    n->next = s->head;
    s->head = n;
    return 0;
}

void *stack_pop(Stack *s)
{
    Node *n = s->head;
    if (!n) return NULL;
    void *d = n->payload;
    s->head = n->next;
    free(n);
    return d;
}
```

- 变长无上限；每次 push **malloc** → 控制面可用，数据面慎用

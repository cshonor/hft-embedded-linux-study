# 17.1 内存分配

## 本节讲什么

ADT **容器专用内存策略**：不透明句柄、堆节点分配、**释放回调** —— 全书综合落地起点。

## 要点

### ADT 两大原则

1. **封装隐藏**：`.h` 仅 `typedef struct Queue Queue;`，`.c` 内完整定义  
2. **接口隔离**：create/destroy/enqueue/dequeue，底层可换链表↔环形数组  

### 不透明指针

```c
/* queue.h */
typedef struct Queue Queue;
Queue *queue_create(size_t cap);
void  queue_destroy(Queue **q, void (*free_cb)(void *));

/* queue.c */
struct Queue {
    struct Node *head, *tail;
    size_t len;
};
```

### 内存与回调

```c
void list_destroy(List **lst, void (*free_cb)(void *))
{
    /* 遍历节点：free_cb(payload); free(node); */
    *lst = NULL;
}
```

- 不传 **free_cb** → 载荷泄漏  
- 容器节点用 **calloc**；数组扩容 **realloc 临时指针**（**ch11**）

### 预分配 vs 动态

| 方式 | 适用 |
|------|------|
| 链表 + malloc | 变长、控制面 |
| 预分配 Ring | **rte_ring** 式数据面 |

# 8.1.11 字符数组的初始化

## 本节讲什么

**字符数组** vs **字符串指针数组** —— 报文缓冲与协议常量表。

## 要点

### 可读写二维字符数组

```c
char pkt_buf[16][64];   /* 16×64 连续可写 */
strncpy(pkt_buf[0], data, 63);
pkt_buf[0][63] = '\0';
```

- 二进制报文无 `\0` 时**勿用 strlen**

### 字符串指针数组（只读表）

```c
const char *pkt_type_name[] = {
    "HEARTBEAT", "QUOTE", "ORDER", "CANCEL"
};
```

- 各字符串在 **.rodata**；修改 → 段错误
- 适合错误码、opcode 名称表

## 与后续衔接

**ch09** 字符串与字节；**8.3** 指针数组。

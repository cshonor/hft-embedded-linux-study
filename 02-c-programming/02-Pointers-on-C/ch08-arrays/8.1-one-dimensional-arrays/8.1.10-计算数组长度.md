# 8.1.10 计算数组长度

## 本节讲什么

**编译期**数组元素个数：`sizeof` 技巧；传参后**不可用**。

## 要点

```c
int arr[10];
size_t n = sizeof(arr) / sizeof(arr[0]);   /* 10 */

void f(int arr[]) {
    /* sizeof(arr)/sizeof(arr[0]) 错误 → 得指针大小 */
}
```

- 宏 **`ARRAY_SIZE(a)`**（内核风格）：`sizeof(a)/sizeof((a)[0])`，**仅用于真实数组非指针**
- 传参必须显式 **`len`**

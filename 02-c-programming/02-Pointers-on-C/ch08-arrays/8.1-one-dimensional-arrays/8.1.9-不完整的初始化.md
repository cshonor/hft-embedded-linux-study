# 8.1.9 不完整的初始化

## 本节讲什么

initializer **少于**元素个数时，剩余元素为 **0**（仅对静态存储期保证；栈数组若部分初始化，C 规则仍补 0 对已初始化列表之后 —— 标准：未显式初始化元素为 0 若整体有 `{}` 或列表后补零）。

## 要点

```c
int a[10] = {1};        /* a[0]=1, a[1..9]=0 */
int b[10] = {0};        /* 全 0 */
```

- 栈数组 **`int c[10];`** 无 `{}` → 全脏
- 工程：**`= {0}`** 清零结构体/数组

# 8.1.6 作为函数参数的数组名

## 本节讲什么

数组作实参时**退化为指针**；形参收到的是**地址副本**，不是数组对象。

## 要点

```c
void f(int arr[], int n);   /* 等价 */
void f(int *arr, int n);
```

- 函数内 **`sizeof(arr)` == sizeof(int*)**，不是数组总字节
- **必须**另传 **`n`** 或约定 `\0` 结尾（字符串）
- 修改 `arr[i]` 会改调用者数组（同一内存）

## DPDK

```c
void handle(struct rte_mbuf *pkts[], uint16_t nb);
/* 实参 pkts 栈上数组，退化为 struct rte_mbuf ** */
```

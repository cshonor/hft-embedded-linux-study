# 8.1 一维数组

## 本节讲什么

**同类型连续内存块**；承接 **ch06** 的 `arr[i] ≡ *(arr+i)`，展开初始化、传参退化、长度计算与字符数组。

## 要点

- 数组名多数情况 → **`&arr[0]`**（指针）
- 不退化：**`sizeof(arr)`**、**`&arr`**
- 局部数组 → **栈**；全局/static → **.data/.bss**
- 传参必带 **长度**（退化后 `sizeof` 失效）

## 工程陷阱（本章汇总）

| 陷阱 | 规避 |
|------|------|
| 传参丢长度 | 显式 `len` |
| 栈上大数组 | `malloc`/静态池 |
| 未初始化栈数组 | `{0}` 或赋值 |
| 大 **VLA** | HFT 禁用，改堆 |

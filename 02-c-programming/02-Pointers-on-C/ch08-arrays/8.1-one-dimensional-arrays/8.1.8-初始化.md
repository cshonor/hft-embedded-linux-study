# 8.1.8 初始化

## 本节讲什么

数组**定义时**初始化；全局/static 未列元素为 **0**。

## 要点

```c
int a[5] = {1, 2, 3};     /* 其余 0 */
int b[] = {1, 2, 3};      /* 长度 3 */
int c[100] = {0};         /* 全部清零 — 工程常用 */
```

| 存储 | 未初始化元素 |
|------|--------------|
| 全局/static | 0 |
| 栈 auto | 脏值 |

### VLA（C99）

```c
void f(int n) {
    char buf[n];   /* VLA，栈上，n 运行时 */
}
```

- **HFT 热路径禁止大 VLA** → 栈溢出
- VLA 不可全局；`sizeof(buf)` 运行时

# 8.1.1 数组名

## 本节讲什么

**数组名**在表达式中的含义：通常为首元素地址；例外见下。

## 要点

```c
int a[10];
int *p = a;      /* a ≡ &a[0] */
```

| 表达式 | 类型/含义 |
|--------|---------|
| `a`（一般） | `int*`，指向 a[0] |
| `sizeof(a)` | `10 * sizeof(int)` |
| `&a` | `int (*)[10]`，整数组地址 |

- **`&a`** 与 **`a`** 值常相同，**类型不同**

# 8.3 指针数组

## 本节讲什么

**指针数组** `Type *arr[N]`：数组元素为指针。DPDK **`rte_rx_burst`** 的 **`pkts[BURST]`** 即此模型。

## 要点

```c
struct rte_mbuf *pkts[32];
pkts[0] = rte_pktmbuf_alloc(pool);

/* 多端口 */
struct rte_mbuf *port_queue[8][16];  /* 8 口 × 16 队列槽 */
```

| 特性 | 说明 |
|------|------|
| 布局 | 连续存放 **指针值**（8 字节×N） |
| 指向 | 各元素**可分散**（堆 mbuf） |
| 传参 | 退化为 **`T **`** |

### 与数组指针

```c
int *a[5];     /* 指针数组 */
int (*b)[5];   /* 数组指针 */
```

- **DPDK burst**：指针数组
- **连续 quote 表 `[row][col]`**：二维数组 / 数组指针

### 字符串指针数组

```c
const char *names[] = {"HEARTBEAT", "QUOTE", "ORDER"};
```

## 实操

模拟 `mbuf *pkts[32]` 赋值遍历；打印 `sizeof(pkts)` vs `sizeof(pkts[0])`。

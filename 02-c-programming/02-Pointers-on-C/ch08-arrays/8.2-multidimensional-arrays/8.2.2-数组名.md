# 8.2.2 数组名

## 本节讲什么

二维数组名 **`buf`** 的类型与退化规则。

## 要点

```c
int buf[4][3];
```

| 表达式 | 含义 |
|--------|------|
| `buf` | 指向 **第一行** `int[3]`，类型 `int (*)[3]` 退化首元素 |
| `buf[i]` | 第 i 行，`int[3]` → `int*` |
| `&buf` | 指向整个 `int[4][3]` |
| `sizeof(buf)` | `4*3*sizeof(int)` |

- **`buf` 等价于 `&buf[0]`**（首行地址）

## 与后续衔接

**8.2.4** `int (*p)[3] = buf;`

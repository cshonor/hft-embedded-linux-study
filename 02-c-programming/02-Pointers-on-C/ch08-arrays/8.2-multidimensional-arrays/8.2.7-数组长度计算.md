# 8.2.7 数组长度计算

## 本节讲什么

多维数组**各维长度**：定义域用 `sizeof`；传参后仅保留已知维。

## 要点

```c
int buf[4][3];
size_t rows = sizeof(buf) / sizeof(buf[0]);      /* 4 */
size_t cols = sizeof(buf[0]) / sizeof(buf[0][0]); /* 3 */
```

- 函数参数 `int a[][3]`：**不能** `sizeof` 得行数，须传 **`rows`**
- 宏计算仅用于**非退化**数组对象

# 8.2.5 作为函数参数的多维数组

## 本节讲什么

二维数组传参：**第一维可省略，其余必须指定** —— 编译器需知**行字节宽**。

## 要点

```c
/* 合法 */
void process(int tbl[][3], int rows);
void process(int (*tbl)[3], int rows);

/* 非法 */
void bad(int tbl[][], int rows);
```

- 形参类型实质：**指向 `int[3]` 的指针**
- **`tbl + i`** 跳 i 行，每行 `3 * sizeof(int)` 字节
- 必须传 **`rows`**（第一维长度）

### 对比一维

```c
void f(int *p, int n);           /* 一维退化 */
void g(int (*p)[COLS], int rows); /* 二维 */
```

## DPDK 类比

连续二维行情表传 **`quote (*table)[COLS]`** + row count。

## 与后续衔接

**K&R 5.9**；误用 `[][]` 编译失败是常见面试点。

# 8.2.4 指向数组的指针

## 本节讲什么

**数组指针** `Type (*p)[N]`：指向含 N 个元素的**一整行**（或固定长数组）。

## 要点

```c
int buf[4][3];
int (*p_row)[3] = buf;   /* 指向第 0 行 */

p_row++;                 /* 下一行，+3*sizeof(int) */
(*p_row)[1] = 42;        /* 当前行第 1 列 */
```

### vs 指针数组

```c
int *a[5];    /* 5 个 int* — 指针数组 */
int (*b)[5];  /* 指向 int[5] — 数组指针 */
```

- 遍历**连续二维块**用 **数组指针**
- **分散 mbuf** 用 **指针数组**（**8.3**）

## 实操

用 `int (*row)[3]` 逐行打印二维表。

# 12.3 双链表

## 本节讲什么

**双向链表**：`next` + **`prev`** —— Linux 内核 **`list_head`**、可 O(1) 删除已知节点。

## 要点

```c
typedef struct dnode {
    uint64_t seq;
    struct dnode *next;
    struct dnode *prev;
} DNode;
```

| 对比单向 | 双向 |
|----------|------|
| 删中间需 prev 遍历 | 已知节点 O(1) 摘除 |
| 内存少一个指针 | 多 `prev`，内核标配 |

### 内核 list_head 思想

- 哨兵头节点：`head.next` / `head.prev` 成环
- 设备、进程、中断源挂在双向链上

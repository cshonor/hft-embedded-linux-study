# 12.3.1 插入节点

## 本节讲什么

双向链表**头插**与**中间插入** —— 维护 `next`/`prev` 四指针改写。

## 要点

### 头插

```c
void dlist_push_front(DNode **head, DNode *n)
{
    n->prev = NULL;
    n->next = *head;
    if (*head)
        (*head)->prev = n;
    *head = n;
}
```

### 在 prev 之后插入 new（已知位置）

```c
void dlist_insert_after(DNode *prev, DNode *n)
{
    n->next = prev->next;
    n->prev = prev;
    if (prev->next)
        prev->next->prev = n;
    prev->next = n;
}
```

- 插入前确认 **prev 仍在链上**（非已 free 节点）

## 与后续衔接

内核 `list_add` / `list_add_tail` 宏封装同类逻辑。

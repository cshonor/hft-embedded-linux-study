# 12.2.2 其他操作

## 本节讲什么

单向链表**遍历、全链销毁、按 seq 删除** —— 撤单/过期清理/防泄漏核心。

## 要点

### 遍历

```c
void list_iterate(Quote *head)
{
    for (Quote *cur = head; cur; cur = cur->next)
        process_quote(cur);
}
```

### 全链销毁（防泄漏）

```c
void list_free_all(Quote **head)
{
    Quote *cur = *head;
    while (cur) {
        Quote *tmp = cur;
        cur = cur->next;   /* 先存 next */
        free(tmp);         /* 再 free */
    }
    *head = NULL;
}
```

**禁止**：`free(cur); cur = cur->next;` → 读已释放内存，UB 崩溃

### 按 seq 删除

```c
int list_remove_by_seq(Quote **head, uint64_t target_seq)
{
    Quote *cur = *head, *prev = NULL;
    while (cur) {
        if (cur->seq == target_seq) {
            if (prev)
                prev->next = cur->next;
            else
                *head = cur->next;
            free(cur);
            return 1;
        }
        prev = cur;
        cur = cur->next;
    }
    return 0;
}
```

### 头删

```c
void list_pop_front(Quote **head)
{
    if (!*head) return;
    Quote *del = *head;
    *head = del->next;
    free(del);
}
```

## 线上陷阱

1. 只 free 外层，漏嵌套堆指针  
2. free 后再读 `next`  
3. 删头用 `Quote *` 无法改外部 head  
4. malloc 未清零 → 野 `next`  

## 实操

valgrind 验证 `list_free_all`；错误 free 顺序复现崩溃。

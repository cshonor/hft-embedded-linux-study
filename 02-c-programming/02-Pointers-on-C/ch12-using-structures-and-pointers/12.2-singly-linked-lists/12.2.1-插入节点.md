# 12.2.1 插入节点

## 本节讲什么

**头插法** `list_push` —— 低延迟链表追加节点的标准写法。

## 要点

```c
void list_push(Quote **head, Quote *new_node)
{
    new_node->next = *head;
    *head = new_node;
}
```

### 用法

```c
Quote *head = NULL;
Quote *n = new_quote();
if (n) {
    n->seq = 1001;
    list_push(&head, n);
}
```

| 要点 | 说明 |
|------|------|
| 入参 **`Quote **`** | 可改外部 head |
| `new_node->next` | calloc 后为 NULL，安全 |

### 对比尾插

- 尾插需遍历 → O(n)；头插 O(1) → 订单池/空闲链常用头插

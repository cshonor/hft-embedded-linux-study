# 12.1 链表

## 本节讲什么

**堆上 struct + 自引用指针** → 链表容器 —— DPDK mbuf 链、内核设备列表、HFT 订单池的公共模型。

## 要点

### 堆上创建节点

```c
typedef struct quote {
    uint64_t seq;
    uint32_t bid, ask;
    struct quote *next;
} Quote;

Quote *new_quote(void)
{
    Quote *node = calloc(1, sizeof *node);
    return node;   /* NULL = 内存耗尽 */
}
```

| 规范 | 说明 |
|------|------|
| **calloc** | 优先于 malloc，指针/数值清零 |
| 判空 | 分配后立即检查 |
| 嵌套堆成员 | **先 free 子对象，再 free 本体** |

### 二级指针 `Quote **`

修改**链表头**、批量指针数组须用 **`**`**：

```c
void list_pop_front(Quote **head);
void batch_handle(Quote **pkt_arr, int cnt);
```

### 嵌套结构体指针

```c
typedef struct msg {
    uint32_t magic;
    struct quote *quote_hdr;   /* 堆子对象 */
    uint8_t *payload;
} Msg;

void free_msg(Msg *m)
{
    if (!m) return;
    free(m->quote_hdr);
    free(m->payload);
    free(m);
}
```

- **内嵌** `struct quote hdr`：随外层一并释放，无需单独 free

### 结构体数组 vs 指针数组

| 类型 | 布局 | 适用 |
|------|------|------|
| `Quote arr[32]` | 连续实体 | 栈上固定小批 |
| `Quote *arr[32]` | 仅存地址 | **rte_rx_burst** 式批量收包 |

- 指针数组**无大 struct 拷贝**，HFT 优先

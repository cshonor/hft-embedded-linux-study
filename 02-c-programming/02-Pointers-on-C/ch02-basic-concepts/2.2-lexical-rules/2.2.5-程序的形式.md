# 2.2.5 程序的形式

## 本节讲什么

**声明 vs 定义**、**语句**、**作用域**、**存储类** —— 本章核心，多文件 DPDK/内核链接与内存生命周期的理论基础。

## 要点

### 声明 vs 定义（链接核心）

| | 声明 | 定义 |
|---|------|------|
| 作用 | 告知类型/签名 | 分配存储或生成代码 |
| 次数 | 可多次 | 整个程序**一次**（外部链接） |
| 例子 | `extern int g_val;` | `int g_val = 10;` |
| | `void func(int x);` | `void func(int x) { }` |
| | `struct rte_mbuf;` | `struct rte_mbuf { ... };` |

- 头文件放 **extern 声明**；某一个 `.c` **定义**
- 重复定义 → **`multiple definition`**

### 语句分类

1. **表达式语句**：`a = 10;`、`printf(...);` — 以 **`;`** 结束
2. **复合语句**：`{ ... }` — if/for/函数体
3. **空语句**：`;`
4. **控制语句**：if/for/while/switch（**ch04**）

### 分号陷阱

```c
if (a > b);   /* 空语句：条件不控制下面块 */
{
    /* 永远执行 */
}
```

《C陷阱与缺陷》典型语法坑。

### 作用域（Scope）

| 类型 | 范围 |
|------|------|
| **文件作用域** | 函数外；全局可见或 `static` 本文件 |
| **块作用域** | `{}` 内；栈生命周期 |
| **函数原型作用域** | 参数名仅声明处 |
| **函数作用域** | **label**（`goto` 目标） |

**嵌套**：内层可访问外层；**内层同名遮蔽外层** —— 内核规范常禁止与全局同名。

```c
int a = 10;
void test(void) {
    int a = 20;      /* 遮蔽 */
    printf("%d", a); /* 20 */
}
```

---

### 存储类（Storage Class）

| 说明符 | 含义 |
|--------|------|
| **auto** | 默认局部；栈；块结束销毁 |
| **static**（局部） | **数据段**；程序全程存活；只初始化一次 |
| **static**（文件） | **内部链接**；符号仅本 `.c` |
| **extern** | **声明**外部定义；不分配内存 |
| **register** | 建议寄存器；**不能 `&` 取址** |

#### static 局部示例（DPDK 单例池）

```c
void pool_get(void) {
    static void *pool = NULL;
    if (!pool)
        pool = alloc_hugepage();
}
```

#### extern 跨文件

```c
/* a.c */ int g_cfg = 1;
/* b.c */ extern int g_cfg;
```

## 专属价值（内核/DPDK/HFT）

1. **链接报错**：90% 多重定义/未定义 → 本章  
2. **内存生命周期**：static 池、一次性硬件 init  
3. **符号隔离**：`static` 私有 helper  
4. **隐蔽 bug**：遮蔽、if 分号、词法最长匹配  

## 实操

1. 多文件 `static` vs 全局，观察链接错误  
2. static 局部循环调用，验证只初始化一次  
3. 制造遮蔽，打印内外层  
4. if 加分号错误  
5. `readelf -S` 看 .text/.data/.bss

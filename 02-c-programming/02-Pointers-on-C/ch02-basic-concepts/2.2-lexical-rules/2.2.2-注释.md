# 2.2.2 注释

## 本节讲什么

两种注释格式；**预处理阶段删除**，不参与编译。内核与用户态工程规范不同。

## 要点

```c
/* 块注释：可跨行，不能嵌套 */
// 行注释：C99+，到行尾
```

| 规范 | 说明 |
|------|------|
| **Linux 内核** | 仅 **`/* */`** |
| **DPDK 应用层** | 常用 **`//`** |
| 预处理 | 注释内关键字、符号**完全无效** |

### 不能嵌套

```c
/* 外层 /* 内层 */  外层仍被误关闭 */   /* 错误示例 */
```

## 与 ch01 关系

`gcc -E` 输出中注释已消失。

## 与后续衔接

**ch14-preprocessor**；好注释解释意图，见 **2.3 程序风格**。

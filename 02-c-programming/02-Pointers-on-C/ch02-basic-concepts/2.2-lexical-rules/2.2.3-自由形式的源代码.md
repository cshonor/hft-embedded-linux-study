# 2.2.3 自由形式的源代码

## 本节讲什么

**空白符**（空格、制表、换行）仅用于**分隔 Token**；C 是**自由格式**源码，排版不影响语义。

## 要点

- 多个连续空白 ≡ 一个空白
- **`int   a   ;`** 与 **`int a;`** 完全相同
- **不可省略空白**：`intx` 是单个标识符，不是 `int x`
- 字符串与预处理指令内换行有特殊规则

### 工程规范

- 内核/DPDK：**4 空格**缩进，**禁止 Tab/空格混用**
- 一行过长 break 时，运算符放行首或行尾（团队统一）

### 与 Token 切分

空白帮助区分 `int` 与标识符；**最长匹配**仍作用于 `++`、`>>=` 等复合操作符。

## 与后续衔接

**2.3 程序风格** 统一排版；preprocessor 行末 `\` 续行。

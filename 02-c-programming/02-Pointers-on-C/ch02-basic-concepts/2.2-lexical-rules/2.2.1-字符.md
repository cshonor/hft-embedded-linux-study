# 2.2.1 字符

## 本节讲什么

**字符集**与**字面量**：整型、字符、字符串在词法层的表示。硬件地址、寄存器、报文解析都从这里开始。

## 要点

### 字符与编码

- C 字符 = 单字节 **`char`**（实现定义 signed/unsigned）
- **`'A'`**：字符常量，类型 `int`，值为 ASCII/编码值
- 驱动 MMIO、单字节寄存器读写基础

### 整型字面量

| 形式 | 含义 |
|------|------|
| `123` | 十进制 |
| `0123` | **八进制**（前导 0，易错） |
| `0x123` | **十六进制**（寄存器、硬件地址高频） |

- 后缀 `U` `L` `LL` 指定宽度（**ch03**）

### 字符串字面量

```c
"abc"   /* 存储为 'a' 'b' 'c' '\0' */
```

- 末尾自动 **`'\0'`**；越界读写最常见根源之一
- 类型为 **`char[]`** 静态存储，通常只读

## 考点

- **`'A'`** vs **`"A"`**（char vs 指针）
- 八进制字面量误用：`09` 非法

## 与后续衔接

**ch09-strings**；**ch03** 整型宽度与转换。

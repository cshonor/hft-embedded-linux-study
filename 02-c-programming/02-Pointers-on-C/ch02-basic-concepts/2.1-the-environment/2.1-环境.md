# 2.1 环境

## 本节讲什么

C 程序所处的**翻译环境**（编译链接）与**执行环境**（OS 加载运行）。承接 ch01 四阶段，本章从「语法规则地基」说明：**词法 → 语法 → 目标文件 → 进程内存模型**。

## 要点

- **翻译环境**：编译器 + 链接器把源码变为可执行文件（2.1.1）
- **执行环境**：内核加载 ELF，分配虚拟地址空间，从 `main` 开始执行（2.1.2）
- 内核/DPDK/HFT 大量**链接报错、符号冲突、内存布局 bug** 根源在本章环境概念

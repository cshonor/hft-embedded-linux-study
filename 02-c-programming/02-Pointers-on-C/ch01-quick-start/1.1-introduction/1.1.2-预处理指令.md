# 1.1.2 预处理指令

## 本节讲什么

以 **`#` 开头**的指令在**编译前**处理。本章最常见 **`#include`**；完整宏与条件编译在 ch14。

## 要点

```c
#include <stdio.h>
#include "myhdr.h"
```

| 形式 | 搜索路径 |
|------|----------|
| `<>` | 系统/标准库目录 |
| `""` | 通常先当前目录，再系统路径 |

### 底层含义

- **只做文本插入**：把头文件内容拼进 `.c`
- 头文件**多为声明**，无实现；实现于 **libc** 或自家 `.c`，**链接**阶段解析符号
- 类比：内核 `#include <linux/module.h>`、DPDK `#include <rte_*.h>`

### 编译四阶段之「预处理」（详见 1.3）

```bash
gcc -E main.c -o main.i
```

- 展开 `#include`、替换 `#define`、删注释、处理 `#ifdef`
- DPDK/内核：海量宏与头依赖在此展开，`.i` 文件可巨大

## 示例

预处理后，`printf` 的**声明**可见，定义仍在 libc。

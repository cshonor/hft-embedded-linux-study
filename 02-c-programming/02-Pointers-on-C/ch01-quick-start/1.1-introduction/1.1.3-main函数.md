# 1.1.3 main 函数

## 本节讲什么

**`main`** 是 C 程序**唯一入口**：OS 加载 ELF 后跳转到此函数；结束时 **`return`** 值交给父进程/shell。

## 要点

```c
int main(void)
{
    /* ... */
    return 0;
}
```

| 要素 | 说明 |
|------|------|
| `int` | 返回类型；退出状态（0 成功，非 0 错误） |
| `(void)` | **显式无参数**；标准 C 推荐，避免旧式 `main()` 隐式参数 |
| `return 0` | 正常退出；省略 return 时 C99+ `main` 等价 return 0 |

### 运行链路（简）

1. shell 执行 `./main` → **`execve`** 加载 ELF
2. 内核建新进程，分配虚拟内存、栈、堆
3. 启动代码初始化后调用 **`main`**
4. **`echo $?`** 查看退出码 —— 交易程序、工具脚本常用

### printf 与低延迟

- **调试/日志**：`printf` 够用
- **极致低延迟网关**：直接 **`write(2, ...)`** 绕过 stdio 缓冲（ch15 深入）

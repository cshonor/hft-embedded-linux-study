# 1.1.4 read_column_numbers 函数

## 本节讲什么

书中示例：从**标准输入**读入一组整数（列宽）。本章**第一次**在工程代码里使用 **`&` 取地址** —— 全书指针章的前置铺垫。

## 要点

### 典型 I/O 模式

```c
#include <stdio.h>

int main(void)
{
    int val;
    if (scanf("%d", &val) == 1)
        printf("input = %d\n", val);
    else
        printf("invalid input\n");
    return 0;
}
```

| 知识点 | 说明 |
|--------|------|
| **`&val`** | 取 `val` 的内存地址；`scanf` 需**指针**才能写入变量 |
| **返回值** | `scanf` 返回成功赋值的**项数**；必须校验，避免非法输入 |
| **无 `&`** | 传错地址 → **段错误**；可用 gdb 定位（1.3） |

### 书中 read_column_numbers 角色

- 循环读入多个列宽，存入数组
- 为 **rearrange** 提供输入数据
- 练习：**循环** + **错误检查** + **&**

## 底层提示

- **stdio 缓冲**：终端 I/O 可能有行缓冲，带来延迟抖动（ch15 展开；HFT 热路径慎用）
- 局部 **`int arr[10000]`** 在栈上 —— 栈有限，大缓冲改全局/堆（ch03）

## 实操

1. 故意写 `scanf("%d", val)` 无 `&`，gdb **`run`** 看崩溃栈
2. 输入字母，观察 `scanf` 返回值 ≠ 1 时的分支

# 1.1.5 rearrange 函数

## 本节讲什么

对读入的列宽数据**重排/排序**，综合运用 **if-else**、**for/while**。语法熟悉即可，优先级陷阱见《C陷阱与缺陷》与 **ch05-operators-and-expressions**。

## 要点

### 控制流（本章仅入门）

```c
/* if-else */
if (a[i] > a[j]) {
    int t = a[i];
    a[i] = a[j];
    a[j] = t;
}

/* for：已知次数 */
for (i = 0; i < n; i++)
    /* ... */

/* while：条件驱动 */
while (i < n && scanf("%d", &a[i]) == 1)
    i++;
```

### rearrange 在示例中的角色

- 接收数组 + 长度（或指针形式，ch06 再 formalize）
- 按题目要求调整列顺序（如升序），供后续格式化输出
- 体会：**main 协调**，子函数**单一职责**

### 容错延伸（实操任务）

- 循环读多组输入，**每次检查 scanf 返回值**
- 简易求和：读直到 EOF 或非法输入

# 1.2 补充说明

## 本节讲什么

本章未展开主题的**速览**：变量与类型、存储位置、分支循环、工程规范。细节在 **ch03-data**、**ch04/ch05**。

## 要点

### 基础类型（极简）

| 类型 | 用途 |
|------|------|
| `char` `short` `int` `long` | 整型（宽度平台相关，定宽用 `stdint.h`） |
| `float` `double` | 浮点 |
| `void` | 无类型（函数无返回值/无参声明） |

完整宽度、符号、转换 → **ch03-data**。

### 存储位置

| 对象 | 典型位置 |
|------|----------|
| 局部自动变量 | **栈** |
| 全局/静态 | **数据段 / BSS** |
| 动态分配 | **堆**（ch11） |

**踩坑**：`main` 内 **`大数组`** 占栈 → 栈溢出；DPDK/嵌入式常见，改静态缓冲或 `malloc`。

### 分支与循环

- **if / else if / else**、**for**、**while**、**do-while**
- 本章示例已覆盖；**运算符优先级**不在此深入

2. **Makefile**：各章通用模板，可复用到 DPDK Demo（见 1.3）
3. **注释**：团队约定 `//` vs `/* */`；公共 API 必文档化

# 6.11 指针表达式

## 本节讲什么

**指针与数组等价**、**指针数组 vs 数组指针**、**字符串与字符指针**。消除「数组≠指针」误区。

## 要点

### arr[i] ≡ *(arr+i)

```c
char buf[64];
buf[3] == *(buf + 3);
/* 3[buf] 合法但禁止写 */
```

### 数组名

| 表达式 | 含义 |
|--------|------|
| `buf`（多数情况） | `&buf[0]`，`char*` |
| `sizeof(buf)` | 整个数组字节 |
| `&buf` | `char (*)[64]` |

### 指针数组 vs 数组指针

```c
struct rte_mbuf *pkts[32];     /* 指针数组：32 个 mbuf* */
struct rte_mbuf (*row)[32];    /* 数组指针：指向 32 元数组 */
```

- DPDK **`rte_rx_burst`** 输出 **`pkts[]`** 即指针数组

### 字符串

```c
char *s = "rodata";   /* 只读，修改 → 段错误 */
char t[] = "writable"; /* 可改副本 */
const char *proto = "FIX.4.2";
```

- 可改缓冲：**`char buf[]`**
- 只读常量：**`const char *`**

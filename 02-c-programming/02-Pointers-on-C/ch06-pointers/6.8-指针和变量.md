# 6.8 指针和变量

## 本节讲什么

指针变量与普通变量一样有**存储**；可**重新指向**不同对象；指针自身也可被取址。

## 要点

```c
int a = 1, b = 2;
int *p = &a;
p = &b;          /* 改指向，不影响 a */
int **pp = &p;   /* pp 指向指针 p */
```

- `p` 的内容是地址；改 `p` 不改变原先指向的对象值（除非已通过 `*p` 写过）
- **`int *const p`**：`p` 不可改指向，内容可改（**6.9**）

# 6.9 指针常量

## 本节讲什么

**const 与指针**四组合 + **`void*`** 通用指针（**ch03** 延伸，本章落地）。

## 要点

### const 四层

| 声明 | 指向内容 | 指针本身 |
|------|----------|----------|
| `const int *p` | 只读 | 可改 |
| `int *const p` | 可改 | 不可改 |
| `const int *const p` | 只读 | 不可改 |
| `const char *restrict p` | 只读 + restrict 优化 | 可改 |

```c
const int *p = &a;
p = &b;      /* OK */
*p = 1;      /* 错误 */

int *const q = &a;
q = &b;      /* 错误 */
*q = 1;      /* OK */
```

- 只读报文：`const uint8_t *pkt`
- DPDK 拷贝 API 常用 **`restrict`** 消除别名

### void*

```c
void *raw = rte_malloc(NULL, 1024, 64);
struct quote_pkt *q = (struct quote_pkt *)raw;
```

- 任意对象地址可赋给 **`void*`**（无需 cast）
- **不能** `*raw`、**不能** `raw++`（不知宽度）
- **malloc** 返回 `void*`；使用前 **cast** 到具体类型

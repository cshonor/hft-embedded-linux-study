# 6.5 未初始化和非法的指针

## 本节讲什么

**野指针**与**悬垂指针** —— 生产环境段错误、堆破坏 **Top 原因**。

## 要点

### 野指针

```c
int *p;        /* 未初始化 → 垃圾地址 */
*p = 1;        /* SIGSEGV */
```

**规范**：局部指针 **`= NULL`** 或立即赋有效地址。

### 悬垂指针

```c
int *p = malloc(sizeof(int));
free(p);
/* p 仍存旧地址 —— 悬垂 */
*p = 10;       /* use-after-free，堆破坏 */
```

**规范**：**`free(p); p = NULL;`**

### 非法指针

- 越界：`arr+100` 后解引用（**UB**）
- 栈局部返回：`return &local;`（**ch07**）
- 修改只读字面量（**6.11**）

## 与后续衔接

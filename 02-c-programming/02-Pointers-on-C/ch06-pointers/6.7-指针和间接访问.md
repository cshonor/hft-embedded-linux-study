# 6.7 指针和间接访问

## 本节讲什么

通过指针**间接读写的完整模式**；函数内改调用者变量（传址）的基础。

## 要点

```c
void inc(int *p) {
    *p += 1;
}

int x = 10;
inc(&x);   /* x 变为 11 */
```

- C **值传递**；要改外部变量传**指针**
- 读：`v = *p`；写：`*p = v`
- 多级间接：`**pp`（**6.10**）

## DPDK/HFT

- 输出参数：`int parse(const uint8_t *in, size_t len, struct quote *out);`
- 批量：`rte_rx_burst(..., struct rte_mbuf *pkts[], ...)`

# 6.3 指针变量的内容

## 本节讲什么

指针变量**存储的内容**是一个**地址值**（通常以无符号整数形式实现，类型为 `uintptr_t` 可互转）。

## 要点

```c
int a = 1, b = 2;
int *p = &a;

printf("%p\n", (void *)p);   /* p 的内容：a 的地址 */
p = &b;                      /* 指针变量自身可被赋值，改存 b 的地址 */
```

- 打印用 **`%p`** + **`(void *)`**
- 指针也有地址：`&p` 是「指向指针的指针」（**6.10**）

## 实操

打印变量地址、指针变量地址、解引用值三者关系。

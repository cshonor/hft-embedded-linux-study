# 6.2 值和类型

## 本节讲什么

**类型**决定：占多少字节、如何解释位模式、指针 **`+1`** 步长。

## 要点

- 同一地址，用 **`int*`** 解引用读 4 字节，用 **`char*`** 读 1 字节 —— **类型是语义**
- **`sizeof(T)`** 得对象字节数（编译期）
- 指针类型：`T*` 表示「指向 T 的指针」

### 声明陷阱

```c
int *a, b;    /* a 是指针，b 是 int */
int *a, *b;   /* 两个指针 */
```

## 与后续衔接

**ch03** 定宽类型；强制 cast 改变解释方式（协议解析）。

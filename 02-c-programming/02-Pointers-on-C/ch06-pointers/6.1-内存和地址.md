# 6.1 内存和地址

## 本节讲什么

**内存 = 字节数组 + 地址**；每个变量占连续若干字节，**地址**标识起始位置。指针是「存放地址的变量」—— DPDK mbuf、内核 struct、HFT 缓冲区的共同语言。

## 要点

- **虚拟地址**：进程看到的地址（**ch18** 物理页映射）
- 64 位 Linux：用户态地址通常 48 位有效；**指针变量占 8 字节**（与指向类型无关）
- **普通变量**存**数据**；**指针变量**存**另一对象的地址**
- `&` 取址；`*` 解引用（**6.4**）

```c
int val = 10;
int *p = &val;   /* p 存 val 的首地址 */
```

## 底层映射

| 场景 | 地址含义 |
|------|----------|
| DPDK mbuf | 报文 data 区起始 |
| 内核 | `task_struct`、链表节点 |
| MMIO | 寄存器映射虚拟地址 |

## 与后续衔接

**ch11** 堆地址；**ch10** 结构体成员偏移。

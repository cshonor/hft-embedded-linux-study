# 6.10 指针的指针

## 本节讲什么

**二级指针** `T **`：指向「指向 T 的指针」。改调用者指针、指针数组、argv 的基础。

## 要点

```c
int x = 10;
int *p = &x;
int **pp = &p;

**pp = 20;   /* 改 x */
*pp = &y;    /* 改 p 的指向 */
```

### 典型场景

| 场景 | 用法 |
|------|------|
| 改指针参数 | `void alloc(char **out)` |
| 指针数组 | `char *argv[]` → `char **` |
| 二维动态数组 | `int **` 行指针 |

## 与后续衔接

**ch13** 多级指针、**main(argc, argv)**；**ch08** 指针数组。

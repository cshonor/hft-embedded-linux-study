# 6.13.2 指针的关系运算

## 本节讲什么

同类型指针 **`== != < > <= >=`**；**相减**得元素个数。

## 要点

### 比较

- **同一数组**（或尾后）内比较有意义
- 不同无关对象比较 → **UB**
- **NULL**：只用 **`==` / `!=`**

```c
if (p == NULL) return;
if (p >= arr && p < arr + n) { /* 在范围内 */ }
```

### 相减

```c
ptrdiff_t n = q - p;   /* 同数组：元素个数 */
```

- 类型 **`ptrdiff_t`**（有符号）
- 不同数组相减 → **UB**

## 与后续衔接

**memchr** 等返回位置指针；**ch17** 容器边界检查。

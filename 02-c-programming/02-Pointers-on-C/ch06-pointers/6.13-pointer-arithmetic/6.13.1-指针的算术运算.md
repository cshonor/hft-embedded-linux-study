# 6.13.1 指针的算术运算

## 本节讲什么

**`p + n`** / **`p - n`**：地址移动 **n 个元素**（× `sizeof(元素类型)`），非 n 字节。

## 要点

```c
int *p;     /* p+1 → +4 字节（通常） */
char *c;    /* c+1 → +1 字节 */
```

### 合法范围

- 结果必须仍指向**同一数组对象**内，或**尾后一位**（C99 one-past-end）
- 越界解引用 → **UB**（线上随机崩溃）

### 用途

- 遍历：`for (p = arr; p < arr+n; p++)`
- 字段偏移：`hdr = (struct eth_hdr *)(pkt + ETH_HLEN)`（需保证布局）

### DPDK

- `mbuf` 数组：`pkts[i]` 或 `*(pkts + i)`

## 与后续衔接

**ch10** 结构体成员指针；ring buffer 索引与指针互转。

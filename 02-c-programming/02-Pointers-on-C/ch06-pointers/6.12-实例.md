# 6.12 实例

## 本节讲什么

**综合实例**：模拟 DPDK 批量收包 + 指针遍历 + 空指针过滤 + const 报文缓冲。

## 要点

```c
#define BURST 32
struct pkt { int type; /* ... */ };

void process_burst(struct pkt *pkts[BURST], int n)
{
    int i;
    for (i = 0; i < n; i++) {
        struct pkt *p = pkts[i];
        if (p == NULL)
            continue;
        if (p->type == HEARTBEAT)
            continue;
        decode(p);
    }
}

/* 指针算术遍历连续数组 */
void walk(struct pkt *base, int n)
{
    struct pkt *end = base + n;
    for (; base < end; base++)
        decode(base);
}
```

### 检查清单

- 局部指针 **NULL** 初始化  
- **`free` 后置 NULL**  
- 只读协议 **`const`**  
- 越界：**`base + n`** 仍在分配范围内  

## 实操

实现 `pkts[32]` 模拟 burst；对比指针数组与数组指针地址。

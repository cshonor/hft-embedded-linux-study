# 6.4 间接访问操作符

## 本节讲什么

**`*` 解引用**：通过指针访问其所指对象；与声明中的 `*` 同名不同义。

## 要点

```c
int num = 100;
int *ptr = &num;

printf("%d", *ptr);   /* 读 num */
*ptr = 200;           /* 写 num */
```

| 上下文 | `*` 含义 |
|--------|----------|
| 声明 `int *p` | p 是指针 |
| 表达式 `*p` | 取 p 所指对象 |

- 解引用前指针必须**有效**（非 NULL、非悬垂、非野）
- `*` 结果是**左值**（可赋值）

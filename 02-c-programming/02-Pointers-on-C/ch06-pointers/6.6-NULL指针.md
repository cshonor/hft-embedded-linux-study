# 6.6 NULL 指针

## 本节讲什么

**NULL** 表示指针**不指向任何对象**；比较、初始化、API 空值的标准写法。

## 要点

```c
#define NULL ((void *)0)   /* 常见定义 */

int *p = NULL;
if (p == NULL) { /* ... */ }
if (p != NULL && p->len > 0) { /* 短路 */ }
```

- **`*NULL`** → 段错误
- C11 可选 **`nullptr`**；C++ 用 **`nullptr`**
- 释放后置 **`NULL`** 使二次 `free` 安全（仍不应 double-free）

### 工程用法

- mbuf/报文不存在：`if (!pkt) return;`
- 链表尾：`next == NULL`

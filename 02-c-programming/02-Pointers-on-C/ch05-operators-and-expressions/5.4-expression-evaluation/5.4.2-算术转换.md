# 5.4.2 算术转换

## 本节讲什么

二元算术/关系运算前，两操作数转为**共同类型**（**寻常算术转换** usual arithmetic conversions）。

## 要点

### 规则概要

1. 若有一边 **long double** → 两边 long double  
2. 否则若有一边 **double** → 两边 double  
3. 否则若有一边 **float** → 两边 float  
4. 整型：按**整数转换等级**向更宽类型提升；同宽则 **signed → unsigned**（若有一边 unsigned）

### 工程影响

- **unsigned + signed** → 常按 **unsigned** 算 → 负数变超大正数
- 混合比较前先 **显式 cast** 到同一有符号/无符号类型

### 指针算术（预告）

- 指针 ± 整数：**ch06**；不是本节算术转换，但同样依赖类型宽度

## 与 HFT

- 时间戳、序列号运算统一 **`uint64_t`**
- 价格运算统一 **`int64_t`** 定点

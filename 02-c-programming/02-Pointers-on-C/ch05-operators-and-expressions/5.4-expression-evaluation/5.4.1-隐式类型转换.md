# 5.4.1 隐式类型转换

## 本节讲什么

编译器自动插入的类型转换：**整型提升**、**赋值转换**、**布尔转换**。

## 要点

### 整型提升

- **`char`/`short`** 参与运算前通常提升为 **`int`**
- 有符号 `char` → `int`：**符号扩展**
- 无符号 `char` → `int`：高位补 0

### 赋值转换

- 右侧值转为**左侧类型**；可能**截断**高位

```c
uint8_t b = 0x123;   /* 截断为 0x23 */
```

### 经典陷阱

```c
unsigned char a = 200;
if (a > -1)   /* a 提升后为大 unsigned，恒真 */
```

### 布尔上下文

- if/while 条件：非 0 即真

## 实操

验证无符号 char 与 `-1` 比较。

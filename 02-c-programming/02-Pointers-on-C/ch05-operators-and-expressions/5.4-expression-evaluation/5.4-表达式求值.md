# 5.4 表达式求值

## 本节讲什么

表达式如何**求值**：隐式转换、算术转换、**优先级**、**结合性**、**副作用**与 **UB**。线上网关逻辑错乱、数值溢出、死循环的头号理论来源。

## 要点

- 表达式 = 运算符 + 操作数 + 求值顺序规则
- **副作用**：求值过程中修改变量（`++`、`=`, 函数调用）
- 两**序列点**之间同一变量**不可多次修改** → **UB**

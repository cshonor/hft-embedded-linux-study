# 5.1 操作符

## 本节讲什么

C **全部运算符**概览：算术、移位、位运算、关系、逻辑、一元、三元、赋值、逗号。DPDK mbuf 标志位、寄存器读写、报文解析的**刚需工具**。

## 要点

### 算术 `+ - * / %`

| 运算 | 规则 |
|------|------|
| 整数 `/` | 截断小数，向 0 取整：`7/3=2`，`-7/3=-2` |
| `%` | 仅整型；符号随**被除数** |
| 溢出 | **无符号**模 2^N 环绕（序列号常用）；**有符号**溢出 → **UB** |

HFT：价格用**整型分厘**，避免 `/` 丢精度。

### 移位 `<<` `>>`

- **`a << n`**：低位补 0，等价 ×2^n（正数/无符号安全）
- **`a >> n`**：无符号逻辑右移（补 0）；**有符号负数**算术右移（实现相关）
- **UB**：移位位数 ≥ 类型位宽

```c
(val >> 8) & 0xFF   /* 取第二字节 */
1u << 5             /* 第 5 位掩码 */
```

### 位运算 `~ & | ^`

| 符 | 用途 |
|----|------|
| `&` | 提取/过滤位 |
| `\|` | 置位 |
| `~` | 掩码取反 |
| `^` | 翻转位、校验 |

```c
flags |= PKT_ERR;
flags &= ~PKT_VLAN;
if (flags & PKT_ORDER) { /* 判位 */ }
```

⚠️ **`if (flags & MASK)`** ≠ **`if (flags && MASK)`**（逻辑与只给 0/1）。

### 关系与判等

- 结果 **`1`/`0`**，无原生 bool（C99 有 `_Bool`）
- **`=` vs `==`**：`if (flag = 1)` 恒真陷阱

### 逻辑 `&&` `||`（短路）

```c
if (mbuf != NULL && mbuf->len > 0)  /* NULL 时不解引用 */
```

- **`A && B`**：A 假则不求 B；**`A || B`**：A 真则不求 B

### 一元 `& * ! ~ sizeof ++ --`

- **`&`** 取址；**`*`** 解引用（**ch06**）
- **`sizeof`**：编译期字节数；数组传参后 **`sizeof(p)`** 是指针宽
- **前置/后置 `++`**：`b=a++` 先赋后增；`b=++a` 先增后赋
- **同一表达式多次 `++` 同一变量 → UB**

### 三元 `?:`

```c
uint64_t price = (bid > 0) ? bid : ask;
```

### 复合赋值 `+= &= <<=` 等

`a += b` 中 **a 只求值一次**；与 `a = a + b` 在 a 为复杂左值时可能不同。

### 逗号 `,`

`for (i=0, j=n; i<j; i++, j--)`；其余场景少用。

## 工程规范

复杂表达式**加括号**；单表达式**最多一次**修改变量。

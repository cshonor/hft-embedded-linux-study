# 5.3 左值和右值

## 本节讲什么

**左值（lvalue）**：有对象标识、可取址、可出现在赋值左侧；**右值（rvalue）**：临时值、常出现在右侧。

## 要点

| | 左值 | 右值 |
|---|------|------|
| 例子 | 变量 `x`、`*p`、数组元素 | 字面量 `42`、`a+b` 结果 |
| 赋值 | `x = 1` 合法 | `3 = x` 非法 |
| 取址 | `&x` 合法 | `&(a+b)` 非法 |

### 与运算符

- **赋值**左操作数必须是左值
- **`++`** 作用于左值
- **函数调用**结果多为右值（返回非数组）

### 副作用

修改左值即**副作用**：`=`、`++`、`--`、`*p = ...`

```c
arr[i++] = i;   /* i 被多次修改，UB 风险 */
```

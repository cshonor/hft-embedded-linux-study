# 5.2 布尔值

## 本节讲什么

C 用**整型 0/1** 表示真假；关系、逻辑运算结果用于 **if/while/for**（**ch04**）。

## 要点

- **0** → 假；**非 0** → 真（不限于 1）
- 关系/逻辑表达式结果类型为 **`int`**：`1` 或 `0`
- C99 **`_Bool`** / **`stdbool.h`** 的 `bool`、`true`、`false`

### 惯用写法

```c
if (mbuf)      /* 指针非 NULL */
if (!err)      /* err == 0 */
if (flags & X) /* 位测试，非逻辑与 */
```

### 陷阱

| 错误 | 后果 |
|------|------|
| `if (x = 1)` | 赋值，恒真 |
| `if (a > -1)`，`a` 为 unsigned | 恒真（**ch03**） |
| `if (flags && MASK)` | 语义错误 |

### 与 ch04

所有控制流条件都是**表达式**；本章运算规则决定条件真假。

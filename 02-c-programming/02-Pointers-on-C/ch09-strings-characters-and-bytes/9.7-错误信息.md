# 9.7 错误信息

## 本节讲什么

**`strerror(errno)`**、**`perror`**：将 **errno** 转为可读文本 —— 系统调用失败诊断。

## 要点

```c
#include <errno.h>
#include <string.h>

if (read(fd, buf, n) < 0)
    fprintf(stderr, "read: %s\n", strerror(errno));

perror("read");   /* 等价前缀 + strerror */
```

- 返回指向**静态存储**的指针（可能非线程安全）；线程环境用 **strerror_r**
- 属于**文本** API，与二进制报文无关

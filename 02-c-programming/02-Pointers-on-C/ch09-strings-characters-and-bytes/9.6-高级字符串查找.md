# 9.6 高级字符串查找

## 本节讲什么

**`strspn` / `strcspn` / `strpbrk` / `strtok`** 等：按字符集跳过、分词。

## 要点

```c
/* 跳过前导空白 */
size_t i = strspn(s, " \t");

/* 分词（会修改字符串，插入 \0） */
char *tok = strtok(buf, ",");
while (tok) {
    process(tok);
    tok = strtok(NULL, ",");
}
```

- **strtok** 非重入、改 buffer —— 多线程/重入用 **strtok_r**
- 仍仅适用于 **C 字符串**

## 与 HFT

- 文本配置、CSV 类日志可用了；二进制 FIX/SBE 用偏移表 + **mem**（**ch10**）

# 9.4 长度受限的字符串函数

## 本节讲什么

**`strncpy` / `strncat` / `strncmp`**：带 **n** 的 str 系列；仍依赖 `\0` 语义，二进制场景不用。

## 要点

### strncpy

```c
strncpy(dst, src, n);
```

- 最多写 **n** 字节；**若 src 长 ≥ n，可能不补 `\0`**
- 截断后必须手动：`dst[n-1]='\0'` 或 `dst[sizeof(dst)-1]='\0'`

### strncat / strncmp

- **strncat(dst, src, n)**：追加最多 n 个来自 src 的字符（仍危险）
- **strncmp**：最多比较 n 字符

### 工程建议

| 场景 | 推荐 |
|------|------|
| 文本拷贝 | **snprintf** / **strlcpy** |
| 二进制 | **memcpy** + len |

## 实操

strncpy 截断后 printf 乱码复现。

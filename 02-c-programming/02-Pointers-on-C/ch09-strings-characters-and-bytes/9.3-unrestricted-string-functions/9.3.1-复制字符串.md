# 9.3.1 复制字符串

## 本节讲什么

**`strcpy(dst, src)`**：拷贝至 `\0`（含），**不检查 dst 大小**。

## 要点

```c
strcpy(dst, src);   /* 生产环境禁用 */
```

| 风险 | 说明 |
|------|------|
| 溢出 | dst 小于 src+1 |
| 二进制 | src 中无 `\0` 或含 0x00 语义错误 |

### 替代

```c
strncpy(dst, src, sizeof(dst)-1);
dst[sizeof(dst)-1] = '\0';   /* strncpy 可能不补 \0 */

/* 更好 */
snprintf(dst, sizeof dst, "%s", src);
memcpy(dst, src, n);         /* 二进制，显式 n */
```

## 实操

strcpy vs memcpy 对含 0x00 的数据对比。

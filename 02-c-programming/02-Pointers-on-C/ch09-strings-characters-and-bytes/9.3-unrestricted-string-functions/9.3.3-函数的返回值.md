# 9.3.3 函数的返回值

## 本节讲什么

字符串函数**返回值**语义：多数返回 **dst** 或 **指针位置**，便于链式写法；比较函数返回 **int**。

## 要点

| 函数 | 返回值 |
|------|--------|
| strcpy/strcat | **dst**（同第一参数） |
| strcmp | `<0` / `0` / `>0` |
| strchr/strstr | 找到位置指针或 **NULL** |
| memcpy/memcpy | **dst** |

```c
char buf[64];
char *p = strcpy(buf, "ok");   /* p == buf */
```

- **勿**对可能 **NULL** 的返回值直接解引用

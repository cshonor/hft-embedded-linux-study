# 9.3.2 连接字符串

## 本节讲什么

**`strcat(dst, src)`**：找到 dst 末尾 `\0`，追加 src（含 `\0`）。

## 要点

- 要求 dst **足够大**容纳原有 + src + `\0`
- 与 strcpy 同类溢出风险
- **`strncat`** 限长但仍需保证 dst 容量（**9.4**）

```c
/* 安全倾向：手动追踪 offset + snprintf */
size_t off = strlen(dst);
snprintf(dst + off, sizeof dst - off, "%s", piece);
```

## 与后续衔接

日志拼接用 **snprintf** 链式写入。

# 9.3.4 字符串比较

## 本节讲什么

**`strcmp(s1, s2)`**：字典序比较，遇 `\0` 停止。

## 要点

```c
if (strcmp(cmd, "quit") == 0) { /* ... */ }
```

- **不能**用于二进制：中间 0x00 会截断语义
- 二进制用 **`memcmp(a, b, n)`**（**9.9**）

| strcmp | memcmp |
|--------|--------|
| 依赖 `\0` | 固定 n 字节 |
| 文本命令 | 报文头魔数 |

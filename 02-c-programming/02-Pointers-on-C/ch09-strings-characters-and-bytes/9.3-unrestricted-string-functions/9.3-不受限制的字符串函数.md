# 9.3 不受限制的字符串函数

## 本节讲什么

**`<string.h>`** 中依赖 **`'\0'`** 边界的经典函数：**无长度上限** → 生产环境高危。

## 要点

| 函数 | 作用 |
|------|------|
| strcpy / strcat | 拷贝 / 拼接 |
| strcmp | 比较 |
| strlen | 长度 |

**工程规范**：

- **文本日志、命令行**：可用（更好用安全替代）
- **二进制报文、mbuf payload**：**禁用**，改用 **9.9 mem\***

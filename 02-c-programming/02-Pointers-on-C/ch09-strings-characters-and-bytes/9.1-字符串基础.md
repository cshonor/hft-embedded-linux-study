# 9.1 字符串基础

## 本节讲什么

**C 标准字符串**：以 **`'\0'`** 结尾的 `char` 连续存储。与 **原始二进制字节流**（显式长度、可含 0x00）严格区分 —— DPDK/HFT 解析的第一道防线。

## 要点

```c
char str[] = "quote";   /* 'q''u''o''t''e''\0' → 6 字节 */
```

| 模型 | 边界 | 0x00 字节 |
|------|------|-----------|
| **C 字符串** | 第一个 `\0` | 表示结束 |
| **二进制 buf** | **`pkt_len`** | 合法数据 |

### 两种存储

```c
char pkt_buf[128] = "order";      /* 数组，可写 */
const char *proto = "heartbeat"; /* 字面量 .rodata，须 const */
```

- 修改字面量 → **SIGSEGV**
- 报文 payload：**`uint8_t *` + len**，不是 `%s`

### 遍历范式（预告）

- 下标至 `\0`：文本日志
- **`p` .. `end`** 指针：DPDK payload（**9.9**）

# 9.2 字符串长度

## 本节讲什么

**`strlen`** vs **`sizeof`** —— 最高频踩坑之一。

## 要点

| 函数 | 含义 | 含 `\0`？ |
|------|------|----------|
| **`strlen(s)`** | 有效字符数 | 否 |
| **`sizeof(arr)`** | 数组总字节 | 是（若 char[]） |

```c
char a[] = "quote";
strlen(a);    /* 5 */
sizeof(a);    /* 6 */

char *p = a;
strlen(p);    /* 5 */
sizeof(p);    /* 8（指针宽） */
```

### 致命风险

- 缓冲**无 `\0`** → `strlen` 越界读
- 二进制报文**禁止 strlen**；用 **`pkt_len`**

### 安全长度

- 文本：`strnlen(s, maxlen)`（POSIX）或自写限长扫描
- 二进制：始终显式 **`size_t len`**

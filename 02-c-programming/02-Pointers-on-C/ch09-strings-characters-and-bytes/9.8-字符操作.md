# 9.8 字符操作

## 本节讲什么

**`<ctype.h>`** 字符分类与大小写转换；ASCII 命令/日志解析。

## 要点

```c
#include <ctype.h>

if (isalpha((unsigned char)c))
if (isdigit((unsigned char)c))
if (isspace((unsigned char)c))

c = tolower((unsigned char)c);
```

### 隐藏坑

- 参数必须是 **`unsigned char`** 值或 **EOF**
- **`signed char` 0x80–0xFF`** 直接传入 → **UB**

## 适用

- 命令行 token、可读 ASCII 字段校验
- **二进制 payload 字节**勿用 is* 当通用判断

## 与后续衔接

**ch05** 整数提升；协议混合 ASCII/二进制分区解析。

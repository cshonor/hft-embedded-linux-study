# 9.5 字符串查找

## 本节讲什么

**`strchr` / `strstr`**：在 **`\0` 终止**字符串中查找字符/子串。

## 要点

```c
char *p = strchr(line, '=');      /* 配置 key=value */
char *q = strstr(haystack, "FIX");
```

- 均依赖 **`\0`** 作为扫描终点
- 二进制缓冲**不可用**

### 指针遍历文本

```c
for (char *p = line; *p; p++)
    if (*p == ',') /* ... */
```

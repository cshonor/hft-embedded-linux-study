# 9.9 内存操作

## 本节讲什么

**`<string.h>`** 中 **mem\*** 系列：**显式长度 n**，**不依赖 `\0`** —— DPDK mbuf、内核、HFT 报文**首选**。

## 要点

| 函数 | 作用 | 重叠 |
|------|------|------|
| **memcpy** | 拷贝 n 字节 | **不可**重叠 |
| **memmove** | 拷贝 n 字节 | **可**重叠 |
| **memset** | 填充 n 字节 | — |
| **memcmp** | 比较 n 字节 | — |

```c
uint8_t pkt_raw[1024];
size_t pkt_len = rte_pktmbuf_data_len(mbuf);
memcpy(pkt_raw, rte_pktmbuf_mtod(mbuf, void *), pkt_len);
```

### DPDK 指针遍历

```c
uint8_t *p = payload;
uint8_t *end = p + pkt_len;
for (; p < end; p++)
    parse_byte(*p);
```

### memset

```c
memset(buf, 0, sizeof buf);   /* mbuf/struct 清零 */
```

### 单字节 / 大小端（铺垫）

```c
uint32_t net = 0x12345678;
uint8_t b0 = (net >> 24) & 0xFF;
uint8_t b1 = (net >> 16) & 0xFF;
```

- 网络序转换：**htonl/ntohl**（**ch03**）

### 工程准则

| 数据类型 | API |
|----------|-----|
| 文本 | str* / snprintf |
| 二进制 | **mem\*** + **len** |

## 实操

1. memcpy vs memmove 重叠区  
2. 含 0x00 数据 strlen vs 手动 len  
3. memset 清零 struct  

## 与后续衔接

**ch10** 协议头 memcmp；**ch15** fread 二进制。

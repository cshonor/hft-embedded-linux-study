# 4.8.4 共享库

## 本节讲什么

共享库 `.so` 的发布涉及 **SONAME**、符号可见性、`rpath/runpath` 与版本脚本。正确安装后多进程共享同一份代码，**apt** 按 SONAME 管理兼容性（**4.5.3**）。

## 要点

### 构建与 SONAME

```bash
gcc -fPIC -shared -Wl,-soname,libfoo.so.1 -o libfoo.so.1.0.0 foo.c
ln -s libfoo.so.1.0.0 libfoo.so.1
ln -s libfoo.so.1 libfoo.so
readelf -d libfoo.so.1.0.0 | grep SONAME
```

### 链接与搜索

```bash
gcc -o prog main.c -L. -lfoo -Wl,-rpath,'$ORIGIN'
ldd prog
LD_LIBRARY_PATH=./ ./prog
```

### 符号导出控制

```c
#define API __attribute__((visibility("default")))
API void foo(void);
```

```bash
nm -D libfoo.so | grep foo       # 动态符号
readelf --dyn-syms libfoo.so
```

### 与静态库对比

```bash
# 同一项目两种产物
ar rcs libfoo.a foo.o
gcc -shared -o libfoo.so foo.o -fPIC
```

# 4.8.2 全局偏移表

## 本节讲什么

**GOT（Global Offset Table）** 是 `.got`/`.got.plt` 段中的 **地址表**：PIC 代码通过「基址 + 偏移」查表取得全局变量与函数的真实地址。链接时填部分项，加载时 `ld.so` 根据 **重定位表 `.rela.dyn`** 补全。

## 要点

### GOT 作用

- 数据访问：`mov var@GOTPCREL(%rip), %rax`
- 与 PLT 配合解析外部函数地址（**4.8.3**）

```bash
gcc -fPIC -shared -o libfoo.so foo.c
readelf -S libfoo.so | grep got
objdump -d libfoo.so | grep -i got
```

### 查看动态重定位

```bash
readelf -r libfoo.so              # .rela.dyn / .rela.plt
readelf -d libfoo.so | grep REL
```

常见项：`R_X86_64_GLOB_DAT`、`R_X86_64_JUMP_SLOT`（函数 GOT/PLT 槽）。

### 链接前后

```bash
gcc -fPIC -c foo.c
readelf -r foo.o
gcc -shared -o libfoo.so foo.o
readelf --dyn-syms libfoo.so
```

### 性能注记

- 经 GOT 多一次间接访存；`static` 内部函数可不走 GOT
- `-fno-plt` 等优化可减少 PLT 层（高级话题）

# 4.8 动态链接

## 本节讲什么

**动态链接** 把共享库 `.so` 在 **运行期** 由 `ld.so` 映射进进程，多进程 **共享代码页**，升级库无需重编主程序。实现依赖 **PIC**、**GOT**、**PLT** 与延迟绑定（**4.8.1–4.8.4**）。Linux 用户态程序、DPDK PMD 插件、内核模块（`insmod`）都属广义「运行时装入」。

## 要点

### 构建与链接

```bash
gcc -fPIC -shared -o libfoo.so foo.c
gcc -o prog main.c -L. -lfoo
readelf -d prog | grep NEEDED
ldd prog
```

### 静 vs 动

| | 静态 `.a` | 动态 `.so` |
|---|-----------|------------|
| 链接时机 | 构建期 | 构建期记依赖 + 运行期加载 |
| 体积 | 可执行大 | 可执行小，库共享 |
| 升级 | 重链 | 换 `.so`（SONAME 兼容） |

### 动态段

```bash
readelf -d libfoo.so
# SONAME、NEEDED、INIT、FINI、PLTGOT、JMPREL...
readelf --dyn-syms libfoo.so
```

### 加载流程（简述）

1. `execve` 映射主程序
2. `ld.so` 加载 `NEEDED` 库
3. 重定位 GOT/PLT（**4.8.2/4.8.3**）
4. 跳 `_start` → `main`

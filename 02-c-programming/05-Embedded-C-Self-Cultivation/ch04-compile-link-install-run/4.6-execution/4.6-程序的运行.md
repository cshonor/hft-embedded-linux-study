# 4.6 程序的运行

## 本节讲什么

链接/安装之后，**运行** 指 OS 或裸机把 ELF 载入内存、建立映射、清零 BSS、从 **入口 `_start`** 启动运行时，最终调用 **`main`**。Linux 上由内核 `execve` + `ld-linux.so` 完成；裸机由启动代码与链接脚本决定（**4.6.2**、**4.14**）。

## 要点

### 运行链路（Linux）

```
shell → execve() → 内核装载 ELF PT_LOAD
      → ld.so 加载 .so、重定位
      → _start → __libc_start_main → main(argc, argv)
```

```bash
readelf -h ./prog | grep Entry
readelf -l ./prog                  # LOAD 段
ldd ./prog
```

### 关键参与者

| 组件 | 作用 |
|------|------|
| **Loader** | 映射段、设权限、ASLR |
| **ld.so** | 动态链接、解析 GOT/PLT |
| **_start** | CRT 入口（非 main） |
| **BSS** | 加载时清零未初始化全局（**4.6.4**） |

### 调试入口

```bash
gdb ./prog
(gdb) break _start
(gdb) break main
(gdb) run
(gdb) info files
```

# 6.10.3 内联函数与宏

## 本节讲什么

**宏** 灵活但缺类型、双求值；**inline 函数** 更安全。内核仍大量宏因 **typeof/语句表达式**；新代码优先 **static inline**。

## 要点

```c
#define MACRO_SQ(x) ((x) * (x))
static inline int sq(int x) { return x * x; }
```

| 需求 | 选宏 | 选 inline |
|------|------|-----------|
| `#x` 字符串化 | 是 | 否 |
| 热点小函数 | 可 | **优先** |

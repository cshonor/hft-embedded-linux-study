# 6.10.4 编译器对内联函数的处理

## 本节讲什么

是否 inline 由 **启发式** 决定：函数大小、`-O` 级别、**`always_inline`**、**LTO**。

## 要点

```bash
gcc -O0 -S foo.c -o O0.s
gcc -O2 -S foo.c -o O2.s
diff O0.s O2.s | head
```

| 因素 | 倾向 |
|------|------|
| `-O2`/`-O3` | 更 aggressive inline |
| 函数体大 | 不 inline |
| **LTO** | 跨文件 inline |

# 6.10.1 属性声明：noinline

## 本节讲什么

**`noinline`** 禁止内联；**`always_inline`** 强制内联。用于 **调试栈回溯**、**控制代码体积** 或 **极热路径必 inline**。

## 要点

```c
__attribute__((noinline))
static int cold_path(int x) { return x * 2; }

static inline __attribute__((always_inline))
int hot_add(int a, int b) { return a + b; }
```

```bash
gcc -O2 -S inline_demo.c -o inline_demo.s
grep -E 'call|hot_add' inline_demo.s
```

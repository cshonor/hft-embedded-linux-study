# 6.3.1 表达式、语句和代码块

## 本节讲什么

C 语法区分 **表达式（有值）** 与 **语句（执行效果）**；**复合语句** `{ ... }` 即代码块。宏只能「文本替换」——若要在宏里写 `if`、临时变量又 **返回一个值**，需要语句表达式桥接二者。

## 要点

| 种类 | 示例 | 能否出现在右值位置 |
|------|------|-------------------|
| **表达式** | `a + b`、`func()` | 是 |
| **语句** | `if (x) y;`、`return 0;` | 否 |
| **代码块** | `{ int t=x; x=t; }` | 单独不行 |

```c
int x = 1 + 2;           /* 表达式语句 */
if (x > 0) { x++; }      /* 语句块 */

/* 错误：块不是表达式 */
/* int y = { int t=1; t; }; */
```

```c
/* GNU：块 + 末尾表达式 → 整体是表达式 */
int y = ({
    int t = x + 1;
    t * 2;
});
```

- 宏 `MAX(a,b)` 若写成三元运算符，**a/b 可能被求值两次**（见 **6.3.3**）
- 内核 `do { ... } while (0)` 是 **语句** 包装，无返回值

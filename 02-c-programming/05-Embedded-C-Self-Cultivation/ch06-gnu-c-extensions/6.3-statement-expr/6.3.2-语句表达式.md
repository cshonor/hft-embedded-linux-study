# 6.3.2 语句表达式

## 本节讲什么

语句表达式 **`({ ... })`** 内可含声明、循环、分支；**最后一个表达式**（带分号）的值类型即为整个语句表达式的类型。GCC 扩展，Clang 兼容。

## 要点

```c
int abs_diff(int a, int b)
{
    return ({
        int _a = a;
        int _b = b;
        _a > _b ? _a - _b : _b - _a;
    });
}
```

```c
/* 指针版本 — typeof 配合 */
#define swap_ptr(a, b) do {             \
    typeof(a) _tmp = (a);               \
    (a) = (b);                          \
    (b) = _tmp;                         \
} while (0)
```

规则摘要：

- 可有多条语句；**最后一条必须是表达式**  followed by `;`
- 内部变量作用域仅在 `({})` 内
- 类型 = 最后表达式的类型（含 const/volatile）
- 在 `-pedantic` 下可用 `__extension__ ({ ... })` 压制警告

```bash
echo 'int x = ({ int a=1; a+1; });' | gcc -std=gnu11 -x c -c - -o /tmp/t.o
echo 'int x = ({ int a=1; a+1; });' | gcc -std=c11   -x c -c - - 2>&1 | head
```

# 6.3 宏构造「利器」：语句表达式

## 本节讲什么

**语句表达式（statement expression）** 语法 `({ statements; expr; })` 把多条语句打包成一个 **有值的表达式**，是 GNU 扩展。内核 **`min`/`max`**、**`container_of`**、原子操作宏都依赖它；没有语句表达式，很多 C 宏无法既安全又返回结果。

## 要点

```c
#define min(a, b) ({                    \
    typeof(a) _a = (a);                 \
    typeof(b) _b = (b);                 \
    _a < _b ? _a : _b;                  \
})
```

- 最后一项 **分号前的表达式** 为整个 `({})` 的值
- 内部可声明临时变量，避免宏参数 **多次求值**
- `-std=gnu11` 默认可用；纯 `-std=c11` 非法

# 6.1.5 编译器对C语言标准的扩展

## 本节讲什么

**GNU C 扩展** 是 GCC（及兼容前端）在 ISO C 之外提供的语法与语义，Linux 内核、U-Boot、Glibc、DPDK 深度依赖。本章 **6.2–6.12** 逐项展开；此处给出总览地图。

## 要点

### 本章覆盖的主要扩展

| 类别 | 示例 | 小节 |
|------|------|------|
| **属性** | `packed` `aligned` `section` `weak` `format` | 6.6–6.9 |
| **语句表达式** | `({ int x=1; x+1; })` | 6.3 |
| **typeof** | `typeof(*ptr) tmp` | 6.4 |
| **零长度数组** | `char buf[0]` | 6.5 |
| **指定初始化** | `[3]=1` `.name=` | 6.2 |
| **内联控制** | `inline` `always_inline` `noinline` | 6.10 |
| **内建函数** | `__builtin_expect` | 6.11 |
| **可变参数宏** | `#define LOG(fmt, ...) ... ##__VA_ARGS__` | 6.12 |

### 其他常见扩展（全书 scattered）

```c
__extension__ ({ ... })              /* 在 -pedantic 下压制扩展警告 */
__asm__ volatile ("mrs %0, cpsr" : "=r"(cpsr))
__attribute__((constructor)) void init(void) { }
__attribute__((noreturn)) void die(void);
__attribute__((deprecated("use foo2"))) void foo(void);
__attribute__((unused)) static int placeholder;
```

- **`__builtin_constant_p(n)`**：编译期常量分支（**6.11.4**）
- **Case range** `case 1 ... 5:`：GNU 扩展，内核 switch 常见
- **Compound literal** `(struct foo){ .x = 1 }`：C99 标准，GNU 早支持

```bash
# 列出常见内建
gcc -dM -E - < /dev/null | grep __builtin | head
```

### 内核/DPDK 视角

- 内核 `include/linux/compiler_attributes.h` 封装 `__attribute__`
- DPDK `rte_branch_prediction.h`：`likely`/`unlikely` → `__builtin_expect`
- 写可移植用户态库：扩展放 **internal header**，对外 API 用标准 C

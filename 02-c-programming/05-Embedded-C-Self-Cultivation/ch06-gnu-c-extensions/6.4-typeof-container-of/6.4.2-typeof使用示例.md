# 6.4.2 typeof使用示例

## 本节讲什么

通过 **链表节点嵌入**、**泛型 min 宏**、**指针赋值** 三类示例掌握 typeof 在工程中的典型用法。

## 要点

```c
struct list_node { struct list_node *next; };

struct task {
    struct list_node link;
    int id;
    char name[16];
};

/* 从 link 指针得到 task — 前置 container_of 概念 */
#define list_entry(ptr, type, member) \
    container_of(ptr, type, member)
```

```c
#define TYPE_SAFE_MIN(a, b) ({              \
    typeof(a) _a = (a);                     \
    typeof(b) _b = (b);                     \
    _a < _b ? _a : _b;                      \
})

/* 不同整数类型自动 promotion */
long lv = TYPE_SAFE_MIN(100L, (short)3);
```

```c
void set_u32(uint32_t *p, uint32_t v) { *p = v; }

#define SET_FIELD(field, value) do {        \
    typeof(field) __v = (value);            \
    (field) = __v;                          \
} while (0)
```

```bash
gcc -std=gnu11 -Wall -c typeof_demo.c
```

- DPDK `RTE_SET_USED(x)` 等与 typeof 无关，但同文件常见 compiler 宏
- 避免 `typeof` 与 **side effect 参数** 同用：先赋临时再运算

# 6.4 typeof与container_of宏

## 本节讲什么

**typeof** 在编译期取得表达式类型；**container_of** 从结构体成员指针反推宿主结构体地址。二者是 Linux 内核 **侵入式链表**、**device model** 的基石；DPDK 部分代码沿用相同技巧。

## 要点

- `typeof(x)` 等价于 `x` 的类型，可声明临时变量
- `container_of(ptr, type, member)` = `ptr` 所在 `type` 的首地址
- 依赖 **GNU typeof** + **语句表达式** + **`offsetof`**

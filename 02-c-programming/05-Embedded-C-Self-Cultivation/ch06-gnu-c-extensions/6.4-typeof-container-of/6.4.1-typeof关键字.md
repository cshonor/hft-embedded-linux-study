# 6.4.1 typeof关键字

## 本节讲什么

**typeof**（GNU 扩展；C23 标准关键字同名）在编译期复制操作数的类型，用于宏中声明与参数同类型的临时变量，避免 `void*` 或多次求值。

## 要点

```c
int x = 42;
typeof(x) y = 0;        /* y 为 int */
typeof(&x) p = &x;      /* p 为 int * */

#define SWAP(a, b) do {                 \
    typeof(a) __t = (a);                \
    (a) = (b);                          \
    (b) = __t;                          \
} while (0)
```

| 对比 | 说明 |
|------|------|
| `typeof(a)` | 保留 const/volatile |
| `__typeof__` | 旧 GNU 拼写，等价 |
| C23 `_Generic` | 标准替代，宏写法更繁 |

```c
/* 内核常用 typeof(*ptr) */
#define assign_ptr(ptr, val) do {       \
    *(ptr) = (typeof(*(ptr)))(val);     \
} while (0)
```

- Clang/GCC 均支持；MSVC 无 typeof，需 typedef 或 C11 `_Generic`
- 与 **`__builtin_types_compatible_p`** 配合做类型断言

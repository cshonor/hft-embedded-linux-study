# 6.7.5 内核中的aligned、packed声明

## 本节讲什么

内核在 `compiler_types.h` / `cache.h` 封装 **`__aligned(x)`**、**`__packed`**、**`____cacheline_aligned_in_smp`** 等，用于 **cache line 隔离、DMA 描述符、协议头、per-CPU 变量**。

## 要点

```c
/* 概念 — 实际宏名以内核树为准 */
#define __aligned(x)    __attribute__((aligned(x)))
#define __packed        __attribute__((packed))
#define ____cacheline_aligned __attribute__((aligned(SMP_CACHE_BYTES)))

struct driver_stats {
    unsigned long rx_packets;
    unsigned long tx_packets;
} ____cacheline_aligned;
```

```c
/* uapi 网络头 — 用户态与内核一致 */
struct tcp_hdr_min {
    __be16 source;
    __be16 dest;
    /* ... */
} __attribute__((packed));
```

| 宏 | 用途 |
|----|------|
| `____cacheline_aligned` | 防 false sharing |
| `__packed` | 协议/on-media 布局 |
| `__aligned(8)` | 原子 uint64_t |

- **per-CPU 变量**：每 CPU 副本 + cache line 对齐
- DPDK **`__rte_cache_aligned`**、`RTE_CACHE_LINE_SIZE` 同目的
- **Device tree / MMIO**：`readl`/`writel` 仍按 **寄存器对齐** 访问，packed 只解决 C struct 布局

```bash
grep -r '__cacheline_aligned' include/linux/ --include='*.h' | head
```

# 6.7 属性声明：aligned

## 本节讲什么

**对齐（alignment）** 决定变量/结构体首址是否为 **2/4/8/64…** 的倍数，影响 **原子访问、SIMD、DMA、cache line**。GCC **`__attribute__((aligned(n)))`** 与 **`packed`** 控制布局；内核 `____cacheline_aligned`、DPDK `__rte_cache_aligned` 同源。

## 要点

- CPU 未对齐访问：ARM 可能 fault 或 kernel fixup；x86 慢路径
- **`aligned(n)`**：最小对齐字节数，须为 2 的幂
- **`packed`**：取消成员间 padding（**6.7.4**）

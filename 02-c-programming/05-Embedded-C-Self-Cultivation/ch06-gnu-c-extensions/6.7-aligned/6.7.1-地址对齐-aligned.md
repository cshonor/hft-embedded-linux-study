# 6.7.1 地址对齐：aligned

## 本节讲什么

**对齐** 指对象地址 mod N == 0。硬件要求 **4 字节 int**、**8 字节 double**、**16 字节 SSE**、**64 字节 cache line** 等于不同 N。编译器默认按 ABI 对齐；`aligned` 属性 **提高** 对齐要求。

## 要点

```c
char c __attribute__((aligned(64)));   /* 64 字节对齐 — cache line */
int  x __attribute__((aligned(8)));

_Static_assert(__alignof__(x) >= 8, "align");
```

```c
/* 栈/堆数组手动对齐 — C11 */
#include <stdalign.h>
alignas(64) char buf[128];
```

| N | 典型用途 |
|---|----------|
| 4 | ARM 32-bit 字 |
| 8 | double / 指针 |
| 16 | NEON / SSE |
| 64 | 防 false sharing（DPDK per-lcore） |

```bash
gcc -std=gnu11 -O2 -c align.c
objdump -t align.o | grep -E 'buf|c '
```

- **`__alignof__(type)`** / **`_Alignof`**：查询对齐
- DMA buffer 常 **`posix_memalign(..., 4096, size)`** 或 **`aligned_alloc`**
- 内核 **`kmalloc`** 保证 `SMP_CACHE_BYTES` 对齐变体

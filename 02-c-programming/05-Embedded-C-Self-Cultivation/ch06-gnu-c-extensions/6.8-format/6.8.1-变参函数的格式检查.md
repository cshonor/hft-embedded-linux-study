# 6.8.1 变参函数的格式检查

## 本节讲什么

标准 **`printf`** 家族在 `<stdio.h>` 已带 format 属性；自定义 **`log_error(fmt, ...)`** 需显式声明，编译器才能比对 **格式说明符** 与 **实参类型**，在编译期发现 `%d` 与指针不匹配等问题。

## 要点

```c
#include <stdio.h>
#include <stdarg.h>

void warn_user(const char *fmt, ...)
    __attribute__((format(printf, 1, 2)));

void warn_user(const char *fmt, ...)
{
    va_list ap;
    va_start(ap, fmt);
    vfprintf(stderr, fmt, ap);
    va_end(ap);
}
```

```c
/* 固定前缀 + 可变 fmt 时，format 索引要跳过前缀 */
void log_err(const char *tag, const char *fmt, ...)
    __attribute__((format(printf, 2, 3)));
```

| format 种类 | 用途 |
|-------------|------|
| `printf` | printf 规则 |
| `scanf` | scanf 规则 |
| `strftime` | 时间格式 |

```bash
gcc -Wall -Wformat=2 -Werror=format -c log.c
# 故意写错实参类型，观察 -Wformat 诊断
```

- **`-Wformat-nonliteral`**：format 非字符串常量时警告（函数指针表驱动 format 串）
- 内核 **`__printf(a,b)`**、**`__scanf(a,b)`** 宏封装 attribute
- **`__attribute__((format(gnu_printf, ...)))`**：GCC 扩展 printf 语义

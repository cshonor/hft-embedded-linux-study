# 6.8.2 变参函数的设计与实现

## 本节讲什么

C **变参函数** 用 **`...`** 与 **`stdarg.h`**（`va_list` / `va_start` / `va_arg` / `va_end`）实现。设计日志 API 时：固定前缀参数（level、tag）+ 可变 fmt/args；format 属性索引要 **跳过固定参**。

## 要点

```c
#include <stdarg.h>
#include <stdio.h>

void vlog(int level, const char *tag,
          const char *fmt, ...)
    __attribute__((format(printf, 3, 4)));

void vlog(int level, const char *tag,
          const char *fmt, ...)
{
    va_list ap;
    va_start(ap, fmt);
    fprintf(stderr, "[%d][%s] ", level, tag);
    vfprintf(stderr, fmt, ap);
    fputc('\n', stderr);
    va_end(ap);
}
```

| 规则 | 说明 |
|------|------|
| 至少一个 named 参数在 `...` 前 | C 要求 |
| default promotion | `char`→`int`，`float`→`double` |
| `va_end` 必须调用 | 栈清理 |

- 内核 **`printk`** 支持 `%pV` 等扩展格式
- DPDK **`RTE_LOG`** 多级 + 编译期 level

# 6.8.3 实现自己的日志打印函数

## 本节讲什么

综合 **format 属性**、**va_list**、**级别过滤**、**文件名行号宏**，实现生产级 **`LOG_INFO(fmt, ...)`** 雏形；对应 **demo06**。

## 要点

```c
#include <stdio.h>
#include <stdarg.h>

enum { LOG_ERR, LOG_WARN, LOG_INFO, LOG_DEBUG };
static int g_log_level = LOG_INFO;

void log_at(int level, const char *file, int line,
            const char *func, const char *fmt, ...)
    __attribute__((format(printf, 5, 6)));

void log_at(int level, const char *file, int line,
            const char *func, const char *fmt, ...)
{
    if (level > g_log_level)
        return;
    va_list ap;
    va_start(ap, fmt);
    fprintf(stderr, "%s:%d %s: ", file, line, func);
    vfprintf(stderr, fmt, ap);
    fputc('\n', stderr);
    va_end(ap);
}

#define LOG_INFO(fmt, ...) \
    log_at(LOG_INFO, __FILE__, __LINE__, __func__, fmt, ##__VA_ARGS__)
```

| 层次 | 职责 |
|------|------|
| `LOG_*` 宏 | 注入 `__FILE__`/`__LINE__`/`__func__` |
| `log_at` | level 过滤 + format 检查 |
| `vfprintf` | 实际格式化 |

```bash
gcc -std=gnu11 -Wall -Wformat=2 -o demo06 demo06_log_macro.c
./demo06
```

- **`##__VA_ARGS__`** 空参时去掉前逗号（**6.12**）
- DPDK **`RTE_LOG(DEBUG, EAL, "port %u\n", port_id);`** 编译期 stripping

# 6.8 属性声明：format

## 本节讲什么

**`__attribute__((format(printf, fmt_idx, arg_start)))`** 让 GCC 对 **自定义 printf 风格函数** 做 **格式串与实参** 编译期检查，捕获 `%d` 配 `"string"` 类错误。内核 `printk`、DPDK `RTE_LOG`、自研日志层都依赖此属性。

## 要点

```c
void my_log(int level, const char *fmt, ...)
    __attribute__((format(printf, 2, 3)));
```

- 第 2 参为 format 串下标（从 1 计）
- 第 3 参为 **第一个可变参** 下标
- 配合 **6.12** 宏转发 `...`

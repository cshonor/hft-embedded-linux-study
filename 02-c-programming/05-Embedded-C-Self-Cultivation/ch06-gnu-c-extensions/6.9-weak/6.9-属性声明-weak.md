# 6.9 属性声明：weak

## 本节讲什么

**弱符号（weak symbol）** 链接时若存在 **强符号** 同名定义则 **被覆盖**；否则使用 weak 默认实现。`__attribute__((weak))` 用于 **可插拔 hook、BSP 默认 stub、单元测试替换**。

## 要点

```c
void board_init(void) __attribute__((weak));
void board_init(void) { /* default empty */ }
```

- 函数与全局变量均可 weak
- 链接阶段决议，非运行时
- **demo03** 演示 weak 覆盖

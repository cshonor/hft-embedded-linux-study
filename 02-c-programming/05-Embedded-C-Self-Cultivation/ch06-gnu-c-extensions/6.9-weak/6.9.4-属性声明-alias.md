# 6.9.4 属性声明：alias

## 本节讲什么

**`__attribute__((alias("sym")))`** 为现有符号创建 **别名**；常与 **weak** 结合导出兼容旧名。

## 要点

```c
void default_handler(int sig) { (void)sig; }

void legacy_handler(int sig)
    __attribute__((alias("default_handler")));
```

| 属性 | 作用 |
|------|------|
| `alias("target")` | 本符号为 target 别名 |
| `weak` | 可被强符号覆盖 |

```bash
nm demo.o | grep handler
```

- **限制**：alias 目标须在同翻译单元

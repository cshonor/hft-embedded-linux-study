# 6.9.3 弱符号的用途

## 本节讲什么

Weak 典型场景：**默认 stub**、**可选 feature**、**测试 mock**、**单内核多 BSP**。避免 `#ifdef` 泛滥；链接时自动选最强定义。

## 要点

```c
void __attribute__((weak)) pm_suspend(void) { }
void __attribute__((weak)) trace_irq(int irq) { (void)irq; }
```

| 场景 | 说明 |
|------|------|
| BSP | 板级 `board_init` 覆盖 weak 空函数 |
| Kconfig 关闭 feature | 仍链 weak stub |
| 单测 | 测试文件 strong 替换 `time_now()` |

- 与 **6.9.4 alias** 组合：`alias("default")` + weak

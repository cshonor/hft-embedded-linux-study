# 6.9.1 强符号和弱符号

## 本节讲什么

链接器规则：**强符号** 唯一定义；**弱符号** 可被强符号 **覆盖** 而不报 duplicate symbol。C 中 **weak attribute** 与 **common block** 行为不同。

## 要点

| 种类 | 典型来源 |
|------|----------|
| **强符号** | 函数定义、已初始化全局 |
| **弱符号** | `__attribute__((weak))` |

```c
/* a.c */
int counter __attribute__((weak)) = 0;
/* b.c — 强符号覆盖 */
int counter = 100;
```

```bash
gcc -c a.c -o a.o && gcc -c b.c -o b.o
gcc a.o b.o -o t && nm t | grep counter
```

- **重复强符号** → 链接错误
- 内核 **`__weak`** 宏广泛用于 arch hook

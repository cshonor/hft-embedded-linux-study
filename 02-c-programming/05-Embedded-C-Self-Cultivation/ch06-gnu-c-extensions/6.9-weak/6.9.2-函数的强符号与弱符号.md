# 6.9.2 函数的强符号与弱符号

## 本节讲什么

**weak 函数** 提供默认空实现；板级提供 **强符号** 同签名函数即可 **无修改链接** 替换。与 **函数指针表** 相比：零 indirection，但 **仅一个** 最终实现。

## 要点

```c
void hook_on_boot(void) __attribute__((weak));
void hook_on_boot(void) { printf("default hook\n"); }

/* board.c 强符号 */
void hook_on_boot(void) { printf("board hook\n"); }
```

| weak 函数 | 函数指针 |
|-----------|----------|
| 零开销直接调用 | 需间接调用 |
| 仅一份实现 | 可运行时切换 |
| 链接期绑定 | 灵活 |

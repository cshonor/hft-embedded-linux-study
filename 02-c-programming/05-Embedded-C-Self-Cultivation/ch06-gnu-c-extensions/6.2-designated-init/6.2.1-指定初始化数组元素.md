# 6.2.1 指定初始化数组元素

## 本节讲什么

数组指定初始化用 **`[index] = value`** 语法，稀疏数组与查找表尤其清晰。ROM 中只占用实际元素，未写明的下标为 0（静态存储期）。

## 要点

```c
/* 稀疏 IRQ 映射表 */
static int irq_map[256] = {
    [0]  = -1,
    [32] = 1,
    [33] = 2,
    [255] = -1,
};

/* 字符串/枚举索引（C99 枚举即 int 常量） */
enum { OP_ADD, OP_SUB, OP_MUL, OP_MAX };
static const char *op_name[OP_MAX] = {
    [OP_ADD] = "add",
    [OP_SUB] = "sub",
    [OP_MUL] = "mul",
};
```

```c
/* 范围初始化：GNU 扩展 case ... 的「表驱动 cousin」 */
static int level[16] = {
    [0 ... 3] = 1,   /* GNU: 连续下标批量赋值 */
    [15] = 99,
};
```

> **注意**：`[0 ... 3]` 为 **GNU 扩展**，`-std=c99 -pedantic` 会警告；内核普遍可用。

```bash
gcc -std=gnu11 -c table.c
objdump -s -j .rodata table.o | less   # 观察稀疏表在 .rodata 的布局
```

- 与 **memset 全表** 相比：源码可读、编译期确定
- DPDK `rte_*` 静态 ops 表偶见 designated 初始化

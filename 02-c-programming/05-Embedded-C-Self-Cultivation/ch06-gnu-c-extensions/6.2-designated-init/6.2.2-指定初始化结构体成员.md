# 6.2.2 指定初始化结构体成员

## 本节讲什么

结构体用 **`.member = value`** 初始化，顺序任意、可只写部分成员。嵌套结构体、数组字段同样支持 designated。这是内核 **`file_operations`**、**`of_device_id`** 等结构最常用写法。

## 要点

```c
struct point {
    int x, y, z;
};

struct point p = { .y = 10, .x = 20 };  /* z 自动为 0 */

struct nested {
    struct point origin;
    const char *label;
};

struct nested n = {
    .label  = "sensor_a",
    .origin = { .x = 1, .y = 2 },
};
```

```c
/* 静态 const 配置 — DPDK 风格示意 */
struct port_conf {
    uint16_t rx_queues;
    uint16_t tx_queues;
    uint32_t promisc : 1;
};

static const struct port_conf eth0 = {
    .rx_queues = 4,
    .tx_queues = 4,
    .promisc   = 0,
};
```

| 写法 | 优点 |
|------|------|
| `.field = v` | 重构字段顺序时 diff 小 |
| 指定 + 省略 | 默认值 0，表项只写差异 |

```c
/* C99 compound literal + designated */
struct point tmp = (struct point){ .x = 1, .y = 2 };
```

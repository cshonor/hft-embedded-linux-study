# 6.2 指定初始化

## 本节讲什么

**指定初始化（designated initializer）** 允许按 **下标或成员名** 初始化聚合类型，不必按顺序填零。C99 纳入标准；GNU 更早支持。内核驱动表、DPDK 静态配置、协议常量表均大量使用。

## 要点

```c
int a[10] = { [3] = 1, [7] = 2 };
struct cfg { int port; const char *name; } c = { .port = 8080, .name = "dpdk0" };
```

- 未指定元素 **零初始化**
- 可与普通初始化混用（C99 起顺序规则需查标准）
- C++ 不支持相同语法（C++20 有不同 designated init）

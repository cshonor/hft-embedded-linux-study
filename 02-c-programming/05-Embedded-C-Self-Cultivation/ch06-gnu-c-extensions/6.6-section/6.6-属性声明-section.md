# 6.6 属性声明：section

## 本节讲什么

**`__attribute__((section("name")))`** 把变量或函数放入 **自定义 ELF 段**，链接脚本 **`SECTIONS`** 再决定加载地址。用于 **init 段回收**、**U-Boot 自复制**、**模块参数表**、**tracepoint 段**。

## 要点

```c
static int cfg __attribute__((section(".my_cfg"))) = 42;
void boot_helper(void) __attribute__((section(".boot.text")));
```

```bash
objdump -h a.out          # 段列表
readelf -S a.out | grep boot
nm a.out | grep boot_helper
```

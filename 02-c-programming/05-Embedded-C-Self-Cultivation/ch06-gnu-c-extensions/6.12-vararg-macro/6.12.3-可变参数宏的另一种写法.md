# 6.12.3 可变参数宏的另一种写法

## 本节讲什么

除 **`##__VA_ARGS__`** 外，可用 **C20 `__VA_OPT__`**、**重载宏计数**、GNU **`args...`** 旧语法。

## 要点

```c
#define LOG(fmt, ...) \
    printf(fmt __VA_OPT__(,) __VA_ARGS__)
```

| 写法 | 标准 |
|------|------|
| `##__VA_ARGS__` | GNU |
| `__VA_OPT__` | C20 |

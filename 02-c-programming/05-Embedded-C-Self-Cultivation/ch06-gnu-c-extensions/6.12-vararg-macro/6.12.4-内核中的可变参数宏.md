# 6.12.4 内核中的可变参数宏

## 本节讲什么

内核 **`printk`**、**`pr_info`/`dev_err`** 等均 **可变参宏 + format 属性**。

## 要点

```c
#define pr_info(fmt, ...) \
    printk(KERN_INFO fmt, ##__VA_ARGS__)
```

| 宏族 | 用途 |
|------|------|
| `pr_*` | 内核通用 log |
| `dev_*` | 带 device 前缀 |

- DPDK **`RTE_LOG`** 编译期 level + **`##__VA_ARGS__`**

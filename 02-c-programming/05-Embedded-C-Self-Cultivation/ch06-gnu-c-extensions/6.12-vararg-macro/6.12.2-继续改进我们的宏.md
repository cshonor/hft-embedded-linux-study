# 6.12.2 继续改进我们的宏

## 本节讲什么

逐步完善 **`LOG` 宏**：加入 **级别**、**`__FILE__`/`__LINE__`/`__FUNCTION__`**、**`##__VA_ARGS__` 空参**、**do-while(0)** 包装，并与 **6.8 format 属性** 底层函数配合。

## 要点

### 问题 1：空可变参

```c
#define LOG(fmt, ...) printf(fmt, ##__VA_ARGS__)
LOG("started\n");   /* GNU ## 去掉多余逗号 */
```

### 问题 2：宏作为单语句

```c
#define LOG(fmt, ...) do {                          \
    printf("[%s:%d] " fmt "\n",                     \
           __FILE__, __LINE__, ##__VA_ARGS__);       \
} while (0)
```

### 问题 3：级别 + 函数名

```c
#define LOG_LEVEL 2
#define LOG(lvl, fmt, ...) do {                     \
    if ((lvl) <= LOG_LEVEL)                          \
        fprintf(stderr, "%s: " fmt, __func__, ##__VA_ARGS__); \
} while (0)
```

| 改进 | 目的 |
|------|------|
| `do { } while (0)` | 安全用于 `if/else` 后 |
| `##__VA_ARGS__` | `LOG("ok")` 无参合法 |
| 底层 `format` 函数 | 编译期格式检查（**demo06**） |

```c
/* 语句表达式版：宏返回值（少用） */
#define max3(a,b,c) ({ int _m = max(a,b); max(_m, c); })
```

# 6.11.2 常用的内建函数

## 本节讲什么

嵌入式/内核/DPDK 常见 **`__builtin_*`**：位计数、字节序、prefetch、return address。

## 要点

```c
int lz  = __builtin_clz(1u);
int pop = __builtin_popcountll(mask);
uint32_t sw = __builtin_bswap32(0x01020304);
__builtin_prefetch(p + 64, 0, 3);
void *ra = __builtin_return_address(0);
```

| Builtin | 场景 |
|---------|------|
| `__builtin_prefetch` | 包处理 prefetch |
| `__builtin_bswap*` | 网络 endian |
| `__builtin_expect` | 分支布局 |

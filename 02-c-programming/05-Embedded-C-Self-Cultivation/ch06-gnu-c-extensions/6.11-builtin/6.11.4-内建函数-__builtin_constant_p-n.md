# 6.11.4 内建函数：__builtin_constant_p(n)

## 本节讲什么

**`__builtin_constant_p(expr)`** 编译期判断 **expr 是否为常量**；为真时 **丢弃假分支**。

## 要点

```c
#define SET_BIT(x, n) do {                              \
    if (__builtin_constant_p(n))                        \
        (x) |= (1UL << (n));                            \
    else                                                \
        (x) |= ((typeof(x))1 << (n));                   \
} while (0)
```

| 返回值 | 含义 |
|--------|------|
| 1 | 编译期已知 |
| 0 | 运行时变量 |

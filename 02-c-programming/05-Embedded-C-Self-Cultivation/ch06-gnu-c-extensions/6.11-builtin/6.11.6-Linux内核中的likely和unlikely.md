# 6.11.6 Linux内核中的likely和unlikely

## 本节讲什么

内核 **`likely`/`unlikely`** 包装 **`__builtin_expect`**，标记 **错误路径 rare**。

## 要点

```c
if (unlikely(!ptr))
    return -EINVAL;
```

```c
/* DPDK — rte_branch_prediction.h */
if (likely(nb_rx > 0)) { /* burst */ }
```

- **不要**在用户态盲目复制：profile 为准

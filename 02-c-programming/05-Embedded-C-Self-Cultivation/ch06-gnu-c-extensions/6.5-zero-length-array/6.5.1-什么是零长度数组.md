# 6.5.1 什么是零长度数组

## 本节讲什么

零长度数组是 **不完整类型数组** 放在结构体末尾的 GNU 写法；C99 标准化为 **flexible array member**（`[]` 无长度）。语义：结构体是 **头部**，实际对象大小 = 头 + 变长尾部。

## 要点

```c
/* GNU */
struct blob_gnu {
    int count;
    char items[0];
};

/* C99 FAM — 推荐新代码 */
struct blob_std {
    int count;
    char items[];
};
```

| 特性 | 说明 |
|------|------|
| **sizeof** | 仅头部，不含 FAM |
| **位置** | 必须是最后一个成员 |
| **C99 前** | ISO 不允许 `[0]`；GNU 扩展 |
| **C11** | FAM 正式入标准 |

```c
/* 非法：中间零长度 */
/* struct bad { char a[0]; int b; }; */
```

- 与 **指针成员** `char *items` 对比：FAM 与头 **同块连续**，缓存友好、一次 free
- **VLA（变长数组）** 不同：在栈上，FAM 在堆/静态自定义大小

# 6.5 零长度数组

## 本节讲什么

**零长度数组**（GNU：`char data[0]`）与 C99 **灵活数组成员（FAM）** `char data[]` 允许结构体 **末尾变长 payload**，一次 `malloc(sizeof(hdr)+payload_len)` 分配。内核 skbuff、NVMe 命令、DPDK mbuf 扩展区都属此类模式。

## 要点

```c
struct pkt_hdr {
    uint32_t len;
    uint8_t  data[0];   /* GNU 零长度；C99 写 data[] */
};
```

- 只能作 **结构体最后成员**
-  sizeof(struct pkt_hdr) **不含** data 占用
- 分配：`malloc(sizeof(*h) + payload_len)`

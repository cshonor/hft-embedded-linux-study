# 6.5.2 零长度数组使用示例

## 本节讲什么

演示 **动态分配**、**访问尾部数据**、**释放** 完整流程；对应 **demo05** 可编译验证 `sizeof` 与布局。

## 要点

```c
#include <stdlib.h>
#include <string.h>
#include <stdint.h>

struct flex_msg {
    uint32_t id;
    uint32_t len;
    uint8_t  payload[0];
};

struct flex_msg *flex_msg_new(uint32_t id, const void *data, uint32_t len)
{
    struct flex_msg *m = malloc(sizeof(*m) + len);
    if (!m)
        return NULL;
    m->id = id;
    m->len = len;
    memcpy(m->payload, data, len);
    return m;
}

void flex_msg_free(struct flex_msg *m)
{
    free(m);
}
```

```c
/* 多个变长段 — 内核常单 FAM；用户态可嵌套设计 */
struct nvme_cmd {
    uint8_t  opcode;
    uint16_t flags;
    uint8_t  cdw[0];   /* 后续 cdw 数量由 flags 决定 */
};
```

```bash
gcc -std=gnu11 -Wall -o demo05 demo05_flex_array.c
./demo05
# 观察 sizeof(struct flex_msg) == 8（32-bit 示例，依对齐而定）
```

| 操作 | 注意 |
|------|------|
| `malloc(sizeof(hdr)+n)` | n 为 payload 字节 |
| 拷贝 | `payload` 与头 contiguous |
| 越界 | 无编译期边界，需 `len` 字段约束 |

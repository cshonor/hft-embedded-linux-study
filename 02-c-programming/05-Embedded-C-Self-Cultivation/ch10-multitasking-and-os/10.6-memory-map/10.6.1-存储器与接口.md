# 10.6.1 存储器与接口

## 本节讲什么

嵌入式系统常用 **哪些存储介质**、通过 **什么总线/接口** 连接 CPU？本节梳理 **SRAM、Flash、eMMC、DDR** 与 **SPI/I2C/并行** 接口，帮助理解 **容量、速度、启动角色** 的差异。

## 要点

**片上 vs 片外**：

| 存储 | 位置 | 特点 |
|------|------|------|
| **SRAM** | 片内/片外 | 快、贵、掉电失 |
| **NOR Flash** | 片内/片外 | 可 XIP、按扇区擦写 |
| **NAND Flash** | 片外 | 大容量、块擦除、需 FS |
| **eMMC/SD** | 片外 | 块设备，Linux 根 FS 常见 |
| **DDR SDRAM** | 片外 | 大 RAM，A 核 Linux 必备 |

**接口协议**：

```
CPU ──AHB/AXI──► 矩阵 ──┬── SRAM 控制器
                        ├── QSPI Flash 控制器
                        ├── FMC 并行 NOR
                        └── SDIO/eMMC 控制器
```

- **QSPI**：4 线 SPI，Flash 代码 XIP 常见
- **FMC**：8080/6800 并行，老 NOR/ LCD
- **I2C/SPI**：小容量 EEPROM、传感器（非主存储）

**Boot 介质选型**：

| 方案 | 适用 |
|------|------|
| 片内 Flash | Cortex-M 量产 |
| 外部 NOR XIP | 中等代码量 |
| NAND/eMMC | Linux 大 rootfs |
| SD 卡 | 开发调试 |

**时序与 wait state**：

- Flash 读慢于 SRAM → 配置 **等待周期** 或 **Prefetch**
- 错误 wait → 取指异常或性能差

**与 **10.7** 分界**：

- **存储器控制器** 管 bulk 读写
- **外设寄存器** 管 UART/GPIO 控制（同属地址空间不同区域）

# 10.7 内存与外部设备

## 本节讲什么

CPU 通过 **内存总线** 访问 **RAM 与外设**；理解 **内存层次、设备分类、MMIO** 是驱动开发与 **10.8 寄存器操作** 的前置。本节总览 **10.7.1–10.7.3**，连接 **RTOS 驱动** 与 **Linux 设备模型**。

## 要点

**内存层次**（**10.7.1**）：

```
寄存器 ──► SRAM ──► DRAM ──► eMMC/Flash（外存）
  最快      快       较大      持久、慢
```

**外部设备**（**10.7.2**）：

- **字符设备**：UART 字节流
- **块设备**：eMMC 扇区
- **网络设备**：MAC/PHY
- **GPIO/ADC/PWM**：控制与采样

**访问方式**：

| 环境 | 访问外设 |
|------|----------|
| 裸机/RTOS | 直接 `*(volatile *)REG` |
| Linux 内核 | `readl/writel(ioremap)` |
| Linux 用户 | `read/write/ioctl` 设备节点 |

**与多任务/中断**：

- 外设 **中断** 唤醒任务（**10.3**）
- 多任务共享 UART → **mutex**（**10.2.4**、**demo04**）

**DPDK 旁路**：

- **UIO/VFIO** 把 NIC 寄存器映射到用户态；**lcore 轮询** 不用中断驱动 RX

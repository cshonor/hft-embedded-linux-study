# 10.7.1 内存与外存

## 本节讲什么

**内存（RAM）** 供程序运行时读写；**外存** 持久保存代码与数据。本节说明 **掉电行为、带宽、在嵌入式中的分工**，以及 **为何不能把 Flash 当 RAM 随便写**。

## 要点

**对比**：

| | RAM | 外存（Flash/eMMC） |
|---|-----|---------------------|
| 速度 | ns 级 | μs～ms（擦写更慢） |
| 掉电 | 丢失 | 保留 |
| 用途 | 栈、堆、缓冲 | 代码、文件、配置 |
| 写入 | 任意字节 | 扇区/页擦除规则 |

**MCU 内存预算**（多任务相关）：

```
Flash:  code + const
RAM:    .data + .bss + heap + 各任务栈 + 内核TCB
```

- **heap**：`malloc` 与 **RTOS 堆** 竞争；多任务需 **mutex 保护**（**10.2.3**）
- **栈溢出** 进 heap → 隐蔽故障（**10.2.2**）

**外存上的内容**：

- **代码段**：XIP 或加载到 RAM
- **文件系统**：配置、日志、OTA 包（**10.5**）
- **裸分区**：Bootloader env、出厂校准

**Cache 与一致性**：

- CPU 写 RAM 可能只在 cache；DMA 读 RAM → **cache invalidate**
- 外存经 **控制器 DMA** 到 RAM，再被任务处理（**queue** 传递）

**Wear leveling**：

- NAND/eMMC 不能无限次写同一逻辑块；FS（UBIFS）或 FTL 负责

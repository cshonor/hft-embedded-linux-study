# 10.8 寄存器操作

## 本节讲什么

嵌入式驱动核心技能：**读改写硬件寄存器**。本节总览 **位运算、MMIO 读写、位域与手册对照**，对应 **10.8.1–10.8.3** 与 **ch06 位操作**、**10.7.3 MMIO**。

## 要点

**寄存器操作三部曲**：

1. **读** 当前值（若需保留其他位）
2. **改** 目标位（OR/AND/XOR）
3. **写** 回寄存器

```c
uint32_t cr = UART->CR;
cr |= UART_CR_TX_EN;
cr &= ~UART_CR_RX_EN;
UART->CR = cr;
```

**头文件组织**（**ch09**）：

```c
/* hw_uart_regs.h — 偏移与位掩码，无逻辑 */
#define UART_CR_TX_EN  (1u << 3)
#define UART_SR_RXNE   (1u << 5)
```

**读清 vs 写 1 清**：

- 手册必须看清：**读 DR 清 RXNE** vs **写 1 清中断标志**
- 写错 → 丢数据或中断风暴

**原子性与并发**：

- 单字段 **RMW** 非原子 → 多任务/ISR 同时改 → **关中断或 mutex** 保护同一寄存器
- 若硬件提供 **SET/CLR 寄存器**（写 1 置位/清位）→ 避免 RMW

**与 Linux 内核**：

- `readl`/`writel` + `regmap` 抽象
- 用户态通过 **ioctl** 间接配置，不直接摸 MMIO（安全）

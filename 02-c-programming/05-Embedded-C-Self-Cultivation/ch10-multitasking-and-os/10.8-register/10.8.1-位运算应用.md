# 10.8.1 位运算应用

## 本节讲什么

寄存器每一位有 **独立含义**；用 **位掩码、移位、与或非** 高效置位/清位/提取字段。本节衔接 **ch06 GNU C 扩展** 与 **ch03 逻辑运算**，给出可复用的宏模式。

## 要点

**常用宏**：

```c
#define BIT(n)          (1u << (n))
#define BITS(h, l)      (((1u << ((h)-(l)+1)) - 1u) << (l))
#define REG_SET(r, m)   ((r) |= (m))
#define REG_CLR(r, m)   ((r) &= ~(m))
#define REG_TOGGLE(r,m) ((r) ^= (m))
#define FIELD_GET(v,h,l) (((v) >> (l)) & BITS((h)-(l),0))
#define FIELD_PREP(val,h,l) (((val) << (l)) & BITS(h,l))
```

**示例：配置 GPIO 模式**：

```c
/* MODER: 每 pin 2 bit，pin5 设为 AF(10) */
uint32_t moder = GPIO->MODER;
moder &= ~(3u << 10);      /* 清 pin5 */
moder |=  (2u << 10);      /* AF 模式 */
GPIO->MODER = moder;
```

**提取状态**：

```c
if (UART->SR & UART_SR_RXNE)
    byte = UART->DR;
```

**注意符号扩展**：

```c
/* 坏：1<<31 在 32 位 int 未定义 */
#define BIT31  (1u << 31)   /* 用 unsigned */
```

**GCC 内置**：

```c
__builtin_popcount(flags);
__builtin_clz(ready_bitmap);   /* 10.2.1 调度 */
```

**与可移植性**：

- 宏放 **hw_*_regs.h**；逻辑放 **driver .c**
- 大端/小端：MMIO 通常 **CPU 字节序**；跨设备协议另论

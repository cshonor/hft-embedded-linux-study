# 10.10.5 小结

## 本节讲什么

**第 10 章 capstone 小结**：从 **裸机前后台** 到 **迷你 RTOS**、再到 **Linux 进程/线程/MMU/syscall**，串起全书嵌入式 C 主线，归纳 **选型、陷阱与 demo 对应**，作为 **《嵌入式 C 内功修炼》** 终章收束。

## 要点

**知识主线回顾**：

```
10.1 裸机多任务（并发）
  → 10.2 OS/RTOS（调度、栈、同步）
  → 10.3 中断（抢占、ISR）
  → 10.4–10.5 syscall/FS（Linux 用户态）
  → 10.6–10.8 存储/MMIO/寄存器（硬件）
  → 10.9 MMU（隔离）
  → 10.10 进程/线程/协程（Linux 抽象）
```

**Demo 与章节映射**：

| Demo | 核心能力 |
|------|----------|
| **demo01_foreground** | 前后台 vs 饿死 |
| **demo02_tcb_coop** | TCB、独立栈、协作调度 |
| **demo03_preempt** | SysTick、上下文切换（setjmp/ARM） |
| **demo04_semaphore** | 同步、阻塞、优先级 |
| **demo05_queue** | 任务间传数据、线程池雏形 |
| **demo06_mini_rtos** | **ch08 TCB OOP + ch09 分层** 终章项目 |

**前置章节如何支撑终章**：

| 章节 | 在 ch10 中的体现 |
|------|------------------|
| **ch03** | ARM `stmfd/ldmfd` 上下文切换 |
| **ch05** | 栈布局、canary 溢出检测 |
| **ch08** | TCB 不透明类型、ops |
| **ch09** | kernel/sync/port/bsp 目录与 Makefile |

**RTOS vs Linux 选型**：

| 维度 | RTOS（FreeRTOS/uC/OS） | Linux |
|------|------------------------|-------|
| 实时性 | μs 级可预期 | ms 级，需 RT patch |
| 内存 | tens～ hundreds KB | MB 级 |
| 隔离 | 弱（同空间） | 强（MMU） |
| 生态 | 驱动 HAL | VFS、网络、容器 |
| 典型 | 传感器 MCU | 网关、MPU |

**多任务四大陷阱**（务必 checklist）：

1. **栈溢出** → 独立栈 + canary + Hook（**10.2.2**）
2. **未保护全局** → mutex/queue（**10.2.4**）
3. **ISR 太长/不可重入** → defer + FromISR（**10.3.3**）
4. **饿死/优先级反转** → 抢占、继承 mutex、合理优先级（**10.2.1**）

**延伸方向**（全书之后）：

- **Linux 内核**：进程调度、驱动、DPDK 用户态驱动
- **AIoT**：MCU RTOS 采集 + Linux 边侧汇聚（**ch09.9**）
- **Rust/Modern C** 在 embedded 的并发安全抽象

# 1.1.4 Vim 的按键映射

## 本节讲什么

**map / imap / nmap** 自定义快捷键 —— 可选，提升重复操作效率。

## 要点

```vim
" ~/.vimrc
nnoremap <F5> :make<CR>
nnoremap <F6> :!./a.out<CR>
inoremap jj <Esc>
```

| 前缀 | 模式 |
|------|------|
| **nmap** | 普通模式 |
| **imap** | 插入模式 |
| **nore** | 不递归映射 |

- 嵌入式课以**默认键位**为主；映射勿与插件冲突

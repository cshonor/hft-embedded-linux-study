# 1.1.3 Vim 配置文件：vimrc

## 本节讲什么

**`~/.vimrc`** 基础配置 —— C 语法高亮、缩进、行号。

## 要点

```vim
syntax on
filetype plugin indent on
set number
set tabstop=4 shiftwidth=4 expandtab
set autoindent
set hlsearch
set incsearch
colorscheme desert
```

| 选项 | 作用 |
|------|------|
| **expandtab** | Tab → 空格，内核风格 |
| **shiftwidth=4** | C 缩进 4 格 |
| **hlsearch** | 搜索高亮 |

```bash
vim ~/.vimrc
```

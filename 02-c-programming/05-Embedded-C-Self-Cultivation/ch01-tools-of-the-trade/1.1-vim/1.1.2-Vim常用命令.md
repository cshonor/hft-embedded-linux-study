# 1.1.2 Vim 常用命令

## 本节讲什么

Vim **模式切换**、移动、编辑、批量操作 —— 纯键盘编辑 C 源码。

## 要点

| 操作 | 命令 |
|------|------|
| 插入 | `i` `a` `o` |
| 退出插入 | `Esc` |
| 保存退出 | `:wq` |
| 不保存退出 | `:q!` |
| 跳转 | `gg` 文首 `G` 文末 `42G` 第 42 行 |
| 删除行 | `dd` |
| 撤销 | `u` |
| 搜索 | `/word` `n` 下一个 |
| 替换 | `:%s/old/new/g` |
| 批量注释 | `Ctrl+v` 块选 → `I//` → `Esc` |

### 与 Shell 配合

```bash
vim main.c
grep -n "error" main.c
```

# 1.1 代码编辑工具：Vim

## 本节讲什么

**Linux Shell + Vim** —— 嵌入式标准开发方式：脱离 IDE，命令行可见、可追溯。

## 要点

### Shell 基础（WSL/Linux）

| 类别 | 命令 |
|------|------|
| 文件 | `cd` `ls` `mkdir` `cp` `rm` `find` `tree` |
| 文本 | `cat` `grep` `sed` `awk` |
| 进程 | `ps` `top` `kill` |
| 权限 | `chmod` `ln -s` |

```bash
grep -rn "main" *.c
find . -name "*.o" -delete
```

- **bash 脚本**为 **1.6 buildenv** 铺垫

### Vim 定位

- 服务器/板卡无 GUI IDE 时的默认编辑器
- 三种模式：**普通 / 插入 / 命令**

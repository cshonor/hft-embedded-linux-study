# 1.1.1 安装 Vim

## 本节讲什么

在 **WSL2 / Linux** 安装 Vim 及 C 开发依赖包。

## 要点

```bash
# Debian/Ubuntu / WSL
sudo apt update
sudo apt install -y vim gcc gdb make cmake git binutils \
    build-essential file hexdump
```

| 包 | 用途 |
|----|------|
| **vim** | 编辑 |
| **gcc binutils** | 编译、objdump/readelf |
| **gdb** | 调试 |
| **make cmake** | 构建 |

### 验证

```bash
vim --version | head -1
gcc --version
```

- Windows 原生开发推荐 **WSL2**，与板端 Linux 工具链一致

# 1.4 ELF 二进制分析工具

## 本节讲什么

**objdump / readelf / nm / size / hexdump** —— 查看段布局、反汇编、符号表；ch04/ch05 内存分析前置。

## 要点

### objdump

```bash
objdump -dS app          # 反汇编 + 源码（需 -g）
objdump -h app           # 段头摘要
objdump -t app           # 符号表
```

### readelf

```bash
readelf -h app           # ELF 头
readelf -S app           # 节 .text .rodata .data .bss
readelf -l app           # 程序头 / LOAD 段
```

### nm / size

```bash
nm app | grep main
size app                 # text data bss 十进制
```

### 原始字节

```bash
hexdump -C app | head
xxd app | head
```

| 工具 | 典型问题 |
|------|----------|
| **readelf -S** | 各段多大？ |
| **objdump -d** | 函数对应哪些指令？ |
| **nm** | 全局变量地址？ |

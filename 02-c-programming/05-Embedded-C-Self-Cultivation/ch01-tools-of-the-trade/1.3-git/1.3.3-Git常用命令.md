# 1.3.3 Git 常用命令

## 本节讲什么

嵌入式工程 **Git 日常流程** 与 **.gitignore** 模板。

## 要点

```bash
git init
git add main.c Makefile
git commit -m "ch01: add gcc demo"
git branch feature-gdb
git checkout feature-gdb   # 或 git switch
git merge main
git log --oneline
git diff
```

### .gitignore（嵌入式）

```gitignore
*.o *.a *.so
*.i *.s
app app.arm
build/
core
*.swp
.vscode/
```

| 忽略 | 原因 |
|------|------|
| ***.o / build/** | 可再编译 |
| **core** | gdb 转储 |

### 冲突

```bash
git status
# 编辑冲突文件 → git add → git commit
```

# 1.2.4 CMake 基础

## 本节讲什么

**CMakeLists.txt** 生成 Makefile —— 跨平台、大型工程标准入口。

## 要点

```cmake
cmake_minimum_required(VERSION 3.16)
project(demo C)

set(CMAKE_C_STANDARD 11)
add_compile_options(-Wall -Wextra -g)

add_executable(app main.c util.c)
```

```bash
mkdir -p build && cd build
cmake ..
make
```

| 步骤 | 作用 |
|------|------|
| **cmake ..** | 生成构建文件 |
| **make** | 编译 |

- 交叉编译：`cmake -DCMAKE_C_COMPILER=arm-linux-gnueabihf-gcc ..`  
- 产物放 **build/**，源码树干净  

## 与后续衔接

DPDK/内核用自有构建系统；CMake 用于用户态练习工程。

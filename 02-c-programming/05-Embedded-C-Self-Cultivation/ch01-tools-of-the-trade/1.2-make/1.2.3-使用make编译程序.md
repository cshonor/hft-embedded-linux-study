# 1.2.3 使用 make 编译程序

## 本节讲什么

**Makefile** 语法：目标、依赖、自动变量 —— 多文件工程模板。

## 要点

```makefile
CC      ?= gcc
CFLAGS  ?= -Wall -Wextra -std=c11 -g
LDFLAGS ?=

app: main.o util.o
	$(CC) $(LDFLAGS) -o $@ $^

main.o: main.c util.h
	$(CC) $(CFLAGS) -c $< -o $@

util.o: util.c util.h
	$(CC) $(CFLAGS) -c $< -o $@

.PHONY: clean
clean:
	rm -f *.o app
```

| 自动变量 | 含义 |
|----------|------|
| **$@** | 目标 |
| **$<** | 首依赖 |
| **$^** | 全部依赖 |

```bash
make all
make clean
```

- **.PHONY`** 声明伪目标  
- 大型工程见 **1.2.4 CMake**

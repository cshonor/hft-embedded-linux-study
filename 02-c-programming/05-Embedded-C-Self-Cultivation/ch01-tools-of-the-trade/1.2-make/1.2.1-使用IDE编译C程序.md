# 1.2.1 使用 IDE 编译 C 程序

## 本节讲什么

对比 **IDE 一键编译**与**命令行工具链** —— 底层开发必须看见每一步。

## 要点

| IDE | 命令行 |
|-----|--------|
| 隐藏预处理/链接 | **gcc -E/-S/-c** 可拆解 |
| 难复现 CI/板端 | 脚本/Makefile 一致 |
| 绑定 GUI | SSH 到板卡只有 Shell |

### 底层思维

- 编译 = **预处理 → 编译 → 汇编 → 链接**（**1.2.2**）  
- 产物 = **ELF**（**1.4** readelf/objdump）  
- 崩溃 = **gdb + core**（**1.5**）

# 7.9.3 指针数组与数组指针

## 本节讲什么

**指针数组**：元素是指针，如 `char *argv[]`；**数组指针**：指向整个数组，如 `char (*row)[cols]`。声明差一个括号，语义截然不同。`main` 的 `argv`、`char **` 字符串列表与二维字符表都建立在此区分之上。

## 要点

```c
char *strs[4];           /* 4 个 char *，指针数组 */
char (*prow)[16];        /* 指向「含 16 char 的数组」，数组指针 */

int *parr[8];            /* 8 个 int * */
int (*pmat)[8];          /* 指向含 8 int 的一行 */
```

| 声明 | 类型 | 常见用途 |
|------|------|----------|
| `T *a[N]` | 指针数组 | `argv`、函数表 |
| `T (*p)[N]` | 数组指针 | 二维数组行指针 |
| `T **` | 指针的指针 | 字符串列表 → **demo02** |

```c
char *devices[] = { "uart0", "spi1", "i2c2" };
char (*matrix)[4] = (char (*)[4])buf;  /* 3x4 逻辑视图 */

/* 函数指针数组 → demo04 */
calc_fn ops[] = { add, sub, mul };
```

```c
/* 错误混淆 */
char *p[10];    /* 10 个指针 */
char (*p)[10];  /* 1 个指针，指向 10 字符数组 */
```

- **typedef**：`typedef char *str_t; str_t tab[10];` → 指针数组
- **sizeof**：`strs` 为 `4 * sizeof(char*)`
- **内核** `const char * const argv[]` 双重 const

# 7.9.1 下标运算符[]

## 本节讲什么

`a[i]` 等价于 `*(a + i)`：下标运算符以指针运算定义，不依赖「数组」语法。因此指针与数组都能用 `[]`；编译器根据左操作数类型计算偏移。理解等价关系是分析缓冲区越界与指针算术的基础。

## 要点

```c
int a[5] = {0, 10, 20, 30, 40};
int *p = a;

a[2] == *(a + 2) == p[2] == *(p + 2);  /* 20 */

/* 甚至 */
2[a] == a[2];   /* *(2 + a) 合法但勿写 */
```

| 左操作数 | `a[i]` 含义 |
|----------|-------------|
| 数组名（decay） | 第 i 个元素 |
| 指针 | 从所指位置偏移 i |
| `struct *` | 少见，通常用 `->` |

```c
void fill(int *dst, size_t n, int val) {
    for (size_t i = 0; i < n; i++)
        dst[i] = val;   /* 指针下标 */
}
```

```gdb
(gdb) print a[3]
(gdb) print *(a+3)
(gdb) set a[0]=99
```

- **越界**：C 不检查，`a[5]` UB
- **负下标**：`p[-1]` 合法若指向非首元素
- **多维**：`m[i][j]` → `*(*(m+i)+j)` → **7.11.3**

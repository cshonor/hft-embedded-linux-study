# 7.7 常量和变量

## 本节讲什么

**变量**是可修改的存储对象；**常量**在编译期或运行期不可（或不应）修改。`const`、`volatile`、`restrict` 修饰符精确表达意图，是驱动、MMIO、并发代码的类型契约。本节总览常量与变量的分类，为 **7.7.1–7.7.3** 与指针 const 四形式铺路。

## 要点

```c
const int max_users = 32;      /* 只读变量，可能在 .rodata */
#define BUF_SIZE 256           /* 预处理替换，无类型 */
enum { TIMEOUT_MS = 1000 };

int counter;                   /* 可写变量 */
static const char msg[] = "ok"; /* 字符串字面量合并 */
```

| 修饰符 | 作用 |
|--------|------|
| `const` | 语义只读，可能放只读段 |
| `volatile` | 每次从内存读/写，禁止优化缓存 → **demo05** |
| `restrict` | 指针独占别名，C99 优化提示 → **7.13** |

**const 指针四形式**（本章必须掌握）：

```c
const int *p1;        /* 指向 const int：*p1 不可改 */
int *const p2 = &x;   /* const 指针：p2 不可改 */
const int *const p3;  /* 都不可改 */
int const *p4;        /* 同 p1 */
```

```c
/* MMIO：volatile + 指针 */
volatile uint32_t *reg = (volatile uint32_t *)0x40001000;
*reg = 1;   /* 每次真实写硬件 */
```

# 7.7.2 常量存储

## 本节讲什么

常量数据可存放在**只读段（.rodata）**、与代码段相邻的 literal pool，或作为立即数编进指令。`const` 全局/静态常量在嵌入式中可链接到 Flash，节省 RAM。字符串字面量、查找表、设备 ID 表是典型常量对象。

## 要点

```c
const int table[] = { 1, 2, 3 };     /* .rodata */
static const char name[] = "uart0";

#define GPIO_BASE 0x40000000         /* 预处理，不占存储 */

const int *p = table;                /* p 可变，*p 不可通过 p 改 */
```

| 形式 | 存储行为 |
|------|----------|
| `const` 全局 | `.rodata`，写保护（MMU 可设只读） |
| 字符串字面量 | 通常 `.rodata`，合并相同串 |
| 局部 `const` | 可能在栈，仍不可写 |
| `enum` / `#define` | 常出现在指令立即数 |

```c
/* 错误：修改字面量（C 标准未定义，常见 SIGSEGV） */
char *s = "hello";
s[0] = 'H';

/* 正确 */
const char *s = "hello";
char buf[] = "hello";  /* 可写数组副本 */
```

```bash
readelf -S a.out | grep rodata
objdump -s -j .rodata a.out
```

- **嵌入式链接脚本**：`.rodata` 映射到 Flash
- **内核 `__initconst`**：初始化后只读数据可释放
- **const 指针四形式** 决定能否改指向与内容 → **7.7**

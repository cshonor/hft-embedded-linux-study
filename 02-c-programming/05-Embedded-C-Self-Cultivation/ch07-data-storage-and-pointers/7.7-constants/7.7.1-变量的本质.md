# 7.7.1 变量的本质

## 本节讲什么

变量是**具名存储对象**：编译器为其分配地址（栈、静态区或寄存器），生命周期由作用域与链接属性决定。理解「变量 = 名字 + 存储期 + 类型 + 地址」是读懂指针的基础——指针里保存的就是这类对象的地址。

## 要点

```c
void demo(void) {
    int a = 10;           /* 自动变量，栈，块作用域 */
    static int call_count; /* 静态存储，程序寿命 */
    register int hint;    /* 寄存器提示，现代编译器常忽略 */
}
int g;                    /* 全局，.bss 或 .data */
```

| 存储类 | 生命周期 | 默认初始化 |
|--------|----------|------------|
| 自动 | 进入/离开块 | 未定义 |
| 静态 | 整个程序 | 零 |
| 线程局部 `_Thread_local` | 线程 | 实现 |
| 堆 | `malloc` 到 `free` | 未定义 |

```bash
# 查看符号放在哪一段
gcc -c file.c -o file.o
nm file.o
readelf -s file.o | grep -E ' g | a '
```

```gdb
(gdb) break demo
(gdb) run
(gdb) print &a
(gdb) info locals
(gdb) x/wx &a          # 查看内存字
```

- **未初始化自动变量**：读值 UB，安全代码显式赋值
- **别名**：同一内存可有多个名字（union、指针）
- **优化**：寄存器变量无固定地址，`&a` 可能迫使 spill 到栈

# 7.5.1 typedef的基本用法

## 本节讲什么

`typedef old_type new_name;` 声明**类型别名**；`new_name` 之后可当类型使用。可与 `struct`、指针、数组、函数指针组合。读内核代码时，能快速展开 typedef 是理解 `DEFINE_SPINLOCK`、`list_for_each_entry` 的前提。

## 要点

```c
/* 基本类型 */
typedef unsigned char byte;

/* 结构体 */
typedef struct node {
    int data;
    struct node *next;
} node_t;

/* 指针 */
typedef int *intptr_alias;   /* intptr_alias 是指针类型，不是 int 的指针变量 */

/* 数组 */
typedef int row_t[4];
row_t matrix;   /* int matrix[4] */

/* 函数指针 — 读法：typedef 返回 (*NAME)(参数) */
typedef int (*signal_handler_t)(int signo);
```

**声明对比**：

```c
int *a, b;           /* a 是指针，b 是 int */
typedef int *ptr_t;
ptr_t c, d;          /* c、d 都是 int * */
```

```c
/* 内核链表 */
struct list_head {
    struct list_head *next, *prev;
};
```

- **多次 typedef**：链式别名 `typedef u8 byte; typedef byte octet;` 可读性下降
- **与 `static`/存储类**：`typedef` 只作用于类型，不含存储期
- **函数指针 typedef** 是 **7.12** / **demo04** 推荐写法

# 7.11.3 二级指针和二维数组

## 本节讲什么

二维数组 `T[row][col]` 在内存中**行优先连续**；数组名 decay 为「指向含 `col` 个 T 的数组的指针」`T (*)[col]`，**不是** `T **`。用 `T **` 接收二维数组或把 `T **` 当矩阵会类型不匹配。模拟二维有时用「指针数组 + 每行 malloc」。

## 要点

```c
int m[3][4] = {0};
int (*p)[4] = m;        /* 指向一行 4 个 int */
int *q = m[0];          /* 首行首元素 */

m[1][2] == *(*(m + 1) + 2);
```

| 类型 | 步长（+1） |
|------|------------|
| `int (*)[4]` | 4 * sizeof(int) 一行 |
| `int *` | sizeof(int) 一元素 |
| `int **` | sizeof(int*) 一个行指针 |

```c
/* 错误：把二维数组传给 char ** */
void print(char **rows);   /* 期望指针数组 */
/* print((char **)matrix);  类型错误/需非自然转换 */

/* 模拟二维：指针数组 */
char **rows = malloc(nrows * sizeof *rows);
for (int i = 0; i < nrows; i++)
    rows[i] = malloc(ncols);
```

```c
/* 单块连续「二维」 */
int *flat = malloc(rows * cols * sizeof *flat);
#define AT(r, c) flat[(r) * cols + (c)]
```

- **VLA**：`int (*fp)[n]` 指向 VLA 行
- **内核** 多维静态表多用一维 + 宏下标
- **DPDK** 多维 lookup table 常扁平化

# 7.11.1 修改指针变量的值

## 本节讲什么

C 按值传递指针时，函数内改**形参指针本身**不影响实参；要让 `malloc`、`realloc` 结果回到调用方，须传**指针的地址**，即二级指针（或返回新指针）。这是资源管理与容器 API 的标准模式。

## 要点

```c
void bad(char *p) {
    p = malloc(100);   /* 只改副本 */
}

void good(char **pp) {
    *pp = malloc(100); /* 改调用方 char * 变量 */
}

char *buf = NULL;
good(&buf);
```

```c
/* realloc 须更新指针 */
int grow(char **list, size_t *cap, size_t need) {
    char **new_list = realloc(*list, need);
    if (!new_list) return -1;
    *list = new_list;
    *cap = need;
    return 0;
}
```

| 需求 | 传参 |
|------|------|
| 改所指对象 | `T *` |
| 改指针变量 | `T **` |
| 改 int 变量 | `int *` |

```c
/* demo02 append_string */
int append_string(char ***list, size_t *count, const char *s);
/* 实现里 *list = realloc(...); 调用 append_string(&argv_like, &n, "uart0"); */
```

- **检查 NULL**：解引用 `*pp` 前确认 `pp != NULL`
- **对称释放**：谁 `malloc` 谁 `free`，列表需逐元素释放
- **三级指针**：极少见，多为通用容器泛型（void ***）

# 7.11.2 二维指针和指针数组

## 本节讲什么

`char **` 常表示「元素为 `char *` 的指针数组」：每元素指向独立字符串，行不必连续，长度可不等。与 `char *matrix[]` 静态表、`main` 的 `argv` 同构。**demo02** 动态构建 `uart0/spi1/i2c2` 列表是典型应用。

## 要点

```
char **argv 布局:

 argv ──→ [ ptr0 ] ──→ "uart0\0"
          [ ptr1 ] ──→ "spi1\0"
          [ ptr2 ] ──→ "i2c2\0"

 顶层是指针数组；每条字符串单独分配。
```

```c
char *names[] = { "uart0", "spi1", "i2c2" };  /* 指针数组，静态 */
char **dyn = NULL;   /* 动态版 demo02 */

for (size_t i = 0; i < n; i++)
    printf("[%zu] %s\n", i, dyn[i]);
```

| 对比 | `char **` | `char (*)[N]` |
|------|-----------|----------------|
| 行连续 | 否 | 是（二维数组） |
| 行长度 | 可不等 | 固定 N |
| 传参 | `char **` | `char (*)[N]` 或 decay |

```c
/* 内核模块参数（示意） */
static char *board_parts[] = { "mmc", "spi", NULL };
```

- **释放顺序**：先 `free` 每行，再 `free` 指针数组
- **`const char **` vs `char * const *`**：常量性位置影响接口契约
- **NULL 结尾**：模仿 `argv` 便于遍历

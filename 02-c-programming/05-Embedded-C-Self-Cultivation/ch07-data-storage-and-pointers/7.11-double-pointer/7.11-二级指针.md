# 7.11 二级指针

## 本节讲什么

**二级指针** `T **` 保存「指针变量的地址」，用于在函数内**修改调用方的指针**、构建指针数组、表达「字符串列表」或「行指针表」。与「指向数组的指针」、二维数组名不同，类型系统上必须逐级匹配。

## 要点

```c
char *s = "uart";
char **pp = &s;       /* pp 指向 char * 变量 s */

void alloc_buf(char **out, size_t n) {
    *out = malloc(n);   /* 改调用方的指针 */
}
```

| 类型 | 含义 |
|------|------|
| `T *` | 指向 T |
| `T **` | 指向「T 的指针」 |
| `T *a[N]` | N 个 T*，数组对象 |

```c
/* demo02 模式 */
char **list = NULL;
size_t count = 0;
append_string(&list, &count, "uart0");  /* &list 是 char *** 语义上传递二级 */
```

```bash
gcc demo02_double_ptr.c -o demo02 && ./demo02
```

- **`int **` 不是二维数组** `int[m][n]` 的传参类型**
- **`realloc` 改指针**：须 `char **` 让调用方看到新地址
- **NULL**：空列表常 `char **p = NULL`

# 7.6.1 使用枚举的三种方法

## 本节讲什么

实践中枚举有三种常见写法：**具名枚举类型**、**匿名枚举常量**、**typedef 包装枚举**。各自适用于状态机类型安全、头文件常量集合、驱动 API 导出。选对写法可减少与整数混用的警告。

## 要点

**方法一：具名 enum 类型**

```c
enum drv_state { DRV_IDLE, DRV_ACTIVE, DRV_ERROR };
enum drv_state st = DRV_IDLE;
void set_state(enum drv_state s);
```

**方法二：匿名 enum（仅要常量）**

```c
enum {
    REG_CTRL = 0x00,
    REG_STATUS = 0x04,
    REG_FIFO = 0x08,
};
```

**方法三：typedef 包装**

```c
typedef enum {
    OP_ADD, OP_SUB, OP_MUL, OP_DIV, OP_COUNT
} op_t;

op_t op = OP_ADD;   /* 少写 enum 关键字 */
```

| 方法 | 适用 |
|------|------|
| 具名 enum | 需要类型区分、函数参数 |
| 匿名 enum | MMIO 偏移、位标志常量 |
| typedef enum | 公共 API、**demo04** |

```c
/* 内核设备 PM 状态（示意） */
typedef enum {
    PM_SUSPEND_PREPARE,
    PM_SUSPEND,
    PM_RESUME,
} pm_message_t;
```

# 7.6.2 枚举的本质

## 本节讲什么

枚举常量在 C 中本质是**整数**，可与 `int` 互相赋值（隐式转换），比较、算术、用作数组下标均可。编译器可为枚举定义**不完全兼容**的其他整数类型检查，但不可依赖枚举阻止所有误用。理解「名义类型、实质整数」有助于设计状态机。

## 要点

```c
enum { A = 0, B = 1, C = 2 };
int x = B;
enum { A, B, C };
int arr[] = { [A] = 10, [B] = 20 };   /* 指定初始化 ch06 */

enum color { RED = 1, GREEN, BLUE };  /* GREEN=2, BLUE=3 */
```

| 特性 | 说明 |
|------|------|
| 默认值 | 首项 0，后续 +1（除非显式赋值） |
| 重复值 | 允许，用于别名 |
| 范围 | 须在 `int` 可表示范围（实现） |
| sizeof(enum) | 常为 `int` 大小 |

```c
/* 误用：超出枚举定义的值 */
enum state { ON, OFF };
enum state s = 99;   /* 往往仅警告，运行逻辑错误 */

if (s == 99) { }    /* 合法比较，说明 enum 非真正封闭 */
```

- **位标志**：大集合用 `#define` 或 `unsigned` + 宏，单选状态用 enum
- **序列化**：存盘/网络用定宽整数，不假设 enum 宽度
- **C++ enum class**：强类型；纯 C 工程用约定 + 审查

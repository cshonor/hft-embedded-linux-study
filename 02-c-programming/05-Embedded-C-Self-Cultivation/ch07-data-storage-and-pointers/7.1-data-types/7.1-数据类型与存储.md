# 7.1 数据类型与存储

## 本节讲什么

本章从**数据在内存中的表示**出发，建立嵌入式与内核编程的底层视角。本节总览 C 基本类型、`sizeof`、存储类别（栈/静态/堆），为后续大小端、溢出、对齐与指针运算打基础。读内核头文件或 DPDK `rte_*` 类型前，必须先弄清「类型决定位宽与解释方式」。

## 要点

| 类别 | 典型类型 | 说明 |
|------|----------|------|
| 整数 | `char` `short` `int` `long` `long long` | 实现定义宽度；用 `<stdint.h>` 固定 |
| 定宽 | `uint8_t` `int32_t` `uint64_t` | 协议、寄存器、DMA 描述符首选 |
| 浮点 | `float` `double` | 内核/DPDK 热路径少用 |
| 指针 | `void *` `char *` | 地址宽度 = 机器字长 |

```c
#include <stdint.h>
#include <stdio.h>

int g_static;          /* .bss，未初始化 */
static int s_file;     /* 文件作用域静态 */
void foo(void) {
    int local;         /* 栈 */
    static int once;   /* 静态存储，只初始化一次 */
    int *heap = malloc(sizeof *heap); /* 堆，ch05 */
}
```

```bash
# 查看本机类型宽度
gcc -dM -E - < /dev/null | grep -E '__SIZEOF_(INT|LONG|POINTER)__'
```

- **存储期**：自动（栈）、静态（`.data`/`.bss`）、动态（`malloc`）
- **对象表示**：同一串比特，按有符号/无符号/浮点解释结果不同 → **7.1.2**
- **跨平台**：勿假设 `int` 32 位、`long` 与指针同宽 → **7.3**、**7.4**

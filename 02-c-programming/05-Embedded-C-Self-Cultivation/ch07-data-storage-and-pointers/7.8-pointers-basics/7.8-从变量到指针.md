# 7.8 从变量到指针

## 本节讲什么

**指针**是保存对象**地址**的变量；`&` 取址、`*` 解引用构成 C 间接访问的核心。从标量变量到指针，是从「直接操作值」到「通过地址操作内存」的跃迁，内核链表、`kmalloc` 返回值、硬件寄存器映射都建立在此。本节建立指针与变量的对应关系。

## 要点

```c
int x = 42;
int *p = &x;    /* p 保存 x 的地址 */

printf("x=%d  *p=%d  &x=%p  p=%p\n", x, *p, (void *)&x, (void *)p);
*p = 100;       /* 通过指针改 x */
```

| 运算符 | 含义 |
|--------|------|
| `&obj` | 取对象地址，类型 `T *` |
| `*ptr` | 解引用，得到 `T` 左值 |
| `ptr->mbr` | 等价 `(*ptr).mbr` |

```c
int *q = NULL;          /* 空指针，未指向有效对象 */
if (q != NULL)
    *q = 1;

void swap(int *a, int *b) {
    int t = *a; *a = *b; *b = t;  /* 通过指针修改调用方变量 */
}
```

```gdb
(gdb) run

(gdb) print p
(gdb) print *p
(gdb) x/wx p           # 查看 p 指向的 int
```

# 7.8.3 指针类型与运算

## 本节讲什么

指针支持**加减整数**、**相减**、**关系比较**（同数组内），步长由指向类型决定。`NULL`、野指针、悬垂指针解引用均为 UB。本节归纳合法运算与 GDB 调试命令，并强调对齐与越界风险。

## 要点

```c
int a[5] = {10, 20, 30, 40, 50};
int *p = a;
p++;           /* 指向下一个 int，地址 +sizeof(int) */
int d = p - a; /* ptrdiff_t，元素个数差 */

char *bp = (char *)a;
bp += 4;       /* 字节步长 1 */
```

| 运算 | 合法条件 |
|------|----------|
| `p + n` / `p - n` | 结果指向同数组或尾后 |
| `p - q` | 同数组内 |
| `p < q` | 同数组内 |
| `p == NULL` | 始终 |

```c
#define NULL ((void *)0)   /* 常见；亦可用 0 */

int *wild;           /* 未初始化 */
*wild = 1;           /* UB */

int *dangling = malloc(sizeof *dangling);
free(dangling);
/* *dangling = 1; */ /* 悬垂 → demo06 */
```

```gdb
(gdb) set $p = (int *)0x7fffffffe000
(gdb) x/5wx $p
(gdb) print $p[2]
(gdb) watch *p              # 写监视点
```

**类型与对齐**：

```c
void *vp = &x;
int *ip = vp;        /* void* 转具体类型 */
/* 须保证对齐；ARM 上 misalign fault */
```

- **指针算术单位是元素**，不是字节（`char *` 除外）
- **`intptr_t`**：指针与整数互转打印
- **内核** `ERR_PTR`/`PTR_ERR` 在指针低位编码错误（架构相关）

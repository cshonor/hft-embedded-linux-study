# 7.8.1 指针的本质

## 本节讲什么

指针变量自身也占内存（通常 4 或 8 字节），其**内容**是某对象的地址。类型 `T *` 告诉编译器：解引用时读多少字节、如何解释比特。没有类型的 `void *` 可保存任意地址，解引用前须转回具体类型。

## 要点

```
内存示意:
  地址 0x7fff1000: [ int x = 42 ]
  地址 0x7fff2000: [ int *p = 0x7fff1000 ]

p 的值是「门牌号」；*p 是打开那扇门看里面的 int。
```

```c
int x = 42;
int *p = &x;

sizeof x;   /* sizeof(int) */
sizeof p;   /* sizeof(int *) — 指针宽度 */
sizeof *p;  /* sizeof(int) — 指向对象的大小 */
```

| 概念 | 说明 |
|------|------|
| 地址 | 字节编址，打印用 `%p` + `(void *)` |
| NULL | 空指针常量，比较用 `== NULL` |
| 无效地址 | 0xdeadbeef、已释放堆块 → UB |

```gdb
(gdb) print &x
(gdb) print p
(gdb) print sizeof(*p)
(gdb) x/4xb p          # 查看指针变量自身字节（小端地址）
(gdb) x/wx (int*)p     # 查看所指 int
```

```c
/* 内核简化 */
void *kmalloc(size_t size, gfp_t flags);
struct device *dev = ...;
/* dev 存某 device 结构体首地址 */
```

- **指针与整数**：`uintptr_t` 转换用于打印/MMIO，勿随意算术
- **双重身份**：`&p` 是「指向指针的指针」→ **7.11**

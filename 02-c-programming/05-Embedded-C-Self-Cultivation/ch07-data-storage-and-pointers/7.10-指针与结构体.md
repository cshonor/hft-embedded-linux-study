# 7.10 指针与结构体

## 本节讲什么

结构体指针用 `->` 访问成员，等价 `(*p).mbr`。`offsetof` 计算成员字节偏移，是协议解析、`container_of`（ch06 **6.4**）、FAM 分配的基础。`packed` 改变偏移规则，与 **7.2** 对齐理论直接相关。**demo03** 对比 normal/packed 布局。

## 要点

```c
struct normal {
    uint8_t  type;
    uint32_t value;
    uint16_t flags;
};

struct normal n = { .type = 1, .value = 0x12345678, .flags = 0xABCD };
struct normal *p = &n;

p->value = 0;              /* (*p).value */
```

```c
#include <stddef.h>
offsetof(struct normal, value);   /* 典型 4；packed 为 1 */

/* ch06 container_of */
#define container_of(ptr, type, member) ({              \
    void *__mptr = (void *)(ptr);                         \
    ((type *)((char *)__mptr - offsetof(type, member))); \
})
```

| 话题 | 要点 |
|------|------|
| `->` | 左操作数为指针 |
| `offsetof` | 编译期常量，单位字节 |
| `packed` | 无 padding，可能未对齐访问 |
| FAM | `struct hdr { int n; char data[]; };` + 一次分配 ch06 **6.5** |

```c
/* FAM 分配示意 */
struct pkt {
    size_t len;
    uint8_t body[];   /* GNU 扩展 / C99 FAM */
};
struct pkt *h = malloc(sizeof *h + payload_len);
```

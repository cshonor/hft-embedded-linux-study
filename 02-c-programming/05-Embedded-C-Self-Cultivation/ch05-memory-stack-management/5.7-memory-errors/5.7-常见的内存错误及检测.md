# 5.7 常见的内存错误及检测

## 本节讲什么

堆栈错误族：**越界读写、use-after-free、double free、未初始化、空指针**——表现从 **段错误** 到 **静默数据损坏**。工具链：**GDB+core、Valgrind、ASAN、mprotect guard**。

## 要点

| 错误 | 典型后果 |
|------|----------|
| 缓冲区溢出 | 踩栈/堆元数据 |
| UAF | 不可预测行为 |
| double free | glibc abort |
| 野指针 | SIGSEGV |

```bash
gcc -fsanitize=address -g -O0 -o t t.c && ./t
valgrind ./t
```

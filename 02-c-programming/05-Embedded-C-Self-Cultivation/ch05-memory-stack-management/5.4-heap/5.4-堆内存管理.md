# 5.4 堆内存管理

## 本节讲什么

**堆（heap）** 供 **动态分配**：运行期按需求申请/释放，生命周期不受函数栈帧约束。实现从裸机 **固定内存池** 到 Linux **glibc ptmalloc**，再到自研 **chunk 链表**；核心问题：**碎片、泄漏、双重释放**。

## 要点

- 用户 API：`malloc`/`free`/`calloc`/`realloc`；C++ 对应 `new/delete`
- 底层：裸机静态数组；RTOS 位图/链表；Linux `brk` + `mmap` arena
- 观测：`malloc_stats`、**5.4.4** 测试程序、**Valgrind** / **mtrace**

```bash
man malloc
ldd $(which ls) | grep libc
```

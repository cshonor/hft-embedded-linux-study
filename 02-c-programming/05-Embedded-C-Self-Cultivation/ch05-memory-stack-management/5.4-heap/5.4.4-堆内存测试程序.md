# 5.4.4 堆内存测试程序

## 本节讲什么

通过 **循环 malloc/free**、不等长块、多线程（拓展）观察 **堆增长、碎片、峰值 RSS**；配合 `size`、`/proc`、Valgrind 量化行为。

## 要点

### 测试思路

```c
/* 碎片：交替分配大小块再释放一半 */
for (int i = 0; i < N; i++) ptr[i] = malloc(i % 2 ? 64 : 1024);
for (int i = 0; i < N; i += 2) free(ptr[i]);
```

```bash
gcc -g -O0 -o heap_test heap_test.c
/usr/bin/time -v ./heap_test          # Max RSS
size heap_test
while ./heap_test; do cat /proc/$!/status | grep VmRSS; done
```

### demo04

```bash
cd demo && make demo04_heap_leak
size demo04_heap_leak
valgrind --leak-check=full --show-leak-kinds=all ./demo04_heap_leak
# 应报告 1024 bytes in 1 blocks definitely lost
```

### 指标

| 指标 | 含义 |
|------|------|
| `size` bss/data | 静态占用 |
| VmRSS | 实际物理驻留 |
| Valgrind `definitely lost` | 确定泄漏 |

- **DPDK**：`rte_malloc_dump_stats()` 看 per-socket 堆

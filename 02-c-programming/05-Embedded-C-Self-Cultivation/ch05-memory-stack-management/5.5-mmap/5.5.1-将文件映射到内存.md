# 5.5.1 将文件映射到内存

## 本节讲什么

**文件 mmap** 把磁盘内容映射为虚拟页：读时缺页从页缓存载入，写 `MAP_SHARED` 回刷文件。适合大文件随机访问、共享只读配置，避免 `read/write` 双拷贝。

## 要点

```c
int fd = open("data.bin", O_RDONLY);
void *p = mmap(NULL, st.st_size, PROT_READ, MAP_PRIVATE, fd, 0);
/* 访问 p[i] 等价读文件 offset i */
munmap(p, st.st_size);
close(fd);
```

```bash
gcc -o fmmap fmmap.c
strace -e mmap,openat ./fmmap 2>&1
cat /proc/self/maps | grep data.bin
```

| 标志 | 行为 |
|------|------|
| `MAP_PRIVATE` | CoW，不写回文件 |
| `MAP_SHARED` | 修改可见于其他映射/文件 |

- **内核** `filp->f_op->mmap` 驱动实现；如 **/dev/mem** 需 root
- **DPDK**：配置文件、 hugepage 文件在 `/dev/hugepages`

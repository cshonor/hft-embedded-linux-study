# 5.5 mmap 映射区域探秘

## 本节讲什么

**mmap** 把 **文件或设备** 映射进进程虚拟地址，或 **匿名映射** 分配零页（heap 延伸、共享内存、栈）。是 Linux **VMA 扩展** 的主要手段，也是 **MMIO 访问硬件** 的用户态入口。

## 要点

```c
void *p = mmap(NULL, len, PROT_READ|PROT_WRITE,
               MAP_PRIVATE|MAP_ANONYMOUS, -1, 0);
munmap(p, len);
```

```bash
cat /proc/self/maps | grep mmap
strace -e mmap,munmap ./demo01_memory_zone 2>&1 | head
```

- 映射区在 maps 中常显示为 **匿名 `[anon]`** 或文件路径
- 大块 `malloc` 底层走 **mmap**（>128KB 量级，依 glibc 版本）

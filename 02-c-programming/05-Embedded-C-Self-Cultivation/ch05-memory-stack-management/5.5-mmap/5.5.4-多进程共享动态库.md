# 5.5.4 多进程共享动态库

## 本节讲什么

**动态库 `.so`** 经 **ld.so** `mmap` 进各进程，**代码段只读共享** 物理页，**数据段 per-process CoW**。多进程同时跑同一 `.so` 省 RAM——与 **ch04 4.8** PIC/GOT/PLT 衔接。

## 要点

```bash
gcc -fPIC -shared -o libfoo.so foo.c
gcc -o app app.c -L. -lfoo
ldd app
cat /proc/self/maps | grep libfoo
```

| 段 | 共享 |
|----|------|
| `.text` / `.rodata` | 多进程共享同一物理页 |
| `.data` / `.bss` | 映射后 CoW，各自副本 |

- **GOT/PLT**：每进程一份 GOT 写权限页；延迟绑定改 GOT 项
- **内核模块**：类似共享库，但在内核空间 `insmod` 链接导出符号
- **DPDK**：静态链 PMD 常见；动态插件亦靠 `dlopen` + mmap

```bash
readelf -d app | grep NEEDED
pmap -x $$ | grep '\.so'
```

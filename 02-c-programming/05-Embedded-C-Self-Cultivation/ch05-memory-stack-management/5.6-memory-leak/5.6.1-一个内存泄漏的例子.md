# 5.6.1 一个内存泄漏的例子

## 本节讲什么

最小泄漏：`malloc` 后 **从不 free**，进程退出时 OS 回收，但 **长期运行** 会耗尽内存。`demo04_heap_leak.c` 故意泄漏 1024 字节供工具抓现行。

## 要点

```c
void *p = malloc(1024);
(void)p;
return 0;   /* 泄漏：无 free，无其它指针指向 p */
```

```bash
cd demo && make demo04_heap_leak
valgrind --leak-check=full --show-leak-kinds=all ./demo04_heap_leak
# ==xxx== 1024 bytes in 1 blocks are definitely lost in loss record 1 of 1
# ==xxx==    at 0x...: malloc (...)
# ==xxx==    by 0x...: main (demo04_heap_leak.c:6)
```

### 变体

- 循环 `malloc(64)` 不 free → RSS 持续升
- 重新赋值丢旧指针：`p = malloc(n); p = malloc(m);` 第一块泄漏

```bash
/usr/bin/time -v ./demo04_heap_leak   # Maximum resident set size
```

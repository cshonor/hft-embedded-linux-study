# 5.3.4 形参与实参

## 本节讲什么

**形参** 是函数定义里的占位名，**实参** 是调用时的表达式。C 只有 **值传递**：形参是实参值的副本；若想改调用方变量，须传 **指针** 并在被调函数里解引用。

## 要点

### 值传递 vs 指针

```c
void swap_wrong(int a, int b) { int t=a; a=b; b=t; }      /* 无效 */
void swap_ok(int *a, int *b)   { int t=*a; *a=*b; *b=t; } /* 有效 */
```

- 修改 **指针形参本身**（`p = NULL`）不影响实参指针变量
- 修改 **`*p` 指向的内容** 影响实参所指对象

### 栈上的副本

```c
void bar(int x) { x = 99; }   /* 只改栈上副本 */
int n = 1;
bar(n);                       /* n 仍为 1 */
```

```bash
gdb -ex 'break bar' -ex run -ex 'print x' -ex 'finish' ./a.out
```

### 常见坑

- `scanf("%d", n)` 应 `scanf("%d", &n)`
- 返回栈上局部指针 → **悬空指针**（UB）
- 大结构体按值传递 → 拷贝开销；内核 API 多用指针

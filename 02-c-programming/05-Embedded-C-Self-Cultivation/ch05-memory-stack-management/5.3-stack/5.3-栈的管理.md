# 5.3 栈的管理

## 本节讲什么

**栈** 是函数调用链的 LIFO 工作区：保存返回地址、旧帧指针、局部变量、部分参数。ARM 上由 **SP/FP/LR** 协作；理解栈帧是读 **GDB `bt`**、内核 oops 栈回溯、嵌入式定栈大小的基础。

## 要点

- 栈通常 **向低地址增长**；`ulimit -s` 限制默认大小（常见 8MB）
- 每次调用：**入栈** 保存 LR/FP → 分配局部 → **出栈** 恢复并返回
- **AAPCS**（ch03）：r0–r3 传参，其余入栈；返回值 r0
- 工具链：`-fno-omit-frame-pointer` 便于 `bt`；`-fstack-protector` 防溢出

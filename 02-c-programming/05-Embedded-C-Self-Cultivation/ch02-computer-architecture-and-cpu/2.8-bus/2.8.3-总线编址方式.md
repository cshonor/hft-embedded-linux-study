# 2.8.3 总线编址方式

## 本节讲什么

**统一编址（Memory-Mapped I/O）vs 独立编址（Port I/O）** —— 外设寄存器如何进入程序地址空间；**MMIO 重点**。

## 要点

| 方式 | 特点 | 平台 |
|------|------|------|
| **统一编址 MMIO** | 外设寄存器占内存地址段 | ARM、RISC-V、PCIe BAR |
| **独立编址 Port** | 专用 in/out 指令 | x86 传统 IO 空间 |

### MMIO 访问要点

```c
#define REG(base, off) (*(volatile uint32_t *)((base) + (off)))
REG(UART_BASE, 0x00) = 0x03;  /* 写控制寄存器 */
while (!(REG(UART_BASE, 0x04) & 0x01)) ;  /* 读状态位 */
```

- 必须 **`volatile`**：防止编译器优化掉对硬件的读写  
- **位域/掩码** 按 Datasheet，勿假设 RAM 语义  
- **对齐与宽度**：8/16/32 bit 访问；错误宽度 → fault 或读错  
- PCIe NIC：**BAR0** 映射 DMA 描述符环与控制寄存器 — DPDK PMD 初始化核心

### 与 DMA

- CPU 写 MMIO **启动 DMA**；数据在 **RAM**，不在寄存器地址  
- Cache 一致性：**2.4.3** flush/invalidate

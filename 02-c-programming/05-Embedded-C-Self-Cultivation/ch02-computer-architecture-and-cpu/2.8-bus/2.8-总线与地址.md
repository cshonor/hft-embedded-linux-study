# 2.8 总线与地址

## 本节讲什么

**地址是什么、总线如何连、外设如何编址** —— C 指针、MMIO、驱动与 DPDK 寄存器访问的硬件基础。

## 要点

| 小节 | 主题 |
|------|------|
| **2.8.1** | 地址 = 总线编号，非物理「位置」 |
| **2.8.2** | 地址/数据/控制总线 |
| **2.8.3** | 统一编址 vs 独立 IO；**MMIO** |
| **2.8.4** | 大小端（已学） |

- `*(volatile uint32_t *)0x40001000 = 1;` — 写 **设备寄存器**，非普通 RAM  
- 内核 `ioremap` / DPDK `rte_write32` 本质相同：**映射到 CPU 可访问地址**

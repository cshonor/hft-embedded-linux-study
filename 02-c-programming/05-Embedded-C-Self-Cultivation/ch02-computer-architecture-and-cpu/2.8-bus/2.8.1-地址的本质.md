# 2.8.1 地址的本质

## 本节讲什么

**地址是总线上的逻辑编号** —— 非「第几号抽屉」，而是 **CPU 发出、译码器路由** 的信号。

## 要点

- CPU 读/写时：**地址线** 选设备/单元，**数据线** 传内容，**控制线** 读/写使能  
- **物理地址** vs **虚拟地址**：OS/MMU 再映射一层（**ch05**）  
- 同一物理地址空间可挂：**RAM、Flash、GPIO、UART、NIC BAR**

```c
uint32_t *p = (uint32_t *)0x20000000;  /* 可能是 SRAM，也可能是外设 — 看 Memory Map */
```

- DPDK：`/proc/iomem` 查看设备占用的物理区间

# 2.8.2 总线的概念

## 本节讲什么

**地址总线、数据总线、控制总线** —— 连接 CPU、内存与外设的「公路系统」。

## 要点

| 总线 | 作用 |
|------|------|
| **地址** | 选从设备（宽度决定寻址空间） |
| **数据** | 传输字/字节（32/64 bit） |
| **控制** | R/W、中断、DMA 请求、时钟 |

- **片上总线**：AMBA AHB/AXI（ARM SoC）；**片外**：PCIe、DDR  
- **带宽** = 频率 × 位宽 × 效率；瓶颈常在 **DDR 与 PCIe**  
- 冯·诺依曼 **指令/数据争用** 同一总线（**2.3.1**）

- HFT/DPDK：NUMA 下 **本地 PCIe 插槽** 到本地内存路径最短

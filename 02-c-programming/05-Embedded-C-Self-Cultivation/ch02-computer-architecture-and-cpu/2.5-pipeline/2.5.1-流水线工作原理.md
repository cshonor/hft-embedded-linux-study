# 2.5.1 流水线工作原理

## 本节讲什么

**取指 → 译码 → 执行 → 访存 → 写回** 五级重叠 —— 同一时刻多条指令处于不同阶段。

## 要点

```
周期:  1    2    3    4    5
I1:   IF   ID   EX   MEM  WB
I2:        IF   ID   EX   MEM  WB
I3:             IF   ID   EX   ...
```

- **吞吐** ≈ 每周期完成一条（稳态）；**延迟** 仍是一条指令多级延迟之和  
- **流水线填满/排空** 时 CPI > 1  
- 嵌入式 Cortex-M 多为 **3 级简流水线**；x86/ARM A 系列 10+ 级

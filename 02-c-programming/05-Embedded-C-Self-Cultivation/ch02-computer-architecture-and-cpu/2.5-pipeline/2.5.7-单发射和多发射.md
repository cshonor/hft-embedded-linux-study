# 2.5.7 单发射和多发射

## 本节讲什么

**每周期发射指令条数** —— 单发射（1 IPC 上限）vs 超标量多发射（>1 IPC）。

## 要点

| 类型 | 特点 |
|------|------|
| **单发射** | 简单 MCU，功耗低 |
| **多发射** | 多执行单元并行，需 OoO + 寄存器重命名 |

- **IPC × 频率 = 吞吐**；多发射受 **指令相关性、端口数** 限制  
- `perf stat` 可看 **instructions/cycles** 估算 IPC  
- DPDK 绑核 + 热路径内联，提高 **指令级并行** 利用率

# 2.5.3 流水线冒险

## 本节讲什么

**结构 / 数据 / 控制冒险** —— 流水线停顿（stall/bubble）的三大来源。

## 要点

| 类型 | 原因 | 缓解 |
|------|------|------|
| **结构** | 硬件资源冲突（单端口内存） | 哈佛分离、双端口 |
| **数据** | 读后写、写后读相关 | 旁路（forwarding）、stall |
| **控制** | 分支/跳转改变 PC | 预测、延迟槽 |

```c
a = b + c;   /* 写 a */
d = a + 1;   /* 读 a — RAW 冒险，可能 stall 或 forwarding */
```

- 编译器 **调度指令** 可插入无关指令填 bubble  
- 内核 **内存屏障** 与流水线无关，约束的是 **可见性/顺序**（别混淆）

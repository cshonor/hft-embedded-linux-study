# 2.5.2 超流水线技术

## 本节讲什么

**加深流水线级数、提高主频** —— 用更长流水线换更高时钟，但分支惩罚更大。

## 要点

- 每级逻辑更少 → **时钟周期更短** → 峰值 GHz 更高  
- 副作用：**分支 mispredict** 需冲刷更多级；**Cache miss** 停顿级数增加  
- Pentium 4 超长流水线 vs ARM 较浅高频效率核  

- 现代趋势：**超流水线 + 乱序 + 多发射** 组合，而非单纯加深  
-  profiling 看 **cycles per instruction** 比只看 GHz 更有意义

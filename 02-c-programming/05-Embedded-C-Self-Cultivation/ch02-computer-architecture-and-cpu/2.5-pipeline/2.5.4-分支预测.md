# 2.5.4 分支预测

## 本节讲什么

**猜测分支方向，提前取指** —— 减少控制冒险冲刷；预测失败是 HFT 热点路径的大敌。

## 要点

| 机制 | 说明 |
|------|------|
| **静态** | 编译器 hint、always/not taken |
| **动态** | 分支历史表 BHT、两级自适应 |
| **失败代价** | 冲刷流水线，10–20+ cycles |

- **`if` 在循环内可预测** vs **随机分支**（协议状态机）性能差一个数量级  
- 技巧：**branchless**（cmov）、查表、位运算代替分支  
- DPDK：`likely/unlikely` 引导编译器布局热路径

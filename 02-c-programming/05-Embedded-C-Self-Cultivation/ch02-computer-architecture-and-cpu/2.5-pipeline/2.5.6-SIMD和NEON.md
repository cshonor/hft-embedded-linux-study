# 2.5.6 SIMD 和 NEON

## 本节讲什么

**单指令多数据（SIMD）** —— 一条指令处理多个数据元素；ARM **NEON**，x86 **SSE/AVX**。

## 要点

```c
/* 概念：4×float 并行加 */
vaddq_f32(a, b);  /* NEON 128-bit 向量 */
```

| ISA | 向量扩展 |
|-----|----------|
| **ARM** | NEON（AArch32/AArch64） |
| **x86** | SSE / AVX / AVX-512 |

- 适用：**checksum、CRC、批量 memcpy、图像/信号**  
- DPDK **`rte_vect`** 封装 SIMD 路径；HFT 批量行情解析可手动 intrinsics  
- 需 **内存对齐**（16/32B）；未对齐可能慢或 fault

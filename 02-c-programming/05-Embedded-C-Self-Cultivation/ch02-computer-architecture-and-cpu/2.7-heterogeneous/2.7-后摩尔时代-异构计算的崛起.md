# 2.7 后摩尔时代：异构计算的崛起

## 本节讲什么

**CPU 之外的专业处理器** —— GPU/DSP/FPGA/NPU 等分工；网络与 AI 时代的算力版图。

## 要点

| 小节 | 加速器 |
|------|--------|
| **2.7.1** | 异构计算概念 |
| **2.7.2–6** | GPU、DSP、FPGA、TPU、NPU |
| **2.7.7** | DPU/SmartNIC 等 XPU |

- **摩尔放缓** → 领域专用架构（DSA）

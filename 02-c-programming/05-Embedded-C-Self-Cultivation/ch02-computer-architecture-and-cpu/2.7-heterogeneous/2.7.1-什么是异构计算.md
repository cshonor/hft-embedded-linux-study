# 2.7.1 什么是异构计算

## 本节讲什么

**多种 ISA/微架构协同** —— 按 workload 选最合适处理器，CPU 作控制面。

## 要点

```
Host CPU ──PCIe/AXI──► GPU / FPGA / NPU / DSP
         调度、DMA、中断、驱动
```

| 维度 | CPU | 加速器 |
|------|-----|--------|
| **擅长** | 控制流、分支、OS | 数据并行、固定算子 |
| **编程** | C/通用 | CUDA/OpenCL/HLS/固件 |

- **统一内存 vs 分立**：影响 zero-copy 与 cache 一致性  
- 嵌入式 SoC：CPU + GPU/VPU 同 Die，共享 DRAM 控制器

# 2.7.3 DSP

## 本节讲什么

**数字信号处理器** —— 乘累加（MAC）硬件、哈佛结构、低延迟 FIR/FFT。

## 要点

| 特点 | 说明 |
|------|------|
| **哈佛** | 指令/数据总线分离（**2.3.2**） |
| **MAC** | 单周期乘加，音频/基带常见 |
| **端序** | 不少 DSP **大端**（**2.8.4**） |

- 嵌入式：TI C6000、Cadence HiFi — 麦克风阵列、Modem  
- 与 CPU 通信：共享内存、Mailbox、**MMIO** 控制寄存器  
- 网络：部分 **SmartPHY** 内置 DSP 做前向纠错

## 与后续衔接

**2.3.2** 哈佛架构；**2.8.4** 大端与 `htonl` 对照。

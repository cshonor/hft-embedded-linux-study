# 2.7.2 GPU

## 本节讲什么

**图形处理器 → 通用 GPGPU** —— 数千轻量核 + 高带宽显存，擅长大规模并行。

## 要点

- **SIMT**：warp 内线程执行相同指令，掩码处理分支  
- **CUDA / OpenCL / Vulkan Compute** 编程模型  
- 瓶颈：**PCIe 传输、kernel launch 延迟** — 不适合 μs 级 HFT 主路径  
- 适用：批量回测、风控 Monte Carlo、AI 推理训练

- DPDK 生态偶有 **GPU packet processing** 研究；生产仍以 CPU + NIC offload 为主

## 与后续衔接

**2.5.6** SIMD 是 CPU 内小规模并行；**2.7.5 TPU** 专用矩阵乘。

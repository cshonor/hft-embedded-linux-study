# 2.7.6 NPU

## 本节讲什么

**神经网络处理单元** —— 手机/嵌入式 SoC 上的 AI 推理加速器（INT8/FP16）。

## 要点

| 特性 | 说明 |
|------|------|
| **算子** | Conv、Depthwise、激活融合 |
| **带宽** | 权重压缩、片上 SRAM |
| **接口** | IOCTL / 共享内存 / **MMIO** 启动 job |

- 代表：Apple ANE、高通 Hexagon、瑞芯微 NPU  
- CPU 跑 **驱动 + 预处理**；NPU 跑推理；实时性看 **job 排队延迟**  
- 与 DPDK 无直接交集；嵌入式视觉/语音常见

# 2.7.4 FPGA

## 本节讲什么

**现场可编程门阵列** —— 硬件逻辑可重配，纳秒级固定延迟流水线。

## 要点

- **LUT + 布线** 实现任意组合/时序逻辑；开发：**Verilog/VHDL → 比特流**  
- 优势：**确定性延迟**、线速包处理、自定义协议  
- HFT：**tick-to-trade**、order gateway 硬件加速常见 FPGA  
- DPDK 侧：**AF_XDP + FPGA**、P4 SmartNIC（Netronome/Broadcom）

- 代价：开发周期长、功耗与面积高于 ASIC；量产量大后常转 **ASIC**

# 2.7.5 TPU

## 本节讲什么

**张量处理单元（Google TPU）** —— 专为矩阵乘/卷积设计的 ASIC 加速器。

## 要点

- **脉动阵列（Systolic Array）**：数据流过 PE 网格，减少访存  
- 面向 **TensorFlow/XLA** 图编译；云 TPU Pod 超大规模训练  
- 与通用 CPU 关系：Host 喂数据、TPU 算子执行、DMA 回写  

- 嵌入式少见；边缘 AI 更多 **NPU（2.7.6）**  
- 网络领域类比：NIC 内 **固定功能流水线** 而非可编程 TPU

## 与后续衔接

**2.7.6 NPU** 端侧推理；**2.7.7** DPU 网络领域专用芯片。

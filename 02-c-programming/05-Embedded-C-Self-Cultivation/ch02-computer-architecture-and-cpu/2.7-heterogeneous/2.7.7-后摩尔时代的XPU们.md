# 2.7.7 后摩尔时代的 XPU 们

## 本节讲什么

**DPU / IPU / SmartNIC / CXL 设备** 等「XPU」—— 网络与数据中心的新兴专用芯片。

## 要点

| XPU | 角色 |
|-----|------|
| **DPU/IPU** | 虚拟化、存储、网络栈 offload |
| **SmartNIC** | RSS、TLS、RoCE、P4 可编程 |
| **CXL** | 内存扩展、池化，改变 NUMA 拓扑 |

- AWS Nitro、NVIDIA BlueField、Intel IPU — **释放 Host CPU** 给业务（含 DPDK app）  
- HFT：定制 **TOE/低延迟 NIC** + kernel bypass  
- 选型问：**哪些包处理在卡上、哪些仍回 CPU**

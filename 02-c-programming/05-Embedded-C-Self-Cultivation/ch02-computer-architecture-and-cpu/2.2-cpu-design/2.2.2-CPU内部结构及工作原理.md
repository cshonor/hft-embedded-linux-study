# 2.2.2 CPU 内部结构及工作原理

## 本节讲什么

**寄存器、PC、PSR/FLAGS、总线** —— C 语句在 CPU 内的执行载体。

## 要点

### 通用寄存器

- CPU **唯一零延迟**存储；运算前须 **Load** 进寄存器  
- 分类：数据、地址、**SP/FP**（栈帧，**ch05**）  
- 32/64 位宽决定一次运算与指针宽度  

### PC（Program Counter）

- 存**下一条指令**地址；顺序执行 PC 自增  
- **函数调用/跳转** = 写 PC  

### PSR / FLAGS

| 标志 | 含义 | C 对应 |
|------|------|--------|
| **Z** | 结果为 0 | `if (a==0)` |
| **N** | 负数 | 有符号比较 |
| **C** | 进位 | 无符号溢出 |
| **V** | 溢出 | 有符号溢出 |

- **if/for** 底层：cmp → 条件跳转读 FLAGS（Demo03 **objdump**）

### 一条赋值语句的硬件流

```c
a = b + c;
```

1. Load `b`,`c` → 寄存器  
2. ALU add  
3. Store → `a` 内存地址  

## 实操

`gdb` 单步 + `info registers`；`objdump -d` 看 cmp/je。

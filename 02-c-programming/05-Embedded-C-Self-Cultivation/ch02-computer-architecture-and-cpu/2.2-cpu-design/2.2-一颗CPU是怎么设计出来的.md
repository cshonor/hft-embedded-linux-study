# 2.2 一颗 CPU 是怎么设计出来的

## 本节讲什么

**理论（图灵机）→ 微架构 → 流片** 的 CPU 设计主线 —— 连接「能算什么」与「怎么造」。

## 要点

| 小节 | 内容 |
|------|------|
| **2.2.1 图灵机** | 可计算性理论基石 |
| **2.2.2 内部结构** | 寄存器、PC、ALU、FLAGS（已学） |
| **2.2.3 设计流程** | 规格 → RTL → 验证 → 综合 → 布局布线 |

- **ISA**（指令集）是软硬件契约；**微架构** 是同一 ISA 的不同实现（如 Cortex-A78 vs X1）  
- 读 **DPDK rte_cycles**、内核 **rdtsc** 时，底层是 CPU 计数器与流水线设计的结果

# 2.2.1 计算机理论基石：图灵机

## 本节讲什么

**图灵机模型** —— 证明「什么可计算」，为 CPU「取指-执行」提供理论原型。

## 要点

```
无限纸带 + 读写头 + 有限状态 + 转移规则
```

| 要素 | 对应现代 CPU |
|------|----------------|
| **纸带** | 内存 |
| **读写头** | 总线访问 |
| **状态** | PC + 寄存器 |
| **规则表** | 指令译码 |

- **丘奇-图灵论题**：可计算函数 ≡ 图灵机可计算  
- 实际 CPU 是 **有限内存 + 有限状态** 的图灵机实现  
- HFT 策略再复杂，最终仍归结为 **有限步骤的指令序列**

# 2.2.3 CPU 设计流程

## 本节讲什么

**规格 → RTL → 验证 → 综合 → 物理设计 → 流片** —— 一颗 CPU 从文档到硅片的工程路径。

## 要点

```
架构规格(ISA/微架构) → HDL(RTL) → 仿真验证 → 逻辑综合 → 布局布线 → 掩膜 → 制造
```

| 阶段 | 产出 |
|------|------|
| **架构** | 流水线级数、Cache 容量、分支预测策略 |
| **RTL** | Verilog/VHDL 可仿真模型 |
| **验证** | 指令级、随机、形式化 |
| **物理** | 时序收敛、功耗分析 |

- **ARM 授权**：多数嵌入式/MCU 购买 Cortex 内核 + 自行集成外设  
- **DPDK 亲和性**：同一 ISA 不同步频/Cache 大小 → `rte_lcore_id` 绑核需了解拓扑

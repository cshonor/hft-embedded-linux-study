# 2.6.2 片上多核互连技术

## 本节讲什么

**多核如何连 Cache、内存控制器与外设** —— Ring、Mesh、NoC 与 NUMA 拓扑。

## 要点

| 拓扑 | 特点 |
|------|------|
| **Ring** | 早期 Intel，实现简单 |
| **Mesh/NoC** | 2D 网格，可扩展 |
| **CCX/CCD** | AMD 簇内/簇间延迟差 |

- **MESI/MOESI**：多核 Cache 一致性协议，跨核写 → **invalidate** 其他核 Cache 行  
- **伪共享**：不同核写同一 cache line 不同变量 → 总线风暴  
- DPDK：`numactl --membind` 本地内存；读 `/sys/devices/system/node`

## 与后续衔接

**2.4.2** 共享 L3；**2.6.5** 核数扩展效率；内核 **SMP** 启动。

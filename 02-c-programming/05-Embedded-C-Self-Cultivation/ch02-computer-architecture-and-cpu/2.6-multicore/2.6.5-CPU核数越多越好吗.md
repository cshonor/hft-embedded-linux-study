# 2.6.5 CPU 核数越多越好吗

## 本节讲什么

**Amdahl 定律与并行效率** —— 核数增加何时收益递减；DPDK 该绑几核。

## 要点

```
加速比 S = 1 / ((1-P) + P/N)   P=可并行比例, N=核数
```

| 因素 | 影响 |
|------|------|
| **串行部分** | 锁、单队列、系统调用 |
| **通信** | 跨核 Cache miss、NUMA |
| **争用** | 同一网卡多队列不足 |

- **8 核跑满 ≠ 2×4 核**：测量 **pps/latency** 找拐点  
- 嵌入式：多核 + RTOS 分区（**AMP**）隔离关键任务  
- 原则：**并行化热点，而非盲目加核**

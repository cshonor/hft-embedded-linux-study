# 2.6.1 单核处理器的瓶颈

## 本节讲什么

**频率墙、功耗墙、ILP 墙** —— 为何不能无限提高单核 GHz。

## 要点

| 瓶颈 | 表现 |
|------|------|
| **频率** | 深流水线 + 线延迟限制 |
| **功耗** | P ∝ CV²f，散热成为硬约束 |
| **ILP** | 指令相关使 IPC 趋近上限 |

- 2005 前后业界转向 **多核** 而非单纯提频  
- 嵌入式：单核 MCU 仍大量存在（成本、实时）  
- 服务器 DPDK：**多核水平扩展** 替代超频

## 与后续衔接

**2.6.2** 多核互连；**2.5** 单核 ILP 优化已接近天花板。

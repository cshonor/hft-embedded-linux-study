# 2.6.4 超线程技术

## 本节讲什么

**SMT（Simultaneous Multi-Threading）** —— 一物理核跑两 **逻辑核**，共享执行单元。

## 要点

- 利用 **不同线程指令混合** 提高执行单元利用率  
- 共享 L1/L2 → 两线程争资源，**峰值性能非 2×**  
- 安全：**Spectre/Meltdown** 等侧信道与 SMT 相关；部分云禁 SMT  

- DPDK 官方建议：**独占物理核**（`isolcpus` + 关 SMT 同核另一线程）  
- HFT：latency 敏感路径避免与兄弟 hyperthread 共享

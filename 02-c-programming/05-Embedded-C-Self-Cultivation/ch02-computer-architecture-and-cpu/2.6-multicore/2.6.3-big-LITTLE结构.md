# 2.6.3 big.LITTLE 结构

## 本节讲什么

**大核（性能）+ 小核（能效）** 同片 —— ARM 移动端与部分嵌入式 SoC 的异构多核。

## 要点

```
big:   深流水线、高 IPC、高功耗  → 突发负载
LITTLE: 浅流水线、低功耗       → 后台/待机
```

- **全局调度**：Linux **EAS** 按负载迁移线程  
- 同 ISA 不同微架构；**频率/Cache** 差异大  
- 服务器 x86 少见；**Ampere/ARM 云实例** 需了解核类型差异  
- DPDK 绑核时避免把 **polling 线程** 放小核

# 2.3.1 冯·诺依曼架构

## 本节讲什么

**五大部件 + 统一存储** —— 现代通用计算机骨架；指令与数据同存于内存，总线分时复用。

## 要点

| 部件 | 作用 |
|------|------|
| **ALU** | 算术/逻辑运算 |
| **CU** | 取指、译码、控制时序 |
| **存储器** | RAM 运行态 + ROM/Flash 固件 |
| **输入** | 按键、串口、传感器 |
| **输出** | 屏、PWM、电机 |

### 数据流

```
取指(PC) → 译码 → 读操作数(内存/寄存器) → ALU → 写回 → 更新 PC
```

- **C 赋值 `a = b + c`**：Load 到寄存器 → Add → Store 回内存（**2.2.2**）

### 特性与代价

- 指令/数据**同一内存** → 结构简单  
- 指令与数据争用总线 → 带宽瓶颈

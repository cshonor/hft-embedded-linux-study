# 2.3.2 哈佛架构

## 本节讲什么

**指令总线 / 数据总线分离** —— MCU 常见；取指与访存可并行，吞吐更高。

## 要点

| 对比 | 冯·诺依曼 | 哈佛 |
|------|-----------|------|
| 总线 | 指令+数据复用 | **分开** |
| 并行 | 取指时难同时读数据 | 可并行 |
| 典型 | PC、服务器 | **Cortex-M**、DSP |

- 许多 MCU **对外**冯·诺依曼、**片上**哈佛（**2.3.3**）  
- 嵌入式固件常放 **Flash 指令** + **RAM 数据**

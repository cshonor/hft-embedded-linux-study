# 2.9.2 什么是微架构

## 本节讲什么

**同一 ISA 的不同硬件实现** —— 流水线深度、Cache、分支预测各异；二进制兼容，性能不同。

## 要点

```
ARMv8-A ISA
  ├── Cortex-A53（小核，低功耗）
  ├── Cortex-A78（大核，高性能）
  └── Apple M-series（定制宽发射）
```

| 微架构决定 | 与 ISA 无关的优化 |
|------------|-------------------|
| Cache 大小 | **2.4** |
| 流水线/OOO | **2.5** |
| 核数/互连 | **2.6** |

- **binfmt 相同** 可在同 ISA 不同芯片运行；benchmark 才反映微架构差异  
- DPDK release notes 常列 **经测 CPU 型号**（NUMA、Cache 相关）

# 2.9.1 什么是指令集

## 本节讲什么

**ISA（Instruction Set Architecture）** —— CPU 支持的指令、寄存器、异常模型；软硬件接口。

## 要点

| ISA | 特点 |
|-----|------|
| **ARM** | 嵌入式/移动/服务器；AArch32/AArch64 |
| **x86** | CISC，复杂寻址；PC 服务器、DPDK 主战场 |
| **RISC-V** | 开放指令集，模块化扩展 |

- **指令字格式**：定长（ARM/RISC-V 32）vs 变长（x86）  
- **特权级**：User/Kernel/Hypervisor — 系统调用、异常入口由 ISA 规定  
- 同一 **GCC 源码**，`-march` 不同 → 不同机器码

# 2.9.3 指令助记符：汇编语言

## 本节讲什么

**助记符、基本指令与寻址方式** —— 连接 C 语句与机器码；`objdump`/GDB 读汇编必备。

## 要点

### 常见助记符（ARM/x86 对照概念）

| 类别 | 示例 | C 对应 |
|------|------|--------|
| **传送** | mov/ldr/str | 赋值、读写内存 |
| **运算** | add/sub/and | `+` `-` `&` |
| **比较分支** | cmp + b.eq/b.ne | `if` `for` |
| **调用** | bl/call | 函数调用 |
| **栈** | push/pop | 局部变量、传参 |

### 寻址方式

| 方式 | 形式 | 用途 |
|------|------|------|
| **立即数** | `#0x10` | 常量 |
| **寄存器** | `r0` | 零延迟操作数 |
| **直接** | `[addr]` | 全局/GPIO 固定地址 |
| **寄存器间接** | `[r1]` | 指针 `*p` |
| **基址+偏移** | `[fp, #8]` | 栈变量、结构体成员 |
| **变址** | `[r1, r2, lsl #2]` | 数组 `arr[i]` |
| **PC 相对** | `.+offset` | 位置无关代码 PIC |

```c
arr[i] = 0;  /* 汇编 ≈ 基址+变址寻址，见 ch03 */
```

- **2.2.2** FLAGS 由 cmp 设置，分支读 Z/N 位  
- HFT 热点：`objdump -d` 确认无意外库调用与多余 load/store

# 2.1.2 PN 结的工作原理

## 本节讲什么

**P 型 / N 型掺杂 + PN 结** —— 晶体管与二极管的物理开关。

## 要点

| 概念 | 说明 |
|------|------|
| **掺杂** | P：缺电子（空穴）；N：多余电子 |
| **耗尽层** | 结区无自由载流子，形成势垒 |
| **正向偏置** | 导通，电流通过 |
| **反向偏置** | 截止，仅极小漏电流 |

- **MOSFET**：Gate 电压控制沟道，实现 **0/1 开关** → 逻辑门 → CPU  
- 漏电流与 **静态功耗** 相关；低功耗 MCU / 移动端 SoC 重点优化

# 2.1 一颗芯片是怎样诞生的

## 本节讲什么

**从硅到封装** 的芯片制造全景 —— 理解「硬件从哪来」，为后续 CPU/总线/寄存器打物理直觉。

## 要点

| 阶段 | 关键 |
|------|------|
| **2.1.1 单晶硅** | 高纯硅锭 → 晶圆（Wafer） |
| **2.1.2 PN 结** | 半导体开关基础 |
| **2.1.3 电路** | 光刻、掺杂、多层金属互联 |
| **2.1.4 封装** | Die → 引脚/BGA，散热与 IO |

- 一颗 **SoC**（手机/MCU/DPDK 网卡芯片）在同一晶圆上集成 CPU、Cache、DMA、MAC  
- 制程（nm）决定晶体管密度与功耗；嵌入式选型先看 **封装引脚** 与 **工作温度**

# 2.1.3 从 PN 结到芯片电路

## 本节讲什么

**光刻 + 掺杂 + 多层互联** 把 PN 结堆叠成完整数字电路。

## 要点

```
设计(RTL) → 版图 → 光刻 → 刻蚀/沉积 → 重复多层 → 测试 → 划片
```

| 步骤 | 作用 |
|------|------|
| **光刻** | 紫外光透过掩膜，定义图案 |
| **离子注入** | 精确掺杂形成 NMOS/PMOS |
| **金属层** | 连接晶体管，构成布线 |

- 一颗 CPU **数十亿晶体管**；时钟、Cache、流水线均为同一硅片上的电路  
- **FPGA/ASIC** 定制网卡（如 SmartNIC）走类似流片流程，周期以月计

## 与后续衔接

**2.2.3** 设计流程中的前端/后端；**2.4** 片上 SRAM 即 Cache 物理实现。

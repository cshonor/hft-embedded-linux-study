# 2.1.1 从沙子到单晶硅

## 本节讲什么

**石英砂 → 高纯单晶硅锭 → 晶圆** —— 芯片的物理起点。

## 要点

1. **提纯**：SiO₂ 还原得冶金硅，再化学提纯至 99.9999999%（9N）  
2. **拉晶（CZ）**：熔融硅中旋转拉出单晶锭，控制晶向（如 ⟨100⟩）  
3. **切片与抛光**：锭 → 薄圆片（Wafer），直径 200/300 mm  
4. **氧化/光刻起点**：表面生长 SiO₂ 作绝缘与掩膜基底  

- 晶圆上划片后得 **Die**；良率影响成本  
- 嵌入式 MCU 常用 8"/12" 线；高端 x86/ARM 服务器芯片用更先进制程

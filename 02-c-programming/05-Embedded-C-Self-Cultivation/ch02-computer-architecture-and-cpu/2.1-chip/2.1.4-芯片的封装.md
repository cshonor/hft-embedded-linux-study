# 2.1.4 芯片的封装

## 本节讲什么

**Die 封装成可焊接/插接的芯片** —— 引脚、散热、电气保护与嵌入式 BOM 选型。

## 要点

| 封装 | 特点 |
|------|------|
| **DIP/QFP** | 引脚外露，手工焊接友好 |
| **BGA/LGA** | 球栅/触阵，高密度（x86、SoC） |
| **QFN** | 底部散热焊盘，MCU 常见 |

- **封装 ≠ 内核**：同 Die 可不同封装（引脚数、电压域）  
- **热设计**：DPDK 跑满核 + 网卡 DMA → 需关注 **TDP 与散热片**  
- 引脚定义见 **Datasheet**：电源、GND、复位、JTAG、外设总线

# 2.4.3 为什么有些处理器没有 Cache

## 本节讲什么

**无 Cache / 仅 I-Cache 的 MCU** —— 何时刻意省略 Cache，以及对 C 代码的隐含约束。

## 要点

| 原因 | 场景 |
|------|------|
| **确定性延迟** | 工业控制、电机 PWM、硬实时 ISR |
| **面积/成本** | 8-bit / 低端 Cortex-M0 |
| **无 MMU** | 简单 Harvard MCU，外设与 RAM 分区固定 |

- 无 D-Cache：**DMA 与 CPU 共享 RAM 无一致性问题**（仍须软件屏障）  
- 有 D-Cache + DMA：**必须 flush/invalidate**，否则网卡/DMA 看到脏数据  
- DPDK 大页、UIO：x86 服务器几乎总有 Cache；嵌入式驱动必考 DMA 一致性

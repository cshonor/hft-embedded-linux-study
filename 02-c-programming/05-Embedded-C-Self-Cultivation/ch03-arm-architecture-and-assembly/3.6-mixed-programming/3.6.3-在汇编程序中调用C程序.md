# 3.6.3 在汇编程序中调用C程序

## 本节讲什么

启动代码、异常向量或裸机 `reset` 里用 **`bl main`** 跳入 C 世界；须先初始化 **栈 SP**、BSS，并遵守 AAPCS 传参。

## 要点

```arm
    .global _start
    .type _start, %function
_start:
    ldr sp, =stack_top      @ 必须：C 需要有效栈
    bl  main                @ 进入 C
    b   .                   @ main 返回后死循环

    .bss
    .space 4096
stack_top:
```

```arm
    mov r0, #3              @ main(int argc, ...) 可设 r0
    mov r1, #0
    bl  main
```

- `bl printf` 等同理；C 运行时可能依赖 `_start` → `__libc_start_main`
- 裸机链接脚本把 `stack_top` 放在 RAM 顶端
- U-Boot 跳内核前设好寄存器（ATAGs/DTB 指针曾用 r0–r2）

## 与后续衔接

- **ch04** 链接脚本 `ENTRY(_start)`；**ch10** OS 启动流程

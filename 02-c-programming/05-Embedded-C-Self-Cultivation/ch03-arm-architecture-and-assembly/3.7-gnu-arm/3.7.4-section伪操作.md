# 3.7.4 .section伪操作

## 本节讲什么

`.section` / `.text` / `.data` / `.bss` / `.rodata` 把代码与数据分入 ELF 段，链接脚本按段布置到 Flash/RAM。

## 要点

```arm
    .section .text.boot, "ax", %progbits
    .section .data.cacheline_aligned, "aw", %progbits
    .previous
    .pushsection .rodata
    .popsection
```

| 段 | 典型内容 |
|----|----------|
| `.text` | 代码 |
| `.rodata` | 常量、字符串 |
| `.data` | 已初始化全局变量 |
| `.bss` | 未初始化，占址不占 ROM |

- 内核 `__cacheline_aligned` 把 per-CPU 变量放入对齐段
- DPDK `__rte_cache_aligned` 同理，避免 false sharing

## 与后续衔接

- **ch04** 链接脚本 `SECTIONS`；**ch05** BSS 清零

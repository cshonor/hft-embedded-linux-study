# 3.7.2 GNU ARM编译器的伪操作

## 本节讲什么

GNU `as` 特有伪操作：控制 ARM/Thumb、literal 池、CPU 特性——写启动代码与内联汇编外围 `.S` 时常用。

## 要点

```arm
.syntax unified
.arm                    @ 后续为 ARM 32 位指令
.thumb                  @ Thumb-2
.ltorg                  @ 在此放置 literal pool
.cpu cortex-a7
.fpu vfpv3-d16
.arch armv7-a
```

| 伪操作 | 作用 |
|--------|------|
| `.ltorg` | 防止 `ldr =` 跳转过远 |
| `.arm` / `.thumb` | 切换指令集 |
| `.code 32` / `.code 16` | 同义切换 |
| `.fnstart` / `.fnend` | 栈展开元数据（异常处理） |

- 内核 `ENTRY(vector)` 宏展开含 `.type`、`.align`
- DPDK 裸机移植 seldom 手写，但读 U-Boot 需认识 `.arch`

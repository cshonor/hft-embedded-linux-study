# 3.7 GNU ARM汇编语言

## 本节讲什么

Linux 工具链（`arm-linux-gnueabihf-gcc/as/ld`）使用 **GNU 汇编语法**：与 Keil/ADS 在伪操作、标号、注释上有所不同。内核、Bootloader、DPDK 平台代码均以此为准。

## 要点

| 主题 | 内容 | 小节 |
|------|------|------|
| 编译流程 | `.S` → `as` → `.o` | **3.7.1** |
| 伪操作 | `.ltorg` `.arm` `.thumb` | **3.7.2** |
| 标号 | 局部 `1:` `b 1b` | **3.7.3** |
| 段 | `.section` `.pushsection` | **3.7.4** |
| 数据格式 | `@` 注释、对齐 | **3.7.5** |
| 数据定义 | `.int` `.asciz` | **3.7.6** |
| 实战 | `objdump -dS` | **3.7.7** |

```bash
arm-linux-gnueabihf-gcc -c add_asm.S -o add_asm.o
arm-linux-gnueabihf-objdump -d add_asm.o
```

- 预处理：大写 `.S` 走 `cpp`；小写 `.s` 不预处理
- 与 **demo/Makefile** 中 `$(CC) -c add_asm.S` 一致

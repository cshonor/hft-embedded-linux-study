# 3.7.3 GNU ARM汇编语言中的标号

## 本节讲什么

GNU 标号规则：局部标号 `1:`/`2:`、向前/向后引用 `1f`/`1b`，减少无意义全局符号。

## 要点

```arm
1:  subs r0, r0, #1
    bne 1b                 @ 向后跳到标号 1
2:  add r1, r1, #4
    cmp r1, r2
    blt 2b
```

- **局部标号**可重复；用 `3f` 指向下一个 `3:`，`3b` 指向上一个
- 全局标号须以字母或 `_` 开头；数字单独为局部
- `ENTRY(irq_handler)` 生成全局 `irq_handler` 供向量表引用

# 3.7.6 数据定义

## 本节讲什么

在汇编中 **静态分配** 数据：字符串、表、中断向量常数——与 C 全局对象一样进入 `.data`/`.rodata`。

## 要点

```arm
    .data
gpio_table:
    .word 0x1000, 0x1004, 0x1008
msg:
    .asciz "Hello\n"
    .align 2
count:
    .int 42
    .skip 16, 0xFF
```

| 伪操作 | 含义 |
|--------|------|
| `.int` / `.long` | 32 位字 |
| `.asciz` | 以 0 结尾字符串 |
| `.skip n, fill` | 预留 n 字节 |
| `.incbin "blob.bin"` | 嵌入二进制 |

- 向量表：`.word reset, undef, swi, prefetch, data, irq, fiq`
- 与 C `const uint32_t gpio_table[]` 可 `extern` 互引

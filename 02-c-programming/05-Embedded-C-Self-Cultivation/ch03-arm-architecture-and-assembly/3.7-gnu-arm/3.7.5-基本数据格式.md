# 3.7.5 基本数据格式

## 本节讲什么

GNU ARM 汇编中 **常数书写、字节序、对齐** 约定；与 C 类型宽度对应，避免 `ldrb`/`ldr` 误用。

## 要点

```arm
.word 0x12345678        @ 32 位，小端内存：78 56 34 12
.short 0xABCD
.byte 0xFF
.quad 0x1234            @ 64 位（AArch64/NEON 上下文）
```

- **@** 行注释；`#` 为立即数前缀（`mov r0, #1`）
- `.align 2` → 2^2 = 4 字节边界；结构体字段对齐影响加载指令选择
- 设备寄存器 32 位宽用 `ldr`；字节设备用 `ldrb` + `volatile` C 封装

# 3.2.6 条件执行指令

## 本节讲什么

ARM32 几乎所有指令可带 **条件码后缀**，仅在 CPSR 标志满足时执行——用 `cmp` + 条件 `mov` 代替分支，减少流水线冲刷（AArch64 已弱化此特性）。

## 要点

```arm
cmp r0, r1
movgt r2, #1          @ r0>r1（有符号）时 r2=1
moveq r2, #0          @ 相等时 r2=0
addeq r0, r0, #1      @ Z=1 时加 1
```

| 后缀 | 条件 |
|------|------|
| eq/ne | 相等/不等 |
| gt/lt/ge/le | 有符号比较 |
| hi/ls | 无符号高于/低于 |
| mi/pl | 负/正 |

- 默认后缀 **AL**（总是执行）可省略
- 内核早期手写汇编常用；现代 C 编译器仍对简单 `cmov` 模式生成条件指令

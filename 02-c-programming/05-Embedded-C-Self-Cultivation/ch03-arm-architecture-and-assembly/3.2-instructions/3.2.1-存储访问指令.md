# 3.2.1 存储访问指令

## 本节讲什么

`LDR`/`STR` 在寄存器与内存间搬运数据；`LDM`/`STM` 一次访问多个寄存器，用于函数序言/尾声和中断现场保存。

## 要点

```arm
ldr r0, [r1]              @ r0 = *r1
ldr r0, [r1, #4]          @ 基址+偏移
str r0, [r1, #4]!         @ 写后 r1 += 4（前变址）
ldrb r0, [r1]             @ 字节加载
ldrh r0, [r1]             @ 半字

stmfd sp!, {r4-r11, lr}   @ 入栈：满递减栈
ldmfd sp!, {r4-r11, pc}   @ 出栈并返回（pc 代替 lr）
```

- **B**：字节；**H**：半字；无后缀为字（32 位）
- `!` 表示写回基址；`{}` 内寄存器列表顺序影响存储布局
- 内核异常入口常用 `stmfd sp, {r0-r12, lr}` 压栈

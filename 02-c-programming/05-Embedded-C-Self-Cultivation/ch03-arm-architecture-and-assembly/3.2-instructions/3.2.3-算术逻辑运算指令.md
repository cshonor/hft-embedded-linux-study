# 3.2.3 算术逻辑运算指令

## 本节讲什么

`ADD`/`SUB`/`AND`/`ORR`/`EOR` 等在寄存器上完成算术与位运算；可带 **S** 后缀更新 CPSR 标志位，供条件执行使用。

## 要点

```arm
add r0, r1, r2        @ r0 = r1 + r2
adds r0, r1, #1       @ 加并更新 N,Z,C,V
sub r2, r1, r3, lsr #1 @ r2 = r1 - (r3>>1)
and r0, r0, #0xFF     @ 掩码低 8 位
orr r0, r0, r1        @ 按位或
eor r0, r0, r1        @ 按位异或
```

- 三操作数格式：`OP Rd, Rn, operand2`
- **S** 后缀：结果写 CPSR；`cmp` 本质是 `subs` 丢弃写回（**3.2.5**）
- DPDK 位图、掩码操作在反汇编中常见 `and`/`orr`

## 与后续衔接

- 带条件：`addeq` 等 **3.2.6**

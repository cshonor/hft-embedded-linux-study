# 3.2.4 操作数：operand2详解

## 本节讲什么

ARM 数据 processing 指令的第三操作数 **operand2** 可以是立即数、寄存器，或 **带移位的寄存器**——一条指令内完成「运算+移位」，减少指令条数。

## 要点

```arm
add r0, r1, #0x100        @ 立即数（8 位旋转）
add r0, r1, r2            @ 寄存器
add r0, r1, r2, lsl #2    @ r2 逻辑左移 2 位后相加
add r0, r1, r2, lsr #4    @ 逻辑右移
add r0, r1, r2, asr #1    @ 算术右移
add r0, r1, r2, ror #8    @ 循环右移
```

- 立即数：8 位值 + 4 位旋转量；**不是任意 32 位数** → 用 `ldr =label`
- 移位量 0–31；`lsl #32` 等价为 0
- 编译器调度：`*8` 常编译为 `lsl #3`

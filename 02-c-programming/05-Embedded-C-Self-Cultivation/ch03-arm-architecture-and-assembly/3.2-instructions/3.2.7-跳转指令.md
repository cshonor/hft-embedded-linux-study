# 3.2.7 跳转指令

## 本节讲什么

`B`/`BL`/`BX` 改变 **PC** 流向：本地分支、函数调用、Thumb/ARM 状态切换。`BL` 将返回地址写入 **LR（R14）**。

## 要点

```arm
b   loop              @ 无条件跳转
beq done              @ 条件分支
bl  printf           @ 调用：lr = 下条指令地址
bx  lr                @ 返回到 lr（可切换 Thumb）

@ demo/add_asm.S 返回范式
add r0, r0, r1
bx  lr
```

| 指令 | 作用 |
|------|------|
| B | 分支，不保存 LR |
| BL | 分支并链接（调用） |
| BX | 跳转并可切换 T 位 |

- **BLX** 可调用 Thumb 函数；GNU 汇编 `.thumb`/`.arm` 指定状态
- 尾调用优化：有时用 `b func` 代替 `bl` 省栈帧

## 与后续衔接

- **3.6.1** 被调者 `bx lr`；**3.6.3** 汇编调 C 用 `bl main`
- **ch04** 重定位类型 `R_ARM_CALL` 解析 `bl` 目标

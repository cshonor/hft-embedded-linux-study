# 3.2 ARM汇编指令

## 本节讲什么

ARM32 指令按功能分为 **存储访问、数据传送、算术逻辑、比较、条件执行、跳转** 六类。运算遵循 load-store：先 `ldr` 取数，再 `add`/`cmp`，最后 `str` 写回。读懂这些指令，才能对照 `objdump -d` 反汇编内核/DPDK 热点函数。

## 要点

### 指令分类总览

| 类别 | 代表指令 | 小节 |
|------|----------|------|
| 存储访问 | `ldr` `str` `ldm` `stm` | **3.2.1** |
| 数据传送 | `mov` `mvn` | **3.2.2** |
| 算术逻辑 | `add` `sub` `and` `orr` `eor` | **3.2.3** |
| 操作数扩展 | `#imm` `Rm` 移位 | **3.2.4** |
| 比较 | `cmp` `cmn` `tst` | **3.2.5** |
| 条件执行 | `addeq` `movne` … | **3.2.6** |
| 跳转 | `b` `bl` `bx` | **3.2.7** |

### 典型片段：两数相加写回

```arm
ldr r1, [r0]          @ 从 r0 指向的地址读
add r2, r1, #1        @ r2 = r1 + 1
str r2, [r0]          @ 写回
```

### 调用约定预览

```arm
mov r0, #3
mov r1, #5
bl  add_asm           @ 见 demo/add_asm.S
                      @ r0 = 返回值，bx lr 返回
```

### 内核/DPDK 常见模式

- **批量访存**：`ldm`/`stm` 保存/恢复寄存器现场（**3.2.1**、**3.9**）
- **无分支优化**：`cmp` + 条件 `mov` 代替 `b`（**3.2.6**）
- **内联汇编**：`asm volatile` 中常用 `mov`/`mcr`/`mrc`（**3.6.2**）

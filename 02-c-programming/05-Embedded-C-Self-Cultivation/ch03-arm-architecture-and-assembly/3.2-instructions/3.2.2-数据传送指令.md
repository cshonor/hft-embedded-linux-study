# 3.2.2 数据传送指令

## 本节讲什么

`MOV`/`MVN` 在寄存器间或立即数到寄存器之间传送数据，不访问内存——load-store 架构中最常用的「赋值」手段。

## 要点

```arm
mov r0, #10           @ r0 = 10（8 位旋转立即数）
mov r0, r1            @ r0 = r1
mov r0, r1, lsl #2    @ r0 = r1 << 2
mvn r0, #0            @ r0 = ~0 = 0xFFFFFFFF
```

- **MOV** 目标、源；源可为 **operand2**（**3.2.4**）
- 立即数受 ARM 编码限制，大常数用 **3.4.1 LDR 伪指令**
- `MOV PC, LR` 已废弃；返回用 `bx lr` 或 `pop {pc}`

## 与后续衔接

- 参数装载：`mov r0, #n` 配合 **3.6.1** AAPCS
- 内联汇编 `asm("mov %0, #1" : "=r"(x))` 见 **3.6.2**

# 3.2.5 比较指令

## 本节讲什么

`CMP`/`CMN`/`TST`/`TEQ` 做减法或位运算但不写回结果，只更新 **CPSR 标志位**，驱动后续条件执行或分支。

## 要点

```arm
cmp r0, r1            @ 等价 subs r0, r0, r1（丢弃写回）
cmp r0, #10
cmn r0, r1            @ 比较 -r1（adds 不写回）
tst r0, #0x1          @ 测试最低位（ands 不写回）
beq label             @ Z=1 则跳转
bgt label             @ 有符号大于
```

| 标志 | 含义 |
|------|------|
| N | 负数 |
| Z | 零 |
| C | 进位/无符号借位 |
| V | 有符号溢出 |

## 与后续衔接

- 条件码后缀 **3.2.6**；分支 **3.2.7**
- C 中 `if (a > b)` 编译为 `cmp` + `ble`/`bgt`

# 3.5 ARM汇编程序设计

## 本节讲什么

从单条指令上升到 **完整汇编源文件**：段布局、标号、伪操作、与 C 的符号衔接。掌握程序结构才能编写 `boot.S`、内核入口和 DPDK 平台相关启动片段。

## 要点

### 源文件三要素

1. **格式**：`.global`、`.type`、标号、指令（**3.5.1**）
2. **符号**：局部/全局标号、外部引用（**3.5.2**）
3. **伪操作**：`.text`、`.data`、`.word` 等（**3.5.3**）

```arm
    .text
    .global add_asm
    .type add_asm, %function
add_asm:
    add r0, r0, r1
    bx  lr
```

- 与 `demo/add_asm.S` 同构：导出符号供 C `extern` 链接
- 注释用 `@`（ARM）或 `/* */`；换行续行用 `\`

# 3.5.1 ARM汇编程序格式

## 本节讲什么

GNU ARM 汇编源文件的标准骨架：指令行、标号、注释、指令与伪操作的分工。

## 要点

```arm
@ 文件头注释
    .syntax unified          @ 可选：统一 ARM/Thumb 语法
    .cpu cortex-a9           @ 目标 CPU
    .fpu neon

    .text
    .align 2                  @ 4 字节对齐
    .global _start
    .type _start, %function
_start:
    mov r0, #0
    bx  lr
```

- **标号**单独一行或以 `:` 结尾；**指令**必须缩进（常以 Tab）
- `.global` 导出链接可见；未导出为局部符号
- 内联在 C 里写汇编时通常省略段伪操作，由编译器放入 `.text`

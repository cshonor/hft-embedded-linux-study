# 3.5.3 伪操作

## 本节讲什么

以 `.` 开头的 **汇编器指令**（非 CPU 指令）：定义段、对齐、数据、宏，把源文件组织成链接器可消费的 `.o`。

## 要点

```arm
    .text
    .data
val:
    .word 0x12345678
    .byte 0x12
    .ascii "OK\0"
    .space 64, 0
    .end
```

| 伪操作 | 作用 |
|--------|------|
| `.text` / `.data` / `.bss` | 段切换 |
| `.align n` | 2^n 字节对齐 |
| `.word` / `.byte` | _emit 数据 |
| `.equ SYM, 0` | 符号等价常量 |

- **3.7** 展开 GNU 专用伪操作（`.section`、`.ltorg`）
- 设备树 blob、中断向量表在 `.rodata`/`.text` 用 `.word` 填入口地址

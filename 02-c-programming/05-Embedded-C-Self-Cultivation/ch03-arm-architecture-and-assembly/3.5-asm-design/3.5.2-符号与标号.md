# 3.5.2 符号与标号

## 本节讲什么

标号代表地址或数据位置；**全局符号**参与链接，**局部符号**仅文件内可见。C 与汇编互调依赖一致的符号命名与类型。

## 要点

```arm
    .global add_asm          @ C: extern int add_asm(int,int);
add_asm:                     @ 函数入口地址
local_loop:                  @ 局部，不导出
    b   local_loop

    .extern printf          @ 引用 C 库
    bl  printf
```

| 概念 | 说明 |
|------|------|
| 标号 | 代表地址，供 `b`/`bl`/`ldr` 引用 |
| `.global` | 全局符号，链接器可解析 |
| `.extern` | 本文件未定义，他处提供 |
| `.type` | 区分函数/对象，利于调试与链接 |

- 内核 `EXPORT_SYMBOL(foo)` 与模块链接同理
- 命名：C 的 `add_asm` 汇编侧勿加多余下划线（ELF ARM 默认不 mangling）

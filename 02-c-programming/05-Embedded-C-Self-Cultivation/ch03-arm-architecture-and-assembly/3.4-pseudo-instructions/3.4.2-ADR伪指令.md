# 3.4.2 ADR伪指令

## 本节讲什么

`ADR Rd, label` 把 **label 与当前指令的偏移地址** 写入 Rd，范围约 ±4KB（ARM），适合 PIC 中取函数/数据地址而不经 literal pool。

## 要点

```arm
adr r0, local_label     @ r0 = PC 相对地址
adr r1, my_func
blx r1                  @ 间接调用
```

- 与 `ldr rd, =label` 区别：`ADR` 算 **相对 PC**；`LDR =` 常经内存常数池
- 超出范围汇编器报错，需改用 `ldr =` 或链接重定位
- 内核 per-CPU 数据有时用 `adr` 取近端表项

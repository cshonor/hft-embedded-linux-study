# 3.4.1 LDR伪指令

## 本节讲什么

`LDR Rd, =imm/label` 加载 **无法用 mov # 编码** 的 32 位立即数或符号地址；汇编器在附近插入 literal pool 并用 `ldr [pc, #off]` 取出。

## 要点

```arm
ldr r0, =0x12345678
ldr r1, =my_buffer
ldr r2, =main
```

- 展开示例：`ldr r0, [pc, #constant_offset]`
- 池太远时需 `.ltorg` 强制就近放置（**3.7.2**）
- 加载 **地址** 而非内容：`ldr r1, =buf`；取内容另需 `ldr r0, [r1]`

# 3.4 ARM伪指令

## 本节讲什么

伪指令由汇编器展开为一条或多条真实 ARM 指令，解决 **大立即数加载** 和 **小范围地址计算** 等编码限制，程序员仍写一条「看起来简单」的语句。

## 要点

| 伪指令 | 展开目的 | 小节 |
|--------|----------|------|
| `LDR Rd, =expr` | 任意 32 位常数/符号地址 | **3.4.1** |
| `ADR Rd, label` | PC 相对小偏移地址 | **3.4.2** |

```arm
ldr r0, =0x12345678       @ → literal pool + ldr [pc, #off]
adr r0, my_func           @ → add/sub 与 PC 相关
```

- GNU 汇编（`arm-linux-gnueabihf-as`）与 Keil 语法略有差异，本章以 **GNU** 为准（**3.7**）
- 内联汇编中尽量用 C 变量约束，少手写 `ldr =`

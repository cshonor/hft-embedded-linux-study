# 3.3.1 寄存器寻址

## 本节讲什么

操作数直接来自寄存器，不访问内存——ARM 运算类指令的默认方式，速度最快。

## 要点

```arm
add r0, r1, r2        @ 三个寄存器
mov r3, r1            @ 寄存器复制
mul r0, r1, r2        @ r0 = r1 * r2
```

- R0–R12 通用；R13 SP、R14 LR、R15 PC 有特殊含义（**3.1**）
- 编译器尽量把局部变量保持在寄存器，减少 `ldr`/`str`

# 3.3.7 相对寻址

## 本节讲什么

以 **PC** 为基的偏移访问，用于位置无关代码（PIC）、literal pool 取常数、与当前指令附近的只读数据。

## 要点

```arm
ldr r0, [pc, #offset]     @ 从 PC 附近 literal 池加载
adr r0, label             @ 伪指令：label 与 PC 的偏移（3.4.2）
```

- ARM 流水线：读 PC 时通常 **PC = 当前指令 + 8**（ARM 状态）
- 共享库、内核模块加载地址随机化依赖 PC 相对重定位（**ch04**）
- `ldr r0, =0x12345678` 汇编器在附近放常数池并用相对寻址加载

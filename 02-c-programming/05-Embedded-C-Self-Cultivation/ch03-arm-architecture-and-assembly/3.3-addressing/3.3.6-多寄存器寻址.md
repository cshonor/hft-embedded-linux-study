# 3.3.6 多寄存器寻址

## 本节讲什么

`LDM`/`STM` 用 `{r0-r3}` 列表一次传送多个寄存器——函数保存现场、中断入栈、块拷贝的高效手段。

## 要点

```arm
stmfd sp!, {r4-r11, lr}    @ 入栈（满递减，先减后存）
ldmfd sp!, {r4-r11, pc}    @ 出栈返回
ldmia r0!, {r2-r5}         @ 从 r0 批量加载，r0 递增
```

- **IA/IB/DA/DB** + **FD/FA/ED/EA**：递增/递减、前/后变址
- `fd` + `sp` 为 GNU/Linux 常用栈约定
- 寄存器列表中 **编号低者→低地址**（`stmfd` 时）

# 3.3.5 基址寻址

## 本节讲什么

`[Rn, #offset]` 等形式：基址 + 立即数或寄存器偏移；`!` 写回、`]` 后偏移等变体控制指针是否前进。

## 要点

```arm
ldr r0, [r1, #8]          @ 结构体字段偏移
ldr r0, [r1], #4          @ 读后 r1 += 4（后变址）
str r0, [r1, #4]!         @ 写前 r1 += 4（前变址）
ldr r0, [sp, #12]         @ 栈帧局部变量
```

| 形式 | 效果 |
|------|------|
| `[Rn, #imm]` | 地址不变 |
| `[Rn], #imm` | 访问后 Rn += imm |
| `[Rn, #imm]!` | 先 Rn += imm 再访问 |

- 函数栈帧中 `fp`/`sp` + 偏移访问参数与局部变量（**3.6.1**）

## 与后续衔接

- **ch05** 栈布局与 `[sp, #off]` 一一对应

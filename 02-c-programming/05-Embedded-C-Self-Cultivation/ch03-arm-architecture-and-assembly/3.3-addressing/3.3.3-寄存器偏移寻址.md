# 3.3.3 寄存器偏移寻址

## 本节讲什么

内存地址 = 基址寄存器 + **经移位的** 索引寄存器，适合数组、动态偏移结构体访问。

## 要点

```arm
ldr r0, [r1, r2]              @ addr = r1 + r2
ldr r0, [r1, r2, lsl #2]      @ addr = r1 + (r2<<2)，int 数组
str r3, [r4, r5, lsr #2]
```

- 仅 **load/store** 使用；移位类型：LSL/LSR/ASR/ROR
- C 代码 `arr[i]` 常编译为 `r_base + (r_i << 2)`

# 3.6 函数实参求值顺序

## 本节讲什么

C 标准 **不规定** 函数实参的求值顺序（C17 前）。同一实参列表里 **多次修改同一变量** → **UB**。

## 要点

```c
int i = 1;
func(i++, i);  /* UB：结果 (1,2) 或 (2,1) 皆可能 */
```

与 [ch02 2.5](../ch02-syntactic-pitfalls/2.5-逗号运算符与分隔符.md) 实参逗号 **非序列点** 一致；[Expert C ch08](../04-Expert-C-Programming/ch08-halloween-vs-christmas/) Christmas 问题同源。

### 工程规范

- 实参 **无副作用**：不自增、不赋值同一变量
- 复杂计算 **拆成临时变量** 再传入

```c
int a = i++;
int b = i;
func(a, b);
```

# 3.7 数组下标越界

## 本节讲什么

C **不做** 数组边界检查。越界读写是 **UB**，常表现为 **覆盖栈/堆相邻数据**、随机崩溃或静默数据损坏。

## 要点

```c
int buf[4];
buf[5] = 100;  /* UB：可能改相邻栈变量 */
```

### 危害场景

| 场景 | 后果 |
|------|------|
| HFT 报文解析 | 订单字段被篡改 |
| 内核栈缓冲区 | 历史提权漏洞类 |
| 嵌入式栈小 | 栈溢出复位 |

### 缓解

- 所有访问 **`index < len`** 显式检查
- `memcpy`/`strcpy` 传 **目标容量**
- 工具：ASan、valgrind（用户态）

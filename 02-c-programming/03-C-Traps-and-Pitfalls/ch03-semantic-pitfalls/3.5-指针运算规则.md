# 3.5 指针运算规则

## 本节讲什么

指针 `+`/`-` 的单位是 **指向类型的大小**，不是 1 字节。`void*` **不能** 直接算术（C99 前）；需转为 `char*` 或具体类型。

## 要点

```c
int *p = ...;
p + 1;    /* +sizeof(int) */

char *cp = ...;
cp + 1;   /* +1 字节 */

void *vp = ...;
/* vp + 1; 非法 */
(char *)vp + 1;  /* 字节遍历内存 */
```

### 同类型指针相减

仅当指向 **同一数组** 内元素时有定义：

```c
int a[5];
int *p1 = &a[0], *p2 = &a[3];
ptrdiff_t d = p2 - p1;  /* 3 */
```

跨对象相减 → UB。

### 与数组

`arr[i]` ≡ `*(arr + i)` — 步长由元素类型决定（[ch02 2.6](../ch02-syntactic-pitfalls/2.6-数组与结构体语法.md)）。

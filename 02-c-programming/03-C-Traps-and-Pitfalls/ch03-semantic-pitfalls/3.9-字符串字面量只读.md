# 3.9 字符串字面量只读

## 本节讲什么

字符串字面量存储在 **只读段（`.rodata`）**。`char *p = "hello"` 后 **修改 `p[i]`** 为 UB，常 **SIGSEGV**。

## 要点

```c
char *p = "hello";
p[0] = 'H';        /* UB / 段错误 */

char arr[] = "hello";
arr[0] = 'H';      /* 合法：栈/数据段副本 */
```

### 工程规范

- 只读文本用 **`const char *`**
- 需要修改 → **`char buf[]`** 或 `malloc` + 拷贝

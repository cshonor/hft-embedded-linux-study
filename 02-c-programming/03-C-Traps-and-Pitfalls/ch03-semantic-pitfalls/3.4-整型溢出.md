# 3.4 整型溢出

## 本节讲什么

**无符号溢出**：标准定义为 **模 2^N 回绕**。**有符号溢出**：标准规定为 **UB**，编译器可假定不发生并激进优化。

## 要点

### 对比

| 类型 | 溢出 |
|------|------|
| `unsigned` | 定义行为：`UINT_MAX + 1 → 0` |
| `signed` | **UB**；`-O2` 下可能删「不可能」分支 |

```c
int x = INT_MAX;
x + 1;  /* UB */
unsigned u = UINT_MAX;
u + 1;  /* 0，定义明确 */
```

### 典型场景

- 计数器、金额、偏移用 `int` 累加越界 → 逻辑错乱
- 编译器见 `if (x + 1 < x)` 推断 `x+1` 溢出不可能，**删掉检查**

### 工程建议

- 可能溢出用 **`long long`** / **`int64_t`** 或库函数检测
- 无符号长度/索引用 **`size_t`**
- 金融/HFT：固定点或 decimal，避免裸 `float`/`int` 混用

# 6.7 条件编译与注释陷阱

## 本节讲什么

预处理指令 **不被 C 注释屏蔽**。把 `#ifdef` 写在 `/* ... */` 或 `//` 注释里，预处理器 **仍会扫描** 并处理，导致意外分支或语法残片。

## 要点

```c
/*
#ifdef DEBUG
    printf("log\n");
#endif
*/
/* 注释无法“注释掉”预处理指令 */
```

### 正确禁用大段代码

```c
#if 0
    /* 任意代码、甚至非法语法，均不进入编译 */
    broken experimental();
#endif
```

### `#ifdef` vs `#if`

```c
#define DEBUG
#ifdef DEBUG    /* 只问“是否定义”，空宏也算真 */
#endif

#define DEBUG 1
#if DEBUG       /* 可区分 0/1 */
#endif
```

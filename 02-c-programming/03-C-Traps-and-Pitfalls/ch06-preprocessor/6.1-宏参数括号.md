# 6.1 宏参数不加括号，优先级错乱

## 本节讲什么

宏是 **纯文本替换**，不参与 C 语法分析，**不会** 给参数自动加括号。参数表达式若含 `+`、`*`、移位等运算符，展开后优先级与结合性与直觉不符。

## 要点

```c
#define SQUARE(x) x * x
SQUARE(a + 1);
/* 展开：a + 1 * a + 1  →  a + (1*a) + 1，不是 (a+1)² */
```

### 标准写法

```c
#define SQUARE(x) ((x) * (x))
```

- **参数** 加括号：`(x)`
- **整体** 加括号：`((x) * (x))`，避免嵌入更大表达式时优先级再错

### 类似陷阱

```c
#define DOUBLE(x) x + x        /* 错 */
#define DOUBLE(x) ((x) + (x))  /* 对 */
```

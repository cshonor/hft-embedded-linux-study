# 6.9 空宏与未定义宏判断

## 本节讲什么

`#ifdef MACRO` / `#ifndef MACRO` 只检测 **是否被 `#define` 过**，**不** 检查宏体是否为空、是否为 0。开关型宏需用 `#if` 显式数值判断。

## 要点

```c
#define DEBUG          /* 空宏 */
#ifdef DEBUG
    /* 始终编译进来 */
#endif
```

### 推荐开关写法

```c
#define DEBUG 1        /* 1=开 0=关 */

#if DEBUG
    fprintf(stderr, "trace\n");
#endif
```

或构建系统传递：`-DDEBUG=0` / `-DDEBUG=1`。

### `#defined` 运算符

```c
#if defined(DEBUG) && DEBUG
```

避免在 `#if` 表达式中裸写标识符（未定义时行为与 `defined()` 不同）。

### `defined` 与 `#ifdef`

| 写法 | 含义 |
|------|------|
| `#ifdef FOO` | FOO 已定义（含空定义） |
| `#if defined(FOO)` | 同 `#ifdef` |
| `#if FOO` | FOO 须为 **整常量** 宏，未定义则视为 0 |

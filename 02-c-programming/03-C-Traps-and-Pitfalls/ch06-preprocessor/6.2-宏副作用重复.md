# 6.2 宏参数含自增/赋值，多次副作用

## 本节讲什么

即使参数已加括号，宏 **仍会重复展开** 同一实参文本。实参含 `++`、`--`、赋值、函数调用等 **有副作用** 表达式时，副作用执行次数不可控。

## 要点

```c
#define SQUARE(x) ((x) * (x))
int i = 2;
int res = SQUARE(i++);
/* 展开：((i++) * (i++)) — i 自增两次，res 不可预测 */
```

### 规范

- **禁止** 向宏传入含副作用的表达式
- 能写 **`static inline` 函数** 就不写宏（C99+）
- 内核等必须宏的场景：文档标明「参数求值多次」

### inline 对比

```c
static inline int square(int x) { return x * x; }
square(i++);  /* i 只自增一次 */
```

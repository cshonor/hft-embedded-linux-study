# 6.4 字符串化 `#` 与连接 `##`

## 本节讲什么

`#` 将宏参数 **字符串化**；`##` 在预处理阶段 **粘贴** 两个 token 成一个标识符。二者仅在宏体内、预处理期生效，易与运行时字符串混淆。

## 要点

### `#` 字符串化

```c
#define PRINT_VAR(x) printf(#x " = %d\n", x)
PRINT_VAR(num);
/* → printf("num" " = %d\n", num);  相邻字符串字面量自动拼接 */
```

坑：参数含空格、逗号会 **原样** 进入字符串；`PRINT_VAR(a + b)` → `"a + b"`，不是 `"a+b"`。

### `##` token 粘贴

```c
#define CAT(a, b) a##b
int val1 = 10;
printf("%d\n", CAT(val, 1));   /* val1 */
```

限制：

- 结果必须是 **合法标识符**
- 不能随意拼数字开头、关键字
- 调试困难，可读性差

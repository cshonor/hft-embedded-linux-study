# 6.8 宏替代函数无类型检查

## 本节讲什么

宏 **不建立函数作用域**，不对实参做类型检查；任意类型 token 均可填入，比较、赋值语义由 **展开后的 C 表达式** 决定，易静默逻辑错误。

## 要点

```c
#define MAX(a, b) ((a) > (b) ? (a) : (b))

MAX("abc", 123);
/* 指针与整数比较 → 无报错，结果无意义 */
```

### 指针陷阱

```c
#define PTR_MAX(p, q) ((p) > (q) ? (p) : (q))
/* 比较的是地址大小，不是指向内容 */
```

### 优先 inline

```c
static inline int max_int(int a, int b) { return a > b ? a : b; }
```

泛型需求：C11 `_Generic`、C++ 模板、或多次重载宏（内核 `min`/`max` 风格）。

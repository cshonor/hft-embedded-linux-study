# 6.6 头文件重复包含防护

## 本节讲什么

同一头文件被多条 `#include` 链 **重复展开** 时，无防护则结构体、枚举、函数声明、**误写的全局定义** 重复出现 → 编译错误或链接多重定义。

## 要点

### 标准 include guard

```c
#ifndef MYMODULE_H
#define MYMODULE_H
/* 内容 */
#endif
```

### `#pragma once`

多数现代编译器支持，简洁；**老嵌入式工具链** 可能不支持，可移植项目优先 `#ifndef`。

### 与 ch04 衔接

[ch04 4.8](../ch04-linking/4.8-头文件保护.md) 从链接角度；本章从 **预处理重复展开** 角度。

### 循环包含

A.h → B.h → A.h：guard 使第二次跳过内容，打破循环。

# 4.6 强符号与弱符号

## 本节讲什么

GNU ld 规则：**强符号** 不能重复；**弱符号** 可与强符号共存；多弱同名取其一。多个文件 **`int cnt;`（tentative）** 可能 **合并为同一全局**——模块间 **静默共享** 计数器。

## 要点

| 符号 | 示例 |
|------|------|
| **强** | `int x = 1;`、函数定义 |
| **弱** | `__attribute__((weak)) int x;`；传统 tentative `int x;`（GCC 10+ `-fno-common` 行为见 Expert C ch05） |

规则：

1. 两 **强** 同名 → `multiple definition`
2. 一 **强** 多 **弱** → 用 **强**
3. 全 **弱** → 任选一个

### 坑

多 `.c` 各写 `int event_cnt;`（无 static）→ 可能 **链接成一份**，你以为每模块独立计数。

### 修复

- 模块私有：`static int event_cnt;`
- 真共享：头文件 `extern int event_cnt;`，**一个** `.c` 定义

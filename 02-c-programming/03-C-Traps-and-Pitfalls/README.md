# 《C 陷阱与缺陷（第 2 版）》

**C Traps and Pitfalls, 2nd Edition** — Andrew Koenig

## 定位

阶段 1 · C 语言常见陷阱。从词法、语法、语义、连接、库函数、预处理器、可移植性等角度，分析「看起来合法、运行出怪事」的典型问题。

## 阅读建议

在 K&R 建立语法基础后阅读；篇幅不大，适合快速扫坑，避免把 C++ 习惯带进 C 时踩雷。

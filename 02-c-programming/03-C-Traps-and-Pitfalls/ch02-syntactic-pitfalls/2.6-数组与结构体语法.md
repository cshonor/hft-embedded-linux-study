# 2.6 数组下标与结构体成员

## 本节讲什么

下标、成员访问的语法糖背后是指针运算；**优先级** 决定 `&st.member` 如何分组。可读性陷阱 `i[a]` 语法合法但应禁止。

## 要点

### 下标等价

```c
a[i]  等价  *(a + i)
i[a]  等价  *(i + a)  /* 语法合法：交换律，禁止在工程代码中使用 */
```

[Expert C ch04](../04-Expert-C-Programming/ch04-arrays-are-not-pointers/) 强调数组 ≠ 指针，但 **下标运算符** 仍是指针算术语法糖。

### 结构体成员优先级

| 写法 | 解析 |
|------|------|
| `st.member` | 成员访问 |
| `p->member` | `(*p).member` |
| `&st.member` | `&(st.member)` — **`.` 高于 `&`** |
| `&p->member` | 正确取成员地址 |

错误理解：`*p.member` 等价 `*(p.member)`（把 member 当指针）—— 实际是 `*(p->member)` 需看 p 类型。

# 2.5 逗号：分隔符 vs 逗号运算符

## 本节讲什么

同一个 `,` 在不同语法位置含义不同：**分隔符**（无组合值） vs **逗号运算符**（从左求值，结果为 **最右** 表达式）。

## 要点

### 逗号运算符

```c
int x = (1, 2, 3);   /* x = 3 */
if (a++, b < 100)    /* 先 a++，再判断 b<100 */
```

### 分隔符（不是逗号运算符）

| 位置 | 作用 |
|------|------|
| 函数实参 `f(a, b)` | 分隔参数；**求值顺序未指定**（C17 前） |
| 声明 `int a, b;` | 声明多个变量 |
| `for (i=0, j=10; ...; i++, j--)` | 初始化/步进里的 **逗号运算符**（合法） |

### 易错

```c
func(i++, ++i);  /* 实参逗号：非序列点，同一变量多次修改 → UB */
```

见 [Expert C ch08](../04-Expert-C-Programming/ch08-halloween-vs-christmas/) demo05。

### 工程建议

- 复杂表达式 **不用逗号运算符** 拼逻辑
- 实参列表 **不** 在多个参数里改同一变量
- `for` 里 `i++, j--` 清晰可读时可用；业务逻辑不要塞进条件

# 2.4 括号缺失与语句体

## 本节讲什么

`if` / `for` / `while` **没有 `{}` 时，语法上仅紧跟的「一条语句」属于体**；下一行已是独立语句，不受控制——与缩进无关。

## 要点

### 经典错误

```c
if (flag)
    a = 100;
    b = 200;   /* 永远执行！不属于 if */
```

等价于：

```c
if (flag)
    a = 100;
b = 200;
```

### 宏展开叠加（ch06 预告）

```c
#define INC(x) x++; x++
if (ok)
    INC(a);   /* 展开后多条语句，仅第一条受 if 控制 */
```

见 **ch06 预处理器**。

### 修复

```c
if (flag) {
    a = 100;
    b = 200;
}
```

### 工程规范

- **Mandatory braces**：内核风格、MISRA、多数 HFT 规范要求控制语句必带 `{}`
- 调试时在 `if` 下加 `log` 行，忘加 `{}` 极易引入新 bug

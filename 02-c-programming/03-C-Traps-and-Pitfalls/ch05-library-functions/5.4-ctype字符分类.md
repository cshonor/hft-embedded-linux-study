# 5.4 字符分类函数 ctype.h

## 本节讲什么

`isdigit` / `isspace` / `isprint` 等接受 **`int`**，合法取值仅为 **`unsigned char` 能表示的值** 或 **`EOF`**。传入 **带符号 `char` 的负值**（如 `0xFF`）会导致内部查表 **越界 → UB**。

## 要点

```c
char ch = 0xFF;           /* 若 char 有符号 → -1 */
if (isprint(ch)) { }      /* 危险：负索引 */

if (isprint((unsigned char)ch)) { }   /* 安全 */
```

### 原因

函数实现通常：

```c
return table[(unsigned char)c];
```

若 `c` 为负，`(unsigned char)c` 在传入前已按 `int` 提升，**负 char 仍是负 int**，索引可能为负。

### 工程规范

- 凡 `ctype.h` 函数：**入参强转 `(unsigned char)`**
- 已确认为 `EOF` 时直接分支，勿传 `EOF` 给 `isprint` 等（仅部分函数明确定义 EOF 行为）

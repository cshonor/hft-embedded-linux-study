# 5.5 文件 I/O：fopen / gets / fgets

## 本节讲什么

标准 I/O 中 **`gets` 已废弃**（C11 移除），`fgets` / `fopen` 仍有典型误用：无长度限制的输入、忽略 `\n`、打开失败未判空。

## 要点

### 5.5.1 gets — 彻底禁用

```c
gets(buf);   /* 无长度参数，任意长输入 → 必溢出 */
```

**一律** 用 `fgets(buf, sizeof(buf), stdin)` 或平台安全 API。

### 5.5.2 fgets 保留换行符

```c
fgets(line, sizeof(line), fp);
/* line 可能为 "hello\n\0" — 比较/存储前需去掉 \n */
line[strcspn(line, "\n")] = '\0';
```

### 5.5.3 fopen 失败返回 NULL

```c
FILE *fp = fopen(path, "r");
if (!fp) { perror(path); return -1; }
/* 未判空直接 fread → 崩溃 */
```

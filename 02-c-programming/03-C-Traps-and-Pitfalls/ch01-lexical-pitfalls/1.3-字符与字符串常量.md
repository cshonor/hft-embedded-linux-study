# 1.3 字符常量与字符串常量

## 本节讲什么

单引号与双引号在词法上是 **不同类别的 token**，类型与用法完全不同。混用会导致 **比较永远为假** 或 **段错误**。

## 要点

### 对比

| 写法 | 类型 | 含义 |
|------|------|------|
| `'a'` | `int`（字符常量） | ASCII/编码值，如 97 |
| `"a"` | `char[]`（字符串） | 字符 `'a'` + 结尾 `'\0'`，占 2 字节 |
| `'ab'` | 多字符常量（实现定义） | 少用，可移植性差 |

### 致命错误

```c
char c = '0';      /* 字符零，值为 48 */
if (c == "0")      /* 错！比较 char 与 char* 地址 */
if (c == '0')      /* 对 */
```

- `"0"` 是 **指针**（字符串首地址），不是字符 `'0'`
- 编译器可能仅给 warning，逻辑 **永远不成立**

### 字符串与只读

```c
char *s = "hello";  /* 指向 .rodata，s[0]='H' 可能 SIGSEGV */
char s[] = "hello"; /* 栈/数据段副本，可修改 */
```

见 [Expert C ch07](../04-Expert-C-Programming/ch07-adventures-in-memory/) `.rodata` 与 [ch04 链接](../04-Expert-C-Programming/ch05-thinking-of-linking/) `extern` 陷阱。

### 底层场景

- 寄存器读回 `char c = (char)REG;`，与 `'0'`/`'1'` 比较 — 勿写 `"0"`
- 协议单字节字段：用 `'A'`，不是 `"A"`

# 7.8 浮点实现差异

## 本节讲什么

`float`/`double` 的 **精度、舍入、异常**  partly 由 IEEE 754 与实现决定；无 FPU 的 MCU 用软件浮点，性能与误差与 x86 硬件 FPU 不同。跨平台用 `==` 比较浮点、依赖 `NaN`/`Inf` 比较顺序，易静默失败。

## 要点

1. **精度**：`float` 约 7 位有效十进制；累积误差
2. **无 FPU**：`double` 运算慢数个数量级
3. **NaN / Inf**：`x != x` 为真（NaN）；不同 lib 对 `printf` 输出可能不同

```c
int x = sqrt(4);   /* 截断小数，见 ch05 5.6 */
```

### 规范

- 比较：`fabs(a-b) < epsilon` 或整数定点运算
- 关键路径：嵌入式优先 **定点 / 整数**
- 链接 `-lm` 见 [ch05 5.6](../ch05-library-functions/5.6-数学库隐式转换.md)

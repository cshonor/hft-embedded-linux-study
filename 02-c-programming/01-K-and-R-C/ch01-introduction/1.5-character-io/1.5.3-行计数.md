# 1.5.3 行计数

## 本节讲什么

统计文本**行数**：把换行符 `\n` 作为行的分隔标记。

## 要点

- 每读到 `'\n'`，行计数加一 —— **行不是库读出来的，是你在字符流里识别 `\n` 数出来的**（对比 `fgets` 一次读一行；本节用 `getchar` 刻意练底层流式逻辑，见 [1.5 总述](./1.5-字符输入输出.md)）
- 最后一行若无换行符，是否算一行取决于题意；K&R 按「见到 `\n` 计一行」
- 与 1.5.2 相同框架，仅增加对 `\n` 的判断

## 示例

```c
nl = 0;
while ((c = getchar()) != EOF)
    if (c == '\n')
        ++nl;
printf("%d\n", nl);
```

# 1.5.1 文件复制

## 本节讲什么

最简 I/O 循环：从标准输入读字符，原样写到标准输出，等价于 `cat` 的无参行为。

## 要点

- 经典写法：`while ((c = getchar()) != EOF) putchar(c);`
- 重定向：`program < infile > outfile` 即可「文件复制」，程序本身不打开文件名
- 体会 **stdin/stdout** 与终端、文件重定向的关系

## 示例

```c
#include <stdio.h>

/* copy input to output; 1st version */
main()
{
    int c;

    while ((c = getchar()) != EOF)
        putchar(c);
}
```

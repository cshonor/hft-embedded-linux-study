# 1.5.4 单词计数

## 本节讲什么

统计**单词数**：以空格、制表符、换行等空白字符分隔；从「非空白」进入「空白」或 EOF 时计为一个单词结束（或等价状态机：进入单词时 `++nw`）。

## 要点

- 需要区分「是否在单词内部」——典型 **状态机** 雏形
- K&R 用 `c == ' ' || c == '\n' || c == '\t'` 判断空白
- 比单纯计行多一层逻辑，是后续词法分析的最简预告

## 思路

```
不在单词中 → 遇到非空白 → 进入单词，nw++
在单词中   → 遇到空白   → 离开单词
```

完整代码见书中 `wc` 风格实现；与 `grep`、`awk` 一类工具同属「按字符扫描文本」家族。

# 1.2 变量与算术表达式

## 本节讲什么

K&R 第一章第二个经典示例：**华氏 → 摄氏温度对照表**。引入 `int` / `float`、四则运算、`for` 循环与 `printf` 格式控制，完成第一个「有计算、有循环、有表格输出」的实用小程序。

核心目标：

1. 认识 **`int` 整型**、**`float` 浮点型**；牢记 C 规则：**变量必须先声明，后使用**
2. 掌握 **`+ - * /`**、运算符优先级与括号
3. 入门控制流：**`if` 条件判断**、**`while` / `for` 循环**（本章以 `for` 为主）
4. 掌握标准循环模板：**初始化 → 循环体计算 → 更新循环变量 → 输出**
5. 打印 **0～300℉** 每隔 20 度对应的摄氏度

## 要点

### 1. 数据类型与变量声明

```c
int fahr;    /* 循环变量：0, 20, 40 … 300，整数步进 */
float celsius;  /* 若单独存结果，用小数类型 */
```

| 类型 | 用途 | 本章示例 |
|------|------|----------|
| **`int`** | 整数 | 华氏温度 `fahr` |
| **`float`** | 单精度浮点 | 摄氏结果 `(5.0/9.0)*(fahr-32)` |

- C **没有**「先用后声明」（与 C++ 局部变量块内乱序不同）；声明通常放在块开头或循环外
- 与 C++ 对比：C++ 有 `double`、初始化列表；本章 C 示例以 **`float` + `printf`** 为主

### 2. 整数截断（本节第一坑）

C 里 **`/` 两侧都是整数 → 结果仍是整数，小数直接丢弃**：

```c
5 / 9;      /* 结果 0，不是 0.555… */
5.0 / 9.0;  /* 浮点除法，结果正确 */
```

温度公式：\(celsius = \frac{5}{9}(fahr - 32)\)

| 写法 | 结果 |
|------|------|
| `(5/9)*(fahr-32)` | **错** — 先算 `5/9==0`，整表全 0 |
| `(5.0/9.0)*(fahr-32)` | **对** — 浮点运算 |

推广：涉及小数的公式，**至少一侧用浮点常量或 cast**，见 [C 陷阱 ch03 语义](../../03-C-Traps-and-Pitfalls/ch03-semantic-pitfalls/)。

### 3. `for` 循环模板

```c
for (fahr = 0; fahr <= 300; fahr = fahr + 20)
    /* 循环体 */
```

等价于三步：

```text
  初始化: fahr = 0
  条件:   fahr <= 300 为真则进入循环体
  更新:   每次末尾 fahr = fahr + 20
```

| 部分 | 示例 | 含义 |
|------|------|------|
| 初始值 | `fahr = 0` | 从 0℉ 开始 |
| 条件 | `fahr <= 300` | 到 300℉ 为止 |
| 更新 | `fahr += 20` 或 `fahr = fahr + 20` | 步长 20 |

`while` 可写同样逻辑，本书先给 `for` 表格式循环；**1.3** 会单独练 `for`。

### 4. `printf` 格式控制（打表格）

```c
printf("%3d %6.1f\n", fahr, celsius);
```

| 格式 | 含义 |
|------|------|
| **`%3d`** | 整数，**至少**占 3 字符宽度，右对齐（不足补空格） |
| **`%6.1f`** | 浮点，总宽 6、**小数 1 位** |
| `\n` | 每行结束换行 |

输出列对齐，便于肉眼对照；调试日志、指标打印同样靠格式符控制列宽。

### 5. 完整可运行代码

```c
#include <stdio.h>

int main(void)
{
    int fahr;

    for (fahr = 0; fahr <= 300; fahr = fahr + 20)
        printf("%3d %6.1f\n", fahr, (5.0 / 9.0) * (fahr - 32));

    return 0;
}
```

```bash
gcc -Wall -Wextra -std=c11 -o fahr fahr.c
./fahr
```

预期片段：

```text
  0  -17.8
 20   -6.7
...
300  148.9
```

## HFT / 底层学习的用处

1. **整型 vs 浮点**：行情价、量、PnL 若误用整数除法或混算，会出现 **静默精度丢失**（与 HFT 里定点/浮点选型同源）
2. **循环模板**：DPDK 收包循环、报文遍历、行情回放 **for/while 骨架相同** — 初始化、条件、更新
3. **格式化输出**：热路径外调试、`printf` / 日志宏格式化指标；格式符与实参类型须匹配（见 [C 陷阱 ch05](../../03-C-Traps-and-Pitfalls/ch05-library-functions/5.1-printf与可变参数.md)）

## 与 C++ 的差异

| 主题 | C++ | C（本节） |
|------|-----|-----------|
| 输出 | `cout << fixed << setprecision(1)` | `printf("%6.1f", x)` |
| 循环 | `for (int fahr = 0; …)` 可块内声明 | C99 前常 **`int fahr;` 在 for 外声明** |
| 除法 | `5/9` 仍为 0（同为整数除法） | 同样规则，**必须** `5.0/9.0` |

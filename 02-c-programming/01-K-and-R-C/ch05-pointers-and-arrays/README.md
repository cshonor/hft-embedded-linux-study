# 第 5 章 指针与数组

**Pointers and Arrays**

## 本章讲什么

全书**核心难点**：指针、数组、地址运算、字符串、函数指针。前面 1–4 章是语法与工程组织，本章是**操作内存的底层工具**。能否看懂 OS 内核、UEFI、HFT、飞控源码，很大程度取决于本章。

## 学习重点

- **`a[i]` ≡ `*(a+i)`**、数组名退化、指针缩放
- **合法地址运算** vs 越界 / 跨数组 UB
- **只读字符串字面量** vs **`char buf[]`**
- **指针数组 vs 数组指针**、**指针函数 vs 函数指针**
- **禁止返回栈局部指针**；`main(argc, argv)` 结构
- 二维：**`*(*(a+i)+j)`**、stride / 帧缓冲偏移

## 场景映射

| 方向 | 本章技能 |
|------|----------|
| OS / UEFI Miniload | ELF 头、页表、GOP 显存指针偏移 |
| HFT | 零拷贝、ring buffer、报文解析、策略回调 |
| 嵌入式 / 飞控 | MMIO 寄存器映射、传感器缓冲、中断 handler |

## 难度

全书最难；必须完全吃透后再进第 6 章结构体与第 8 章系统接口。

# 7.2 格式化输出 printf 函数

## 本节讲什么

**`printf`** 按格式串把各类型转为文本输出；**`fprintf`** 写到指定 `FILE*`，**`sprintf`/`snprintf`** 写到内存缓冲。日志、调试、配置生成的基础。

## 要点

### 常用转换说明符

| 符 | 类型 |
|----|------|
| `%d` `%i` | int |
| `%u` `%o` `%x` `%X` | unsigned / 八进制 / 十六进制 |
| `%c` | char |
| `%s` | 字符串 |
| `%f` `%e` `%g` | float/double |
| `%p` | 指针（实现定义格式，常十六进制） |
| `%%` | 字面 `%` |

### 修饰

- **宽度 / 精度**：`%8d`、`%.2f`、`%-10s`（左对齐）
- **长度修饰**：`%ld`、`%zu`、`%llx` 等
- **返回值**：成功输出的**字符数**；出错返回负值

### 文件与内存变体

```c
fprintf(stderr, "error: %s\n", msg);
fprintf(fp, "%d\n", n);

char buf[64];
snprintf(buf, sizeof buf, "tick=%lld", tick);  /* 推荐：有边界 */
```

## 考点

- **`sprintf` 无边界** → 缓冲区溢出；现代代码用 **`snprintf`**
- 格式与实参类型不匹配 → 未定义行为
- HFT 热路径避免 `printf`（锁 + 格式化开销），用预分配缓冲或二进制日志

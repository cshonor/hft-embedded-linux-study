# 7.3 变长参数表

## 本节讲什么

**`printf(format, ...)`** 的 **`...`** 表示变参；`<stdarg.h>` 提供 **`va_list` / `va_start` / `va_arg` / `va_end`**，用于实现或封装自定义格式化函数。

## 要点

```c
#include <stdarg.h>

void warn(char *fmt, ...)
{
    va_list ap;
    va_start(ap, fmt);   /* fmt 为最后一个固定参数 */
    vfprintf(stderr, fmt, ap);
    va_end(ap);
}
```

- **`va_start(ap, last_fixed)`**：`last_fixed` 必须是 `...` 前最后一个具名参数
- **`va_arg(ap, type)`**：按 **type** 取下一个实参（类型必须正确）
- **`va_end(ap)`**：清理
- C99 另有 **`va_copy`**

## 设计约束

- 变参函数**无法**从 `...` 自动得知参数个数 → 靠 **format 串**（如 `printf`）或约定
- 封装日志时常用：`log(level, fmt, ...)` 内部 `vfprintf`

## 与后续衔接

理解变参有助于读 `printk` 宏展开；底层 ABI 中变参通过栈/寄存器传递，与第 5 章栈帧相关。

## 难度

偏实现细节；会用 `vfprintf`/`vsnprintf` 包装即可，不必手写完整 printf。

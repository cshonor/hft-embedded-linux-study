# 7.7 行输入和行输出

## 本节讲什么

**`fgets`/`fputs`** 按行读写文本；**`gets`** 已废弃。安全读配置、日志行、协议文本行的首选 stdio 方式。

## 要点

### fgets

```c
char line[256];
if (fgets(line, sizeof line, fp) != NULL) {
    /* line 含内容，可能含 '\n' */
}
```

- 最多读 **`size-1`** 个字符，保留 **`'\0'`**
- 遇 **`'\n'`** 或 EOF 停止；**超长行保留 `\n` 并截断**
- 返回值：`NULL` 表示 EOF 且无数据读入，或出错

### fputs

```c
fputs(line, fp);   /* 不自动追加换行 */
```

- 与 **`fputs(str, stdout)`** 输出一行（若 `str` 已含 `\n`）

### 避免 gets

- **`gets(buf)`** 无长度限制 → **永远不要用**
- 替代：`fgets` 或 POSIX **`getline`**

## 示例：读配置

```c
while (fgets(line, sizeof line, cfg) != NULL) {
    if (line[0] == '#' || line[0] == '\n')
        continue;
    /* sscanf 或手工解析 */
}
```

## 与后续衔接

文本协议「一行一条」；二进制 tick 用 7.5 `fread`。嵌入式配置文件、串口 AT 指令行常用 `fgets` 模式。

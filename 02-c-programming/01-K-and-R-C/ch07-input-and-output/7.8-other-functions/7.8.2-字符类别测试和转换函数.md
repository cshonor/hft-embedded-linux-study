# 7.8.2 字符类别测试和转换函数

## 本节讲什么

**`<ctype.h>`** 按字符属性分类与转换，解析文本协议、lexer、配置文件时常用。

## 要点

| 函数 | 含义 |
|------|------|
| `isalpha(c)` | 字母 |
| `isdigit(c)` | 数字 |
| `isspace(c)` | 空白（空格、`\t`、`\n` 等） |
| `isalnum` `isupper` `islower` | 组合/大小写 |
| `tolower(c)` `toupper(c)` | 转换 |

- 参数为 **`int`**，须为 **`unsigned char`** 值或 **`EOF`**
- 对 **`char`** 先转 **`unsigned char`** 再传入，避免负值 `char` 越界表访问

## 示例

```c
while (isspace(*p))
    p++;
if (isdigit(*p))
    /* 解析数字 */
```

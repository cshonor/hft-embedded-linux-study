# 7.8 其它函数

## 本节讲什么

K&R 本章末尾概览**标准库**若干类别：字符串、字符类、`ungetc`、**`malloc`/`free`**、数学、随机数等。详细列表见附录 B；此处建立索引，便于查手册。

## 要点

| 小节 | 头文件 | 用途 |
|------|--------|------|
| 7.8.1 | `<string.h>` | `strlen` `strcpy` `strcmp` `strncpy` … |
| 7.8.2 | `<ctype.h>` | `isalpha` `isdigit` `tolower` … |
| 7.8.3 | `<stdio.h>` | `ungetc` 推回流 |
| 7.8.4 | `<stdlib.h>` | `system` 执行 shell 命令 |
| 7.8.5 | `<stdlib.h>` | **`malloc` `calloc` `realloc` `free`** |
| 7.8.6 | `<math.h>` | `sin` `sqrt` `pow` … |
| 7.8.7 | `<stdlib.h>` | `rand` `srand` |

## 底层开发取向

- **字符串 / 内存**：内核有自实现 `strlcpy`；用户态 HFT 注意 `strcpy` 边界
- **malloc**：第 8 章与 `sbrk`/堆；热路径预分配池，避免频繁 `malloc`
- **system**：安全风险大，生产环境慎用

6.5 链表、6.6 符号表依赖 7.8.5 堆分配；第 8 章系统接口是 I/O 的 UNIX 层替代实现。

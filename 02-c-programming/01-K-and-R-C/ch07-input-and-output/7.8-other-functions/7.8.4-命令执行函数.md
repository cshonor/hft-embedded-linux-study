# 7.8.4 命令执行函数

## 本节讲什么

**`system(const char *cmd)`** 调用 shell 执行命令，等待结束并返回状态。简单脚本集成用，**安全与可移植性有限**。

## 要点

```c
int status = system("ls -l");
```

- 依赖 **`/bin/sh`**（或平台 shell）
- 返回值编码实现相关；需用宏 **`WEXITSTATUS`** 等（`<sys/wait.h>`，第 8 章）解析
- **安全风险**：若 `cmd` 含用户输入 → 命令注入
- 嵌入式/内核/UEFI **无 system**；HFT 生产环境通常禁用

## 与后续衔接

第 8 章 **`fork` + `exec`** 是正确、可控的进程启动方式；`system` 仅是便捷封装。

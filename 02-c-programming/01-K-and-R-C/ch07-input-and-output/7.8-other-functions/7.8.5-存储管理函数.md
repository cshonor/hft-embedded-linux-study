# 7.8.5 存储管理函数

## 本节讲什么

**`malloc` `calloc` `realloc` `free`** —— **堆区**分配（[4.3.1 内存布局](../../ch04-functions-and-program-structure/4.3.1-程序内存布局.md)），6.5 链表、6.6 符号表、动态缓冲的基础。第 8 章从系统角度讲堆与 `sbrk`。

## 要点

```c
void *malloc(size_t n);           /* 未初始化 */
void *calloc(size_t n, size_t sz); /* 清零 */
void *realloc(void *p, size_t n);  /* 调整大小，可能搬迁 */
void free(void *p);
```

- 失败返回 **`NULL`**，必须检查
- **`free(NULL)`** 合法无操作
- **禁止**：`free` 非堆指针、double-free、use-after-free
- 分配大小 **`n * sz`** 注意溢出

## 底层取舍

| 场景 | 策略 |
|------|------|
| HFT 热路径 | 预分配对象池，避免 malloc |
| 内核 | `kmalloc`/`kfree`，非 stdlib |
| UEFI | `AllocatePool`/`FreePool` |
| 工具/解析器 | malloc 可接受 |

## 与后续衔接

`fread` 读变长记录前常 `malloc` 缓冲；第 8 章 `brk`/`sbrk` 揭示堆与内核边界。

# 7.8.7 随机数发生器函数

## 本节讲什么

**`rand()`** 返回伪随机整数；**`srand(seed)`** 设种子。非密码学安全，适合仿真、测试、负载抽样。

## 要点

```c
srand((unsigned)time(NULL));
int r = rand() % N;   /* 0..N-1，% 有轻微偏差 */
```

- 周期与质量实现相关；**不要**用于安全/token
- C99 **`random()`** / **`srandom()`**（部分系统）质量更好
- 需要可重复测试：**固定 seed**

## 与 HFT / 回测

- 回测用固定 seed 复现随机成交
- 生产撮合随机性用硬件 RNG 或专用库，非 `rand()`

Monte Carlo 仿真、 fuzz 测试入门；密码学见 `<stdint.h>` + OS 接口。

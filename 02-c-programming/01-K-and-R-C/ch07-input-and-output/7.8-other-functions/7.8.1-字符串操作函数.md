# 7.8.1 字符串操作函数

## 本节讲什么

**`<string.h>`** 中 C 字符串核心 API：长度、拷贝、比较、连接。第 5 章用指针实现，此处为标准库封装。

## 要点

| 函数 | 作用 |
|------|------|
| `strlen(s)` | 长度（不含 `'\0'`） |
| `strcpy(dst, src)` | 拷贝（**不检查 dst 大小**） |
| `strncpy(dst, src, n)` | 最多 n 字节，可能不补 `'\0'` |
| `strcat(dst, src)` | 连接 |
| `strcmp(s1, s2)` | 比较，返回 `<0` `0` `>0` |
| `strncmp` | 限长比较 |
| `strchr` / `strstr` | 查找字符/子串 |

## 安全习惯

- 优先 **`strncpy` + 手动 `'\0'`** 或 **`snprintf`**
- 内核常用 **`strlcpy`/`strscpy`**（非 K&R 标准 C）

## 与后续衔接

报文定长字段、符号表 `name` 比较都依赖这些原语；二进制数据用 `memcpy`（`<string.h>`）。

# 7.8.6 数学函数

## 本节讲什么

**`<math.h>`** 浮点数学：`sin` `cos` `sqrt` `pow` `log` `floor` `ceil` 等。链接时常需 **`-lm`**。

## 要点

- 多数函数操作 **`double`**；`float` 版本常有 **`sinf`** 等（C99）
- **`pow(x,y)`** 通用但非最快；热路径可用专用近似
- 检查 **`errno`** 或返回值处理域错误（如 `sqrt(-1)`）

## 与飞控 / 仿真的关系

- 姿态解算、PID、滤波会用到三角与开方
- 嵌入式可换 **CMSIS-DSP** 或定点库替代 double

## 与后续衔接

本章 I/O 与数学并列属标准库概览；数值性能不是 K&R 重点，工程上另选库。

# 7.8.3 ungetc 函数

## 本节讲什么

**`ungetc(c, fp)`** 把字符 **`c`** **推回**输入流，下次读会先读到它。实现「读多看一、再退回」的 parser 必备。

## 要点

- 每个流通常只允许**少量**推回（常 1 个，实现定义）
- 成功返回 `c`；失败返回 `EOF`
- 与 **`fgetc`** 配合：试探下一字符，若非预期则 `ungetc`

## 示例

```c
int c = fgetc(fp);
if (c == '/')
    c = fgetc(fp);
else {
    ungetc(c, fp);
    /* 按其它规则处理 */
}
```

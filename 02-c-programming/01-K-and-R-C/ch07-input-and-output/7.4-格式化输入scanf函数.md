# 7.4 格式化输入 scanf 函数

## 本节讲什么

**`scanf`** 按格式从 **stdin** 解析并写入变量；**`fscanf`** 对文件，**`sscanf`** 对内存字符串。与 `printf` 对称，但**更易出错**。

## 要点

- 除 **`%s` `%[]`** 等外，实参必须是**指针**：`scanf("%d", &n);`
- **数组名**已是地址：`scanf("%s", buf)` 无需 `&buf`
- **返回值**：成功匹配并赋值的**项数**；`EOF` 表示在读入前即结束
- **空白**：格式串中的空白匹配任意空白；`%d` 等前也会跳过 leading 空白

### 常用格式

| 符 | 含义 |
|----|------|
| `%d` | int |
| `%f` `%lf` | float / double |
| `%s` | 无空白字符串（**无长度上限，危险**） |
| `%c` | 字符（不跳过空白，常需 `" %c"`） |
| `%x` | 十六进制 |

### 变体

```c
fscanf(fp, "%d %s", &n, name);
sscanf(pkt, "MD:%lf", &price);   /* 报文内解析，无磁盘 IO */
```

## 考点

- **`%s` 溢出** → 用 **`%63s`** 限宽或改用 `fgets` + 解析
- 与 `printf` 类型不匹配 → UB
- 留字符在流中未读 → 下次 `scanf` 行为诡异

## 与后续衔接

HFT 简单文本协议可用 `sscanf`；二进制行情应用 `fread` + struct（7.5）而非 scanf。

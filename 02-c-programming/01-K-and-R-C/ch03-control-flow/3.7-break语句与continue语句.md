# 3.7 break 与 continue

## 本节讲什么

在循环和 `switch` 中改变正常控制流：**`break`** 提前结束当前结构；**`continue`** 跳过本轮循环剩余语句，进入下一轮。嵌套循环时，二者 **只作用于离它最近的那一层** —— 笔试与读代码的高频考点。

**前置**：[3.5 while / for](./3.5-while循环与for循环.md)、[3.4 switch](./3.4-switch语句.md)、[3.6 do-while](./3.6-do-while循环.md)。

## 要点

### 1. `break`：跳出 **当前这一层**

| 用在哪 | 效果 |
|--------|------|
| **`switch`** | 结束整个 `switch`，不再执行后续 `case`（见 [3.4](./3.4-switch语句.md)） |
| **循环**（`while` / `for` / `do-while`） | 跳出 **最内层**（enclosing）的那一圈循环 |

**只跳一层** —— 不会一次跳出两层、三层嵌套。

### 2. `continue`：跳过本轮，进入 **本层** 下一轮

- **仅用于循环**，**不能** 用于 `switch`
- 跳过 **本次迭代** 中 `continue` **之后的语句**
- 然后进入 **同一层循环** 的下一轮（不是外层）

| 循环类型 | `continue` 之后 |
|----------|----------------|
| **`while` / `do-while`** | 回到 **条件判断** |
| **`for`** | 先执行 **第三部分步进**，再 **条件判断** |

```c
for (i = 0; i < n; i++) {
    if (a[i] < 0)
        continue;   /* 跳过负数；i++ 仍会执行 */
    sum += a[i];
}
```

### 3. 嵌套循环：只影响 **最近一层**

两层 `for` 嵌套时，在内层写的 `break` / `continue` **只管内层**，外层照常继续。

```c
for (i = 0; i < 3; i++) {
    for (j = 0; j < 3; j++) {
        if (j == 1)
            break;      /* 只跳出内层 for(j) */
        printf("(%d,%d) ", i, j);
    }
    printf("| ");       /* 外层 for(i) 每轮都会执行到这里 */
}
```

**输出**（`break` 在内层 `j == 1` 时触发）：

```
(0,0) | (1,0) | (2,0) |
```

内层每次只打印 `j == 0`，然后 `break`；**外层 `i` 仍 0→1→2 完整跑完**。

#### `continue` 在内层：只跳过内层本轮

```c
for (i = 0; i < 2; i++) {
    for (j = 0; j < 4; j++) {
        if (j % 2 == 1)
            continue;   /* 跳过内层 j 为奇数的那次 */
        printf("%d%d ", i, j);
    }
}
```

**输出**：

```
00 02 10 12
```

`j == 1`、`j == 3` 时被 `continue` 跳过，**内层 `j` 的下一次（2、4…）照常进行**；外层 `i` 不受影响。

| 写法位置 | `break` | `continue` |
|----------|---------|------------|
| **内层循环** | 只结束 **内层**；外层继续 | 只跳过 **内层本轮**；内层下一轮正常 |
| **外层循环** | 结束 **外层**（内层已不在） | 跳过 **外层本轮** 剩余代码 |

### 4. 逐步理解：内层 `break`，外层继续

```c
for (i = 0; i < 3; i++) {          /* 外层 */
    for (j = 0; j < 5; j++) {      /* 内层 */
        if (some_error(j))
            break;                 /* ① 只跳出内层 */
        process(i, j);
    }
    /* ② 内层 break 后到这里；外层 i++，开始新一轮内层 */
}
```

- 内层 `break` ≠ 程序结束，也 ≠ 两层一起退出
- 外层 **不知道** 内层是正常结束还是 `break` 出来的 —— 若需通知外层，用 **标志变量** 或 [3.8 `goto`](./3.8-goto语句与标号.md)

### 5. 想一次跳出多层怎么办？

C 的 `break` **没有标签**，不能写 `break outer`（Java 可以）。

| 做法 | 说明 |
|------|------|
| **多次 `break`** | 不行 —— 一层 `break` 只出一层 |
| **标志变量** | 内层 `break` 前置 `done = 1`，外层每轮检查 |
| **`goto` 标号** | 内核/错误处理常见，跳到多层外（3.8） |
| **`return`** | 直接结束 **整个函数** |
| **改成函数** | `break` 变 `return`，用返回值表示是否提前结束 |

```c
int done = 0;
for (i = 0; i < n && !done; i++) {
    for (j = 0; j < m; j++) {
        if (bad(i, j)) {
            done = 1;
            break;       /* 出内层 */
        }
    }
}
```

### 6. 与 `return`、`goto` 对比

| 语句 | 作用范围 |
|------|----------|
| **`break`** | 当前 `switch` 或 **最内层** 循环 |
| **`continue`** | **最内层** 循环的下一轮 |
| **`return`** | 结束 **整个函数** |
| **`goto label`** | 跳到标号处（可跨多层循环，慎用） |

### 7. 基本示例（单层）

```c
/* continue：过滤 */
for (i = 0; i < n; i++) {
    if (a[i] < 0)
        continue;
    sum += a[i];
}

/* break：退出 */
for (;;) {
    c = getchar();
    if (c == EOF)
        break;
    putchar(c);
}
```

## 易混点

| # | 要点 |
|---|------|
| 1 | 嵌套时 `break` / `continue` 只作用于 **最近一层** 循环 |
| 2 | 内层 `break` → 外层 **继续**；不是两层一起停 |
| 3 | 内层 `continue` → 只跳过 **内层本次**；内层下一次照常 |
| 4 | `for` 里 `continue` **仍执行步进**（第三部分） |
| 5 | `continue` **不能** 用于 `switch` |
| 6 | 一次跳出多层 → 标志变量 / `goto` / `return`，不能多层 `break` |

## HFT / 内核提示

- 扫描报文字段：内层 `for` 遇非法 `break`，外层换下一包 —— 注意外层是否仍需 `continue` 跳过本包剩余逻辑
- 热路径避免深层嵌套 + 多个 `break`/`continue` 交织；可读性差易引入逻辑 bug
- 内核搜索循环：`break` 表示「找到了」；`continue` 表示「跳过本项」—— 与嵌套层级的对应关系审代码时要画清楚

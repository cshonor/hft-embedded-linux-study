# 3.3 else-if 多分支结构

## 本节讲什么

把多层嵌套 `if` 写成 **`if ... else if ... else`** 链，按顺序测试条件，命中即执行并跳过后续分支。适合多状态、多档位判断。

## 要点

- 语法糖：`else { if (...) ... }` 写成 `else if (...) ...`
- **自上而下**依次求值，**第一个为真的分支**执行后，整条链结束
- 各分支互斥；若可能同时满足多个条件，顺序即优先级
- 末尾可选 `else` 作兜底（未匹配任何条件时执行）
- 比深层嵌套 `if` 更易读、更少 else 配对错误

## 示例

```c
if (n < 0)
    sign = -1;
else if (n == 0)
    sign = 0;
else
    sign = 1;
```

## 与 switch 的选择

| 场景 | 更合适 |
|------|--------|
| 离散整型常量（状态码、枚举） | `switch`（3.4） |
| 区间、不等式、浮点比较 | `else if` 链 |
| 条件复杂且各不相干 | `else if` 或拆成独立 `if` |

## 与后续衔接

协议解析、调度状态机常用 `else if` 或 `switch`；先理清互斥与顺序，再考虑是否用表驱动替代（第 5 章数组、函数指针可进一步抽象）。

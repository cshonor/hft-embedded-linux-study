# 2.5 算术运算符

## 本节讲什么

二元算术：`+ - * / %` 及与整型、浮点混合时的行为。

## 要点

- **`/` 整数除法截断** toward zero：`5/2 == 2`，`-5/2` 实现定义前 C89 需注意
- **`%` 取模仅用于整数**；符号随实现，C99 后 `(a/b)*b + a%b == a`
- 浮点除法至少一侧为浮点：`5.0/2` → `2.5`
- 溢出：有符号溢出 **UB**；无符号溢出模 \(2^n\)

## 示例

```c
9 / 2;    /* 4 */
9 % 2;    /* 1 */
9 / 2.0;  /* 4.5 */
```

HFT/内核里计时、对齐常用 `%` 与 `/` 组合（如 `% align`）。

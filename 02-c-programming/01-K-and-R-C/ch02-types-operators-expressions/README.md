# 第 2 章 类型、运算符与表达式

**Types, Operators, and Expressions**

## 本章讲什么

C 的底层核心：类型如何在内存中存储、表达式如何求值、运算符如何组合。**无指针、无复杂项目**，全是数值与位层面的细节，为第 5 章指针打基础。

## 学习重点（Linux / OS / HFT）

- **位运算**、**有符号/无符号转换**、**整型提升** —— 寄存器、驱动、DPDK、内核天天用
- **短路求值**、**优先级** —— 避免隐蔽 bug
- 理解各类型**宽度**与符号性，是读内存布局、ABI 的前提

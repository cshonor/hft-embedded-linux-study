### ① 解读鼠标数据 · 让鼠标动起来

Day 7 收到 **hex 瀑布**；Day 8 **按协议组包**。

#### 字节流形态

| 阶段 | 内容 |
|------|------|
| 激活后首字节 | **`0xFA`**（ACK） |
| 之后 | **每 3 字节一组** 运动/按键包 |

#### `MOUSE_DEC` 结构体

```c
/* 示意字段 — 以原书为准 */
struct MOUSE_DEC {
    unsigned char phase;  /* 当前收到第几字节 0/1/2 */
    char x, y;            /* 解析后的位移 */
    char btn;             /* 按键状态 */
};
```

从 **FIFO** 逐字节喂入状态机 → 凑满 3 字节 → 得到一次 **完整事件**。

#### 防错位 · 第一字节校验

连线不良 → **丢字节 → 整包错位**。

**规律：** 每包 **第 1 字节** 与 **`0xC8` 做 AND** 必为 **`0x08`** — 不满足则 **丢弃/重同步**，快速纠正相位。

#### 位运算解析

| 字段 | 解析 |
|------|------|
| **第 1 字节低 3 位** | **左 / 右 / 中键** |
| **第 2、3 字节** | **X、Y 位移**（带符号） |
| **Y 轴** | 屏幕 Y 与鼠标 Y **方向相反** → 代码里 **取反** |

```
FIFO ──► phase 0/1/2 ──► MOUSE_DEC { btn, x, y }
                              │
                              ▼
                         mx += x;  my -= y;  (示意)
```

### ② 移动鼠标指针 · 边界与重绘

| 变量 | 含义 |
|------|------|
| **`mx`, `my`** | 当前指针 **屏幕坐标** |
| **`mdec.x`, `mdec.y`** | 本包 **位移** |

**新坐标** = 当前 + 位移；**`if` 钳制** 在 `[0, width-1]` / `[0, height-1]` — **不出屏**。

**绘制流程（每帧/每事件）：**

1. 用 **背景色** 覆盖 **旧位置** 指针（隐藏）
2. 在 **(mx, my)** **重画** 16×16 箭头（Day 5）

→ **鼠标可自由移动、可点** — **交互里程碑**。

#### 遗留问题 · 无图层

指针移到 **底部任务条** 时 **直接写 VRAM** → **破坏任务栏像素** — **没有叠加/合成**。  
**伏笔：** 后续章节 **图层处理** 再修。

**HFT：** 类似 **无双缓冲 UI 直接 blit** 的撕裂/覆盖问题 — 要有 **独立 surface 或 damage region**。

### ③ 揭秘 asmhead.nas · 32 位模式之路

Day 3 **用过** 32 位结果；Day 8 **补全引导里 ~100 行汇编**。

#### A20GATE · 突破 1MB

| 背景 | 操作 |
|------|------|
| 早期 PC **兼容模式** 默认 **只认 ~1MB** | 开 **A20 地址线** 才能访问 **更高物理内存** |
| 经 **键盘控制器附属端口** | 写 **`0xDF`** 等序列（原书流程）→ **A20 有效** |

#### 保护模式 · `CR0` + 复制 bootpack

| 步骤 | 说明 |
|------|------|
| 设 **`CR0`** 相应位 | 进入 **32 位保护模式** |
| 汇编 **`memcpy`** | 把 **`bootpack.hrb`** 拷到 **`0x00280000`** |
| **`far-JMP`** | 跳到 **OS 本体代码段 `0x1b`** 等入口 → **开始跑 C（HariMain 链）** |

```
实模式 asmhead（BIOS、读盘、A20、GDT…）
        │
        ▼ CR0=1 · 32bit
memcpy bootpack → 0x00280000
        │
        ▼ far-JMP
HariMain / bootpack.c（C 世界）
```

### Day 8 小结

| 问题 | 答案 |
|------|------|
| 鼠标包多长？ | 激活 **`0xFA`**；之后 **3 字节/组** |
| 防错位？ | 首字节 **`& 0xC8 == 0x08`** |
| 按键与移动？ | **低 3 位 btn**；**byte2/3 = x/y**；**Y 取反** |
| 怎么动指针？ | **mx/my ± 位移** · 边界 **if** · 擦旧画新 |
| 任务栏花屏？ | **无图层** — 后文解决 |
| A20 干嘛？ | 打开 **>1MB** 物理寻址 |
| bootpack 放哪？ | **`0x00280000`** · **`far-JMP` 进 C** |

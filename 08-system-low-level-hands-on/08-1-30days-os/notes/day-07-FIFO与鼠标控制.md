### ① 获取真实按键编码 · 中断必须快

Day 6 能 **响应中断**，还不知道 **按了哪键**。

#### 读键盘数据端口

| 端口 | 作用 |
|------|------|
| **`0x0060`** | 键盘控制器 **数据端口** — 读 **扫描码（scancode）** |

| 动作 | 示例码（原书） |
|------|----------------|
| 按下 **A** | **`0x1E`** |
| 松开 **A** | **`0x9E`** |

#### 底层铁律：ISR 越短越好

| ❌ 错误 | ✅ 正确 |
|---------|---------|
| 在中断里 **putstr / 画屏** | ISR：**读端口 → 写入缓冲 → 立刻 `IRETD`** |
| 中断占用长 → **卡顿、丢中断** | **HariMain** 主循环空闲时 **慢慢显示/处理** |

```
键按下 ──► IRQ ──► ISR: 读 0x60 → 丢进 FIFO ──► 返回
                      │
HariMain ◄────────────┘ 轮询/取 FIFO → sprintf → 屏幕
```

**HFT 对照：** 网卡 **NAPI / 收包 ISR 只 enqueue**、策略线程 ** dequeue 算信号** — 同一 **「中断短、业务慢」** 分工。

→ [02 SysPerf 热路径](../../02-Systems-Performance-2nd/chapter-02-methodologies/) · 别在 IRQ 里做重活

### ② FIFO 缓冲区 · 从排队到环形

#### 问题：单变量不够

| 场景 | 后果 |
|------|------|
| 连按 / 键鼠同时 | 主程序还没读，**下一中断覆盖** 上一字节 |
| 只用一个 **`unsigned char buf`** | **丢键、丢鼠标包** |

#### 第一版 FIFO：移位（慢）

每次 **读走队头**，后面数据 **全体前移一格** — **O(n) 搬运**，CPU 浪费。

#### 改良：环形缓冲区（Ring buffer）

| 指针 | 作用 |
|------|------|
| **写指针（wp）** | 中断 **写入** 位置 |
| **读指针（rp）** | 主程序 **读出** 位置 |

写满末尾 → **指针绕回开头**（覆盖 **已读、可复用** 的空间）— **不移动数据**，**O(1)**。

```
      rp          wp
       ▼           ▼
buf [·|已读|数据数据|空|·]
       └─写绕回──► 环形
```

封装为通用结构 **`FIFO8`**（8 位数据、固定容量）— **键盘一条、鼠标一条**，随时 **`Fifo8_Put` / `Fifo8_Get`**。

**HFT：** **SPSC/MPSC ring buffer**（行情、订单、日志）— 本 OS 课是 **最简 8 位单生产者单消费者** 原型。

→ [01-CSAPP Ch12 / 并发结构](../../01-CSAPP-3rd/chapter-12-concurrent-programming/) · [14-HFT 无锁队列](../../14-HFT-Low-Latency-Practice/)

### ③ 激活并接收鼠标数据

#### 鼠标挂在键盘控制器上

历史原因：**鼠标电路附属于键盘控制器** — 激活要走 **键盘控制器端口** 发命令：

| 步骤（示意） | 含义 |
|--------------|------|
| 发 **`0xd4`** 等 | 告诉控制器「接下来给鼠标」 |
| 发 **`0xf4`** 等 | **启用鼠标数据报告** |
| 收到 **`0xFA`** | 鼠标 **ACK，激活成功** |

#### IRQ12 + 大 FIFO

| 属性 | 说明 |
|------|------|
| **IRQ12** | 鼠标硬件中断（Day 6 已映射 **INT 0x2c**） |
| **数据形态** | 一动鼠标 → **连续 3 字节/包**，量大 |
| **缓冲** | 专用 **`FIFO8`，128 字节** — 防 burst 覆盖 |

ISR：**只把 3 字节流拆进 FIFO**；主循环：**批量读出** → 屏幕 **十六进制瀑布**。

#### Day 7 里程碑

**晃动鼠标** → 屏上 **刷鼠标原始 hex** — **信号已进系统**；**指针还没跟手移动**（Day 8 解析 3 字节包 + 改坐标）。

### Day 7 小结

| 问题 | 答案 |
|------|------|
| 键码从哪读？ | 端口 **`0x0060`**（如 **1E / 9E**） |
| ISR 该干啥？ | **只接收进 FIFO**，显示放 **HariMain** |
| 为何 FIFO？ | 防 **快输入覆盖**、键鼠 **并发** |
| 环形 vs 移位？ | **读写指针绕回**，不搬数据 |
| 通用封装？ | **`FIFO8`** |
| 鼠标怎么开？ | 经 **键盘控制器** 命令 → **`0xFA`** |
| 鼠标 FIFO 多大？ | **128 字节**（3 字节包 burst） |
| 今天验收？ | **鼠标动 → hex 瀑布**；指针 Day 8 |

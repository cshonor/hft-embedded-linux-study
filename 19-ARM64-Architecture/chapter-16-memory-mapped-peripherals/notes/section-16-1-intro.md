### 内存映射 I/O (MMIO)

ARM **没有** x86 式 **`IN`/`OUT` 端口指令**。

```
CPU 把外设寄存器当作普通 RAM 地址
        ↓
LDR  读状态/数据
STR  写控制/数据
```

**链接脚本 / 数据手册** 给出 **`0xE000C000`** 这类 **物理/总线地址** — 与 [Ch4 伪指令](../chapter-04-assembler-rules-directives/) **`= label`** 常量同源。

---

### 典型 Bring-up 流水线

```
1. 时钟 / 电源（Tiva 必须；LPC 较隐式）
2. 引脚复用 PINSEL（UART/DAC/GPIO 功能）
3. 外设模式寄存器（波特率、DAC 使能、GPIO DIR）
4. 数据路径：轮询状态位 或 使能中断（Ch15）
```

---

### 三节为何选三种外设

| 外设 | 教什么 |
|------|--------|
| **UART** | **串口调试** — 裸机 `printf` 前身 · **状态位轮询** |
| **DAC** | **模拟输出** · **查表 + 定点换算** |
| **GPIO** | **数字 IO** · **M4 时钟门控** · **位级安全写** |

---

### 可复述要点

1. **MMIO** = 驱动硬件的 **Load/Store**，不是新指令集。  
2. 永远 **先查手册**：地址、位域、上电顺序。  
3. 本章代码是 **Linux 驱动 MMIO** 的 **裸机祖先**。

## §16.3 LPC2132 — D/A 转换器生成正弦波

> **Ch 16 · 内存映射外设** · [章导读](../README.md) · [本章概述](./section-0-本章完整概述.md)

---

### LPC2132 与 DAC

LPC2132 与 LPC2104 **同族**（ARM7 + VPB），本节外设为 **10 bit DAC**：

| 项目 | 说明 |
|------|------|
| **模拟输出** | **AOUT** 引脚 — 电压 ∝ 数字码 |
| **参考** | **VREF** — 满量程基准 |
| **分辨率** | **10 bit** → 0…1023（有效位在寄存器 **[15:6]**） |
| **DACR 地址** | **`0xE006C000`** |

---

### 引脚与 DACR

**PINSEL1** — 将某引脚设为 **AOUT**（读手册具体 pin / 位域）。

**DACR 写入：**

```
位 [15:6] = 10 bit DAC 值
位 [5:0]  通常保留/控制（使能等 — 查手册）
```

**半字写：** 常用 **`STRH`** 写入 **[15:0]** 有效域。

---

### 结合 Ch12 正弦查找表

**目标：** 逻辑分析仪 / 示波器上看到 **模拟正弦波**。

```
循环 r1 = 0 … 359（角度步进 1°）
        ↓
LDR sin_val, [SinTable, r1, LSL #2]   ; Q31，约 [-1, 1)
        ↓
定点换算 → 无符号 10 bit
        ↓
STRH → DACR
        ↓
延时 / 定时器定采样率
```

---

### Q31 → 10 bit 无符号缩放

DAC 要 **0…1023**（中心 512），表值 **有符号 Q31**：

**公式（书中思想）：**

```
v = 512 × sin(θ) + 512     ; 映射到 [0, 1024)
```

**汇编实现思路：**

| 步骤 | 指令级 |
|------|--------|
| 取 sin | **`LDR`** 查表（[Ch12 §12.2](../chapter-12-tables/notes/section-12-2-int-lookup.md) 象限归约若只存 0–90°） |
| 乘 512 | **`SMULL`/`MUL`** 或 **算术移位**（Q 格式下常 **>> 位数** 合并） |
| 加 512 偏置 | **`ADD`** |
| 限幅 | **`USAT`** 或比较（防溢出） |
| 对齐到 [15:6] | **左移 6 bit** 再 **`STRH`** |

**口述：** **查表负责角度→sin**；**乘加移位负责 sin→DAC 码** — [Ch7 Q](../chapter-07-integer-logic-arithmetic/notes/section-7-7-fractional.md) 实战。

---

### 采样率

循环 **无延时** → 频率取决于 **CPU 循环速度**；生产用 **Timer 中断**（[Ch15 §15.7](../chapter-15-exception-handling-v7m/notes/section-15-7-nvic.md)）定 **固定 fs**。

---

### 可复述要点

1. **DACR @ 0xE006C000**，有效数据 **[15:6]**。  
2. **sin 表 (Ch12) + 缩放 (Ch7) + STRH (Ch5)** 三连。  
3. **`v = 512·sin + 512`** 把 **双极正弦** 变成 **单极 DAC 码**。

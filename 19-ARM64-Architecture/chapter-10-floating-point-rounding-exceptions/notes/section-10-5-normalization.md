## §10.5 规格化与抵消 (Normalization and Cancelation)

> **Ch 10 · 浮点舍入与异常** · [章导读](../README.md)

---

### 后规格化 (Post-normalization)

**尾数相乘** 等运算 → 中间积尾数可能 **≥ 2.0**（超出 **1.xxx** 规范形式）。

**硬件：**

```
尾数右移 1 bit，指数 +1  →  恢复 1.f 格式
```

**与 Ch9 Normal** — 维持 **隐含 leading 1** 约定；用户不可见，影响 **舍入位 G/S** 位置。

---

### 抵消 (Cancelation)

**两个接近的数相减**（或 **异号大数相加** = 有效减法）：

```
  1.2345678
- 1.2345670
─────────────
  0.0000008   ← 高位大量消掉，前导零爆炸
```

| 后果 | 说明 |
|------|------|
| **有效位数暴跌** | 相对误差巨大 — **灾难性抵消** |
| **尾数左移重规格化** | 指数减小 |
| **可能 UFC/下溢** | 指数减太多 → subnormal 或 0 |
| **IXC** | 重规格化常伴随舍入 |

**经典：** `sqrt(x+1)-sqrt(x)` 用代数等价式改写 — 数值分析课内容；汇编手写 float 同样要避免。

---

### 与飞控 / 滤波

| 场景 | 风险 |
|------|------|
| **高通 = 输入 - 低通** | 接近频率处抵消 |
| **差分求导** | 小 Δt 放大噪声 |
| **协方差更新** | 需稳定算法（Kalman 平方根形式等） |

**无 FPU：** [Ch7 Q 格式](../chapter-07-integer-logic-arithmetic/notes/section-7-7-fractional.md) 同样怕 **相近大数相减** — 先 **scale** 再算。

---

### 可复述要点

1. **乘后尾数>2 → 右移+指数增**（后规格化）。  
2. **相近相减 → 前导零 → 精度损失 + 可能下溢**。  
3. 算法层 **改公式** 比改舍入模式更重要。

## §5.4 操作数寻址 — 前变址 · 后变址

> **Ch 5 · 加载、存储与寻址** · [章导读](../README.md)

---

### 有效地址 (Effective Address)

```asm
        LDR/STR  Rt, [Rn, #offset]    ; 多种形式
```

**Rn** = 基址寄存器；**offset** = 立即数或 **寄存器 ± 移位**。

---

### 前变址 (Pre-Indexed)

**先** 用 `Rn ± offset` 作地址访问，可选 **写回 Rn**：

```asm
        STR     r0, [r1, #12]!      ; EA = r1+12；然后 r1 ← r1+12
        LDR     r2, [r1, -#4]!       ; EA = r1-4 ；然后 r1 ← r1-4
```

| 要素 | 说明 |
|------|------|
| **`!` 后缀** | **Pre-indexed with write-back** — 访问后基址更新 |
| **用途** | **栈 push/pop**、**顺序写数组**、构造链表指针 |

**栈增长（满递减栈常见）：**

```asm
        STR     r0, [sp, #-4]!        ; 先减 sp，再存 — 压栈
        LDR     r0, [sp], #4          ; 先读，再增 sp — 出栈（后变址）
```

→ **Ch13** 系统化 PUSH/POP 与 APCS。

---

### 后变址 (Post-Indexed)

**先** 用 **Rn 原值** 作地址，**再** Rn ← Rn ± offset：

```asm
        STR     r0, [r1], #12         ; Mem[r1]←r0；然后 r1←r1+12
        LDR     r0, [r1], #4          ; r0←Mem[r1]；然后 r1←r1+4
```

| 对比 | Pre (`!`) | Post (`, #off`) |
|------|-----------|-----------------|
| 地址计算 | 访问 **前** 加偏移 | 访问 **后** 加偏移 |
| 遍历数组 | 两者皆可；post 常像 `for` 里先读 *p 再 p++ |

**数组扫描模式：**

```asm
        ; r1 = &arr[0], 循环
        LDR     r0, [r1], #4          ; 读 word，指针自动 +4
```

---

### 寄存器偏移 + 移位

偏移不必是常数 — 可用 **另一寄存器经移位**：

```asm
        LDR     r0, [r1, r2, LSL #2]  ; EA = r1 + (r2<<2) — 字索引 i
```

| 场景 | 偏移形式 |
|------|----------|
| **arr[i]** (word) | `r2, LSL #2` |
| **查表 table[i]** | 同上 + 基址 = 表头 |
| **结构体 field** | 基址 + `#offsetof` 立即数 |

**Ch12 查表** 直接依赖此模式。

---

### 可复述要点

1. **Pre + `!`**：先算新地址再访问，并 **写回基址**。  
2. **Post `], #n`**：先用旧基址访问，再 **基址 += n**。  
3. **`[Rn, Rm, LSL #2]`** = 数组下标 × 元素大小 — 驱动/内核遍历常用。

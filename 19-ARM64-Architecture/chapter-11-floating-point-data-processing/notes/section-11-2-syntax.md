## §11.2 浮点指令语法

> **Ch 11 · 浮点数据处理** · [章导读](../README.md)

---

### 统一格式

```asm
        V<op>{cond}.F32   {<Sd>}, <Sn>, <Sm>
        ; 或 .F 作为 .F32 简写
```

| 元素 | 说明 |
|------|------|
| **`V`** | 向量/FPU 前缀（M4 单精度标量） |
| **`{cond}`** | 可选 ARM 条件码 — **浮点算术本身不产标志**（§11.4） |
| **`.F32` / `.F`** | **单精度** — M4 本章主战场 |
| **`<Sd>`** | 目标 **s** 寄存器 — 可省略（仅副作用，如 VCMP） |
| **`<Sn>, <Sm>`** | 源 — **s0–s31**；**Sd 可与 Sn/Sm 相同**（in-place） |

**示例：**

```asm
        VADD.F32    s0, s1, s2      ; s0 = s1 + s2
        VMUL.F      s4, s4, s5      ; s4 *= s5
```

---

### 与 Ch9 搬运指令分工

| 类 | 例 | 作用 |
|----|-----|------|
| **Load/Store** | VLDR/VSTR | Mem ↔ s（Ch9） |
| **Convert** | VCVT | int/fixed ↔ float（Ch9） |
| **Data processing** | VADD/VMUL… | **本章** — s ↔ s 运算 |

---

### 无双精度硬件运算（典型 M4）

**d0–d15** 别名存在 — **64 bit 运算** 常软件或 **仅存储**；指令助记符本章以 **`.F32`** 为准。

---

### 可复述要点

1. **`Vop.F32 Sd, Sn, Sm`** — 三寄存器形式为主。  
2. **仅单精度** 数据处理；位操作回 **整数 r**。  
3. **目标可与源相同** — 累乘累加常见。

## §3.4 示例程序 2 — 阶乘 (Factorial)

> **Ch 3 · 指令集简介** · [章导读](../README.md)

---

### 本程序学什么

| 目标 | 内容 |
|------|------|
| **循环** | 递减计数直到条件满足 |
| **比较与标志** | `CMP` 设置 **N/Z/C/V** |
| **条件执行** | ARM7：`MULGT`/`SUBGT` 等 · M4：**IT 块** |

算法：`n!` = n × (n−1) × … × 1，或累乘循环。

---

### CMP 与标志位

```asm
        CMP     r0, #1          ; 伪：r0 - 1，只改标志不改 r0
        ; r0 == 1 → Z=1（EQ）
        ; r0 >  1 → GT（有符号）
        ; r0 <  1 → LT
```

| 标志 | 阶乘里常见用途 |
|------|----------------|
| **Z** | 是否减到 / 等于 1（结束） |
| **N / V** | 有符号 **GT/LT** |
| **C** | 无符号比较（本例多用有符号 GT） |

**CMP ≠ SUB：** `SUB` 改 r0；`CMP` **只刷标志**。系统展开 → [Ch7 §7.3](../../chapter-07-integer-logic-arithmetic/notes/section-7-3-compare.md)

与循环搭配：

```asm
loop
        CMP     r0, #1
        MULGT   r1, r1, r0
        SUBGT   r0, r0, #1
        BGT     loop
```

---

### ARM7TDMI：条件后缀

ARM **32 bit 模式** 可在助记符后加条件码，**不满足则当作 NOP**：

```asm
        MULGT   r1, r1, r0      ; 仅当 GT（有符号大于）时执行
        SUBGT   r0, r0, #1
        BGT     loop            ; 分支也条件化
```

**常见后缀：** `EQ` `NE` `GT` `LT` `GE` `LE` — 读 **CPSR N,Z,C,V**。

**优势：** 少分支 → 少流水线冲刷（历史 ARM 卖点）。  
**Ch8 专章** 展开循环与条件。

---

### Cortex-M4：IT 块 (IF-THEN)

Thumb-2 **没有** 所有指令都带条件后缀；用 **IT** 修饰后续 1–4 条：

```asm
        CMP     r0, #1
        IT      GT
        MULGT   r1, r1, r0      ; IT 块内条件执行
        SUBGT   r0, r0, #1
```

| 要点 | 说明 |
|------|------|
| **`IT` 掩码** | `ITTE EQ` 等指定后续几条是否条件执行 |
| **块内指令** | 必须带与 IT 一致的条件后缀 |
| **密度** | 替代 ARM 模式整条指令条件位 |

**易错：** IT 块中间不能插入无关指令；调试时 **单步看清 IT 掩码**。

---

### 与 C 对照

```c
while (n > 1) {
    result *= n;
    n--;
}
```

编译器生成 **CMP + 条件分支**；ARM 条件后缀/IT 是 **把分支折叠进指令** 的手写优化（现代 C 编译器也会选 B.cond）。

---

### 可复述要点

1. **`CMP` 只改标志** — 比较的两操作数之一常用立即数。  
2. **ARM7：指令后缀条件执行**；**M4：IT 块** 包装 Thumb 指令。  
3. 阶乘 = **循环 + 比较 + 乘法** — Ch3 第一个「小算法」。

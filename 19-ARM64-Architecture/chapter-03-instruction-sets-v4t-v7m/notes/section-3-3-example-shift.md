## §3.3 示例程序 1 — 数据移位 (Shifting Data)

> **Ch 3 · 指令集简介** · [章导读](../README.md)

---

### 本程序学什么

| 目标 | 内容 |
|------|------|
| **汇编程序骨架** | 段/入口/结束 — 第一次见完整 `.s` 结构 |
| **常数进寄存器** | `MOV` 加载立即数 |
| **移位** | `LSL` 逻辑左移 — 等价于 ×2ⁿ |

---

### 典型程序框架（Keil/ARM 语法风格）

```asm
        AREA    Prog1, CODE, READONLY   ; 命名代码段
        ENTRY                           ; 告诉链接器入口符号
start
        MOV     r0, #5                  ; r0 ← 5
        MOV     r1, r0, LSL #2          ; r1 ← r0 << 2  (= 20)
        ; ...
        END                             ; 源文件结束
```

| 伪指令 | 作用 | 详述 |
|--------|------|------|
| **`AREA`** | 定义代码/数据块名与属性 | → **Ch4** |
| **`ENTRY`** | 指定程序入口标签 | 链接器设 Reset/启动 PC |
| **`END`** | 源文件结束 | 不再汇编其后内容 |

**GNU 对照：** `.section .text` · `.global _start` · 无 `END` — 语法不同，**角色相同**。

---

### 核心指令

| 指令 | 含义 | 备注 |
|------|------|------|
| **`MOV Rd, #imm`** | 立即数 → 寄存器 | 立即数范围受编码限制（大数 → **Ch6** `LDR`=`MOVW/MOVT`） |
| **`LSL`** | Logical Shift Left | 低位补 0；可写 `MOV r1, r0, LSL #n` |

**Ch7 扩展：** 还有 LSR、ASR、ROR；标志位 **N/Z/C** 可能更新（视后缀 `S`）。

---

### 调试习惯

- 单步后看 **r0、r1** — 验证 `5 → 20`。
- 对照 [Ch2 §2.3](../../chapter-02-programmers-model/notes/section-2-3-arm7tdmi.md)：**PC 递增、寄存器即状态**。

---

### 可复述要点

1. 汇编程序 = **段声明 + 入口标签 + 指令序列 + END**。  
2. **`MOV` + 移位** 是第一批「不用访存就算数」的工具。  
3. 伪指令是 **给汇编器看的** — 不一定对应一条机器指令。

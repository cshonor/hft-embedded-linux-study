## §7.3 比较指令 — CMP · CMN · TST · TEQ

> **Ch 7 · 整数逻辑与算术** · [章导读](../README.md)

---

### 共性：只改标志，不写结果

四条 **比较/测试** 指令执行 ALU 运算，**丢弃结果**，仅更新 **N Z C V**。

---

### CMP — Compare（减）完整解读

> **一句话：** CMP 是 **专门生成判断条件** 的指令 — **本身不跳转、不分支**，只更新标志位。

#### 0. 生成条件 vs 使用条件（勿混）

| 角色 | 谁 | 干什么 |
|------|-----|--------|
| **生成条件** | **`CMP`**（或 `ADDS`/`SUBS`…） | 算大小/相等，写入 **N/Z/C/V**；**无** 分支、无改 GPR |
| **使用条件** | `MULGT` / `SUBGT` / `BGT`… | **读** 已有标志，决定执不执行 / 跳不跳 |

```asm
        CMP     r0, #1        ; ① 生成条件：刷标志（r0 不变）
        MULGT   r1, r1, r0    ; ② 使用条件：GT 才乘
        SUBGT   r0, r0, #1    ; ② 使用条件：GT 才减
        BGT     loop          ; ② 使用条件：GT 才跳
```

没有前头的 CMP（或其它 `S` 指令），后面的 `GT` 读的是 **旧标志** → 判断错。

#### 1. 核心功能

```asm
        CMP     Rn, Operand2    ; ≈ SUBS 临时, Rn, Op2（结果丢掉）
```

1. 内部：`Rn − Op2`  
2. **只改** CPSR/APSR 的 **N/Z/C/V**  
3. **不改** 任何通用寄存器（Rn 数值不变）  

这些标志就是后面所有带条件后缀指令的依据。

#### 3. 最简例子

```asm
        CMP     r0, #1          ; 算 r0−1，只改标志，r0 不变
```

| (r0 − 1) 直觉 | 标志/条件 | 含义 |
|---------------|-----------|------|
| 结果 = 0 | **Z=1** → **EQ** | r0 **==** 1 |
| 结果 > 0（有符号） | **GT** | r0 **>** 1 |
| 结果 < 0（有符号） | **LT** | r0 **<** 1 |

有符号用 **GT/LT/GE/LE**（看 N,V,Z）；无符号用 **HI/HS/LO/LS**（看 C,Z）。

#### 4. 与阶乘代码搭配（→ [Ch3 §3.4](../../chapter-03-instruction-sets-v4t-v7m/notes/section-3-4-example-factorial.md)）

```asm
loop
        CMP     r0, #1      ; 对比 r0 与 1，刷新标志
        MULGT   r1, r1, r0  ; 仅 r0>1 时乘
        SUBGT   r0, r0, #1
        BGT     loop        ; 仍 >1 则跳回
```

**CMP** 是循环判断核心：先比，再条件乘/减/跳。

#### 5. 配套四标志（CMP 后怎么读）

| 标志 | CMP 语境 |
|------|----------|
| **Z** | 差为 0 → **两数相等** |
| **N** | 有符号差为负 |
| **C** | 无符号大小（减法：C=¬借位）→ **HS/HI** 等 |
| **V** | 有符号减法溢出 → 影响 **GT/LT** 判定 |

完整标志语义 → [§7.2](./section-7-2-flags.md)

#### 6. 关键区分：SUB vs CMP · 谁还能生成条件

| | `SUB` / `SUBS` | `CMP` |
|--|----------------|-------|
| 运算 | r0 − 1 | r0 − 1（结果丢弃） |
| **改 r0？** | **是** | **否** |
| 改标志？ | `SUB` 否；**`SUBS` 是** | **永远改** |

> **CMP = 专为判断的「影子减法」= 丢弃结果的 SUBS。**

**不止 CMP 能生成条件**（带 `S` 也能刷标志）：

| 指令 | |
|------|--|
| `ADDS` / `SUBS` | 加减后更新标志 |
| `ANDS` / `ORRS` / `EORS` | 逻辑后更新标志 |

但要做 **大小对比且不改动寄存器** → 标准写法就是 **`CMP`**。

**极简：** CMP = **条件准备工具**；真正执行/跳转的是后面带条件后缀的指令。

```asm
        CMP     r0, r1          ; 伪：r0 - r1，只设标志
        CMP     r0, #10
```

| 后续 | 含义 |
|------|------|
| **EQ** | r0 == op2（Z=1） |
| **NE** | 不等 |
| **GT/GE/LT/LE** | **有符号** |
| **HI/HS/LO/LS** | **无符号** |

---

### CMN — Compare Negative（加）

```asm
        CMN     r0, r1          ; r0 + r1，只设标志
```

用于 **与 -r1 比较** 等等价变换；较少见但对称于 CMP。

---

### TST — Test bits（与）

```asm
        TST     r0, #0xFF       ; r0 AND #mask，只设标志
        BEQ     no_low_byte     ; Z=1 → 低 8 位全 0
```

| 用途 | 说明 |
|------|------|
| **某位是否为 1** | `TST r0, #BIT` → **NE** |
| **掩码后是否全 0** | **EQ** |

**驱动：** 读 MMIO 后 `TST` 状态位 — 再分支（常配合 **Ch16** Load）。

---

### TEQ — Test Equivalence（异或）

```asm
        TEQ     r0, r1          ; r0 EOR r1，只设标志
        BEQ     equal           ; Z=1 → 每一位相同
```

**判断两寄存器是否 bitwise 相等** — 比 `CMP` 更适合「全等」语义（含符号无关的位模式）。

---

### 与 C 对照

| C | 汇编 |
|---|------|
| `if (a == b)` | `CMP a,b` / `TEQ` + `BEQ` |
| `if (a > b)` signed | `CMP` + `BGT` |
| `if (x & MASK)` | `TST` + `BNE` |

---

### 可复述要点

1. **CMP 只生成条件（刷标志），本身不跳转。**  
2. **MULGT/BGT… 使用条件**；前面须有 CMP/`S` 指令刷标志。  
3. **CMP=减、CMN=加、TST=AND、TEQ=EOR** — 都只更新标志。  
4. 比较后 **紧跟 B.cond / 条件后缀 / IT**（Ch8）。

## §7.2 标志位 — N · V · Z · C · S 后缀

> **Ch 7 · 整数逻辑与算术** · [章导读](../README.md) · [本章概述](./section-0-本章完整概述.md)

---

### CPSR 是什么（ARM32）

**CPSR** = **Current Program Status Register**（当前程序状态寄存器）。

| 存什么 | 例子 |
|--------|------|
| 运算结果标记 | **N Z C V**（最高 4 bit） |
| 处理器模式 | `M[4:0]`（User/IRQ/SVC… → [Ch2 §2.3](../../chapter-02-programmers-model/notes/section-2-3-arm7tdmi.md)） |
| 中断屏蔽 | **I / F** 等 |
| 执行状态 | **T**（Thumb）等 |

你前面的 `CMP`、`GT`/`EQ`/`LT` **全部靠 NZCV 判断**。

| 架构 | 标志放哪 |
|------|----------|
| **ARM7TDMI** | **CPSR[31:28]** = N Z C V |
| **Cortex-M4** | **APSR**（xPSR 应用视图）里的 N Z C V；逻辑同 |
| **饱和 DSP** | 另有 **Q** — §7.4/7.5 |

⚠ **NZCV 在最高 4 bit（31…28），不是「低 8 位」。**

```
CPSR  bit31  bit30  bit29  bit28  …  (模式/中断等更低位)
        N      Z      C      V
```

---

### NZCV 四位详解

#### N — Negative（bit 31）

| | |
|--|--|
| **置 1** | 结果当作有符号时为 **负**（最高位=1） |
| **置 0** | 正数或 0 |
| **用途** | 有符号正负；配合 V → GT/LT |

例：`CMP r0, #5`，r0=3 → `3−5=−2` → **N=1**。

#### Z — Zero（bit 30）

| | |
|--|--|
| **置 1** | 结果 **全 0** |
| **置 0** | 非 0 |
| **用途** | **EQ / NE**（`CMP a,b` 差为 0 → Z=1 → 相等） |

#### C — Carry（bit 29）

| 场景 | C |
|------|---|
| **加法** | 最高位产生进位 → **C=1** |
| **减法 / CMP** | **C = ¬借位** — 无符号够减 → C=1；不够减 → **C=0** |

专给 **无符号** 比较（**HI / LO / HS / LS**）。  
⚠ 不要记成「有借位就 C=1」— ARM 减法是 **有借位则 C=0**。

#### V — Overflow（bit 28）

有符号加减 **超出可表示范围** → **V=1**。  
例：正+正得负、负+负得正 → 溢出。  
配合 **N** → 有符号 **GT / LT / GE / LE**。

```
0x7FFFFFFF + 1 → 0x80000000
有符号：最大正 +1 →「负」→ V=1
```

**C 与 64 位链：**

```asm
        ADDS    r0, r0, r1      ; 低字，C=进位
        ADC     r2, r2, r3      ; 高字 + C（§7.4）
```

---

### 条件码如何由 NZCV 算出

#### 有符号（CMP 后 · 可负整数）

| 条件 | NZCV 规则 |
|------|-----------|
| **EQ** | Z = 1 |
| **NE** | Z = 0 |
| **GT** | Z=0 **且** N == V |
| **LT** | N **≠** V |
| **GE** | N == V |
| **LE** | Z=1 **或** N ≠ V |

#### 无符号（地址、长度）

| 条件 | NZCV 规则 |
|------|-----------|
| **HS**（≥） | C = 1 |
| **LO**（<） | C = 0 |
| **HI**（>） | C=1 **且** Z=0 |
| **LS**（≤） | C=0 **或** Z=1 |

口述表（EQ/GT/LT…）→ 下文「常用大小比较条件码」；分支 → [Ch8](../../chapter-08-branches-loops/notes/section-8-2-branches.md)。

---

### 完整例子：CMP 后读标志

```asm
        MOV     r0, #4
        CMP     r0, #6          ; 4 − 6  → 有符号 −2
```

| 标志 | 值 | 理由 |
|------|-----|------|
| **N** | **1** | 有符号结果为负 |
| **Z** | **0** | 非 0 |
| **C** | **0** | 无符号 4<6，减法有借位 → C=¬借位=0 |
| **V** | **0** | 未溢出 |

**N ≠ V** → **LT** 成立 → `ADDLT` / `BLT` 会执行（r0 < 6）。

---

### 关键总结

1. **CPSR** = 状态总寄存器；**NZCV** = 里做大小判断的 4 个开关（bit31–28）  
2. **`CMP` / `SUBS` / `ADDS`…** 刷新 NZCV  
3. 所有 `GT`/`EQ`/`LT`/`HI`… 后缀 = **读 NZCV 决定是否执行**  
4. **N+V** 管有符号；**C** 管无符号；**Z** 管相等  

---

### 如何更新标志

| 方式 | 例子 |
|------|------|
| **比较类** | `CMP`/`CMN`/`TST`/`TEQ` — 只改标志，**不写结果寄存器**（§7.3） |
| **`S` 后缀** | `ADDS` `SUBS` `MOVS` — 写结果 **并** 更新 N,Z,C,V |

```asm
        ADDS    r0, r1, r2      ; r0=r1+r2，更新标志
        ADD     r0, r1, r2      ; 同上，但 **不** 改标志（默认）
```

**习惯：** 需要分支/条件执行前用 **`ADDS`/`SUBS`** 或先 **`CMP`**。

### 常用大小比较条件码（配套 CMP）

> **生成条件**用 `CMP`；**使用条件**用后缀。一组标志可同时喂给多条 `xxEQ`/`xxGT`…  

#### 一、相等：EQ / NE

| 后缀 | 触发 | 含义 |
|------|------|------|
| **EQ** | **Z=1** | 相等（差为 0） |
| **NE** | **Z=0** | 不等 |

```asm
        CMP     r0, #5
        ADDEQ   r1, r1, #1      ; r0 == 5 才加
        BEQ     label           ; 相等就跳

        CMP     r0, #5
        BNE     loop            ; r0 ≠ 5 就循环
```

#### 二、有符号（可负）：GT / LT / GE / LE

| 后缀 | 含义（`CMP r0, r1` 后） |
|------|-------------------------|
| **GT** | r0 **>** r1（有符号）— [MULGT/SUBGT](../../chapter-03-instruction-sets-v4t-v7m/notes/section-3-4-example-factorial.md) |
| **LT** | r0 **<** r1 |
| **GE** | r0 **≥** r1 |
| **LE** | r0 **≤** r1 |

```asm
        CMP     r0, #3
        MOVLT   r0, #0          ; r0 < 3 → 清零
        BLT     exit
```

**GT 标志条件：** `Z==0 && N==V`（详 [§7.3](./section-7-3-compare.md)）。

#### 三、无符号（地址、长度、缓冲区尺寸）：HI / LO / HS / LS

| 后缀 | 含义 |
|------|------|
| **HI** | 无符号 **大于**（Higher） |
| **LO** | 无符号 **小于**（Lower） |
| **HS** | 无符号 **≥**（Higher or Same）— 常 ≡ **CS**（C=1） |
| **LS** | 无符号 **≤**（Lower or Same） |

比指针/字节数用 **HI/LO**；比温度用有符号 **GT/LT** — **别混**。

#### 四、日常三基础（有符号）

| | |
|--|--|
| **EQ** | 等于 |
| **GT** | 大于 |
| **LT** | 小于 |

```asm
        CMP     r0, #10
        ADDEQ   r1, r1, #1      ; ==10 → +1
        ADDGT   r2, r2, #1      ; >10  → +1
        ADDLT   r3, r3, #1      ; <10  → +1
```

（注意：ARM 的 `ADD` 一般要写满 `Rd, Rn, Op2`，不能只写 `ADDEQ r1, #1`。）

#### 补充

1. 一次 `CMP` 之后，后面多条条件指令 **共用同一组标志**  
2. 有符号 **GT/LT/GE/LE**；无符号 **HI/LO/HS/LS**  
3. 后缀通用：`ADD`/`SUB`/`MUL`/`B`…（Thumb-2 放 **IT** 里）→ [Ch8](../../chapter-08-branches-loops/notes/section-8-2-branches.md)

---

### Q 标志（饱和，预告）

**`SSAT`/`USAT`** 等把结果 **钳位** 到范围内时 **Q=1** — 音频/DSP 检测失真（§7.4–7.5）。

---

### 与 Ch8 / HFT

| 用途 | 标志 |
|------|------|
| `if (a>b)` signed | `CMP` + **GT/LE** 读 N,V,Z |
| `if (a==0)` | **EQ** 读 Z |
| 无锁 CAS 重试 | 常读 **Z** 或 **NE** |
| 64 位 counter | **ADC** 链读 **C** |

---

### 可复述要点

1. **CPSR[31:28] = N Z C V**；条件后缀读这四位。  
2. **N/V** 有符号；**C** 无符号；**Z** 相等。  
3. 减法 **C=¬借位**（有借位 → C=0）。  
4. **`S` / CMP 族** 更新标志；再接 `B.cond` / 条件后缀（Ch8）。

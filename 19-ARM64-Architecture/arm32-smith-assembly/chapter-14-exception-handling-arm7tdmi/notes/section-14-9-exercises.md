### 练习方向（原书典型）

| 类型 | 练什么 |
|------|--------|
| **向量表** | 写 **8 项 `B handler`** 最小表 |
| **IRQ stub** | **STMDB/LDMIA + SUBS pc,lr,#4** |
| **优先级** | 场景题：Abort + IRQ 同时谁先做 |
| **LR 修正** | 为何 Data Abort 用 **#8**、IRQ 用 **#4** |
| **VIC** | 画 **LDR pc ← VIC** 相对 **轮询 IRQ** 的路径 |
| **SVC** | 实现 **`SVC #1` 打印单字符**（Semihosting 概念） |

### 常见错误

| 错误 | 后果 |
|------|------|
| 用 **`BX lr`** 直接返 IRQ | **CPSR 未恢复**、模式错 |
| **SUBS #imm** 用错 | 返回到 **错误指令** |
| 忘记 **清外设中断** | **IRQ 风暴** 重入 |
| 向量槽写 **地址常数** 而非 **B/LDR PC** | CPU 当指令执行 **数据** → fault |

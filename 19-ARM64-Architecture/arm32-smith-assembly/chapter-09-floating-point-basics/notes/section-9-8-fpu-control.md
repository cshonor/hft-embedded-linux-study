### FPSCR — 浮点状态与控制

类似整数 **APSR**，浮点侧 **FPSCR** 含：

| 类别 | 内容 |
|------|------|
| **比较标志** | **N Z C V**（浮点比较后） |
| **舍入模式 RMode** | 向 nearest/even 等（→ **Ch10**） |
| **FZ (Flush-to-Zero)** | 次正常 → 0 |
| **DN (Default NaN)** | NaN 传播策略 |
| **累积异常标志** | **IDC/OFC/UFC/OFC/DZC/IOC** 等 — 除零、溢出、下溢、不精确 |

**读/写 FPSCR** — 特权 **MRS/MSR** 类到系统寄存器（具体助记符见手册）。

### 可复述要点

1. **CPACR 开 CP10/11** — FPU 电源/门禁。  
2. **FPSCR** = 浮点标志 + 舍入 + 异常累积。  
3. 裸机启动代码常 **Init FPU** 一段 — 与 Init 时钟同级。

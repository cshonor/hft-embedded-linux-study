## §9.6 特殊值 — 无穷大 · NaN

> **Ch 9 · 浮点简介** · [章导读](../README.md)

---

### 4. 无穷大 (Infinities)

**exp 全 1，fraction 全 0** → **+∞** / **-∞**。

| 来源 | 例子 |
|------|------|
| 溢出 | 极大运算结果 |
| 除法 | **有限 / 0**（非 NaN 路径） |

**比较：** ∞ 参与 ordered compare 有特殊规则（Ch11 `VCMPE`）。

---

### 5. 非数字 (NaN — Not a Number)

**exp 全 1，fraction ≠ 0** → **NaN**。

| 类型 | 行为 |
|------|------|
| **sNaN (signaling)** | **触发异常** — 非法使用时告警 |
| **qNaN (quiet)** | **静默传播** — `NaN op x → NaN` |

**典型产生：**

- `0.0 / 0.0`  
- `∞ × 0`  
- 未初始化 float 内存（若碰巧 bit 模式像 NaN）  
- 非法 `sqrt(-1)` 等  

**调试：** 一旦出现 NaN，**整条链污染** — 飞控/PID 必须 **检测与 clamp**。

---

### 五类一览

| 类 | exp | frac | 例 |
|----|-----|------|-----|
| Normal | 中间 | 任意 | 3.14 |
| Subnormal | 0 | ≠0 | 极小 |
| ±0 | 0 | 0 | +0, -0 |
| ±∞ | 全1 | 0 | overflow |
| NaN | 全1 | ≠0 | 0/0 |

→ **Ch10** 舍入/异常标志在 **FPSCR** 累积。

---

### 可复述要点

1. **∞ = 溢出/除零的极端**；**NaN = 未定义或未初始化**。  
2. **qNaN 静默传播** — C 里 `isnan()` 必备。  
3. 嵌入式：**别让控制环跑出 NaN** — 与 Ch7 **SSAT** 同类「防崩」。

### 汇编器操作符（Assembly-time Operators）

在 **汇编阶段** 对 **常量表达式** 求值 — **CPU 不执行**这些运算。

| 操作符（Keil 风格） | 含义 | 典型用途 |
|---------------------|------|----------|
| **`:SHL:` / `:SHR:`** | 逻辑左/右移 | `MASK EQU (1 :SHL: 5)` |
| **`:AND:` / `:OR:`** | 按位与/或 | 合并寄存器位域掩码 |
| **`:MOD:`** | 取模 | 对齐/分页常量 |
| **`+ - * /`** | 算术 | `TABLE_SIZE EQU ROWS * COLS` |

**配合伪指令：**

```asm
GPIO_BASE    EQU     0x40020000
PIN_MASK     EQU     (1 :SHL: 3) :OR: (1 :SHL: 7)
config_word  DCD     PIN_MASK
```

**C 对照：** 与 `#define BIT(n) (1U << (n))` 同一层 — **编译/汇编前** 就算出数。

**GNU：** 用 C 预处理器或 gas 表达式 `(1 << 3) | (1 << 7)`。

---

### 与子程序/指令的区别

```
EQU 表达式里的 :SHL:     → 汇编器算 → 写入 .o 的常数
指令 LSL r0, r1, #2      → CPU 运行时移位
```

混淆两者 = 读 listing 时 **分不清哪行进了 Flash**。

### CCS 数学函数（选读 · TI 专用）

CCS 汇编器可在 **数据段生成期** 调用内置函数，例如：

| 函数 | 用途 |
|------|------|
| **`$$sin` / `$$cos`** | 生成 sin/cos **浮点常数表** |
| **`$$sqrt`** | 开方表 |

**场景：** **浮点查找表**（→ **Ch12 Tables**）— 汇编时算好 ` .float` 序列，避免 MCU 启动后逐点计算。

**本路线：** WSL + GNU **无** 同等内置 — 用 Python/CMake 生成 `.inc` 或 C 数组更常见（[24 飞控](../../../24-Motion-Control-Motor/) 姿态表）。

---

### 与 HFT / 驱动的联系

| 场景 | 汇编期常量 |
|------|------------|
| **MMIO 位掩码** | `EQU` + `:SHL:` — 所有平台 |
| **DMA 描述符 flags** | 同 |
| **查表 sin/cos** | CCS 汇编器函数 or 离线生成 |

---

### 可复述要点

1. **`:SHL:` 等只在 EQU/DCD 表达式里** — 构建配置字，不是运行时指令。  
2. **CCS `$$sin` 等** = 汇编时生成浮点表 — 可选，非 GNU 标配。  
3. Linux 驱动更常 **设备树 + C 宏** 表达掩码；读 U-Boot 仍常见 `EQU`/`.equ`。

### 二分法 (Bisection) 求根

**问题：** 连续函数 **f(x)** 在 **[a,b]** 异号 → 中间有根。

```
while (b-a > tol):
    mid = (a+b)/2
    if f(mid) 与 f(a) 同号:
        a = mid
    else:
        b = mid
```

**汇编要素：**

| 步骤 | 浮点指令 |
|------|----------|
| 中点 | **VADD** + **VMUL** 0.5 或 **VADD** 半区间 |
| 判符号 | **VCMP** + **VMRS** + **B** / **IT** |
| 循环 | **Ch8** `SUBS`/`BNE` 或 **CBNZ** 计数 |

**与 [Ch12 二分查找表](../../chapter-12-tables/)：** 同名思想 — 表是 **离散 key**，此处 **连续 f(x)**。

---

### 子程序：泰勒级数 `sin(x)`

**f(mid)** 需 **sin** — 示例用 **泰勒展开** 多项式近似：

```
sin(x) ≈ x - x³/3! + x⁵/5! - …
```

**展示：**

- **VMUL/VFMA** 累加多项式项  
- **VNEG/VADD** 交替符号  
- **`BL` 子程序**（Ch13）传参 **s0** 返回 **s0**  
- 精度 vs 项数 — **IXC** 每步舍入

**工程现实：** 生产用 **C libm** 或 **查表+插值**（Ch12）— 示例为 **指令练习**。

### 可复述要点

1. **二分 = VCMP 判符号 + 区间减半循环**。  
2. **泰勒 sin = MAC 密集型** — 练 VFMA/VMLA。  
3. 综合例 **衔接 Ch8 分支 + Ch13 BL** — 浮点三章闭环。

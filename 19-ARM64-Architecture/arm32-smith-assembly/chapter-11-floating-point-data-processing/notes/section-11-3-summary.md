## §11.3 浮点指令摘要

> **Ch 11 · 浮点数据处理** · [章导读](../README.md)

---

### 按功能分类（Cortex-M4 FPU）

| 类别 | 代表指令 | 小节 |
|------|----------|------|
| **比较** | `VCMP`, `VCMPE` | §11.4 |
| **非算术** | `VABS`, `VNEG` | §11.6 |
| **加减** | `VADD`, `VSUB` | §11.7 |
| **乘** | `VMUL` | §11.7 |
| **除** | `VDIV` | §11.7 |
| **平方根** | `VSQRT` | §11.7 |
| **链式 MAC** | `VMLA`, `VMLS`, … | §11.7 — **中间舍入** |
| **融合 MAC** | `VFMA`, `VFNMA`, … | §11.7 — **一次舍入** |
| **移标志** | `VMRS`, `VMSR` | §11.4 — APSR ↔ FPSCR |

**无：** `VAND`/`VEOR` 等 — **VMOV 到 r**，用整数逻辑。

---

### 异常关联（Ch10）

| 运算 | 常见标志 |
|------|----------|
| **VDIV** | **DZC**（非零/0）· **IXC** |
| **VSQRT** 负数 | **IOC** → NaN |
| **大数 VMUL** | **OFC** |
| **任意** | **IXC** 若舍入 |

---

### 可复述要点

1. M4 FPU = **算术 + compare + MAC + sqrt/div** — 非完整 SIMD。  
2. **逻辑/bit 在整数侧** 做。  
3. 读手册 **指令列表** 时按上表归类即可。

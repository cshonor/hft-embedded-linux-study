## §13.2 堆栈 — LDM/STM · PUSH/POP · 满/空 · 递增/递减

> **Ch 13 · 子程序与堆栈** · [章导读](../README.md) · [本章概述](./section-0-本章完整概述.md)

---

### SP 与 LIFO

- **r13** 固定作 **堆栈指针 (SP)**（[Ch2 程序员模型](../chapter-02-programmers-model/)）。  
- 概念：**后进先出** — 最后 `PUSH` 的项，第一个 `POP` 弹出。  
- 物理：SP 指向 RAM 中一段连续区域；**满递减** 时向 **低地址** 生长（C 默认）。

---

### LDM / STM — 多寄存器传输

相对单字的 **`LDR`/`STR`**：

| 指令 | 含义 |
|------|------|
| **STM** | Store Multiple — 多个 **Rt** 写入内存 |
| **LDM** | Load Multiple — 从内存读回多个 **Rt** |

**好处：** 一条指令完成 **保存/恢复 8 个寄存器** — 代码更小、常更快（一次取指）。

**语法示例：**

```asm
    STMDB   sp!, {r4, r5, r6, lr}   ; 压栈（事前递减）
    LDMIA   sp!, {r4, r5, r6, pc}   ; 出栈并返回
```

**`!`** — 写回基址（SP 随操作更新）。

---

### 寻址后缀：IA · IB · DA · DB

基址寄存器（常为 **SP**）相对存取时刻如何变化：

| 后缀 | 全称 | 行为 |
|------|------|------|
| **IA** | Increment After | 传送 **后** 基址 **+** |
| **IB** | Increment Before | 传送 **前** 基址 **+** |
| **DA** | Decrement After | 传送 **后** 基址 **−** |
| **DB** | Decrement Before | 传送 **前** 基址 **−** |

**ARM7TDMI：** 四种全支持。  
**Cortex-M3/M4：** 为精简仅 **IA + DB** — 恰好对应 **`POP`/`PUSH`** 所需。

---

### PUSH 与 POP（汇编器助记符）

| 助记符 | 实际指令 | 典型用途 |
|--------|----------|----------|
| **`PUSH {…}`** | **`STMDB sp!, {…}`** | 压栈 |
| **`POP {…}`** | **`LDMIA sp!, {…}`** | 出栈 |

**口述：** 写 **`PUSH`/`POP`** 即可；反汇编时看到 **STMDB/LDMIA** 不要陌生。

---

### 13.2.3 堆栈的四种类型

两个维度组合：

| 维度 | 选项 |
|------|------|
| **方向** | **递增 (Ascending)** — 向高地址长 · **递减 (Descending)** — 向低地址长 |
| **SP 含义** | **满 (Full)** — SP 指向 **最后有效项** · **空 (Empty)** — SP 指向 **下一空闲槽** |

| 类型 | 助记符后缀 | 说明 |
|------|------------|------|
| Full Descending | **FD** | **ARM C/C++ 默认** |
| Empty Descending | ED | |
| Full Ascending | FA | |
| Empty Ascending | EA | |

**配对示例（FD）：**

```asm
    STMFD   sp!, {r4-r7, lr}    ; 压栈 ≡ PUSH（满递减）
    LDMFD   sp!, {r4-r7, pc}    ; 出栈；pc←lr 返回
```

**满递减口述：** SP 初始指向栈区 **高地址一端**；每次压栈 **SP 先减** 再写 — 栈向低地址 **长**。

---

### 与 Ch5 单字访问的关系

- 小数据、随机访问 → **`LDR/STR`** + 偏移  
- **函数序言/尾声**、异常保存 → **`STM/LDM`** 批量  

内核 **`push {r0-r12, lr}`**（AArch32）思想相同。

---

### 可复述要点

1. **SP = r13**；**PUSH = STMDB**，**POP = LDMIA**。  
2. C 用 **满递减 (FD)** — **`STMFD`/`LDMFD`** 成对。  
3. **M 系只有 IA/DB** — 不影响正常栈操作。  
4. 列表里写 **`pc`** 等价于 **`POP` 返回地址到 PC** — 子程序尾声常用。

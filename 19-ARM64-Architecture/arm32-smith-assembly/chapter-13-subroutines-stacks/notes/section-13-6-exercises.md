## §13.6 练习题

> **Ch 13 · 子程序与堆栈** · [章导读](../README.md) · [本章概述](./section-0-本章完整概述.md)

---

### 练习方向（原书典型）

| 类型 | 练什么 |
|------|--------|
| **PUSH/POP** | 手动 **`STMFD`/`LDMFD`** 与助记符对照 |
| **子程序** | 写 **`Delay`** / **`Max`** — 序言、尾声、**`pc` 返回** |
| **递归** | 阶乘或 Fib — 每帧 **保存 LR + 局部在栈** |
| **传参** | 同一函数：**全寄存器** vs **部分压栈** — 画栈图 |
| **AAPCS** | 实现 **`int swap(int *a,int *b)`** 供 C 调用 |
| **浮点** | **`float dot3`** — 哪些 **s** 寄存器要 **VPUSH** |

---

### 自测 Checklist

- [ ] 解释 **PUSH** 对应哪种 **STM** 寻址（**DB**）  
- [ ] 说明 **满递减 (FD)** 栈 **SP 指向哪里**、向哪生长  
- [ ] 写 **`STMDB sp!, {r4-r7,lr}`** 对应的 **`LDMIA … pc`** 返回  
- [ ] 说明子程序内再 **`BL`** 时为何必须 **保存 LR**  
- [ ] 背出 **r0–r3** 与 **r4–r8,r10,r11** 在 AAPCS 中的角色  
- [ ] 说明 **8 字节栈对齐** 的原因  

---

### 常见错误

| 错误 | 后果 |
|------|------|
| 用了 **r4** 不保存 | C 局部变量/循环变量被破坏 |
| 忘记保存 **LR** 又 **BL** | 返回跳进随机地址 |
| **`LDM … pc`** 列表顺序错 | 返回地址进错寄存器 |
| 栈不对齐调 **printf** | HardFault 或 subtle bug |

## §3.7 示例程序 5 — 整数与浮点寄存器传数

> **Ch 3 · 指令集简介** · [章导读](../README.md)  
> **标签：** **选做** — 同 §3.6，FPU 路线

---

### 本程序学什么

| 目标 | 内容 |
|------|------|
| **整 ↔ 浮 搬运** | 通用寄存器与 **s** 寄存器之间传比特模式 |
| **浮点常数加载** | **`VLDR.F`** 从内存加载（如阿伏伽德罗常数） |

---

### VMOV 在整数与浮点间

```asm
        VMOV    S0, R0          ; 将 r0 的位模式拷到 s0（reinterpret）
        VMOV    R0, S0          ; 反向
```

| 注意 | 说明 |
|------|------|
| **不是类型转换** | 多数形式是 **位拷贝**；`VCVT` 才是 int↔float 转换（Ch9） |
| **ABI** | 函数返回值 float 在 **s0/r0** 等约定 — **Ch13 APCS** / AAPCS |

---

### VLDR.F — 加载浮点常数

```asm
        VLDR    S1, =6.022e23   ; 伪：从 literal pool 加载 32-bit float 编码
```

| | `VMOV.F #imm` | `VLDR.F` |
|---|---------------|----------|
| 常数大小 | 有限编码范围 | **任意单精度**（经内存池） |
| 类比整数 | `MOV #small` | `LDR =big`（§3.5） |

**Ch6 文字池：** 整数与浮点 **literal pool** 机制相同 — PC 相对取常数。

---

### 五程序递进关系

```
§3.3 MOV/LSL     → 整数基础
§3.4 CMP/循环    → 控制流
§3.5 EOR/LDR=    → 逻辑 + 大常数
§3.6 CPACR/VADD  → 启用 FPU + 运算
§3.7 VMOV/VLDR   → 两套寄存器文件协作
```

---

### 可复述要点

1. **VMOV** 可在 **R ↔ S** 间搬位模式；真正换算用 **VCVT**。  
2. **VLDR.F** 类似整数 **`LDR =`**，用于大/精确浮点常数。  
3. 整数主线读完 **§3.5** 即可进 **Ch4**；浮点留 Ch9 或飞控时再补。

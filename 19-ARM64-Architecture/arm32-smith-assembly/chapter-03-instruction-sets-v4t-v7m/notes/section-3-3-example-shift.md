## §3.3 示例程序 1 — 数据移位 (Shifting Data)

> **Ch 3 · 指令集简介** · [章导读](../README.md)

---

### 本程序学什么

| 目标 | 内容 |
|------|------|
| **汇编程序骨架** | 段/入口/结束 — 第一次见完整 `.s` 结构 |
| **常数进寄存器** | `MOV` 加载立即数 |
| **移位** | `LSL` 逻辑左移 — 等价于 ×2ⁿ |

---

### 典型程序框架（Keil/ARM 语法风格）

```asm
        AREA    Prog1, CODE, READONLY   ; 命名代码段
        ENTRY                           ; 告诉链接器入口符号
start
        MOV     r0, #5                  ; r0 ← 5
        MOV     r1, r0, LSL #2          ; r1 ← r0 << 2  (= 20)
        ; ...
        END                             ; 源文件结束
```

| 伪指令 | 作用 | 详述 |
|--------|------|------|
| **`AREA`** | 定义代码/数据块名与属性 | → **Ch4** |
| **`ENTRY`** | 指定程序入口标签 | 链接器设 Reset/启动 PC |
| **`END`** | 源文件结束 | 不再汇编其后内容 |

**GNU 对照：** `.section .text` · `.global _start` · 无 `END` — 语法不同，**角色相同**。

---

### 核心指令 · MOV 详解

| 指令 | 含义 | 备注 |
|------|------|------|
| **`MOV Rd, Op2`** | 复制到寄存器（不做算术） | 源 = 寄存器或合法立即数 |
| **`LSL`** | Logical Shift Left | 低位补 0；可写 `MOV r1, r0, LSL #n` |

#### MOV — 数据传送（核心）

```asm
        MOV     Rd, Operand2    ; Rd ← Op2（纯搬运，不运算）
```

| | |
|--|--|
| **Rd** | 目标寄存器 |
| **Op2** | 源：另一寄存器，或 **可编码立即数**；也可带移位：`MOV r1, r0, LSL #2` |

**基础示例：**

```asm
        MOV     r0, #5          ; r0 = 5
        MOV     r1, r0          ; r1 = r0（寄存器拷贝）
        MOV     r2, #0          ; 清零常用（亦常见 EOR r2,r2,r2）
```

**与阶乘联动（→ [§3.4](./section-3-4-example-factorial.md)）：**

```asm
        MOV     r1, #1          ; 累乘初值
        MOV     r0, #6          ; 算 6!
loop
        CMP     r0, #1
        MULGT   r1, r1, r0
        SUBGT   r0, r0, #1
        BGT     loop
```

#### 限制（ARM32 / Thumb）

| 点 | 说明 |
|----|------|
| **立即数** | 经典 ARM：`#imm` 须落在 **8 bit + 偶数旋转** 可编码集合（不是「任意 32 位数」） |
| **大常数** | `MOVW`/`MOVT` 拼 32 位，或 `LDR Rd, =const`（→ [Ch6](../../chapter-06-constants-literal-pools/)） |
| **不能碰内存** | 内存 → 寄存器用 **LDR**；寄存器 → 内存用 **STR**。MOV **只管寄存器↔立即数/寄存器** |
| **条件后缀** | ARM32：`MOVGT` / `MOVEQ`…（M4 放 IT 块） |

```asm
        CMP     r0, #1
        MOVGT   r2, r0          ; 仅 r0>1 时拷贝
```

#### MOV vs LDR vs STR

| 指令 | 方向 |
|------|------|
| **MOV** | 立即数/寄存器 → 寄存器 |
| **LDR** | **内存 → 寄存器**（加载） |
| **STR** | **寄存器 → 内存**（存储） |

**口诀：** MOV 做「寄存器桌面上的搬运」；过内存必须 LDR/STR。

**Ch7：** `MOVS` 可刷 N/Z；移位还有 LSR、ASR、ROR。

### 可复述要点

1. 汇编程序 = **段声明 + 入口标签 + 指令序列 + END**。  
2. **`MOV` + 移位** 是第一批「不用访存就算数」的工具。  
3. 伪指令是 **给汇编器看的** — 不一定对应一条机器指令。

## §16.5 练习题

> **Ch 16 · 内存映射外设** · [章导读](../README.md) · [本章概述](./section-0-本章完整概述.md)

---

### 练习方向（原书典型）

| 类型 | 练什么 |
|------|--------|
| **UART** | 写 **`SendString`** — 循环 **`BL SendByte`** |
| **PINSEL** | 画 **R-M-W** 改 **2 bit 域** 而不伤其它 pin |
| **DAC** | 改查表步进 / 缩放 — 观察 **频率/幅度** 变化 |
| **GPIO** | 三色 **顺序闪烁** · 用 **地址掩码** 代替 BIC/ORR |
| **综合** | UART **打印** 当前 DAC 角度 / LED 状态 |

---

### 自测 Checklist

- [ ] 解释 **MMIO** 为何用 **LDR/STR** 而非 I/O 指令

### 可复述要点

1. 精读章 — 三节 **各写一条最小 bring-up 链** 即达标。  
2. **轮询 UART** 是 **bring-up 第一技能**；**GPIO+时钟** 是 **M4 第二技能**。  
3. 所有板卡 **地址/位域不同**，流程 **可迁移**。

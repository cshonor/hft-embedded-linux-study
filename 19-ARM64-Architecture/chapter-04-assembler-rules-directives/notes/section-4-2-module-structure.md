## 一、核心：单行汇编四分段标准格式

```
{标签}  {指令/伪指令}  {操作数}  {; 注释}
```

四个字段均为 **可选**；**有指令/伪指令时必须缩进** — ARM 汇编（Keil）与 GNU 汇编通用基础规范。

### 各字段强制规则

| 字段 | 书写规范 | 关键禁忌 |
|------|----------|----------|
| **标签** | 必须写在 **行首（第 1 列）**，无前置空格 | 前面有空格会被当成指令 → 报错；末尾常加 `:`（部分汇编器可省略） |
| **指令/伪指令** | 前方必须有 **空格/Tab 缩进**，不能顶格 | 顶格会被当成 **标签** → 语法错误 |
| **操作数** | 跟在指令后，逗号分隔；无操作数可省略 | — |
| **注释** | **Keil/ARM：** `;` 至行尾<br>**GNU as：** `@` 单行 · `/* */` 块注释 | `;` / `@` **只作用本行**，不跨行 |

**背一句：**

> **顶格 = 标签；缩进 = 指令。**

---

## 二、示例逐行解析

```asm
loop            ; loop 标签：行首无空格，跳转目标
    ADD r0, r0, #1  ; ADD：前置 Tab；r0 = r0 + 1
    B   loop        ; 无条件跳转到 loop
```

| 行 | 作用 |
|----|------|
| `loop` | **标签** — 给当前指令地址起别名；`B`/`BL`/调用可引用 |
| `ADD r0, r0, #1` | 算术指令；必须缩进 |
| `B loop` | 无条件分支 → 标签 `loop` |

## 三、关键拓展

### 1. 标签的本质

标签 = **当前行代码的内存地址别名**；汇编阶段换成地址/偏移。  
对应 [Ch3](../../chapter-03-instruction-sets-v4t-v7m/) 示例里的 `start`、`loop` — 分支、链接、函数入口全靠标签。

### 2. 两种工具链注释

| 工具链 | 注释 |
|--------|------|
| **ARM / Keil MDK** | `;` … 至行尾 |
| **GNU as**（GCC、Linux、裸机） | `@` 单行 · `/* … */` 块 |

工程里按工具链选用；混用源时别用错注释符导致「整行被当注释」。

### 3. 易错坑 ⚠

1. **指令顶格** → 当成标签 → 未定义符号 / 语法错  
2. **标签前加空格** → 标签失效，无法作跳转目标  
3. **行首先写 `;`** → **整行注释掉**，后面的标签也不会生效  

### 4. 模块结构整体意义

整个 `.s` / `.asm` = 多行上述格式 + **段伪指令**（Keil `AREA` / GNU `.text` · `.data`）划分存储区 → 完整可汇编模块。  
裸机、内核 entry、U-Boot 汇编底层都循此规范 → [§4.4 伪指令](./section-4-4-directives.md)

## 四、常量表示（本节约）

| 类型 | 典型写法 |
|------|----------|
| 十进制 | `42` |
| 十六进制 | Keil `0xFF` · CCS 或见 `0xFFh` |
| 字符 | `'A'` |
| 字符串 | `"Hello"`（NUL / 转义随工具链略异） |

C 对照：`.rodata` 的 `"Hello\n\0"` ↔ Keil `DCB "Hello",0` / GNU `.asciz "Hello"`。

## 五、模块 = 一个源文件

| 概念 | 说明 |
|------|------|
| **多文件** | 每文件若干 **段**；链接器合并（→ [Ch1 §1.7](../../chapter-01-overview-computing-systems/notes/section-1-7-tools.md)） |
| **符号** | LOCAL / EXPORT / GLOBAL — 跨文件靠全局符号 |
| **与 C 混合** | `.c` + `.s` → 各自 `.o` → 链接（→ **Ch18**） |

### GNU (gas) 行格式示例

```gas
.global _start
_start:                     @ 标签（可带 :）
    mov     r0, #1          @ 指令须缩进
.Lloop:
    b       .Lloop
```

| Keil | GNU |
|------|-----|
| 标签第 1 列 | 同上 |
| `;` | `@` 或 `/* */` |

---

## 六、记忆要点

1. **顶格 = 标签，缩进 = 指令**  
2. 标签是 **地址别名**；`B` / `BL` 靠标签定位  
3. 注释符跟 **工具链**（Keil `;` · GNU `@`）  
4. 源文件 = 标签 + 指令 + 伪指令（段）→ `.o`

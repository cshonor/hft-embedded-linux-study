## §7.4 数据处理 — 逻辑 · 移位 · 加减 · 乘除 · 饱和

> **Ch 7 · 整数逻辑与算术** · [章导读](../README.md)

---

### 布尔逻辑

| 指令 | 操作 | 典型用途 |
|------|------|----------|
| **`AND`** | 与 | 清位、掩码 |
| **`ORR`** | 或 | 置位 |
| **`EOR`** | 异或 | 翻转位、交换（Ch3） |
| **`BIC`** | Bit Clear — `Rd = Rn AND NOT Rm` | 清指定位 |
| **`MVN`** | 按位取反 | 常数扩展（Ch6） |

**寄存器形式常带移位：** `AND r0, r1, r2, LSL #2` — **第二操作数经桶形移位器**。

---

### 桶形移位器 — 同一周期移位

| 移位 | 助记符 | 效果 |
|------|--------|------|
| 逻辑左 | **LSL** | 低位补 0 |
| 逻辑右 | **LSR** | 高位补 0 |
| 算术右 | **ASR** | 高位 **符号扩展** |
| 循环右 | **ROR** | 低位回卷到高位 |

**两种写法：**

```asm
        MOV     r0, r1, LSL #2          ; r0 = r1 << 2
        LSL     r0, r1, #2              ; 独立移位指令（M/Thumb-2）
```

**Ch6：** 立即数编码也用 **循环右移** — 同一硬件。

---

### 加减法

| 指令 | 含义 |
|------|------|
| **`ADD` / `SUB`** | 32 bit ± |
| **`ADDS` / `SUBS`** | ± 并更新标志 |
| **`ADC` / `SBC`** | 带进/借位 — **64 bit 长整数** 链 |
| **`RSB` / `RSC`** | **Reverse Sub** — `Rd = op2 - op1`（常数减寄存器） |

**64 bit 加（概念）：**

```asm
        ADDS    r0, r0, r2            ; 低字
        ADC     r1, r1, r3            ; 高字 + C
```

**HFT/内核：** 64 bit 计数器、时间戳 — `ADC` 链常见。

---

### 乘法

| 指令 | 结果 |
|------|------|
| **`MUL`** | 32×32 → 低 32 bit |
| **`MLA`** | 乘加 `Rd = Ra + Rn*Rm` |
| **`SMULL` / `UMULL`** | 32×32 → **64 bit** 到两个寄存器 |

**乘常数优化（无 MUL）：**

```asm
        ; r0 = r1 * 5  ==  r1 + (r1<<2)
        ADD     r0, r1, r1, LSL #2
        ; *7 == r1 + (r1<<1) + (r1<<2) 等
```

**原因：** 早期 **乘法器慢/贵** — 编译器与你都会 **shift-add** 优化。

**Cortex-M4：** 单周期 **`MUL`** 仍可用；小常数优化仍省代码/周期。

---

### 除法

| 平台 | 除法 |
|------|------|
| **ARM7 等早期** | **无硬件除法** — 软件 **移位-减法** 例程 |
| **Cortex-M4** | **`UDIV` / `SDIV`** 硬件指令 |

**驱动注意：** 除零 → UsageFault — 与 C `/0` 不同，需显式检查。

---

### 饱和数学 (Saturated Math)

DSP/音频中 **溢出 wrap** → 波形 **咔嗒** 失真；**饱和** = **钳到最大/最小可表示值**。

| 指令 | 作用 |
|------|------|
| **`SSAT Rd, #n, Rm`** | **有符号** 饱和到 n 位范围 |
| **`USAT Rd, #n, Rm`** | **无符号** 饱和 |

**Q 标志：** 发生饱和时 **Q=1** — 可查询是否 clip。

**与 float clip：** `fminf`/`fmaxf` 同类思想 — 无 FPU 时用 **SSAT/USAT**。

---

### 可复述要点

1. **逻辑 + 移位** 可在一个 ALU 操作里组合（第二操作数移位）。  
2. **`ADC/SBC`** = 64 bit 算术；**RSB** = 反向减。  
3. **小常数乘法** 用 shift-add；M4 有 **MUL/UDIV/SDIV**。  
4. **`SSAT/USAT`** 饱和 + **Q 标志** — 定点/DSP 防失真。

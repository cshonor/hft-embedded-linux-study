## §7.7 分数表示法 — Q 格式 (Q Notation)

> **Ch 7 · 整数逻辑与算术** · [章导读](../README.md)  
> **标签：** **选读** — 无 FPU 时 **定点**；有 float/NEON 可压缩

---

### 问题：没有浮点，如何表示小数？

**Q 格式（定点）：** 在 32 bit 里 **固定假想二进制小数点** — 仍用 **整数指令** 运算，按规则解释比特。

---

### Qn 含义

**Q15（16 bit 有符号常用）示意：**

```
| S | 整数部分 | 小数部分 (15 frac bits) |
  ↑
符号
```

| 格式 | 范围/精度 tradeoff |
|------|---------------------|
| **Q15** | 小数多、动态范围小 |
| **Q31** | 32 bit 中 1 符号 + 31 小数 — 常见 DSP |

**常数存储：** π、e 等 **预缩放** 为整数 bit 模式 — `DCD` 表里是 **已乘 2^n 的整数**。

---

### 乘法规则（核心）

**Qn × Qm → Q(n+m)** 乘积 — **小数点“相加”**：

```
两个 Q15 相乘 → 有效 Q30 结果（在 32 bit 寄存器里）
        ↓
通常需 **右移 n 位**（或左移，视约定）重新 **归一化到 Q15**
        ↓
配合 **SSAT** 防溢出 clip
```

**例：** PID 的 `Kp * error` — 全 int，最后 shift 回工程单位。

---

### 与浮点 / 飞控

| | Q 定点 | IEEE float |
|---|--------|------------|
| **硬件** | 任意 M4 | M4F / Cortex-A |
| **代码** | 手动 shift + SSAT | C `float` |
| **本书** | §7.7 + §7.5 DSP | Ch9–11 浮点章（可跳过） |
| **[24 飞控](../../../24-Motion-Control-Motor/)** | 极端 constrained MCU | 常见 **float/double** + Kalman |

**HFT 对照：** 价格 **fixed tick size**（1/10000）≈ 定点 — 与 float 性能权衡同源。

---

### 可复述要点

1. **Q 格式 = 整数比特 + 约定小数点位置**。  
2. **乘法后必须 shift 重新对齐** Q 阶。  
3. 无 FPU 飞控/PID 靠 **Q + SSAT**；有 FPU 优先 C float。

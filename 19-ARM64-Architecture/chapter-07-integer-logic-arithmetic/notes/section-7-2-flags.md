## §7.2 标志位 — N · V · Z · C · S 后缀

> **Ch 7 · 整数逻辑与算术** · [章导读](../README.md)

---

### 标志位在哪里

| 架构 | 寄存器 |
|------|--------|
| **ARM7TDMI** | **CPSR** 低 8 位中的 **N Z C V** |
| **Cortex-M4** | **APSR**（xPSR 的应用视图）中的 **N Z C V** |
| **饱和 DSP** | 另 **Q 标志** — §7.4/7.5 |

**Ch3 阶乘** 已用 `CMP` 改 Z 等 — 本章系统定义含义。

---

### 四核心条件标志

| 标志 | 名 | 何时置 1（典型） |
|------|-----|------------------|
| **N** | Negative | 结果视为 **有符号数** 时 MSB=1（负数） |
| **Z** | Zero | 结果 **32 bit 全 0** |
| **C** | Carry | **无符号** 加产生进位；减 **无借位**（≥ 比较） |
| **V** | oVerflow | **有符号** 溢出（两正加得“负”等） |

**N 与补码：** [Ch1 §1.5](../../chapter-01-overview-computing-systems/notes/section-1-5-representation.md) — N 反映 **interpretation as signed**。

**V 示例：**

```
0x7FFFFFFF + 1  →  0x80000000
有符号：最大正 + 1 → 负数  →  V=1
无符号：合法 wrap     →  V 语义仍按有符号规则置位
```

**C 与 64 位：**

```asm
        ADDS    r0, r0, r1      ; 低 32 位加，C=进位
        ADC     r2, r2, r3      ; 高 32 位 + 进位（§7.4）
```

**Z：** 循环 `CMP r0, #0` / 倒计时到零 → **BEQ** 退出（→ **Ch8**）。

---

### 如何更新标志

| 方式 | 例子 |
|------|------|
| **比较类** | `CMP`/`CMN`/`TST`/`TEQ` — 只改标志，**不写结果寄存器**（§7.3） |
| **`S` 后缀** | `ADDS` `SUBS` `MOVS` — 写结果 **并** 更新 N,Z,C,V |

```asm
        ADDS    r0, r1, r2      ; r0=r1+r2，更新标志
        ADD     r0, r1, r2      ; 同上，但 **不** 改标志（默认）
```

**习惯：** 需要分支/条件执行前用 **`ADDS`/`SUBS`** 或先 **`CMP`**。

---

### 常用大小比较条件码（配套 CMP）

> **生成条件**用 `CMP`；**使用条件**用后缀。一组标志可同时喂给多条 `xxEQ`/`xxGT`…  

#### 一、相等：EQ / NE

| 后缀 | 触发 | 含义 |
|------|------|------|
| **EQ** | **Z=1** | 相等（差为 0） |
| **NE** | **Z=0** | 不等 |

```asm
        CMP     r0, #5
        ADDEQ   r1, r1, #1      ; r0 == 5 才加
        BEQ     label           ; 相等就跳

        CMP     r0, #5
        BNE     loop            ; r0 ≠ 5 就循环
```

#### 二、有符号（可负）：GT / LT / GE / LE

| 后缀 | 含义（`CMP r0, r1` 后） |
|------|-------------------------|
| **GT** | r0 **>** r1（有符号）— [MULGT/SUBGT](../../chapter-03-instruction-sets-v4t-v7m/notes/section-3-4-example-factorial.md) |
| **LT** | r0 **<** r1 |
| **GE** | r0 **≥** r1 |
| **LE** | r0 **≤** r1 |

```asm
        CMP     r0, #3
        MOVLT   r0, #0          ; r0 < 3 → 清零
        BLT     exit
```

**GT 标志条件：** `Z==0 && N==V`（详 [§7.3](./section-7-3-compare.md)）。

#### 三、无符号（地址、长度、缓冲区尺寸）：HI / LO / HS / LS

| 后缀 | 含义 |
|------|------|
| **HI** | 无符号 **大于**（Higher） |
| **LO** | 无符号 **小于**（Lower） |
| **HS** | 无符号 **≥**（Higher or Same）— 常 ≡ **CS**（C=1） |
| **LS** | 无符号 **≤**（Lower or Same） |

比指针/字节数用 **HI/LO**；比温度用有符号 **GT/LT** — **别混**。

#### 四、日常三基础（有符号）

| | |
|--|--|
| **EQ** | 等于 |
| **GT** | 大于 |
| **LT** | 小于 |

```asm
        CMP     r0, #10
        ADDEQ   r1, r1, #1      ; ==10 → +1
        ADDGT   r2, r2, #1      ; >10  → +1
        ADDLT   r3, r3, #1      ; <10  → +1
```

（注意：ARM 的 `ADD` 一般要写满 `Rd, Rn, Op2`，不能只写 `ADDEQ r1, #1`。）

#### 补充

1. 一次 `CMP` 之后，后面多条条件指令 **共用同一组标志**  
2. 有符号 **GT/LT/GE/LE**；无符号 **HI/LO/HS/LS**  
3. 后缀通用：`ADD`/`SUB`/`MUL`/`B`…（Thumb-2 放 **IT** 里）→ [Ch8](../../chapter-08-branches-loops/notes/section-8-2-branches.md)

---

### Q 标志（饱和，预告）

**`SSAT`/`USAT`** 等把结果 **钳位** 到范围内时 **Q=1** — 音频/DSP 检测失真（§7.4–7.5）。

---

### 与 Ch8 / HFT

| 用途 | 标志 |
|------|------|
| `if (a>b)` signed | `CMP` + **GT/LE** 读 N,V,Z |
| `if (a==0)` | **EQ** 读 Z |
| 无锁 CAS 重试 | 常读 **Z** 或 **NE** |
| 64 位 counter | **ADC** 链读 **C** |

---

### 可复述要点

1. **N/Z** 看结果符号与零；**C** 偏无符号进位；**V** 偏有符号溢出。  
2. **`S` 后缀** 或 **CMP 族** 更新标志。  
3. **Ch8 条件码** 直接读这四位的组合。

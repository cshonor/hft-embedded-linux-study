## §7.3 比较指令 — CMP · CMN · TST · TEQ

> **Ch 7 · 整数逻辑与算术** · [章导读](../README.md)

---

### 共性：只改标志，不写结果

四条 **比较/测试** 指令执行 ALU 运算，**丢弃结果**，仅更新 **N Z C V**。

---

### CMP — Compare（减）完整解读

#### 1. 核心功能

```asm
        CMP     Rn, Operand2    ; 等价：Rn − Operand2
```

- **只做减法**，结果 **丢掉**
- **只更新 CPSR / APSR 的 N Z C V**
- **不修改 Rn**，也不写任何目的寄存器

#### 2. 作用

专门做 **大小 / 相等** 判断，给后续：

- 条件后缀指令（`MULGT`…）
- 条件分支（`BGT` / `BEQ`…）
- M4 的 **IT 块**

提供标志依据。

#### 3. 最简例子

```asm
        CMP     r0, #1          ; 算 r0−1，只改标志，r0 不变
```

| (r0 − 1) 直觉 | 标志/条件 | 含义 |
|---------------|-----------|------|
| 结果 = 0 | **Z=1** → **EQ** | r0 **==** 1 |
| 结果 > 0（有符号） | **GT** | r0 **>** 1 |
| 结果 < 0（有符号） | **LT** | r0 **<** 1 |

有符号用 **GT/LT/GE/LE**（看 N,V,Z）；无符号用 **HI/HS/LO/LS**（看 C,Z）。

#### 4. 与阶乘代码搭配（→ [Ch3 §3.4](../../chapter-03-instruction-sets-v4t-v7m/notes/section-3-4-example-factorial.md)）

```asm
loop
        CMP     r0, #1      ; 对比 r0 与 1，刷新标志
        MULGT   r1, r1, r0  ; 仅 r0>1 时乘
        SUBGT   r0, r0, #1
        BGT     loop        ; 仍 >1 则跳回
```

**CMP** 是循环判断核心：先比，再条件乘/减/跳。

#### 5. 配套四标志（CMP 后怎么读）

| 标志 | CMP 语境 |
|------|----------|
| **Z** | 差为 0 → **两数相等** |
| **N** | 有符号差为负 |
| **C** | 无符号大小（减法：C=¬借位）→ **HS/HI** 等 |
| **V** | 有符号减法溢出 → 影响 **GT/LT** 判定 |

完整标志语义 → [§7.2](./section-7-2-flags.md)

#### 6. 关键区分：SUB vs CMP

| | `SUB r0, r0, #1` / `SUBS` | `CMP r0, #1` |
|--|---------------------------|--------------|
| 运算 | r0 − 1 | r0 − 1（结果丢弃） |
| **改 r0？** | **是** | **否** |
| 改标志？ | `SUB` 默认否；**`SUBS` 才改** | **永远改** |

> **CMP = 专为判断的「影子减法」。**

```asm
        CMP     r0, r1          ; 伪：r0 - r1，只设标志
        CMP     r0, #10
```

| 后续 | 含义 |
|------|------|
| **EQ** | r0 == op2（Z=1） |
| **NE** | 不等 |
| **GT/GE/LT/LE** | **有符号** |
| **HI/HS/LO/LS** | **无符号** |

---

### CMN — Compare Negative（加）

```asm
        CMN     r0, r1          ; r0 + r1，只设标志
```

用于 **与 -r1 比较** 等等价变换；较少见但对称于 CMP。

---

### TST — Test bits（与）

```asm
        TST     r0, #0xFF       ; r0 AND #mask，只设标志
        BEQ     no_low_byte     ; Z=1 → 低 8 位全 0
```

| 用途 | 说明 |
|------|------|
| **某位是否为 1** | `TST r0, #BIT` → **NE** |
| **掩码后是否全 0** | **EQ** |

**驱动：** 读 MMIO 后 `TST` 状态位 — 再分支（常配合 **Ch16** Load）。

---

### TEQ — Test Equivalence（异或）

```asm
        TEQ     r0, r1          ; r0 EOR r1，只设标志
        BEQ     equal           ; Z=1 → 每一位相同
```

**判断两寄存器是否 bitwise 相等** — 比 `CMP` 更适合「全等」语义（含符号无关的位模式）。

---

### 与 C 对照

| C | 汇编 |
|---|------|
| `if (a == b)` | `CMP a,b` / `TEQ` + `BEQ` |
| `if (a > b)` signed | `CMP` + `BGT` |
| `if (x & MASK)` | `TST` + `BNE` |

---

### 可复述要点

1. **CMP=减、CMN=加、TST=AND、TEQ=EOR** — 都只更新标志。  
2. **TST/TEQ** 是 **位级** 测试利器。  
3. 比较后 **紧跟 B.cond 或 IT**（Ch8）。

## 6.4.1 通用组织结构

> **Ch6 §6.4.1** · [章导读](../README.md) · 上节 [§6.3 ←](./section-6.3-层次结构与缓存概念.md) · 下节 [§6.4.2 →](./section-6.4.2-直接映射.md)

---

地址划分为（从低位到高位）：

```
| block offset (b) | set index (s) | tag (t) |
```

- **S = 2^s** 组，每组 **E** 条 cache line，每条 **B = 2^b** 字节
- 查 cache：**index 选组 → tag 比较 → valid 位**

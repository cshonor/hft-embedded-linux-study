## 5.4 消除循环的低效率

> **Ch5 §5.4** · [章导读](../README.md) · 上节 [§5.3 ←](./section-5.3-程序示例.md) · 下节 [§5.5 →](./section-5.5-减少过程调用.md)

---

**典型杀手：循环内重复计算**

```c
// 慢：每次迭代 strlen
for (i = 0; i < strlen(s); i++) ...

// 快：不变量外提
int len = strlen(s);
for (i = 0; i < len; i++) ...
```

- **归纳变量优化** — 编译器常做，但依赖 `-O` 与可见性
- HFT：循环边界、buffer 长度 **在循环外算好**；tick 处理别每包 `vector.size()` 若可缓存

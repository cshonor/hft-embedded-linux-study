## 5.8 循环展开 (Loop Unrolling)

> **Ch5 §5.8** · [章导读](../README.md) · 上节 [§5.7 ←](./section-5.7-现代处理器抽象模型.md) · 下节 [§5.9 →](./section-5.9-提高并行性.md)

---

**手动或编译器展开：**

```c
// 2 路展开
for (i = 0; i < n-1; i += 2) {
    acc = (acc + data[i]) + data[i+1];
}
// 处理尾部 n%2
```

**收益：**

- 减少 **分支**（每迭代处理多元素）
- 增 **指令级并行** — 多条独立 load/add 可并行发射
- 常与 **多累加器** 联用

**代价：** 代码膨胀、I-cache 压力、寄存器压力（→ 5.11）

```bash
gcc -O3 -funroll-loops ...
```

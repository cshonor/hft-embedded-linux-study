## 8.6 非本地跳转

> **Ch8 §8.6** · [章导读](../README.md) · 上节 [§8.5 ←](./section-8.5-信号.md) · 下节 [§8.7 →](./section-8.7-操作进程的工具.md)

---

```c
#include <setjmp.h>
jmp_buf env;
if (setjmp(env) == 0) {
    // 正常路径
} else {
    // longjmp(env, 1) 跳回此处
}
longjmp(env, 1);
```

- **不还原栈展开** — 与 C++ **异常、RAII** 不兼容；**跳过析构**
- 用途：深层错误快速回退（老代码）；现代 C++ 用 **异常** 或 **Result 类型**

**HFT：** 新代码 **避免 setjmp**；协程/状态机用显式枚举更清晰。

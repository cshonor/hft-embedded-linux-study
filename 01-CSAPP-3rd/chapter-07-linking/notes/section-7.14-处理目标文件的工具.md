## 7.14 处理目标文件的工具

> **Ch7 §7.14** · [章导读](../README.md) · 上节 [§7.13 ←](./section-7.13-库打桩.md) · 下节 [§7.15 →](./section-7.15-小结.md)

---

| 工具 | 用途 |
|------|------|
| `ar` | 创建/查看静态库 |
| `nm` | 符号表 |
| `objdump -d/-r/-t` | 反汇编、重定位、符号 |
| `readelf -a` | ELF 头、节、段、动态段 |
| `size` | 各段大小 |
| `strings` | 可打印串 |
| `ldd` | 动态依赖 |
| `strip` | 去符号减体积 |

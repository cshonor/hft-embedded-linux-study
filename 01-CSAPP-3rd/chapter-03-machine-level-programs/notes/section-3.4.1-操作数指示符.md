## 3.4.1 操作数指示符（gas / AT&T）

> [章导读](../README.md) · 上节 [§3.3 数据格式](./section-3.3-数据格式.md) · 下节 [§3.4.2 传送](./section-3.4.2-数据传送指令.md)

> **一条线：** 操作数写法 = 值从哪来 — ABI 在机器码层的底座之一。

---

| 类型 | 写法 | 含义 | 例 |
|------|------|------|----|
| **立即数** | `$常数` | 编译期常量，只读 | `movl $10, %eax` |
| **寄存器** | `%名` | 寄存器里的值（最快） | `%rax` |
| **绝对地址** | 裸地址 | 固定内存址（现代少见；多用 RIP 相对） | `0x601000` |
| **间接** | `(%reg)` | ≡ C `*p` | `movq (%rax), %rdi` |
| **基址+偏移** | `imm(%reg)` | ≡ `*(base+imm)` — **栈最常用** | `8(%rbp)` |

再往上（热路径）：变址 `(%rax,%rcx,4)` = `rax + rcx*4` — 数组 `A[i]`。

**栈口语：** `%rbp` 常作帧基址；`8(%rbp)` 等取 **更高地址**（视帧布局）；负偏移多取局部变量。`%rsp` = **栈顶**；栈向 **低地址** 增长。

```asm
addq 8(%rbp), %rax   # CISC：内存可直接作算术源 — ARM 禁止，须先 ldr
```

**vs ARM32：** 偏移在括号前 `8(%rbp)` ↔ ARM `[r11, #8]`；宽度用后缀 ↔ `ldrb/ldrh/ldr`。

### 和 ABI / CSAPP 考点串联

| 条款 | 在本节约等于 |
|------|----------------|
| `int` 4B、指针 8B（LP64） | 后缀 **`l` / `q`**（→ [§3.3](./section-3.3-数据格式.md)） |
| 前 6 整型/指针参数 | `%rdi, %rsi, %rdx, %rcx, %r8, %r9` |
| 栈局部 / 多余参数 / 返回地址 | `n(%rsp)` / `n(%rbp)` |
| 内存参与运算 | x86 **允许**；ARM **Load/Store 强制分步** |
| 浮点 | `%xmm0`–`%xmm15`（→ [§3.11](./section-3.11-浮点代码.md)） |
| 虚拟地址 | 典型 **48-bit canonical**（≠ 满 64 bit） |

手写 asm / 读 `objdump`：**先对后缀，再对寻址**。

## 纠正：不要用错的「口述八章主题」

| 错记（旧口述） | **正确（本书目录）** |
|----------------|----------------------|
| Ch5 = ARM 架构 | **Ch5 = 常见数字模块**（加法器/ALU/存储器阵列…） |
| Ch6 = 存储器+I/O | **Ch6 = 体系结构**（ARM 汇编/机器语言；另有 x86 对照） |
| Ch7 = 微处理器 | **Ch7 = 微结构**（单周期/多周期/**流水线+冲突**）✓ 主题对、名称以书为准 |

自底向上路径（书末总结）：二进制/门 → 组合/时序 → HDL → 数字模块 → **ARM 体系结构** → **微结构** → **存储器** →（在线）I/O。

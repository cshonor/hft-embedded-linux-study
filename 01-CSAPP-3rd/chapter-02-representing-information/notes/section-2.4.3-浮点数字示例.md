## 2.4.3 数字示例

> **Ch2 §2.4** · [章导读](../README.md) · 上节 [§2.4.2 IEEE 754](./section-2.4.2-IEEE754浮点表示.md) · 下节 [§2.4.4 舍入](./section-2.4.4-舍入.md)

> **怎么读例子：** 先写成 \(±1.M × 2^{E}\)，再填 **S / EXP(=E+127) / frac(=M)**。字段含义 → [§2.4.2](./section-2.4.2-IEEE754浮点表示.md)

---

### 例 1：`2.0f`（正、规格化、尾数全 0）

\[
2 = 1.0 × 2^{1} \Rightarrow S=0,\ E=1,\ EXP=128=10000000_2,\ M=0
\]

```text
0 10000000 00000000000000000000000
S   EXP              frac
```

**配合：** 符号正；指数把小数点定在「×2」；尾数没有额外小数位。

---

### 例 2：`0.5f`（\(E=−1\)）

\[
0.5 = 1.0 × 2^{-1} \Rightarrow EXP = -1+127 = 126 = 01111110_2,\ frac=0
\]

```text
0 01111110 00000000000000000000000
```

---

### 例 3：`−5.75f`（负、尾数有内容）

\[
5.75 = 101.11_2 = 1.0111_2 × 2^{2}
\]

| 字段 | 值 |
|------|-----|
| S | `1` |
| E | `2` → EXP=`129`=`10000001` |
| frac | `0111` 后补 0 → `0111000…0` |

```text
1 10000001 01110000000000000000000
S   EXP            frac
```

**验算：** \((-1)×(1 + 1/4 + 1/8 + 1/16)×2^{2} = -1.4375×4 = -5.75\) ✓

---

### 例 4：特殊值（对照）

| 值 | 位模式直觉（float） |
|----|---------------------|
| `+0` | `0 00000000 000…0` |
| `-0` | `1 00000000 000…0` |
| `+Inf` | `0 11111111 000…0` |
| `-Inf` | `1 11111111 000…0` |
| NaN | `? 11111111` + frac **非全 0** |

---

### 量级与精度（记数量级）

| | float | double |
|--|-------|--------|
| 有效十进制位（约） | **~7** | **~15–16** |
| 尾数字段 | 23 (+隐含1) | 52 (+隐含1) |
| 上溢 / 下溢 | → Inf / →0（或非规格） | 同逻辑 |

超过有效位数 → **不能精确存**（回到 [§2.4.1](./section-2.4.1-分数二进制数.md) 的 `0.1`）。

**HFT：** 看盘价用整数 tick；`float` 只适合对精度不敏感的统计/信号，别当金额真值。

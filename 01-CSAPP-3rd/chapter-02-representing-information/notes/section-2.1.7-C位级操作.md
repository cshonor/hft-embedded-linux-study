## 2.1.7 C 语言中的位级操作

> **Ch2 §2.1** · [章导读](../README.md) · 上节 [§2.1.6 布尔](./section-2.1.6-布尔代数简介.md) · 下节 [§2.1.8 逻辑](./section-2.1.8-C逻辑操作.md)

这张表就是 C 里专门操作 **二进制位** 的运算符 — 和 [§2.1.8](./section-2.1.8-C逻辑操作.md) 的 `&&`/`||`/`!`（逻辑真假）不是一套。

---

| 运算符 | 含义 | 典型用途（口述对应） |
|--------|------|----------------------|
| `&` | 按位与 | **掩码**：提取某几个特定位（`x & mask`） |
| `\|` | 按位或 | **合并/置位**：把标志位设进去（`x \| flag`） |
| `^` | 按位异或 | **切换/翻转** 某几位（同位再异或一次还原） |
| `~` | 按位取反 | 把整个数的 **0↔1 全部颠倒**（常配合掩码求补） |
| `<<` `>>` | 左/右移 | **×2 / ÷2（2 的幂）**，或把数据 **挪到位字段位置** → [§2.1.9](./section-2.1.9-C移位操作.md) |

```c
/* 取第 k 位（从 0 起） */
int get_bit(unsigned x, int k) { return (x >> k) & 1; }

/* 第 k 位置 1 */
unsigned set_bit(unsigned x, int k) { return x | (1u << k); }

/* 第 k 位翻转 */
unsigned toggle_bit(unsigned x, int k) { return x ^ (1u << k); }

/* 协议常见：从 word 中抠 bit 域 */
uint32_t field = (word >> shift) & mask;
```

**HFT：** 操作寄存器标志、解析网络包 / SBE header — **shift + mask** 是热路径基本动作；保证 `mask` 与 `shift` 来自 schema，别手写魔数散落各处。

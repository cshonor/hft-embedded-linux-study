## 2.1.9 C 语言中的移位操作

> **Ch2 §2.1** · [章导读](../README.md) · 上节 [§2.1.8 逻辑](./section-2.1.8-C逻辑操作.md) · 下节 [§2.2 整数表示](./section-2.2-整数表示与类型转换.md)

---

- **左移 `<<`：** 低位补 0；无符号等价 **× 2^k**（超出宽度则截断，模 2^w）
- **右移 `>>`：**
  - **无符号：** 逻辑右移，高位补 0 → 等价 `/ 2^k`（向下取整 toward 0）
  - **有符号：** C 标准对 **负数右移** 留坑（算术 vs 逻辑）；关键代码用无符号或显式分支

```c
unsigned u = 16u;  u >> 2;   /* 4 */
int s = -8;        s >> 1;   /* 实现相关，x86 gcc 常为 -4（算术右移） */
```

**HFT：** 价格 **tick 整数化** 优先 `/ tick_size` 或定点缩放，少依赖 float（→ [§2.4](./section-2.4.1-分数二进制数.md)）。

## 2.1.4–2.1.9 字符串、代码、布尔与 C 位/逻辑/移位

> **本节一条线：** 字符/机器码也是字节 → 布尔/位运算 → 别混淆 `&` 与 `&&`、移位与除法。

---

### 2.1.4 字符串表示

- C 字符串 = **字节序列 + 末尾 `'\0'` (0x00)**
- **ASCII** — 英文子集；**UTF-8** — 变长编码，兼容 ASCII 前缀
- 字符 `'0'`…`'9'` 的 ASCII 码是 **0x30…0x39**，与整数 0…9 的数值编码 **不是一回事**

**和 §2.1.3 / 大小端：**

> **普通 `char` 字符串没有大小端。** 每个字符 1 字节，按书写顺序低→高存放（`"AB"` → `41 42`），与 CPU LE/BE 无关。  
> 只有 **多字节整数**（及 `wchar_t` 等宽码元）才谈 endian。细节 → [§2.1.3 · 字符串无大小端](./section-2.1.3-寻址与字节序.md#字符串没有大小端只有多字节整数才有)

**二进制协议：**

- 通常 **不是** C 字符串 — 多为 **定长字段** 或 `length + payload`，**无** `\0`
- Symbol/account 常 **定长 8/16 字节**，空格或 `\0` 填充
- FIX 等 ASCII 字段：**无需** `htons`/`htonl`；整数字段才要

**HFT：** 用 `strcmp` 前确认是否以 `\0` 结尾；定长字段用 `memcmp` + 已知长度。

---

### 2.1.5 代码表示

- **机器码也是字节序列** — 与数据无本质区别，靠 **PC 指向、页权限 (X/D)** 区分指令 vs 数据
- **NX/DEP、W^X** — 数据段不可执行，减轻 shellcode 风险

→ 执行视角：[Ch 3 机器级程序](../../chapter-03-machine-level-programs/)

---

### 2.1.6 布尔代数简介

- 与 / 或 / 非 / 异或 — 对应 `& | ~ ^`
- 德摩根：`\~(A & B) == ~A | ~B` — 读位掩码优化、硬件电路时有用
- **HFT：** 时间紧可略读；与 2.1.7 位操作重复处以后者为准

---

### 2.1.7 C 语言中的位级操作

| 运算符 | 含义 | 典型用途 |
|--------|------|----------|
| `&` | 按位与 | **掩码** 取字段 |
| `\|` | 按位或 | **合并** 标志位 |
| `^` | 按位异或 | 切换位、简单 checksum |
| `~` | 按位取反 | 掩码求补 |
| `<<` `>>` | 左/右移 | 乘除 2 的幂、字段抽取 |

```c
/* 取第 k 位（从 0 起） */
int get_bit(unsigned x, int k) { return (x >> k) & 1; }

/* 第 k 位置 1 */
unsigned set_bit(unsigned x, int k) { return x | (1u << k); }

/* 协议常见：从 word 中抠 bit 域 */
uint32_t field = (word >> shift) & mask;
```

**HFT：** SBE / 自定义 header 解码 — **shift + mask** 是热路径基本动作；保证 `mask` 与 `shift` 来自 schema，别手写魔数散落各处。

---

### 2.1.8 C 语言中的逻辑操作

- `&&` `||` `!` — **短路求值**，结果为 **0 或 1**，**不** 对每一位运算
- **易混：**

```c
if (x & y)   /* 位与：结果可能任意非零 bit 模式 */
if (x && y)  /* 逻辑与：x、y 是否「真」 */
```

- `if (flag & MASK)` 常用；`if (a && b)` 用于条件分支，别混用语义

---

### 2.1.9 C 语言中的移位操作

- **左移 `<<`：** 低位补 0；无符号等价 **× 2^k**（超出宽度则截断，模 2^w）
- **右移 `>>`：**
  - **无符号：** 逻辑右移，高位补 0 → 等价 `/ 2^k`（向下取整 toward 0）
  - **有符号：** C 标准对 **负数右移** 留坑（算术 vs 逻辑）；关键代码用无符号或显式分支

```c
unsigned u = 16u;  u >> 2;   /* 4 */
int s = -8;        s >> 1;   /* 实现相关，x86 gcc 常为 -4（算术右移） */
```

**HFT：** 价格 **tick 整数化** 优先 `/ tick_size` 或定点缩放，少依赖 float（→ §2.4）。

## 2.4.2 IEEE 754 浮点表示

> **Ch2 §2.4** · [章导读](../README.md) · 上节 [§2.4.1 分数二进制](./section-2.4.1-分数二进制数.md) · 下节 [§2.4.3 数字示例](./section-2.4.3-浮点数字示例.md)

---

**float（32 bit）：**

```
| S (1) | exp (8) | frac (23) |
```

**double（64 bit）：** exp 11 位，frac 52 位，bias **1023**

### `exp`：不是补码，也不是「带符号整数」

IEEE 754 的 **阶码字段用无符号偏移编码 (biased / excess)**：

\[
\text{存进去的 } exp = E + bias
\]

| 格式 | `exp` 位数 | bias | 真实指数 \(E\)（规格化时） |
|------|------------|------|---------------------------|
| float 32 | 8 | **127** | \(E = exp - 127\) |
| double 64 | 11 | **1023** | \(E = exp - 1023\) |

- **不是** 补码，也 **不是** 单独再挂一个符号位的有符号数  
- 存的是 **无符号整数**（把真实指数平移到非负区间）  
- **好处：** 浮点大小常可先比符号，再 **直接按无符号整数比 `exp`（再比 `frac`）** — 硬件/软件排序快

规格化数：

\[
V = (−1)^S × 1.M × 2^{E},\quad E = exp - bias
\]

（`exp` 既非全 0 也非全 1）

---

### 特殊值：`Inf` 与 `NaN`（float / double 同一套规则）

**通用：** `exp` 全 1 = 特殊值  

1. 尾数（`frac`）**全 0** → **Inf**  
2. 尾数 **不全 0** → **NaN**

| 类别 | exp | frac | 含义 |
|------|-----|------|------|
| 规格化 | 中间值 | 任意 | 正常数 |
| 非规格化 | 0 | ≠0 | 接近 0 |
| 零 | 0 | 0 | ±0（看符号位） |
| **Inf** | **全 1** | **全 0** | 无穷大 |
| **NaN** | **全 1** | **≠0** | Not a Number（不是数字） |

float / double / 更长 IEEE 浮点 **都有**；口误成「long」时，说的就是 **NaN**。

#### float 32 位位模式例

```text
+Inf:  0 11111111 00000000000000000000000
-Inf:  1 11111111 00000000000000000000000
NaN:   0 11111111 00010000000000000000000   /* frac 任意非全 0 */
```

#### Inf 运算例

```c
#include <stdio.h>
#include <math.h>

double pos_inf = 1.0 / 0.0;    /* +Inf */
double neg_inf = -1.0 / 0.0;   /* -Inf */

pos_inf + 100;                 /* +Inf */
pos_inf * 5;                   /* +Inf */
pos_inf - 999;                 /* +Inf */
pos_inf + neg_inf;             /* NaN：无穷±无穷无意义 */
pos_inf * neg_inf;             /* -Inf */
```

#### NaN 产生场景（非法 / 无实数解）

```c
sqrt(-5.0);    /* 负数开方 → NaN */
0.0 / 0.0;     /* → NaN */
pos_inf - pos_inf;  /* Inf−Inf → NaN */
pos_inf * 0.0; /* Inf×0 → NaN */
log(-1.0);     /* → NaN */
```

#### 最反直觉：`NaN != NaN`

```c
double n = sqrt(-1.0);
n == n;        /* 假！NaN 不等于任何值，包括自己 */
isnan(n);      /* 真 — 必须用库函数判断 */
```

#### 实用口诀

1. **除以 0** → 正/负无穷（看分子符号）  
2. **无实数解 / 不定式** → **NaN**  
3. 判断 NaN **不能** `==`，必须 **`isnan`**  
4. Inf 参与常规 ±×÷ 有限数 **仍常是 Inf**；`Inf±Inf`、`Inf×0` 等 → **NaN**

**HFT：** 行情/风控路径遇 NaN 要显式拦截；别靠 `==` 抓 NaN。金额仍优先定点/整数 tick。

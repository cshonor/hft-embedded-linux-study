## 2.4.1–2.4.4 二进制小数、IEEE 754 与舍入

> **本节一条线：** 很多十进制小数在二进制里 **不能精确表示** → IEEE 754 存 **近似值** → 舍入规则影响累积误差。

---

### 2.4.1 分数二进制数

- 二进制小数：位 i（小数点后）代表 **2^(−i)**
- 类似十进制里 1/3 = 0.333…，**0.1 十进制** 在二进制里 **无限循环**，只能存近似

```
0.625 (十) = 0.101 (二)   ✓ 精确
0.1   (十) = 0.00011… (二) ✗ 只能近似
```

**因此 `0.1 + 0.2 != 0.3`（float/double）** — 不是编译器 bug，是 **表示极限**。

---

### 2.4.2 IEEE 754 浮点表示

**float（32 bit）：**

```
| S (1) | exp (8) | frac (23) |
```

**规格化数（exp 既非全 0 也非全 1）：**

\[
V = (−1)^S × 1.M × 2^{E-127}
\]

**double（64 bit）：** exp 11 位，frac 52 位，bias 1023

| 类别 | exp | frac | 含义 |
|------|-----|------|------|
| 规格化 | 中间值 | 任意 | 正常数 |
| 非规格化 | 0 | ≠0 | 接近 0 |
| 零 | 0 | 0 | ±0 |
| Inf | 全 1 | 0 | 无穷 |
| NaN | 全 1 | ≠0 | Not a Number |

---

### 2.4.3 数字示例

- **FLT_MAX / FLT_MIN** — 量级差大；下溢→0、上溢→Inf
- **有效十进制位数：** float ≈ **7** 位，double ≈ **15–16** 位 — 超过即 **不能精确存**

---

### 2.4.4 舍入 (Rounding)

- 默认 **向偶数舍入 (round to nearest, ties to even)** — 减少长期 bias
- 历史坑：x87 寄存器 **80 bit** 扩展精度 → 同表达式不同优化级别结果可能不同
- 缓解：统一 SSE/AVX、`-ffloat-store`（慢）、或 **不用 float 做账本**

**HFT 铁律（常见）：**

| 场景 | 建议 |
|------|------|
| 价格、PnL、风控阈值 | **整数 tick** 或 decimal 库 / `rust_decimal` |
| 指标、统计 | double 可接受，知悉误差 |
| 相等比较 | **禁止** `a == b`；用 epsilon 或整数 tick 差 |

## 2.4.1–2.4.4 二进制小数、IEEE 754 与舍入

### 2.4.1 分数二进制数

- 二进制小数：**位 i 代表 2^(−i)** — 很多十进制小数 **不能精确表示**（类似 1/3 在十进制）

```
0.625 (十进制) = 0.101 (二进制)  ✓ 精确
0.1 (十进制)   = 无限循环二进制  ✗ 只能近似
```

### 2.4.2 IEEE 754 浮点表示

**单精度 float（32 bit）布局：**

```
| S (1) | exp (8) | frac (23) |
```

**值（规格化数，exp ≠ 0 且 ≠ 255）：**

\[
V = (−1)^S × 1.M × 2^{E-127}
\]

**双精度 double（64 bit）：** exp 11 位，frac 52 位，bias 1023

| 类别 | exp | frac | 含义 |
|------|-----|------|------|
| 规格化 | 1…254 | 任意 | 正常数 |
| 非规格化 | 0 | ≠0 | 接近 0 |
| 零 | 0 | 0 | ±0 |
| Inf | 255/2047 | 0 | 无穷 |
| NaN | 255/2047 | ≠0 | Not a Number |

### 2.4.3 数字示例

- **FLT_MAX / FLT_MIN** — 量级差巨大；下溢变 0、上溢变 Inf
- **精度：** float 有效约 **7 位十进制**，double 约 **15–16 位**

### 2.4.4 舍入 (Rounding)

- 默认 **向偶数舍入 (round to nearest, ties to even)** — 减少累积 bias
- 运算结果在寄存器 **可能 80 bit 扩展精度**（x87 历史坑）— `-ffloat-store`、统一 SSE/AVX、或 **别用 float 做账本**

**HFT 铁律（常见）：**

| 场景 | 建议 |
|------|------|
| 价格、PnL、风控阈值 | **整数 tick** 或 **decimal 库** / Rust `rust_decimal` |
| 指标、信号、统计 | double 可接受，知悉误差 |
| 相等比较 | **禁止** `a == b`；用 epsilon 或整数差 |

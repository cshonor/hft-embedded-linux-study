## 2.4.5 浮点运算

> **Ch2 §2.4** · [章导读](../README.md) · 上节 [§2.4.4 舍入](./section-2.4.4-舍入.md) · 下节 [§2.4.6 C 浮点](./section-2.4.6-C语言浮点数.md)

> **一条线：** 浮点运算是 **近似算术** → 非结合 → HFT 账本/热路径优先整数。

---

- **非结合、非分配：** `(a+b)+c` 可能 ≠ `a+(b+c)` — **大数 swallow 小数**
- **NaN 传播：** 任一操作数为 NaN → 结果为 NaN；比较为 **无序**
- **Inf** 有定义规则，仍易出 bug

```c
float a = 1e30f, b = -1e30f, c = 1.0f;
(a + b) + c;   /* 0 + 1 = 1 */
a + (b + c);   /* 可能仍 ≈ 0 — 非结合 */
```

**HFT：** 多 venue partial fill 算 VWAP — 用 **整数分子/分母** 或 Kahan summation；别裸 `float` 连加 PnL。

# 对象移动（C++11）

## 本节讲什么

本节讲解「对象移动（C++11）」。

## 要点

- 新标准引入对象「移动」而非「拷贝」，提升性能
- **右值引用**概念
- `std::move`：避免不必要的内存拷贝（重新分配内存、处理不可共享资源等）

## 示例

```cpp
#include <iostream>
#include <vector>
int main() {
    std::vector<int> v1{1, 2, 3};
    std::vector<int> v2 = std::move(v1);
    std::cout << v2.size() << " " << v1.size() << std::endl;
    return 0;
}
```

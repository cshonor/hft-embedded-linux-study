# 其他应用

## 本节讲什么

本节讲解「其他应用」。

## 要点

- 定义 `swap` 以配合标准库算法
- 实战示例：动态内存管理类、邮件处理程序中的 `Message` 和 `Folder` 类

## 示例

```cpp
#include <iostream>
#include <string>
class String {
public:
    String(const char *s) : data_(s ? s : "") {}
    String(String &&other) noexcept : data_(std::move(other.data_)) {}
    const std::string &str() const { return data_; }
private:
    std::string data_;
};
int main() {
    String s("hello");
    String t(std::move(s));
    std::cout << t.str() << std::endl;
    return 0;
}
```

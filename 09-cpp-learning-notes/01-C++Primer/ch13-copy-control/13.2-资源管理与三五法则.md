# 资源管理与三/五法则

## 本节讲什么

本节讲解「资源管理与三/五法则」。

## 要点

- 如何决定类是否需要自定义这些控制成员
- 若类需管理动态分配内存或其他外部资源，通常将这几个操作作为一个整体定义

## 示例

```cpp
#include <iostream>
#include <memory>
class Blob {
public:
    void push_back(int i) { data->push_back(i); }
private:
    std::shared_ptr<std::vector<int>> data =
        std::make_shared<std::vector<int>>();
};
int main() {
    Blob b;
    b.push_back(42);
    return 0;
}
```

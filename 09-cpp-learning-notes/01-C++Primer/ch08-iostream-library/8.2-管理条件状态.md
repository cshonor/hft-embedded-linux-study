# 管理条件状态

## 本节讲什么

本节讲解「管理条件状态」。

## 要点

- 每个 IO 对象维护一组状态标志（有效、文件尾、错误等）
- `rdstate`、`setstate`、`clear` 等：检测和复位流状态

## 示例

```cpp
#include <iostream>
int main() {
    int ival;
    while (std::cin >> ival)
        std::cout << ival << std::endl;
    if (std::cin.eof())
        std::cout << "EOF reached" << std::endl;
    return 0;
}
```

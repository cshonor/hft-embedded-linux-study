# IO 类体系

## 本节讲什么

继承机制使得 `fstream` 或 `sstream` 可替代接受 `iostream` 引用参数的地方。

## 要点

| 头文件 | 用途 |
|--------|------|
| `iostream` | 读写控制台流（`cin`、`cout`、`cerr`） |
| `fstream` | 读写命名文件 |
| `sstream` | 读写内存中的 `string` 对象 |

## 示例

```cpp
#include <iostream>
int main() {
    int ival;
    double dval;
    std::cin >> ival >> dval;
    std::cout << ival << " " << dval << std::endl;
    return 0;
}
```

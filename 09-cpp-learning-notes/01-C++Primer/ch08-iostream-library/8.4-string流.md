# string 流

## 本节讲什么

本节讲解「string 流」。

## 要点

- **`istringstream`**：从 `string` 读取数据（逐行读取后按单词分解）
- **`ostringstream`**：向 `string` 写入数据（逐步构造格式化输出后一次性打印）

## 示例

```cpp
#include <sstream>
#include <iostream>
#include <string>
int main() {
    std::istringstream iss("42 3.14");
    int i; double d;
    iss >> i >> d;
    std::ostringstream oss;
    oss << "i=" << i << ", d=" << d;
    std::cout << oss.str() << std::endl;
    return 0;
}
```

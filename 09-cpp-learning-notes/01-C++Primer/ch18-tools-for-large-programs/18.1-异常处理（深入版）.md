# 18.1 异常处理（深入版）

## 本节讲什么

第 5 章仅为入门，本章深入：

## 要点

- 完整机制：`throw` 抛出、多层函数栈自动回溯、`try-catch` 捕获
- **noexcept**：标记函数是否会抛出异常
- 异常对象、继承体系（`exception`、`runtime_error`、`logic_error` 等）
- 工程准则：RAII 配合异常保证资源不泄漏
- 异常安全三级：**基础保证**、**强保证**、**无异常保证**

## 示例

```cpp
#include <iostream>
#include <stdexcept>
double safe_divide(double a, double b) {
    if (b == 0) throw std::invalid_argument("b is zero");
    return a / b;
}
int main() {
    try {
        std::cout << safe_divide(10, 2) << std::endl;
    } catch (const std::exception &e) {
        std::cerr << e.what() << std::endl;
    }
    return 0;
}
```

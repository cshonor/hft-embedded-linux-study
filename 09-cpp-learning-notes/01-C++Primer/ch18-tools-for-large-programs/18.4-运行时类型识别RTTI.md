# 18.4 运行时类型识别 RTTI

## 本节讲什么

适用：多态体系里判断实际对象类型。开销略大，能靠虚函数解决就尽量不用 RTTI。

## 要点

| 运算符 | 用途 |
|--------|------|
| `dynamic_cast` | 安全向下转型，失败返回 `nullptr` |
| `typeid` | 获取 `type_info` 类型信息 |

## 示例

```cpp
#include <iostream>
#include <typeinfo>
class Base { public: virtual ~Base() = default; };
class Derived : public Base {};
int main() {
    Derived d;
    Base &b = d;
    std::cout << typeid(b).name() << std::endl;
    if (auto *p = dynamic_cast<Derived *>(&b))
        std::cout << "ok\n";
    return 0;
}
```

# 18.2 命名空间 namespace

## 本节讲什么

解决大型项目**命名冲突**：

## 要点

- 自定义命名空间、嵌套命名空间、内联命名空间
- `using 声明` vs `using namespace 指令`，滥用全局 using 的坑
- 标准库 `std` 原理、库版本兼容、第三方库整合避重名

## 示例

```cpp
#include <iostream>
namespace utils {
    void hello() { std::cout << "hello from utils\n"; }
}
int main() {
    utils::hello();
    using utils::hello;
    hello();
    return 0;
}
```

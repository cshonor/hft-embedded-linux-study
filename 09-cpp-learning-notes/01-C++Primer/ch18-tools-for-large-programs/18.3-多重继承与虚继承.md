# 18.3 多重继承与虚继承

## 本节讲什么

1. **多重继承**：一个类继承多个父类 → 菱形继承、成员名冲突、二义性
2. **菱形继承**：公共祖先多份拷贝、数据冗余
3. **虚继承 `virtual public`**：公共基类只保留一份实例
4. 多重继承下的构造/析构顺序、类型转换、虚函数重写规则

## 示例

```cpp
#include <iostream>
class Base {
public:
    virtual void f() { std::cout << "Base\n"; }
};
class D1 : public Base {};
class D2 : public Base {};
class Most : public D1, public D2 {};
int main() {
    Most m;
    m.D1::f();
    return 0;
}
```

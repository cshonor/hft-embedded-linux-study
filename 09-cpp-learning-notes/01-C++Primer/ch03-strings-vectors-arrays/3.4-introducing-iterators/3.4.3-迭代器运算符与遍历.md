# 3.4.3 迭代器运算符与遍历

## 本节讲什么

迭代器必考运算符与全书通用的 `for` 遍历模板。

## 基础运算符（必考）

| 运算符 | 含义 |
|--------|------|
| `*it` | 解引用，得到所指元素的引用 |
| `it->成员` | 等价 `(*it).成员`，访问类/结构体成员 |
| `++it` / `it++` | 移到**下一个**元素（**前置 `++` 更高效**） |
| `--it` / `it--` | 移到**上一个**元素（`vector` / `string` / `list` 等支持；因容器而异） |
| `it1 == it2` / `!=` | 是否指向**同一位置** |

> **规范**：遍历一律写 **`it != v.end()`**，不要用 **`it < v.end()`**——链表等容器的迭代器**不支持** `<` 比较（随机访问迭代器才支持，见 [3.4.4](./3.4.4-随机访问算术运算.md)）。

### 前置 `++` vs 后置 `++`

```cpp
++it;   // ✅ 推荐：直接前移，无多余临时
it++;   // 后置会拷贝旧迭代器再前移，性能更差
```

## 标准遍历模板（全书通用）

```cpp
std::vector<int> v = {1, 2, 3};
for (auto it = v.begin(); it != v.end(); ++it) {
    std::cout << *it << std::endl;
}
```

只读版本：

```cpp
for (auto cit = v.cbegin(); cit != v.cend(); ++cit)
    std::cout << *cit << " ";
```

### `string` 同样适用

```cpp
std::string s = "hello";
for (auto it = s.begin(); it != s.end(); ++it)
    *it = static_cast<char>(std::toupper(static_cast<unsigned char>(*it)));
```

C++11 范围 for 等价于对 `begin`/`end` 的封装（见第 5 章）：

```cpp
for (int x : v) { /* ... */ }   // 底层即迭代器遍历
```

# 3.4 迭代器介绍

## 本节讲什么

之前只用**下标**访问 `string` / `vector`，但下标**不通用**（`list`、`map`、`set` 不支持下标）。本节引入 **迭代器 iterator**——所有标准容器通用的元素访问机制；`string` 虽不算容器，也完整支持迭代器。

迭代器本质类似指针：间接访问元素，可移动、可读/写（或只读）。

## 要点

- **`begin()`** → 首元素；**`end()`** → **尾后**迭代器（不指向元素，勿解引用）
- **`cbegin()` / `cend()`**：只读迭代器，只遍历不修改时优先用
- **运算符**：`*it`、`it->成员`、`++it`、`==` / `!=`；遍历写 **`it != end()`**，不用 `<`
- **算术**（仅 `string` / `vector` / `array` / `deque`）：`it + n`、`end - begin` 得元素个数
- **易错**：尾后迭代器禁止 `*`；`vector` 扩容 / `erase` 后迭代器**失效**

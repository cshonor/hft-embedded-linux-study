# 3.4.2 begin、end 与常量迭代器

## 本节讲什么

`begin` / `end` / `cbegin` / `cend` 的语义、返回类型与读写权限。

## begin() — 指向第一个元素

- 返回类型：`容器::iterator`（如 `std::vector<int>::iterator`）
- 指向**第一个有效元素**
- 可读写：`*it = value;`

```cpp
std::vector<int> v{1, 2, 3};
auto it = v.begin();
*it = 10;   // v 变为 {10, 2, 3}
```

## end() — 尾后迭代器（终止标记）

- 返回**尾后（past-the-end）迭代器**
- **不指向任何有效元素**，仅表示「遍历到这里结束」
- **禁止解引用**：`*v.end()` → **未定义行为**（可能崩溃）
- **空容器**：`begin() == end()`，遍历循环体**一次都不执行**

```cpp
std::vector<int> empty;
if (empty.begin() == empty.end())  // true，等价 empty.empty()
    /* 无元素 */;
```

## cbegin() / cend()（C++11 常量迭代器）

- 返回 `容器::const_iterator`，**只读**
- 即使容器本身**不是** `const`，也不能通过 `cit` 修改元素：`*cit = 10;` **编译报错**
- **只遍历、不修改**时优先用，防止误改数据

| 调用 | 返回类型 | 能否改元素 |
|------|----------|------------|
| `v.begin()` | `vector::iterator` | ✅ 可读写 |
| `v.cbegin()` | `vector::const_iterator` | ❌ 只读 |
| `const vector<int> cv; cv.begin()` | `vector::const_iterator` | ❌ 只读 |

```cpp
std::vector<int> v{1, 2, 3};
for (auto cit = v.cbegin(); cit != v.cend(); ++cit) {
    std::cout << *cit;
    // *cit = 0;  // ❌ 编译错误
}
```

## 类型书写（`auto` 之前）

```cpp
std::vector<int>::iterator it = v.begin();
std::vector<int>::const_iterator cit = v.cbegin();
std::string::iterator sit = s.begin();
```

现代 C++ 直接 **`auto`** 推导即可。

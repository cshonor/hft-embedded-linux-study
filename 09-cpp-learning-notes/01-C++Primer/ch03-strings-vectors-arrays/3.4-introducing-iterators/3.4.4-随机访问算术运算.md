# 3.4.4 随机访问算术运算

## 本节讲什么

`string` / `vector` 等**连续内存**容器支持的迭代器加减与距离——**随机访问迭代器**专属能力。

## 哪些容器支持

| 支持 `it + n`、`it1 - it2`、`<` 比较 | 不支持（双向或前向迭代器） |
|--------------------------------------|---------------------------|
| `string`、`vector`、`array`、`deque` | `list`、`map`、`set`、`forward_list` 等 |

## 算术运算

| 运算 | 含义 |
|------|------|
| `it + n` / `n + it` | 向前跳 `n` 个元素 |
| `it - n` | 向后跳 `n` 个元素 |
| `it += n` / `it -= n` | 复合赋值 |
| `it1 - it2` | 两迭代器间**元素个数**，类型 `difference_type`（有符号整数） |
| `it1 < it2` / `>` / `<=` / `>=` | 位置前后比较（仅随机访问） |

```cpp
std::vector<int> v{1, 2, 3, 4, 5};

auto mid = v.begin() + v.size() / 2;   // 指向中间元素
std::cout << *mid << std::endl;        // 3

auto dis = v.end() - v.begin();        // 5，元素总数
```

### HFT 示意

对 tick 缓冲做**二分**、取**中间档位**时，随机访问迭代器 + 算术比手写下标更通用（算法库第 10 章大量依赖此类迭代器）。

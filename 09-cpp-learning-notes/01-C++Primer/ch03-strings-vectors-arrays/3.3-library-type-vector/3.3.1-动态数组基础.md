# 3.3.1 动态数组基础

## 本节讲什么

`vector` 是什么、与 `string` 的相似点，以及 `push_back` / `size()` / `empty()` 等基本操作。

## 是什么

`std::vector<T>` 是标准库提供的**动态数组**：

- **同类型元素**连续存放在内存中（缓存友好）
- **长度可变**：运行时 `push_back` 增、也可 `pop_back` 删（第 9 章详讲）
- **RAII**：析构时自动释放内部缓冲

与 [3.2 string](../3.2-library-type-string/3.2-标准库类型string.md) 类似：都是可变长序列容器；`string` 专用于字符，`vector` 泛用于任意类型 `T`。

## 基本用法

```cpp
#include <vector>

std::vector<int> v;           // 空 vector，size()==0
v.push_back(42);              // 末尾追加，必要时内部扩容
v.push_back(100);

std::cout << v.size();        // 2
std::cout << v.empty();       // false
std::cout << v[0];            // 42（下标访问已有元素，见 3.3.3）
```

| 操作 | 含义 |
|------|------|
| `push_back(x)` | 末尾追加元素（**正确增容方式**） |
| `emplace_back(args...)` | 末尾**原地构造**（少一次拷贝） |
| `size()` | 当前元素个数 |
| `empty()` | 是否无元素 |
| `reserve(n)` | 预留容量，减少多次扩容（见 [3.2.3 reserve](../3.2-library-type-string/3.2.3-拼接与内存优化.md)） |

增删改查完整 API 见第 9 章 [顺序容器](../ch09-sequential-containers/)。

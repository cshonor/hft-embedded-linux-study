# 3.2.6 HFT 场景拓展

## 本节讲什么

FIX 报文切分、日志拼接、合约/订单文本字段；本章背诵极简版与综合示例。

## 1. FIX 报文（简化示意）

FIX 是交易所常见的**键值对 + 分隔符**文本（`|` 或 `\001` SOH）。`string` 适合整段持有报文，再 `find` / `substr` 取字段：

```cpp
#include <string>

// 简化：用 '|' 代替 SOH
std::string fix = "8=FIX.4.2|35=D|11=OID1001|55=IF2406|38=10|";

std::string tag_11;   // ClOrdID
auto pos = fix.find("11=");
if (pos != std::string::npos) {
    auto start = pos + 3;
    auto end = fix.find('|', start);
    tag_11 = fix.substr(start, end - start);   // "OID1001"
}
```

生产环境会用专业 FIX 库；这里说明 **`string` 是报文容器 + 字段切分的基础**。

## 2. 日志拼接

```cpp
std::string log;
log += "[";
log += std::to_string(order_id);   // C++11
log += "] send ";
log += symbol;
log += " qty=";
log += std::to_string(qty);
// 或一次拼接："[123] send IF2406 qty=10"
```

热路径上若频繁拼接，用 **`reserve` + `append`**（见 [3.2.3](./3.2.3-拼接与内存优化.md)）；固定长度字段用 **`char[N]` / `std::array`** 避免动态分配（见 [3.2.4](./3.2.4-string与C风格字符串.md)）。

## 3. 合约 / 订单文本

- 合约代码：`"IF2406"`、`"600519.SH"` —— `string` 比较、哈希、作 map 键
- 与 [2.6 Sales_data](../ch02-variables-and-basic-types/2.6-custom-data-structures/2.6.1-Sales_data入门与聚合.md) 的 `bookNo` 同类：**业务文本字段用 `string`，数值用算术类型**

## 背诵极简版

```
C 字符串：裸内存+手动；string：封装+RAII+运算符
HFT 一般 string；热路径固定 char[N]/array + RAII 折中
HFT 热路径：reserve+append 代替 +；少临时、少碎片
s3=s1+s2：至少 1 临时 + 多次分配；+= 改 s1 更省
C char*：指首字符；声明用 char *p（勿写 char* p,q 坑）
const int*p 指向常量；int*const p 常量指针（见 2.4.1）
读：>> 连续空白当分隔符，逐字段；getline 保留空格读到换行
HFT：cin>> 拆订单号/价/量；getline 读备注与日志
拼接 +/+=  比较 ==  长度 size/empty
vs char*：string 安全；C 接口用 c_str() 过渡
cctype：isalpha/isdigit；参数 unsigned char→int
HFT：校验合约全字母、价格全数字
```

## 示例

```cpp
#include <iostream>
#include <string>
#include <cctype>

int main() {
    std::string s1 = "hello", s2;
    s2 = s1;
    s1 = "world";
    std::cout << s1 << " " << s2 << std::endl;  // world hello

    std::string greeting = s1 + ", " + s2;
    std::cout << greeting << std::endl;

    std::string line;
    std::getline(std::cin, line);

    for (unsigned char c : s1) {
        if (std::isalpha(c))
            std::cout << static_cast<char>(std::toupper(c));
    }
    std::cout << std::endl;

    return 0;
}
```

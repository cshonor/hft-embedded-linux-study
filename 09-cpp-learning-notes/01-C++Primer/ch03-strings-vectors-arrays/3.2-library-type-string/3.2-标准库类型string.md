# 3.2 标准库类型 string

## 本节讲什么

`std::string` 是 C++ 处理文本的**核心类**，也是后面写 HFT 策略、处理**订单 / 行情 / 日志**文本的基础。重点：**初始化**、**读写**、**拼接与内存**、**与 C 字符串对比**、**`<cctype>` 校验**。

## 要点

- **`#include <string>`**；可变长，**RAII 自动释放**内存
- **初始化**：默认空串、拷贝、字面值、`string(n, c)` 重复字符
- **读写**：`cin >> s` 读单词；`getline(cin, s)` 读整行；`cout << s` 输出
- **操作**：`+` / `+=` 拼接、`==` 比较、`size()` / `empty()` / `[]` / `substr`
- **vs C 字符串**：`char*` + 手动分配 vs `string` 随内容增长自动扩缩
- **HFT**：FIX 报文、日志拼接、合约代码；热路径 `reserve`+`append`、固定 `char[N]`
- **`<cctype>`**：`isalpha`/`isdigit`；参数须 **`unsigned char`**

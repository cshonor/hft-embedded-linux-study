# 3.1 命名空间的 using 声明

## 本节讲什么

介绍如何通过 **`using` 声明**（如 `using std::cin;`）简化代码，使得在使用标准库名字时无须每次都添加 `std::` 前缀。这是进入标准库编程前的常用写法，但应避免在头文件中滥用 `using namespace std;`。

## 要点

- 命名空间 `std` 封装了标准库中的名字，避免与用户代码冲突
- **`using std::cin;`**：将 `cin` 引入当前作用域，之后可直接写 `cin` 而非 `std::cin`
- 可对多个名字分别做 using 声明：`using std::cout; using std::endl;`
- **不要**在头文件中使用 `using namespace std;`，以免污染所有包含该头文件的代码
- 在 `.cpp` 源文件中局部使用 using 声明是常见且安全的做法

## 示例

```cpp
#include <iostream>
#include <string>
using std::cin;
using std::cout;
using std::endl;
using std::string;

int main() {
    string line;
    getline(cin, line);
    cout << line << endl;
    return 0;
}
```

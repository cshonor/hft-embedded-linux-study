# 3.5.5 易错点与示例

## 本节讲什么

教材高频易错点、背诵极简版与完整示例代码。

## 高频易错点汇总

1. **维度不能用普通变量**，须 `constexpr` 等常量表达式  
2. **`char` 数组接字符串字面量**要预留 **`'\0'`** 空间：`"abc"` 需要 4 字节  
3. **数组传参退化为指针**，函数内 `sizeof` 得到的是指针大小，**丢失长度**  
4. **`&arr` 与 `&arr[0]`**：地址数值可能相同，**类型不同**，`+1` 步长不同  
5. **数组不能整体复制**，只能循环逐元素拷贝  
6. **越界无报错**，属于隐蔽 UB——与 `vector[]` 同类风险  

## 背诵极简版

```
内置数组：编译期固定大小，不能扩容、不能整体拷贝赋值
T a[N] 或 T a[]={...}；维度须常量；char "abc" 占 4 含 \0
下标 0..N-1；arr[i]==*(arr+i)；越界 UB，无检查
数组名多数退化为 &a[0]；例外 sizeof(a) &a 初始化
&arr 类型 int(*)[N]；&arr[0] 类型 int*
begin/end 遍历数组；优先 vector/string
传参丢长度；业务用 vector，C 接口/固定缓冲才用数组
```

## 完整示例

```cpp
#include <iostream>
#include <iterator>
#include <string>

int main() {
    int arr[3] = {1, 2, 3};

    // 下标
    arr[0] = 10;
    std::cout << arr[0] << "\n";

    // 退化：数组名 → 指针
    int *p = arr;
    std::cout << *p << " " << *(p + 1) << "\n";

    // sizeof / 地址类型
    std::cout << "sizeof(arr)=" << sizeof(arr)
              << " sizeof(p)=" << sizeof(p) << "\n";

    // begin/end 遍历
    for (auto it = std::begin(arr); it != std::end(arr); ++it)
        std::cout << *it << " ";
    std::cout << "\n";

    // 字符数组
    char msg[] = "FIX";   // 'F','I','X','\0'
    std::cout << msg << "\n";

    return 0;
}
```

## 课后核心考点

1. 数组 vs `vector`：固定大小、不能拷贝、传参退化  
2. 列表初始化、字符数组与 `'\0'`  
3. 下标与指针等价、`arr[i]` ≡ `*(arr+i)`  
4. 退化规则与 **3 种例外**  
5. `std::begin`/`std::end` 遍历  
6. **现代 C++**：优先 `vector`/`string`

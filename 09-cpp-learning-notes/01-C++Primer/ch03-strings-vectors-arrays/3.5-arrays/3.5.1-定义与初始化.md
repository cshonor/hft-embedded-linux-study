# 3.5.1 定义与初始化

## 本节讲什么

内置数组的语法、维度规则、两种初始化方式，以及**不能拷贝/赋值**的致命限制。

## 语法格式

```cpp
T 数组名[常量表达式];

int arr[10];              // 10 个 int
constexpr int n = 5;
double nums[n];           // ✅ 维度是常量表达式

// int m; double a[m];    // ❌ m 是运行时变量，非法 → 见 2.4.3 constexpr
```

**规则：**

- 元素类型**不能是引用**：不存在 `int &ref_arr[5];`
- 数组尺寸必须 **> 0**，**编译期确定**（须 `constexpr` 等常量表达式，见 [2.4.3 constexpr 详解](../../ch02-variables-and-basic-types/2.4-const-qualifier/2.4.3-constexpr详解.md)）
- 局部数组通常在**栈**上；过大易**栈溢出**（HFT 超大缓冲慎用栈数组）

## 列表初始化

可**省略维度**，由初值个数推断：

```cpp
int a1[] = {1, 2, 3};       // 维度 = 3
int a2[5] = {1, 2};         // 前 2 个为 1、2，后 3 个**值初始化为 0**
std::string s[3] = {"hi"};  // 第一个 "hi"，后两个空串 ""

// int err[2] = {1, 2, 3};  // ❌ 初值个数超过数组长度
```

## 字符数组与字符串字面量

用双引号字符串初始化会**自动追加 `'\0'`**：

```cpp
char c1[] = "abc";                    // 实际 4 字节：'a' 'b' 'c' '\0'
// char c2[3] = "abc";              // ❌ 需要 4 字节，放不下结束符
char c3[] = {'a', 'b', 'c', '\0'};  // 等价 c1
```

与 [3.2.4 string 与 C 风格字符串](../3.2-library-type-string/3.2.4-string与C风格字符串.md) 衔接：业务字符串优先 `std::string`；必须 `char[]` 时要**自己保证 `\0` 与边界**。

## 数组不能拷贝、不能整体赋值

```cpp
int a[3] = {1, 2, 3};
// int b[3] = a;   // ❌ 非法：不能拷贝初始化
// b = a;          // ❌ 非法：不能整体赋值
```

只能**逐元素**循环拷贝；这也是传参易丢长度、应优先 `vector` 的原因之一。

# 3.5 数组

## 本节讲什么

介绍 C++ **内置原生数组**：与 `vector` 一样存**同类型连续元素**，但尺寸**编译期固定、不可扩容**。重点：定义与初始化、下标、**数组名退化为指针**、`std::begin`/`end`；现代 C++ **优先 `vector`/`string`**。

## 要点

- **语法**：`T name[常量表达式]`；维度须编译期常量，**不能拷贝/整体赋值**
- **初始化**：列表初始化可省略维度；`char[]` 用字符串字面量会**自动加 `\0`**
- **下标**：从 0 开始，`arr[i]` ≡ `*(arr+i)`；**无边界检查**，越界 UB
- **退化**：数组名多数情况变**指向首元素的指针**；`sizeof(arr)`、`&arr`、初始化时不退化
- **`begin(arr)`/`end(arr)`**：C++11 全局函数，原生数组也能按迭代器风格遍历
- **vs vector**：数组固定、传参丢长度；业务开发优先 vector（见 [3.3](../3.3-library-type-vector/3.3-标准库类型vector.md)）

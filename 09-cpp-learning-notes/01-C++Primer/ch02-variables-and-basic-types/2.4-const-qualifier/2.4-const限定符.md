# 2.4 const 限定符

## 本节讲什么

介绍如何使用 `const` 定义值不能被改变的变量。深入探讨**常量引用**、**const 指针**，并明确区分**顶层 const**（对象自身只读）和**底层 const**（通过指针/引用不能改所指对象）。最后引入 C++11 的 **`constexpr`** 编译期常量。

## 要点

- **`const int ci = 0;`**：顶层 const，自身不可改，**必须初始化**
- **`const int &r`**：底层 const，不能通过 `r` 改对象；**可绑字面量**
- **`const int *p`**：底层 const，不能 `*p` 改值，`p` 可改指向
- **`int *const p`**：顶层 const，地址固定，可 `*p` 改值
- **顶层** = 自己不能变；**底层** = 指向/绑定的对象不能通过它变
- **`constexpr`**：强制编译期求值，可作数组长度、模板实参

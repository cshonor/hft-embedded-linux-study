# 小结

## 本节讲什么

回顾第 2 章核心内容：数据类型决定程序中数据和操作的意义；内置类型、复合类型、const 限定、类型别名与推断机制，以及用 `struct` 自定义数据结构，是后续学习类与高级特性的基础。

## 要点

- **内置类型**：算术类型与字面值
- **变量**：声明 vs 定义、`extern`、作用域
- **复合类型**：引用（别名）与指针（间接访问）
- **const**：顶层/底层 const、`constexpr`
- **类型工具**：`typedef`/`using`、`auto`、`decltype`
- **自定义类型**：`struct`、头文件与 include guard

## 示例

```cpp
#include <iostream>
int main() {
    // 第 2 章综合回顾：引用、指针、auto、decltype
    int x = 10;
    int &rx = x;
    int *px = &x;
    rx = 20;

    auto y = 30;
    decltype(y + 1) z = 31;

    std::cout << *px << " " << z << std::endl;
    return 0;
}
```

# 返回类型

## 本节讲什么

本节讲解「返回类型」。

## 要点

- 函数如何返回结果
- 返回数组指针（或引用）的特殊函数声明

## 示例

```cpp
#include <iostream>
void print(const int *beg, const int *end) {
    while (beg != end)
        std::cout << *beg++ << " ";
}
int main() {
    int arr[] = {1, 2, 3};
    print(std::begin(arr), std::end(arr));
    std::cout << std::endl;
    return 0;
}
```

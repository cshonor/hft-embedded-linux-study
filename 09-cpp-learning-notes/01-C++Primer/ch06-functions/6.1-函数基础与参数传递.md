# 函数基础与参数传递

## 本节讲什么

本节讲解「函数基础与参数传递」。

## 要点

- 返回类型、函数名、形参列表
- **传值参数**：拷贝实参，形参改动不影响实参
- **传引用参数**：绑定实参，可直接改变实参
- 指针形参、数组形参
- `main` 命令行选项：`argc`、`argv`
- 可变形参：`initializer_list`、省略符形参

## 示例

```cpp
#include <iostream>
int fact(int n) {
    return n <= 1 ? 1 : n * fact(n - 1);
}
int main() {
    std::cout << fact(5) << std::endl;  // 120
    return 0;
}
```

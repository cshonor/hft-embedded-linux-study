# 函数重载（Overloaded Functions）

## 本节讲什么

本节讲解「函数重载（Overloaded Functions）」。

## 要点

- 同一作用域内，函数名相同、形参列表不同
- 编译器通过**函数匹配**决定选用哪个版本（最佳匹配、无匹配、二义性调用）

## 示例

```cpp
#include <iostream>
void print(int i) { std::cout << "int: " << i << std::endl; }
void print(double d) { std::cout << "double: " << d << std::endl; }
int main() {
    print(42);
    print(3.14);
    return 0;
}
```

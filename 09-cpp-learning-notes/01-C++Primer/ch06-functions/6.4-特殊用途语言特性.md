# 特殊用途语言特性

## 本节讲什么

本节讲解「特殊用途语言特性」。

## 要点

- 默认实参
- **内联函数（`inline`）**：优化规模较小的操作
- `constexpr` 函数：编译时计算结果
- 调试辅助：`assert` 宏、`NDEBUG` 预处理变量

## 示例

```cpp
#include <iostream>
constexpr int size() { return 42; }
int main() {
    int arr[size()];  // 编译期常量作数组大小
    std::cout << sizeof(arr) / sizeof(arr[0]) << std::endl;
    return 0;
}
```

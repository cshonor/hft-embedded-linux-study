# 核心类型

## 本节讲什么

本节讲解「核心类型」。

## 要点

| 类别 | 容器 |
|------|------|
| 有序关联容器 | `map`、`set`、`multimap`、`multiset`（基于红黑树/平衡树） |
| 无序关联容器（C++11） | `unordered_map`、`unordered_set` 等（基于哈希函数） |

## 示例

```cpp
#include <iostream>
#include <map>
int main() {
    std::map<std::string, int> word_count;
    ++word_count["hello"];
    ++word_count["world"];
    std::cout << word_count["hello"] << std::endl;
    return 0;
}
```

# 特有操作

## 本节讲什么

本节讲解「特有操作」。

## 要点

- 对关键字类型的限制
- `insert`、`emplace` 插入元素（检查返回值）
- `erase` 删除元素
- 下标操作与查找函数：`find`、`count`、`lower_bound`、`equal_range`

## 示例

```cpp
#include <iostream>
#include <set>
int main() {
    std::set<int> s{3, 1, 4, 1, 5};
    auto it = s.lower_bound(3);
    std::cout << *it << std::endl;
    return 0;
}
```

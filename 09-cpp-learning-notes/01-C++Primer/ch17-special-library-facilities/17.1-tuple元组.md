# 17.1 tuple 元组

## 本节讲什么

本节讲解「tuple 元组」。

## 要点

- 存放**不同类型**的一组数据，长度固定，比 `pair` 支持更多成员
- `get<索引>(tuple对象)` 获取元素，`tuple_size` 获取元素个数
- 场景：函数一次性返回多个不同类型结果、简易异构数据打包

## 示例

```cpp
#include <iostream>
#include <tuple>
#include <string>
int main() {
    std::tuple<std::string, int, double> t{"book", 10, 9.9};
    std::cout << std::get<0>(t) << " "
              << std::get<1>(t) << std::endl;
    return 0;
}
```

# 17.2 bitset 位集

## 本节讲什么

本节讲解「bitset 位集」。

## 要点

- 固定长度二进制位容器，专门做位运算（与、或、异或、置 1、清 0、翻转）
- 按下标访问某一位、统计 1 的个数、整体转为整数/字符串
- 场景：权限掩码、状态标记、高效布尔数组

## 示例

```cpp
#include <bitset>
#include <iostream>
int main() {
    std::bitset<8> bs("10101010");
    std::cout << bs.count() << std::endl;
    bs.set(0);
    std::cout << bs << std::endl;
    return 0;
}
```

# 17.4 随机数

## 本节讲什么

摒弃 `rand()`，现代 C++ 随机数架构：
1. **随机引擎**（如 `mt19937` 梅森旋转）：生成底层无符号随机原始数
2. **分布器**（均匀整数/浮点、正态、伯努利等）：映射到目标数值区间
优势：可控种子、分布均匀、可复现，适合仿真、抽奖、测试。

## 示例

```cpp
#include <iostream>
#include <random>
int main() {
    std::default_random_engine eng;
    std::uniform_int_distribution<int> dist(1, 6);
    for (int i = 0; i < 5; ++i)
        std::cout << dist(eng) << " ";
    std::cout << std::endl;
    return 0;
}
```

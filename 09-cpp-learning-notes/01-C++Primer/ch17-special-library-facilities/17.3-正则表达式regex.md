# 17.3 正则表达式 regex

## 本节讲什么

C++11 正则全套工具：

## 要点

- `regex`：正则表达式对象；`smatch`：字符串匹配结果
- `regex_match`：完整匹配；`regex_search`：查找子串；`regex_replace`：替换
- 支持迭代遍历所有匹配项，语法兼容 ECMAScript 正则规则

## 示例

```cpp
#include <iostream>
#include <regex>
#include <string>
int main() {
    std::regex pat(R"(\d+)");
    std::string s = "order 42 items";
    std::smatch m;
    if (std::regex_search(s, m, pat))
        std::cout << m.str() << std::endl;
    return 0;
}
```

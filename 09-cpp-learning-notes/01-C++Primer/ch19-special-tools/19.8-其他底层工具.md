# 19.8 其他底层工具

## 本节讲什么

`type_traits` 类型萃取、编译期类型判断、`void_t`、`decltype` 高阶用法、编译期编程基础。

## 示例

```cpp
#include <iostream>
#include <memory>
int main() {
    alignas(16) char buf[64];
    int *p = new (buf) int(42);  // placement new
    std::cout << *p << std::endl;
    p->~int();
    return 0;
}
```

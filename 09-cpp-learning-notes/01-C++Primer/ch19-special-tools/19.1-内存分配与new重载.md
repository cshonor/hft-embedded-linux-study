# 19.1 内存分配与 new 重载

## 本节讲什么

本节讲解「内存分配与 new 重载」。

## 要点

- 全局 `operator new` / `operator delete` 底层内存分配
- 类内重载专属分配器：自定义内存池、对齐分配
- **定位 new（placement new）**：在已分配内存上构造对象，不申请堆内存
- 内存释放、析构与 placement new 配套使用

## 示例

```cpp
#include <iostream>
void *operator new(std::size_t sz) {
    std::cout << "alloc " << sz << " bytes\n";
    return ::operator new(sz);
}
int main() {
    int *p = new int(42);
    delete p;
    return 0;
}
```

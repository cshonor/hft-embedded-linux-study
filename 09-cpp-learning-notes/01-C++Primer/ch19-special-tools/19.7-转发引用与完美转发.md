# 19.7 转发引用与完美转发

## 本节讲什么

结合可变参数，把参数原值类别（左值/右值）完整转发给内部函数，`std::forward` 原理应用。

## 示例

```cpp
#include <iostream>
#include <utility>
void process(int &)  { std::cout << "lvalue\n"; }
void process(int &&) { std::cout << "rvalue\n"; }
template<typename T>
void forward_call(T &&arg) {
    process(std::forward<T>(arg));
}
int main() {
    int x = 1;
    forward_call(x);
    forward_call(2);
    return 0;
}
```

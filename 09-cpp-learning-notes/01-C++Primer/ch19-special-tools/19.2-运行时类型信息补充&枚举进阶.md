# 19.2 运行时类型信息补充 & 枚举进阶

## 本节讲什么

本节讲解「运行时类型信息补充 & 枚举进阶」。

## 要点

- 强类型枚举 **`enum class`**：对比传统无作用域 `enum`，类型更安全、不会隐式转整型

## 示例

```cpp
#include <iostream>
enum class Color { Red, Green, Blue };
int main() {
    Color c = Color::Red;
    if (c == Color::Red)
        std::cout << "red\n";
    return 0;
}
```

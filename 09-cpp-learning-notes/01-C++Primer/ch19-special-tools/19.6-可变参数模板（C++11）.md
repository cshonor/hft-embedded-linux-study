# 19.6 可变参数模板（C++11）

## 本节讲什么

模板参数包 `Args...`，接收任意数量、任意类型参数。现代 STL 大量基于它：`tuple`、`emplace_back`、万能转发等。

## 示例

```cpp
#include <iostream>
template<typename... Args>
void print(Args... args) {
    ((std::cout << args << " "), ...);
    std::cout << std::endl;
}
int main() {
    print(1, 2.5, "hi");
    return 0;
}
```

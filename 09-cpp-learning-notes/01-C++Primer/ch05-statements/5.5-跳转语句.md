# 5.5 跳转语句

## 本节讲什么

本节讲解「跳转语句」。

## 要点

- **break**：终止最近循环或 `switch`
- **continue**：终止当前迭代，开始下一次迭代
- **goto**：无条件跳转（书中建议尽量避免）

## 示例

```cpp
#include <iostream>
int main() {
    for (int i = 0; i < 10; ++i) {
        if (i == 5)
            break;
        if (i % 2 == 0)
            continue;
        std::cout << i << " ";
    }
    std::cout << std::endl;
    return 0;
}
```

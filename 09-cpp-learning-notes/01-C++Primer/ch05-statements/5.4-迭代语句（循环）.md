# 5.4 迭代语句（循环）

## 本节讲什么

本节讲解「迭代语句（循环）」。

## 要点

| 语句 | 特点 |
|------|------|
| **while** | 条件为真时重复执行；适合迭代次数未知的场景 |
| **传统 for** | 初始化、条件、表达式三部分；适合按步骤迭代 |
| **范围 for（C++11）** | 简便遍历容器、数组或其他序列（`vector`、`string` 等） |
| **do while** | 先执行循环体再检查条件；保证至少执行一次 |

## 示例

```cpp
#include <iostream>
int main() {
    for (int i = 0; i != 5; ++i)
        std::cout << i << " ";
    std::cout << std::endl;
    int j = 0;
    while (j < 5) {
        std::cout << j << " ";
        ++j;
    }
    std::cout << std::endl;
    return 0;
}
```

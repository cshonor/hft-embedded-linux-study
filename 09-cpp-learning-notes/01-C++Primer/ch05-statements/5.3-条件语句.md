# 5.3 条件语句

## 本节讲什么

### if 语句
### switch 语句

## 要点

- 根据条件决定执行路径
- `if-else` 嵌套
- **悬垂 else（dangling else）**问题及花括号的正确使用
- 在若干固定选项（整数类型常量）中做出选择
- `case` 标签、`break` 的阻断作用、`default` 标签
- 在 `switch` 内部定义变量的规则

## 示例

```cpp
#include <iostream>
int main() {
    int score = 85;
    if (score >= 90)
        std::cout << "A" << std::endl;
    else if (score >= 60)
        std::cout << "Pass" << std::endl;
    else
        std::cout << "Fail" << std::endl;
    return 0;
}
```

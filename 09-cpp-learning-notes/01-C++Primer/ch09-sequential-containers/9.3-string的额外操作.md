# string 的额外操作

## 本节讲什么

本节讲解「string 的额外操作」。

## 要点

- 将 `string` 视作特殊容器
- 额外构造方法、子串操作
- 修改操作：`append`、`replace` 等
- 搜索操作、与数值类型之间的转换

## 示例

```cpp
#include <iostream>
#include <string>
int main() {
    std::string s = "hello";
    s += " world";
    std::cout << s.substr(0, 5) << std::endl;
    return 0;
}
```

# 容器概览与操作

## 本节讲什么

本节讲解「容器概览与操作」。

## 要点

- 各类顺序容器：`vector`、`list`、`deque`、`forward_list`、`array` 等
- 公共接口与特有操作：添加、访问、删除、改变容器大小

## 示例

```cpp
#include <iostream>
#include <vector>
int main() {
    std::vector<int> v = {1, 2, 3};
    v.push_back(4);
    for (auto x : v)
        std::cout << x << " ";
    std::cout << std::endl;
    return 0;
}
```

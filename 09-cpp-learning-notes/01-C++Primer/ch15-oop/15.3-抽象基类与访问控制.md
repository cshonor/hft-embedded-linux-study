# 抽象基类与访问控制

## 本节讲什么

本节讲解「抽象基类与访问控制」。

## 要点

- 含纯虚函数的**抽象基类**：无法实例化，仅表示接口
- `protected` 访问说明符：允许派生类访问，向普通用户隐藏

## 示例

```cpp
#include <iostream>
class Instrument {
public:
    virtual void play() const = 0;  // 纯虚函数
    virtual ~Instrument() = default;
};
class Piano : public Instrument {
public:
    void play() const override { std::cout << "Piano\n"; }
};
int main() {
    Piano p;
    p.play();
    return 0;
}
```

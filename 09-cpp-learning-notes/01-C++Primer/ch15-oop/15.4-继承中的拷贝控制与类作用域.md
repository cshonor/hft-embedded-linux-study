# 继承中的拷贝控制与类作用域

## 本节讲什么

本节讲解「继承中的拷贝控制与类作用域」。

## 要点

- 基类通常需定义**虚析构函数**，保证动态分配对象正确销毁
- 派生类对象的作用域嵌套关系
- 合成的拷贝/移动操作规则

## 示例

```cpp
#include <iostream>
class Base {
public:
    virtual ~Base() { std::cout << "~Base\n"; }
};
class Derived : public Base {
public:
    ~Derived() { std::cout << "~Derived\n"; }
};
int main() {
    Base *p = new Derived;
    delete p;  // 虚析构保证先 ~Derived 再 ~Base
    return 0;
}
```

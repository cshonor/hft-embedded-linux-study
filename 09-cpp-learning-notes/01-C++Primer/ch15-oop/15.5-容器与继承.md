# 容器与继承

## 本节讲什么

本节讲解「容器与继承」。

## 要点

- 容器无法直接保存不同类型对象（派生类赋给基类会被「切掉」）
- 存放具有继承关系的对象时，应在容器中放置基类的**智能指针**

## 示例

```cpp
#include <iostream>
class A { public: void f() { std::cout << "A\n"; } };
class B : public A {};
class C : public A {};
class D : public B, public C {};  // 菱形继承
int main() {
    D d;
    d.B::f();
    d.C::f();
    return 0;
}
```

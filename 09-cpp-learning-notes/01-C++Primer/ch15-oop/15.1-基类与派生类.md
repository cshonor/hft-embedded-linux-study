# 基类与派生类

## 本节讲什么

本节讲解「基类与派生类」。

## 要点

- 通过继承构建类的层次关系
- 派生列表声明继承（如 `public` 继承）
- `final` 关键字：防止继承发生

## 示例

```cpp
#include <iostream>
class Quote {
public:
    virtual double net_price(std::size_t cnt) const {
        return cnt >= bulk ? price * (1 - disc) * cnt : price * cnt;
    }
    virtual ~Quote() = default;
protected:
    double price = 0.0;
    std::size_t bulk = 0;
    double disc = 0.0;
};
int main() {
    Quote q;
    std::cout << q.net_price(10) << std::endl;
    return 0;
}
```

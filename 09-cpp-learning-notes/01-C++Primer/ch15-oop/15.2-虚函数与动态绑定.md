# 虚函数与动态绑定

## 本节讲什么

本节讲解「虚函数与动态绑定」。

## 要点

- 多态的实现
- 基类引用或指针调用**虚函数（virtual function）**时，运行时根据绑定对象的实际动态类型决定执行哪个版本
- 动态绑定（运行时绑定）

## 示例

```cpp
#include <iostream>
class Base {
public:
    virtual void print() const { std::cout << "Base\n"; }
    virtual ~Base() = default;
};
class Derived : public Base {
public:
    void print() const override { std::cout << "Derived\n"; }
};
int main() {
    Derived d;
    Base *bp = &d;
    bp->print();  // Derived
    return 0;
}
```

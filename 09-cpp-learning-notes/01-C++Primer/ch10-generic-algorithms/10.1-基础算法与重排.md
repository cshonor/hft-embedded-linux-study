# 基础算法与重排

## 本节讲什么

本节讲解「基础算法与重排」。

## 要点

- 查找、计数、填充、拷贝：`find`、`accumulate` 等
- 排序和重排：`sort`、`unique` 等
- **泛型算法本身永远不会改变底层容器的大小**

## 示例

```cpp
#include <algorithm>
#include <iostream>
#include <vector>
int main() {
    std::vector<int> v{3, 1, 4, 1, 5};
    std::sort(v.begin(), v.end());
    for (int x : v) std::cout << x << " ";
    std::cout << std::endl;
    return 0;
}
```

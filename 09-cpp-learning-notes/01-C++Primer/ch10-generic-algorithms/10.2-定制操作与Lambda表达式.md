# 定制操作与 Lambda 表达式

## 本节讲什么

本节讲解「定制操作与 Lambda 表达式」。

## 要点

- 默认运算符（`<`、`==`）不适用时，向算法传递自定义谓词函数
- C++11 **Lambda 表达式**：捕获列表、参数绑定（`bind`）机制

## 示例

```cpp
#include <algorithm>
#include <iostream>
#include <vector>
int main() {
    std::vector<int> v{1, 2, 3, 4, 5};
    auto it = std::find_if(v.begin(), v.end(),
        [](int n) { return n > 3; });
    if (it != v.end())
        std::cout << *it << std::endl;
    return 0;
}
```

# 再探迭代器

## 本节讲什么

本节讲解「再探迭代器」。

## 要点

| 类型 | 用途 |
|------|------|
| 插入迭代器 | 自动执行插入操作以扩展容器 |
| 流迭代器 | 绑定到 IO 流 |
| 反向迭代器 | 逆序遍历 |
| 移动迭代器 | 移动语义遍历 |

## 示例

```cpp
#include <iostream>
#include <vector>
int main() {
    std::vector<int> v{1, 2, 3, 4, 5};
    auto mid = v.begin() + v.size() / 2;
    std::cout << *mid << std::endl;
    return 0;
}
```

# 特定容器算法

## 本节讲什么

本节讲解「特定容器算法」。

## 要点

- `list`、`forward_list` 因内存结构特殊，提供专用成员函数算法
- 链表专用：`sort`、`merge`、`splice` 等，比通用算法性能更好

## 示例

```cpp
#include <algorithm>
#include <iostream>
#include <vector>
int main() {
    std::vector<int> v{5, 4, 3, 2, 1};
    std::sort(v.begin(), v.end(), std::greater<int>());
    for (int x : v) std::cout << x << " ";
    std::cout << std::endl;
    return 0;
}
```

# 模板参数与成员模板

## 本节讲什么

本节讲解「模板参数与成员模板」。

## 要点

- 类型参数与非类型参数
- 模板默认实参
- 类成员也可以是模板（成员模板）

## 示例

```cpp
#include <iostream>
template<typename T>
class Stack {
public:
    void push(const T &v) { data_[top_++] = v; }
    T pop() { return data_[--top_]; }
private:
    T data_[100];
    int top_ = 0;
};
int main() {
    Stack<int> s;
    s.push(1); s.push(2);
    std::cout << s.pop() << std::endl;
    return 0;
}
```

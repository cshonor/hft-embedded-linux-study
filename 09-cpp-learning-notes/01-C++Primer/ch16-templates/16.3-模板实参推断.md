# 模板实参推断

## 本节讲什么

本节讲解「模板实参推断」。

## 要点

- 编译器从函数调用推断模板实参类型
- 尾置返回类型
- 引用折叠（与 `std::move`、实参转发原理相关）

## 示例

```cpp
#include <iostream>
#include <vector>
template<typename C>
void print(const C &c) {
    for (const auto &e : c)
        std::cout << e << " ";
    std::cout << std::endl;
}
int main() {
    std::vector<int> v{1, 2, 3};
    print(v);
    return 0;
}
```

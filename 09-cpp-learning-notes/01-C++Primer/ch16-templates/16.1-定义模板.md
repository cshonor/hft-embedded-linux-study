# 定义模板

## 本节讲什么

本节讲解「定义模板」。

## 要点

- **函数模板**：`template <typename T>`，通过模板类型参数
- **类模板**：如自定义 `Blob` 模板类
- 编译器通过模板实参实例化特定函数或类版本

## 示例

```cpp
#include <iostream>
template<typename T>
T maximum(const T &a, const T &b) {
    return a > b ? a : b;
}
int main() {
    std::cout << maximum(3, 7) << std::endl;
    std::cout << maximum(3.14, 2.71) << std::endl;
    return 0;
}
```

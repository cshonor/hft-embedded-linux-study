# 高级特性

## 本节讲什么

本节讲解「高级特性」。

## 要点

- 模板重载
- 可变参数模板：接受可变数量参数
- 模板特例化：为特定类型提供与通用模板不同的定制实现

## 示例

```cpp
#include <iostream>
template<typename T>
void foo(const T &) {}
template<>
void foo<int>(const int &) {
    std::cout << "int specialization\n";
}
int main() {
    foo(3.14);
    foo(42);
    return 0;
}
```

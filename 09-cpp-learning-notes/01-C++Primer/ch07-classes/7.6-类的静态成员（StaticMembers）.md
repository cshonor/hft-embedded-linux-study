# 类的静态成员（Static Members）

## 本节讲什么

本节讲解「类的静态成员（Static Members）」。

## 要点

- 存在于所有对象之外，属于整个类共享
- 静态数据成员、静态函数成员

## 示例

```cpp
#include <iostream>
class Account {
public:
    static int object_count() { return count_; }
    Account() { ++count_; }
private:
    static int count_;
};
int Account::count_ = 0;
int main() {
    Account a, b;
    std::cout << Account::object_count() << std::endl;
    return 0;
}
```

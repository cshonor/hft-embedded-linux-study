# 定义抽象数据类型

## 本节讲什么

本节讲解「定义抽象数据类型」。

## 要点

- 接口与实现分离
- **隐式 `this` 指针**：成员函数内部访问调用该函数的对象
- **常量成员函数**：参数列表后加 `const`，确保不修改对象内容

## 示例

```cpp
#include <iostream>
#include <string>
class Sales_data {
public:
    std::string isbn;
    unsigned units_sold = 0;
    double revenue = 0.0;
    double avg_price() const {
        return units_sold ? revenue / units_sold : 0;
    }
};
int main() {
    Sales_data item;
    item.isbn = "978-0-201-78345-0";
    item.units_sold = 10;
    item.revenue = 250.0;
    std::cout << item.avg_price() << std::endl;
    return 0;
}
```

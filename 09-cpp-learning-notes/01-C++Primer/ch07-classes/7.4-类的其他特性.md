# 类的其他特性

## 本节讲什么

本节讲解「类的其他特性」。

## 要点

- 自定义类型成员
- 内联成员函数
- **可变数据成员（`mutable`）**：即使在 `const` 对象中也可修改
- 返回 `*this` 的成员函数设计

## 示例

```cpp
#include <iostream>
class X {
public:
    X() { std::cout << "default ctor\n"; }
    X(const X &) { std::cout << "copy ctor\n"; }
};
int main() {
    X a;
    X b = a;
    return 0;
}
```

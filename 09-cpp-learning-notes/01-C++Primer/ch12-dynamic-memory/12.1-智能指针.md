# 智能指针

## 本节讲什么

定义在 `<memory>` 头文件：
大大简化异常发生时的内存安全管理。

## 要点

| 类型 | 特点 |
|------|------|
| `shared_ptr` | 多个指针共享同一对象，引用计数自动释放内存 |
| `unique_ptr` | 独占所指向的对象 |
| `weak_ptr` | 弱引用，不增加引用计数 |

## 示例

```cpp
#include <iostream>
#include <memory>
int main() {
    std::shared_ptr<int> p1 = std::make_shared<int>(42);
    std::shared_ptr<int> p2 = p1;
    std::cout << *p1 << " use_count=" << p1.use_count() << std::endl;
    return 0;
}
```

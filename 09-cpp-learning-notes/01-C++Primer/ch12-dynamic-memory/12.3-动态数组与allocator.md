# 动态数组与 allocator

## 本节讲什么

本节讲解「动态数组与 allocator」。

## 要点

- `new T[]` 分配动态数组
- `allocator` 类：将内存分配与对象构造相分离，获得更高灵活性和性能

## 示例

```cpp
#include <iostream>
#include <memory>
int main() {
    std::unique_ptr<int[]> up(new int[5]{1, 2, 3, 4, 5});
    for (int i = 0; i < 5; ++i)
        std::cout << up[i] << " ";
    std::cout << std::endl;
    return 0;
}
```

# 函数调用运算符与函数对象

## 本节讲什么

本节讲解「函数调用运算符与函数对象」。

## 要点

- 重载 `operator()`
- 重载该运算符的类对象称为**函数对象**，可像函数一样调用
- C++11 **Lambda 表达式**本质：编译器生成的未命名函数对象类的未命名对象

## 示例

```cpp
#include <iostream>
class Counter {
public:
    Counter &operator++() { ++val_; return *this; }
    int operator++(int) { int old = val_; ++val_; return old; }
    int value() const { return val_; }
private:
    int val_ = 0;
};
int main() {
    Counter c;
    std::cout << ++c << " " << c++ << " " << c.value() << std::endl;
    return 0;
}
```

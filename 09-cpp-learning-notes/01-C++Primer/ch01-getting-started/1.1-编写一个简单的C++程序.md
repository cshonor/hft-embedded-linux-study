# 1.1 编写一个简单的 C++ 程序

## 本节讲什么

介绍 C++ 程序的基本结构。每个程序必须包含一个名为 `main` 的函数——操作系统通过调用它来运行程序。函数定义包含四部分：**返回类型**、**函数名**、**形参列表**和**函数体**。在大多数系统中，`main` 返回 `0` 表示程序执行成功。

## 要点

- 每个 C++ 程序至少有一个 `main` 函数，程序从 `main` 开始执行
- 函数定义四要素：返回类型、函数名、形参列表、函数体
- `return 0;` 通常表示程序正常结束（成功）
- 程序需经编译、链接后才能运行

## 示例

```cpp
#include <iostream>
int main() {
    return 0;  // 返回 0 表示程序正常结束
}
```

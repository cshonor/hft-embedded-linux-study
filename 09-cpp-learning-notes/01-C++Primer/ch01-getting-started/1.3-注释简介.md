# 1.3 注释简介

## 本节讲什么

讲解 C++ 中的两种注释方式：**单行注释**（以 `//` 开始）和**界定符对形式的注释**（以 `/*` 开始，以 `*/` 结束）。书中特别警告：界定符对形式的注释**不能嵌套**使用，否则会导致编译器产生令人迷惑的错误。

## 要点

- 单行注释：`//` 从该行注释符到行尾均为注释
- 界定符对注释：`/*` … `*/` 之间的内容为注释，可跨行
- **禁止嵌套** `/* */` 注释（如在一个 `/*` … `*/` 内再写 `/*`）
- 注释不影响程序行为，用于说明代码意图；编译器会**完全忽略**注释
- **单行 `//`**：适合短说明，如 `// 读取委托价格和数量`
- **多行 `/* */`**：适合模块说明，如标清「行情处理」「发单逻辑」等段落

## 示例

```cpp
// 单行注释：读取两个交易参数
#include <iostream>
/*
 * 多行注释：委托处理模块
 * 注意 /* */ 不能嵌套，否则编译报错
 */
int main() {
    double price = 0;
    int qty = 0;
    std::cin >> price >> qty;  // 按顺序读入价格和数量

    // 下面这行被注释掉，编译器不执行
    // std::cout << "debug" << std::endl;

    std::cout << price << " " << qty << std::endl;
    return 0;
}
```

# 小结与术语表

## 本节讲什么

回顾并巩固第 1 章涉及的核心概念，包括内置类型、命名空间、实参、缓冲区等，帮助读者在继续学习第 2 章之前建立完整的知识框架。

## 要点

- **内置类型（built-in type）**：语言自带的类型，如 `int`、`double`、`char`
- **命名空间（namespace）**：如 `std`，避免名字冲突；`std::cin` 表示标准库中的 `cin`
- **实参（argument）**：调用函数时传入的值；**形参（parameter）**：函数定义中的参数
- **缓冲区（buffer）**：IO 流在读写时使用的内存区域；`endl` 会刷新缓冲区
- **类（class）**：用户自定义的数据类型，本章通过 `Sales_item` 初步体验
- **头文件（header）**：如 `<iostream>`、`"Sales_item.h"`，提供声明与定义

## 示例

```cpp
// 第 1 章综合：循环读取不定数量输入并求和
#include <iostream>
int main() {
    std::cout << "Enter numbers (Ctrl+Z then Enter on Windows to end):\n";
    int sum = 0, value = 0;
    while (std::cin >> value)   // 读至 EOF
        sum += value;
    std::cout << "Sum = " << sum << std::endl;
    return 0;
}
```

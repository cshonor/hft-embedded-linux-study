# 1.6 书店程序

## 本节讲什么

全章实战总结：将前面学到的变量、输入输出、控制流以及 `Sales_item` 类结合起来，编写一个完整程序——读取多条书籍销售记录，按书号汇总，输出各本书的**总销售额**、**售出册数**和**平均售价**。

## 要点

- 综合运用：`cin`/`cout`、循环、条件、`Sales_item`
- 读入第一条记录作为当前汇总对象
- 循环读入后续记录：书号相同则累加（`total += trans`），不同则先输出当前汇总再换新书
- 循环结束后输出最后一本书的汇总

## 示例

```cpp
#include <iostream>
#include "Sales_item.h"

int main() {
    Sales_item total;
    if (std::cin >> total) {               // 读入第一条记录
        Sales_item trans;
        while (std::cin >> trans) {        // 读入剩余记录
            if (total.isbn() == trans.isbn())
                total += trans;            // 同一本书，累加
            else {
                std::cout << total << std::endl;  // 不同书，输出上一本汇总
                total = trans;             // total 重置为新书
            }
        }
        std::cout << total << std::endl;   // 输出最后一本
    }
    return 0;
}
```

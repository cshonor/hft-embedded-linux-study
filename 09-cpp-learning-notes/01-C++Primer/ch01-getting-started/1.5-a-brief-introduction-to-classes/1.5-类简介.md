# 1.5 类简介

## 本节讲什么

初步引入 C++ 最重要特性之一的**类（class）**：通过定义类来自定义数据结构。本节通过书中预设好的 `Sales_item` 类，教导读者**无需关心内部实现**，只需了解如何定义类类型的对象、如何利用点运算符 `.` 调用成员函数（如提取书号的 `isbn()`），以及如何使用运算符完成对象间的读取、打印和相加操作。

## 要点

- 类是 C++ 的核心机制，用于定义用户自定义类型
- 定义类对象：`Sales_item item;`
- 点运算符 `.` 调用成员函数：`item.isbn()`
- `Sales_item` 支持 `>>` 读取、`<<` 打印、`+` 相加、`=` 赋值
- 本节只**使用**类，类的定义与实现留待后续章节

# 1.2 初识输入输出

## 本节讲什么

C++ 通过标准库中的 `iostream` 提供 IO 机制，其中包含表示输入流的 `istream` 和表示输出流的 `ostream` 基础类型。讲解如何使用 `std::cin` 与输入运算符 `>>` 读取数据，使用 `std::cout` 与输出运算符 `<<` 打印消息，使用 `std::cerr` 输出错误信息，以及使用操纵符 `std::endl` 结束当前行并刷新缓冲区。

## 要点

- 标准库头文件 `<iostream>` 提供 IO 支持；`istream` / `ostream` 是输入流与输出流的基础类型
- **管道比喻**：`cin`/`cout` 是连接键盘、屏幕和程序的管道；**箭头方向 = 数据流入/流出的方向**
  - `std::cin >> v`：`>>` 指向变量，数据从键盘管道**流进** `v`
  - `std::cout << x`：`<<` 指向 `cout`，数据从程序**流出**到屏幕管道
  - `std::cerr << err`：错误信息流出到屏幕，通常不缓冲，立刻显示
- **变量是装数据的盒子**：`int v1 = 0, v2 = 0;` 先声明两个能装整数的盒子；`std::cin >> v1 >> v2;` 让你依次敲两个数，第一个流进 `v1`，第二个流进 `v2`
- **`cin >>` 遇空白就停**（空格 / Tab / 回车都算），不是「不按回车就一直读」；读带空格的整行用 `getline`（见 [3.2.2 读写操作](../ch03-strings-vectors-arrays/3.2-library-type-string/3.2.2-读写操作.md)）
- **输出顺序：从左到右，离 `cout` 最近的先输出**
  - `std::cout << v1 << " + " << v2;` 从左往右读：先输出 `v1`，再输出 `" + "`，最后输出 `v2`
  - 若 `v1=12`、`v2=34`，屏幕显示 `12 + 34`
  - 把 `<<` 看成「递给 `cout`」：`cout` 左边紧挨着的那个最先被递出去
- `std::endl`：输出换行并刷新缓冲区

## 通俗理解：管道与数据流

可以把标准输入输出想成**连接程序和外部的管道**——`iostream` 库负责把键盘、屏幕等默认设备和你的程序连起来。

```
键盘 ──管道──► 程序          std::cin  +  >>
程序 ──管道──► 屏幕          std::cout +  <<
程序 ──管道──► 屏幕（错误）   std::cerr +  <<
```

**箭头方向就是数据流的方向：**

| 写法 | 数据往哪流 | 含义 |
|------|-----------|------|
| `std::cin >> v` | 键盘 → 变量 `v` | 从输入通道**读进**程序 |
| `std::cout << x` | 程序 → 屏幕 | 从程序**写出**到屏幕 |
| `std::cerr << err` | 程序 → 屏幕 | 写出错误信息（通常不缓冲，立刻显示） |

- **`std::cin`**：输入管道，键盘敲的内容沿 `>>` 流进变量
- **`std::cout`**：输出管道，程序里的内容沿 `<<` 流到屏幕
- **`std::cerr`**：错误输出管道，专门打错误/警告；不像 `cout` 那样先攒在缓冲区里，而是尽快显示，方便快速定位问题

连写多个 `>>` 或 `<<` 时，**一律从左到右**依次处理：

```cpp
std::cin >> v1 >> v2;
// ① 先读进 v1  ② 再读进 v2

std::cout << v1 << " + " << v2;
// ① 先输出 v1  ② 再输出 " + "  ③ 最后输出 v2
//    ▲ 紧挨 cout 的第一个，最先出现在屏幕上
```

**输出顺序记忆法**：`cout` 是要接东西的人，`<<` 是递东西的手——从左边开始递，**离 `cout` 最近的那个最先输出**，不是离得最远的先输出。

**输入 vs 输出的顺序比喻：**

| | 比喻 | 含义 |
|---|------|------|
| **输出** `cout << v1 << v2` | **先进先出**（管道/队列） | 先「放进」管道的 `v1` 先从另一端流到屏幕，再是 `v2` |
| **输入** `cin >> v1 >> v2` | **按顺序填格子** | 像铺地砖：先敲的第一个数填进 `v1`，第二个数填进 `v2`，严格按代码先后顺序 |

## 三条 IO 线：cin / cout / cerr

`iostream` 默认给程序连了**三条线**：

```
        ┌── cin  ──► 程序（键盘 → 程序）
键盘 ───┤
        └── （标准输入）

        ┌── cout ──► 屏幕（程序 → 屏幕，正常输出，有缓冲）
程序 ───┤
        └── cerr ──► 屏幕（程序 → 屏幕，错误输出，通常立刻显示）
```

| 线 | 方向 | 用途 | 缓冲 |
|----|------|------|------|
| **`std::cin`** | 键盘 → 程序 | 读入用户输入 | — |
| **`std::cout`** | 程序 → 屏幕 | 正常提示、结果打印 | 有缓冲区，可能攒一批再显示 |
| **`std::cerr`** | 程序 → 屏幕 | 错误、警告 | **无缓冲（或立即刷新）**，马上显示 |

**举例（委托单）：**

```cpp
#include <iostream>

int main() {
    double price = 0;
    int qty = 0;

    std::cout << "请输入委托价格和数量: ";
    std::cin >> price >> qty;                    // cin：读入价格和数量

    if (price < 0) {
        std::cerr << "错误：委托价格不能为负\n";  // cerr：立刻报错，不缓冲
        return 1;
    }

    std::cout << "委托已提交：价格=" << price     // cout：正常业务提示
              << " 数量=" << qty << std::endl;
    return 0;
}
```

- **`cin`**：读客户敲进来的价格和数量
- **`cout`**：打印「委托已提交」等正常信息
- **`cerr`**：价格非法时**立刻**打出错误，方便第一时间发现问题（生产环境常用专门日志库替代，但原理类似）

## 示例

```cpp
#include <iostream>
int main() {
    std::cout << "Enter two numbers: ";

    int v1 = 0, v2 = 0;   // 两个「盒子」，用来装从键盘流进来的整数
    std::cin >> v1 >> v2; // 依次输入：第一个数 → v1，第二个数 → v2

    std::cout << v1 << " + " << v2 << " = " << v1 + v2 << std::endl;
    // 输出顺序（从左到右）：v1 → " + " → v2 → " = " → (v1+v2) → 换行
    // 例如 v1=12, v2=34 时屏幕显示：12 + 34 = 46
    return 0;
}
```

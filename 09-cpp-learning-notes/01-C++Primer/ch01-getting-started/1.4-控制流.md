# 1.4 控制流

## 本节讲什么

介绍允许程序跳出顺序执行、实现更复杂执行路径的语句。详细讲解 `while` 循环（反复检测条件并执行循环体直至条件为假）、`for` 循环（将初始化、循环条件和变量修改集中在循环头以简化代码）以及支持条件执行的 `if` 语句。还演示了如何使用 `while (std::cin >> value)` 不断读取数量不定的输入，直至遇到文件末尾（EOF）。

## 要点

- `while`：先判断条件，为真则重复执行循环体
- `for`：循环头包含初始化、条件、迭代语句，适合计数循环
- `if` / `else`：按条件选择执行路径
- `while (std::cin >> value)`：循环读取输入，直至 EOF 或输入失败
- Windows 下 EOF 常用 `Ctrl+Z` 后回车；Unix 下为 `Ctrl+D`
- **条件表达式的规则（与 Rust 不同）**：
  - `if` / `while` / `for` 的条件最终要当成**布尔值**来用
  - C++ 允许**隐式转换**：`0` → 假，`非 0` → 真
  - 所以 `while (1)` **合法**，表示一直循环（`1` 永远为真）
  - Rust 则**严格要求** `bool`，不能写 `while 1`
  - 现代 C++ 写死循环更推荐 `while (true)` 或 `for (;;)`，意图更清晰

## 与 Rust 对照

| 写法 | C++ | Rust |
|------|-----|------|
| `while (1)` | ✅ 合法，死循环 | ❌ 编译错误，必须是 `bool` |
| `while (true)` | ✅ 推荐 | ✅ `while true` |
| `if (0)` | ✅ 合法，分支不执行 | ❌ 必须是 `bool` |
| `if (price < 0)` | ✅ | ✅ |

**记忆**：C++ 很**宽松**——非 0 即真；Rust 很**严格**——必须是纯 `bool`。

## 通俗理解

- **`while (1)`** = 死循环：`1` 永远是非 0，一直被当成「真」
- **HFT 行情监听**：常用 `while (1)` 或 `while (true)` 做主循环，一直盯盘，满足条件时 `break` 退出
- **`if` / `for`** 同样适用：只有 `0` 是假，其他整数都算真

```cpp
while (1) {
    // poll_market();   // 拉行情
    // if (should_stop) break;  // 主动退出
}
```

## 示例

```cpp
#include <iostream>
int main() {
    // while：累加 1 到 10
    int sum = 0, val = 1;
    while (val <= 10) {
        sum += val;
        ++val;
    }
    std::cout << "Sum 1..10 = " << sum << std::endl;

    // for：同样效果
    sum = 0;
    for (int i = 1; i <= 10; ++i)
        sum += i;
    std::cout << "Sum 1..10 = " << sum << std::endl;

    // if：按分数输出等级
    int score = 85;
    if (score >= 90)
        std::cout << "A" << std::endl;
    else if (score >= 60)
        std::cout << "Pass" << std::endl;
    else
        std::cout << "Fail" << std::endl;

    // while(1) 死循环（C++ 允许，1 当 true）
    // while (true) { /* 轮询行情 */ break; }  // 更推荐的写法

    return 0;
}
```

# 4.10 逗号运算符

## 本节讲什么

逗号运算符 `,` 的求值顺序与返回值；与函数参数逗号、初始化列表逗号的区别。

## 规则

- 从左到右依次求值每个子表达式  
- **整个表达式的值 = 最右侧**子表达式的值  
- 优先级**最低**（见 [4.12](../4.12-operator-precedence/4.12-运算符优先级表.md)）

```cpp
int a = (1, 2, 3);   // a == 3
```

## 典型用途：`for` 多变量

```cpp
for (int i = 0, j = 10; i < j; ++i, --j)
    /* ... */;
```

## 不是逗号运算符的情况

| 场景 | 逗号含义 |
|------|----------|
| `f(a, b)` | 函数**参数分隔** |
| `int a{1, 2, 3}` | 初始化列表（容器/数组） |
| `int i, j;` | **声明**分隔多个变量 |

教材建议：**少滥用**逗号运算符写复杂表达式，可读性差。

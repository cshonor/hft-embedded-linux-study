# 4.7 条件运算符

## 本节讲什么

C++ **唯一三元运算符** `cond ? expr1 : expr2`。

## 规则

- 条件为 **真** → 计算并返回 `expr1` 的值  
- 条件为 **假** → 计算并返回 `expr2` 的值  
- 两分支类型须能统一（可能隐式转换）

## 嵌套

```cpp
// grade > 90 ? "A" : grade > 60 ? "B" : "C"
```

## 优先级陷阱（必考）

`?:` 优先级**很低**，与 `<<` 混用时必须加括号：

```cpp
int grade = 75;
std::cout << (grade < 60 ? "fail" : "pass");  // ✅
// std::cout << grade < 60 ? "fail" : "pass"; // ❌ 等价 (cout<<grade)<60 ? ...
```

# 4.6 成员访问运算符

## 本节讲什么

`.` 点运算符与 `->` 箭头运算符；与 `*` 的优先级陷阱。

## `.` 与 `->`

| 运算符 | 作用于 | 示例 |
|--------|--------|------|
| **`.`** | 对象 / 引用 | `obj.member`、`s.size()` |
| **`->`** | 指针 | `ptr->member`，等价 `(*ptr).member` |

```cpp
#include <string>

std::string s = "test";
std::string *p = &s;

s.size();      // 点：对象
p->size();     // 箭头：指针
(*p).size();   // 与 -> 等价

// *p.size();  // ❌ . 优先级高于 *，等价 *(p.size())，编译错误
```

`string`、`vector`、自定义类成员访问都遵循此规则（见 [3.2](../ch03-strings-vectors-arrays/3.2-library-type-string/3.2-标准库类型string.md)、[7.1 类基础](../ch07-classes/)）。

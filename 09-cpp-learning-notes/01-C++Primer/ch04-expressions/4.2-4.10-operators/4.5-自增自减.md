# 4.5 自增自减

## 本节讲什么

`++` `--` 前置与后置的区别——**重点考点**。

## 前置 `++i` / `--i`（推荐）

- **先**自增/自减，**再**返回修改后的对象
- 返回**左值**（可继续链式使用）
- **无**多余临时副本，性能更好

## 后置 `i++` / `i--`

- **先**返回**原值**的副本（右值），**再**修改变量
- 多一次临时，热路径避免

```cpp
int i = 10;
i += 2;      // i = 12
++i;         // i = 13
int j = i++; // j = 13，i 变为 14
// cout << i << " " << j;  →  14 13
```

## 完整示例解析

```cpp
#include <iostream>

int main() {
    int i = 10;
    i += 2;    // 复合赋值，i = 12
    ++i;       // 前置自增，i = 13
    int j = i++;  // 后置：j 得 13，i 变 14
    std::cout << i << " " << j << std::endl;  // 14 13
    return 0;
}
```

## 禁忌

**同一表达式中对同一变量多次自增/自减** → **未定义行为**：

```cpp
// int k = i++ + i++;  // ❌ UB
```

教材建议：默认写 **前置 `++i`**。

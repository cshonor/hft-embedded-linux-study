# 4.3 关系与逻辑运算符

## 本节讲什么

关系比较与逻辑组合；**短路求值**是核心考点。

## 关系运算符 `< <= > >= == !=`

- 结果类型为 **`bool`**（打印时可能显示 `0`/`1`）
- **错误写法**：`a < b < c` 等价 `(a < b) < c`，不是数学上的「介于两者之间」

```cpp
int a = 1, b = 2, c = 3;
// if (a < b < c)  // 先 (a<b)→true→1，再 1<c，逻辑常错
if (a < b && b < c)  // ✅
```

## 逻辑运算符 `!` `&&` `||`

| 运算符 | 含义 | 短路 |
|--------|------|------|
| `!` | 逻辑非，优先级在 `&&`/`||` 之上 | — |
| `&&` | 与 | 左侧为 **false**，右侧**不执行** |
| `||` | 或 | 左侧为 **true**，右侧**不执行** |

```cpp
if (b != 0 && a / b) { /* 左侧假则不算 a/b，防除零 */ }
```

**注意**：位运算 `&` `|` **没有**短路（见 [4.8](./4.8-位运算符.md)）。

# 4.8 位运算符

## 本节讲什么

`~` `<<` `>>` `&` `|` `^`——教材建议**只对无符号整数**使用。

## 各运算符

| 运算符 | 含义 |
|--------|------|
| `~` | 按位取反 |
| `<<` | 左移：右侧丢弃，左侧补 0；等价 × 2ⁿ（无溢出时） |
| `>>` | 右移：**无符号**左侧补 0；**有符号**补符号位（实现相关，不推荐） |
| `&` | 按位与（掩码） |
| `|` | 按位或 |
| `^` | 按位异或 |

```cpp
unsigned flags = 0x01;
flags |= 0x02;           // 置位
if (flags & 0x01) { }    // 测位
```

## 与逻辑运算符区别

- `&` `|` `^`：**无短路**，两侧都求值  
- `&&` `||`：短路（见 [4.3](./4.3-关系与逻辑运算符.md)）

## HFT 典型用途

协议标志位、权限掩码、快速 ×2 / ÷2（无符号左移/右移）；解析二进制字段时常与 `unsigned` 掩码配合。

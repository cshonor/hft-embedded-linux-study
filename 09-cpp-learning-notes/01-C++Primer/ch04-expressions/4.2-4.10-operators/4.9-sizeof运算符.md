# 4.9 sizeof 运算符

## 本节讲什么

计算类型或表达式占用的**字节数**；返回 `size_t`（无符号）。

## 两种写法

```cpp
sizeof(int);     // 类型
sizeof(var);     // 表达式（一般不求值 var，见下）
sizeof var;      // 变量可省略括号（不推荐与类型混淆）
```

## 两大特性（必考）

### 1. 数组名 vs 指针

```cpp
int arr[10];
sizeof(arr);     // 整个数组：10 * sizeof(int)
int *p = arr;
sizeof(p);       // 指针大小，如 8（64 位）
```

数组**不退化**的例外之一（见 [3.5.3](../ch03-strings-vectors-arrays/3.5-arrays/3.5.3-数组退化与指针.md)）。

### 2. 不对指针解引用

```cpp
int *p = nullptr;
sizeof(*p);      // ✅ 只算 int 的大小，不访问 *p，不崩溃
```

## 用途

- 估算结构体/数组占用  
- `sizeof(arr)/sizeof(arr[0])` 得数组元素个数（**仅当 `arr` 仍是数组名时**）

# 4.4 赋值运算符

## 本节讲什么

`=` 与复合赋值 `+=` `-=` `*=` `/=` `%=` `<<=` `>>=` `&=` `^=` `|=`。

## 基本规则

1. **`=` 左侧必须是可修改左值**

```cpp
// 10 = i;           // ❌ 字面量不能当左值
const int a = 1;
// a = 2;            // ❌ const 不可改
```

2. **复合赋值**等价展开：`i += 2` → `i = i + 2`（但**不是**简单文本替换，复合形式对类类型可能更高效）

3. **赋值返回左侧对象**，支持连续赋值：

```cpp
int a, b, c;
a = b = c = 0;   // 从右向左：c=0，返回值赋给 b，再赋给 a
```

复合赋值也返回左侧引用（类类型时常返回 `*this`）。

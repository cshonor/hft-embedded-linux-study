# 4.2 算术运算符

## 本节讲什么

二元 `+` `-` `*` `/` `%` 与一元 `+` `-` 的规则、整型提升与边界坑点。

## `+` `-` `*` `/`（二元）

- **整数除法**截断小数：`5 / 2 == 2`
- **浮点除法**保留小数：`5.0 / 2 == 2.5`
- 混合类型会**隐式转换**（见 [4.11.1](../4.11-type-conversions/4.11.1-隐式类型转换.md)）

## 一元 `-x`（取相反数）

### 核心语义

对操作数求相反数，数学上等价于 `0 - x`：

- 正数 → 负数：`-5 == -5`
- 负数 → 正数：`-(-3) == 3`
- `0` 不变：`-0 == 0`

### 底层规则（补码）

1. 先对操作数执行**整型提升**（`char` / `short` → `int`，见下文一元 `+`）；
2. **有符号**整数：按位取反再加 1（补码）；
3. **无符号**整数：数学上 `-u` 等价于 `2^位数 - u`（仍可能再参与有符号运算，注意混合类型）。

### 边界坑：`INT_MIN` 溢出

32 位 `int` 范围：`[-2147483648, 2147483647]`。数学上 `-INT_MIN == 2147483648`，超出 `int` 上限 → **有符号整数溢出，未定义行为（UB）**：

```cpp
#include <climits>
int a = INT_MIN;
int b = -a;   // UB（C/C++ 标准）
```

**HFT**：对行情偏移、盈亏取反时，若数据可能取极值，先用更宽类型（`long long`）或显式检查，不要对 `INT_MIN` 直接一元取负。

### 适用类型

所有算术类型：`char` / `short` / `int` / `long` / `float` / `double` 等均可；浮点直接变号，不涉及补码。

## 一元 `+x`（无数学效果，触发整型提升）

### 数学作用：几乎无意义

`+x` 直接返回操作数本身，值不变：

```cpp
+5 == 5;
+(-2) == -2;
```

日常极少单独写 `+x`；它与二元加法 `a + b` 里的 `+` 是**不同运算符**（一元 vs 二元），语法上靠上下文区分，例如 `+a + b` 中第一个 `+` 是一元，第二个是二元加。

### 唯一实用特性：强制整型提升

一元 `+`、一元 `-`、按位取反 `~` 都会对操作数执行**整型提升**（integral promotion）：

- `char` / `signed char` / `unsigned char` / `short` / `unsigned short`，且其值域能完整放入 `int` → 提升为 `int`；
- 若 `int` 与 `unsigned int` 同宽容纳所有 `unsigned short` 值 → `unsigned short` 提升为 `unsigned int`。

```cpp
char ch = static_cast<char>(0x80);  // 有符号 char 常为 -128
auto t1 = ch;    // 类型 char
auto t2 = +ch;   // 一元 + 触发提升 → 类型 int，值仍为 -128
```

`decltype(t1)` 是 `char`，`decltype(t2)` 是 `int`；`sizeof(t2)` 通常大于 `sizeof(t1)`。

### 典型场景

1. **只做类型提升、不改数值**：把 `char` / `short` 提到 `int` 再参与位运算或打印；
2. **与二元 `+` 区分**：正负号 vs 加法；
3. 保持表达式语法完整（教材/规范写法）。

## 一元 `+` / `-` 对比

| 特性 | 一元 `-x` | 一元 `+x` |
|------|-----------|-----------|
| 数学输出 | 数值取反 | 数值不变 |
| 整型提升 | 一定触发 | 一定触发 |
| 溢出风险 | `INT_MIN` 取负 → UB | 无 |
| 日常用途 | 取相反数 | 几乎只做类型提升 |
| 浮点 | 支持（`-3.14`） | 支持（`+2.5`，少见） |

## `%` 取余

- **仅整型**；浮点不能用 `%`
- 余数符号与**被除数**一致：`-5 % 2 == -1`

## 易错点

| 坑 | 说明 |
|----|------|
| **除 0** | 未定义行为 |
| **`-INT_MIN`** | 有符号最小值取负 → 溢出 UB |
| 负数取余 | 符号跟被除数，不是数学模运算 |
| 混合类型 | 自动提升，可能意外变 `double` |
| 一元 `+` | 误以为会「变正」；实际只提升类型，不改变符号 |

**HFT**：价格比例用整数除法前先确认是否该用浮点；用 `if (b != 0 && a / b)` 防除零（见 [4.3](./4.3-关系与逻辑运算符.md)）。

## 示例

### 整数除法 vs 浮点除法

```cpp
#include <iostream>

int main() {
    std::cout << 5 / 2 << "\n";      // 2  整数除法，截断小数
    std::cout << 5.0 / 2 << "\n";    // 2.5
    std::cout << 5 / 2.0 << "\n";    // 2.5  int 提升为 double

    double price = 3750.5;
    int lots = 10;
    // std::cout << price / lots;    // 375.05 浮点结果
    std::cout << static_cast<int>(price) / lots << "\n";  // 375 先截断再除
    return 0;
}
```

### 取余 `%`

```cpp
#include <iostream>

int main() {
    std::cout << 10 % 3 << "\n";    // 1
    std::cout << -10 % 3 << "\n";   // -2  符号跟被除数
    std::cout << 10 % -3 << "\n";   // 1

    int total_qty = 25;
    int lot_size = 10;
    int full_lots = total_qty / lot_size;   // 2 整手
    int remainder = total_qty % lot_size;   // 5 零股
    std::cout << full_lots << " " << remainder << "\n";
    return 0;
}
```

### 一元 `+` `-` 与整型提升

```cpp
#include <climits>
#include <iostream>
#include <type_traits>

int main() {
    char c = -10;
    short s = 100;

    std::cout << "-c = " << -c << "\n";   // 10，char 提升 int 再取反
    std::cout << "-s = " << -s << "\n";   // -100

    std::cout << "+c = " << +c << "\n";   // -10，值不变，类型变 int
    std::cout << "+s = " << +s << "\n";   // 100

    std::cout << std::boolalpha
              << std::is_same_v<decltype(c), char> << " "
              << std::is_same_v<decltype(+c), int> << "\n";  // true true

    int min = INT_MIN;
    // int bad = -min;                    // UB，勿对 INT_MIN 一元取负
    (void)min;
    return 0;
}
```

### 一元 `+` 与二元 `+` 区分

```cpp
int a = 3, b = 4;
std::cout << +a + b << "\n";   // 7：+a 是一元（提升），+b 是二元加法
```

### 防除零（配合逻辑短路）

```cpp
int a = 100, b = 0;
if (b != 0 && a / b > 10) {   // b==0 时不会算 a/b
    /* ... */
}
// int bad = a / b;             // ❌ b==0 → UB
```

### 与优先级结合（见 [4.12.1](../4.12-operator-precedence/4.12.1-优先级与结合律表.md)）

```cpp
int a = 2, b = 3;
std::cout << a + b * 4 << "\n";   // 14，先算 b*4 再加 a
std::cout << (a + b) * 4 << "\n"; // 20，括号改变结合
```

# 4.12.1 优先级与结合律表

## 本节讲什么

C++ 运算符优先级（**从高到低**）与结合方向——教材考点 + 日常查表用。

## 速记口诀（粗粒度）

```
作用域 :: 
后缀 ++ -- () [] . ->
前缀 ++ -- ! ~ 一元 * &
乘除模 * / %
加减 + -
移位 << >>
关系 < <= > >=
相等 == !=
位与 &
位异或 ^
位或 |
逻辑与 &&
逻辑或 ||
三元 ?:
赋值 = += -= ...
逗号 ,
```

## 完整优先级表（Primer 核心）

| 优先级 | 运算符 | 名称 | 结合律 |
|:------:|--------|------|--------|
| 1 | `::` | 作用域解析 | 左 |
| 2 | `++` `--` `()` `[]` `.` `->` `typeid` `const_cast` 等 | 后缀 / 成员 / 下标 / 调用 | 左 |
| 3 | `++` `--` `+` `-` `!` `~` `*` `&` `sizeof` `static_cast` 等 | 前缀一元 | **右** |
| 4 | `*` `/` `%` | 乘、除、模 | 左 |
| 5 | `+` `-` | 加、减 | 左 |
| 6 | `<<` `>>` | 移位 | 左 |
| 7 | `<` `<=` `>` `>=` | 关系 | 左 |
| 8 | `==` `!=` | 相等 | 左 |
| 9 | `&` | 按位与 | 左 |
| 10 | `^` | 按位异或 | 左 |
| 11 | `\|` | 按位或 | 左 |
| 12 | `&&` | 逻辑与 | 左 |
| 13 | `\|\|` | 逻辑或 | 左 |
| 14 | `?:` | 条件三元 | **右** |
| 15 | `=` `+=` `-=` `*=` `/=` `%=` `<<=` `>>=` `&=` `^=` `\|=` | 赋值 | **右** |
| 16 | `,` | 逗号 | 左 |

> 同一单元格内运算符优先级相同；不同编译器文档可能把 `<<` 与 `<` 的相对关系再细分，考试以教材表为准。

## 经典例题

### `*` 高于 `+`

```cpp
int a = 2, b = 3;
std::cout << a + b * 4 << std::endl;  // 2 + 12 = 14，不是 20
```

### `.` 高于 `*`

```cpp
// *p.size();     // ❌ 等价 *(p.size())
(*p).size();      // ✅ 或 p->size()
```

### `?:` 低于 `<<`

```cpp
std::cout << (grade < 60 ? "fail" : "pass");  // ✅
```

### 赋值右结合

```cpp
a = b = c = 0;    // a = (b = (c = 0))
```

### 逻辑 vs 位运算

```cpp
a & b && c;       // 先 & 再 &&
a && b || c;      // && 高于 ||
```

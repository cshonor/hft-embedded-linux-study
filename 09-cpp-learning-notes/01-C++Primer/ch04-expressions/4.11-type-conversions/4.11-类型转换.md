# 4.11 类型转换

## 本节讲什么

两种不同类型参与运算、赋值、传参时需统一类型：**隐式转换**（编译器自动）与**显式强制转换**（程序员写明）。对比 C 风格强转与 C++ 四种命名转换的适用场景与风险。

## 要点

- **隐式**：整型提升、算术转换、数组→指针、指针/const 转换、标量→`bool`、赋值截断
- **显式**：优先 `static_cast` / `const_cast` / `reinterpret_cast` / `dynamic_cast`，拒绝 `(type)expr`
- **`static_cast`**：算术互转、`void*`↔指针；**不能**去 `const`
- **`const_cast`**：唯一能增删 `const`/`volatile`；改原本 `const` 对象是 **UB**
- 数组退化例外见 [3.5.3](../ch03-strings-vectors-arrays/3.5-arrays/3.5.3-数组退化与指针.md)

## 阅读建议

1. **优先 1–20**：基础、运算符、异常、性能——高性能开发 80% 的坑在此
2. **21–31**：面向对象架构 + 智能指针底层——做框架、封装组件重点看
3. **32–35**：架构顶层思维——中后期拔高

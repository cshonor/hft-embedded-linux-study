## 笔记

汇编 = 机器语言的人类可读形式。

### 6.2.1 指令

常见操作如 `ADD`、`SUB`。一条 ARM 指令大致：

```
助记符   目的操作数, 源操作数…
例: ADD  R0, R1, R2     ; R0 = R1 + R2
```

### 6.2.2 操作数（数据在哪）

| 位置 | 说明 |
|------|------|
| **寄存器** | 16×32-bit：**R0–R15**（本书 AArch32） |
| **存储器** | 经 LDR/STR 访问（下节） |
| **立即数** | 常数嵌在指令编码里 |

特殊寄存器习惯名（后文函数调用会反复用）：

| 编号 | 别名 | 角色 |
|------|------|------|
| R13 | **SP** | 栈指针 |
| R14 | **LR** | 链接寄存器（返回地址） |
| R15 | **PC** | 程序计数器 |

对照：x86 通用寄存器更少、命名不同（见 6.8）；AArch64 是 X0–X30 等（书 6.7 / 19-ARM64）。

## 笔记

高级语言 → 可执行文件的流水线：

```
源码 ──编译器──► 汇编
      ──汇编器──► 目标文件（机器码 + 重定位）
      ──链接器──► 可执行（合并库、定全局地址）
      ──OS 加载──► 内存中运行
```

### ARM 典型内存映射（直觉）

| 区域 | 内容 |
|------|------|
| **文本段** | 指令（代码） |
| **全局数据** | 已初始化 / BSS 等 |
| **堆** | 动态分配（向上长，常见） |
| **栈** | 局部/调用帧（向下长，常见） |

与 CSAPP / 嵌入式 Linux 进程映像同构；裸机则由链接脚本钉死地址。

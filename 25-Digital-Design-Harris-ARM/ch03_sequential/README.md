# 第3章 时序逻辑设计

> **定位：** Ch1–2 = **组合**（无记忆）；本章 = **时序**（有存储、有时钟、有状态）。  
> **根基：** CPU 寄存器 / 流水线锁存 / 外设 FSM / 同步约束 — 嵌入式 Linux 与 HFT 都用得到。

**一句话：** 锁存→D-FF→寄存器；同步范式；FSM；**setup/hold · 亚稳态 · 同步器**。

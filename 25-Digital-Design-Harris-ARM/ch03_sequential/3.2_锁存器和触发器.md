## 笔记

时序最基础的 **1 bit / 多 bit** 存储。

**底层脉络：** SR 锁存（最基础）→ D 锁存（电平）→ **D 触发器（边沿）**。  
存储本质：**正反馈环路** —— 两反相器首尾互锁电平。

---

### 3.2.0 记忆的物理根源：反相器正反馈

```
IN1 ──NOT──> IN2
IN2 ──NOT──> IN1
```

| 稳态 | 如何自锁 |
|------|----------|
| IN1=0 → IN2=1 → IN1 保持 0 | |
| IN1=1 → IN2=0 → IN1 保持 1 | |

无外部干预则 **永久维持** 该电平 → 数字「记忆」的根源。  
SR 锁存 = 在此反馈环上加 S/R，用来 **改写** 状态。

---

### 3.2.1 SR 锁存器（一切触发器的源头）

**定位：** 时序电路最基础的 1 bit 存储；无时钟 → **异步**时序。  
结构：两与非（或两或非）**交叉耦合正反馈**。

#### 版本 1：与非门 SR（更常用，**低电平有效**）

输入 \(\bar S\)（置位）、\(\bar R\)（复位）；输出 \(Q,\bar Q\)。

| \(\bar S\) | \(\bar R\) | Q（次态） | 功能 |
|------------|------------|-----------|------|
| 1 | 1 | **保持** | 记忆（正反馈维持） |
| 0 | 1 | **1** | Set 置位 |
| 1 | 0 | **0** | Reset 复位 |
| 0 | 0 | **非法** | 禁止：\(Q=\bar Q=1\)；同时回 1 时竞争、次态不定 |

#### 版本 2：或非门 SR（**高电平有效**）

输入 S、R 高有效。

| S | R | Q⁺ | 功能 |
|---|---|-----|------|
| 0 | 0 | Q | 保持 |
| 1 | 0 | 1 | Set |
| 0 | 1 | 0 | Reset |
| 1 | 1 | 非法 | 禁用 |

#### 正反馈（同 3.2.0）

Q 送入另一门输入，另一门输出再送回；稳定在 0/1 后，无外部 S/R 则 **永久锁住**。

#### SR → D 锁存：为何要改

痛点：存在 **非法输入**（与非版 \(\bar S=\bar R=0\)；或非版 S=R=1）。  
改造：加组合逻辑 \(S=D\)，\(R=\overline{D}\)（或等价），合并为单输入 D，消非法组合 → 再加 EN → **D 锁存器**。

#### 链条（理顺）

| 器件 | 要点 |
|------|------|
| **SR 锁存** | 异步、无时钟、有非法态 |
| **D 锁存** | SR 改造 + EN；**电平**触发 |
| **主从 D-FF** | 两级 D 锁存串联；**边沿**触发；CPU 寄存器单元 |

**易考点：** 本质=交叉耦合正反馈；有非法态，实际少直接用；是 D 锁存/各类 FF 的底层原型；无时钟=异步。

---

### 3.2.2 D 锁存器（电平敏感）

结构：SR 锁存 + 组合逻辑 → 单一 **D** + 使能 **EN**。

| EN | 行为 |
|----|------|
| **1（高）** | **透明**：Q 跟随 D；反馈环暂时被旁路 |
| **0（低）** | 切断输入；只剩内部反相器正反馈 → **锁住** 当前 Q |

缺陷：EN 持续高时 D 变 Q 就变，易捕获毛刺。

---

### 3.2.3 D 触发器（边沿敏感 · 主流：主从）

**CPU 标准：** 仅在时钟 **上升/下降沿** 采样并锁存；其余时间 Q 不变。

#### 主从结构（Master-Slave）

两级 D 锁存串联，时钟反相控制：

```
D →【主锁存器】 →【从锁存器】 → Q
     ↑ clk              ↑ ~clk
```

**上升沿触发，分阶段：**

| 阶段 | 主锁存 | 从锁存 | 效果 |
|------|--------|--------|------|
| **clk=低** | EN 有效，采样 D | EN 关，Q **保持** | 主在采、从在守 |
| **clk 上升沿 0→1** | 立刻关闭，**锁死**刚采到的 D | ~clk 变低 → 打开，把主的值传到 Q | **沿上完成一次采样+交接** |
| **clk=高全程** | 持续关；外部 D 抖动进不来 | 开着但输入已固定 → Q 稳 | 隔绝外部 |

**关键结论：**

- D-FF **只在时钟沿瞬间采样一次 D**  
- 采样后靠两级锁存器内部 **正反馈** 维持 → 实现保存  
- 之后外部 D 怎么变，**无法**改内部状态  

#### 形象总结

| 角色 | 比喻 |
|------|------|
| 正反馈环 | 锁住电平的 **保险柜** |
| 主锁存 | clk 低时先把数据 **临时拿进来** |
| 上升沿 | **关上柜门**，数据封存 |
| 从锁存 | 把封存数据送到输出 Q |
| 下一上升沿 | 才允许换新数据 |

#### 锁存 vs 触发器（考试易混）

| | D 锁存器 | D 触发器（主从） |
|--|----------|------------------|
| 控制 | **电平** | **边沿** |
| 高电平时 | 持续 **透明**，环可被改写 | 主关、外 D 进不来 |
| 毛刺 | 透明窗易进 | 采样后隔离 |

**CPU 通用寄存器、栈指针相关状态、流水线级间寄存器 — 全是 D-FF 搭的。**

---

### 3.2.4 寄存器

多个 D-FF 并排 → 多 bit。  
例：**32 位寄存器 = 32 个独立 D-FF**，共同一 clk；沿到 → 一次载入 32 bit，之后保持到下一拍。

### 3.2.5 带使能触发器

`enable=1` 时边沿才更新 → **条件写** 寄存器。

### 3.2.6 带复位触发器

同步/异步复位；开机清空；外设与 CPU 复位电路必备。

### 3.2.8 小结

| | 锁存器 | 触发器 |
|--|--------|--------|
| 触发 | **电平** | **边沿** |
| 透明 | 有（易进毛刺） | 无（隔离毛刺） |
| 工业 CPU | 少作主存状态 | **标准存储器件** |

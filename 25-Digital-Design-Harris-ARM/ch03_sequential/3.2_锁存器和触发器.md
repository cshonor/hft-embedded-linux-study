## 笔记

时序最基础的 **1 bit / 多 bit** 存储。

### 3.2.1 SR 锁存器

两与非（或或非）交叉耦合，存 1 bit。  
缺陷：存在非法输入（如 S=R=1）；实际极少单独用。

### 3.2.2 D 锁存器（电平敏感）

用 D 替代 SR，消非法态。  
**高电平透明：** clk 为高时 D→Q；clk 降后锁住。  
毛刺可在透明窗口串入。

### 3.2.3 D 触发器（边沿敏感 · CPU 标准）

仅在 **时钟上升/下降沿** 采样并锁存；其余时间 Q 不变。  

**CPU 通用寄存器、栈指针相关状态、流水线级间寄存器 — 全是 D-FF 搭的。**

### 3.2.4 寄存器

多个 D-FF 并排 → 多 bit（如 32 位 ARM 寄存器 = 32 个 D-FF）。

### 3.2.5 带使能触发器

`enable=1` 时边沿才更新 → **条件写** 寄存器。

### 3.2.6 带复位触发器

同步/异步复位；开机清空；外设与 CPU 复位电路必备。

### *3.2.7 晶体管级（可选）

CMOS 底层实现；做极致功耗/电路优化再看。

### 3.2.8 小结

| | 锁存器 | 触发器 |
|--|--------|--------|
| 触发 | **电平** | **边沿** |
| 透明 | 有（易进毛刺） | 无（隔离毛刺） |
| 工业 CPU | 少作主存状态 | **标准存储器件** |

## 笔记

### Module（模块）

硬件最小封装单元，类似 C 的函数/结构体边界；模块之间 **连线** 完成信号交互。

### HDL 两类用途

| | 仿真 Simulation | 综合 Synthesis |
|--|-----------------|----------------|
| 干什么 | 模拟运行、找逻辑/时序 bug | 代码 → 门/触发器网表 |
| 产出 | 波形 / 报告 | 可烧 FPGA 或流片 |

### 最大误区：Verilog 不是 C

| | C | Verilog |
|--|---|---------|
| 本质 | **串行**软件指令 | **并行**描述硬件结构/行为 |
| 赋值 | 按语句顺序执行 | 多处赋值可 **同时**「发生」（硬件并发） |

写 Verilog 时始终想：**这段代码综合成什么电路？**

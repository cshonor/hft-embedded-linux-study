## 笔记

处理带小数的有理数：硬件里两种主方案。

### 5.3.1 定点数

- 二进制小数点位置**固定**（隐含）：前面若干位整数、后面若干位小数。  
- 加减像整数；乘除要盯对位与溢出。  
- **HFT / 金融：** 价格、数量常用定点（避免浮点舍入与非确定性）。

### 5.3.2 浮点数（IEEE 754）

科学计数法硬件化：

| 字段 | 作用 |
|------|------|
| 符号位 | 正负 |
| **阶码（指数）** | 常带 **偏置（bias）** 存储 |
| 尾数（有效数字） | 规格化表示 |

| 格式 | 位宽 |
|------|------|
| 单精度 | 32 |
| 双精度 | 64 |

特殊情况（硬件与软件都要认）：

| 编码 | 含义 |
|------|------|
| 全 0 等 | **±0** |
| 阶码全 1、尾数 0 | **±∞** |
| 阶码全 1、尾数非 0 | **NaN** |

## 笔记

存大量数据的规整阵列；第7章寄存器堆、第8章层次存储的硬件底座。

### 5.5.1 基本结构

| 概念 | 作用 |
|------|------|
| **位单元阵列** | 每位一个存储单元排成矩阵 |
| **字线 wordline** | 选中某一行（读/写哪一行） |
| **位线 bitline** | 该行各位数据进出 |
| **多端口** | 多套字线/位线 → 并发读写（寄存器堆常用） |

### 5.5.2 DRAM（动态 RAM）

- 典型 **1T1C**（1 晶体管 + 1 电容）存 1 bit。  
- **密度高、成本低、较慢**；电容漏电 → 必须**定期刷新**。  
- 用途：主存、大容量缓冲。

### 5.5.3 SRAM（静态 RAM）

- 交叉耦合反相器，常见 **6T** 单元。  
- **快、不刷新**；面积大、贵。  
- 用途：Cache、片上缓冲、微处理器内 **寄存器文件（Register file）**。

| | DRAM | SRAM |
|--|------|------|
| 密度 | 高 | 低 |
| 速度 | 慢 | 快 |
| 刷新 | 要 | 不要 |
| 典型 | 主存 | 寄存器堆 / Cache |

### 5.5.6 ROM 与演进

- **非易失**：断电不丢。  
- 现代多可写：PROM → EEPROM → **Flash**（嵌入式固件、启动代码）。  

（书中中间小节若谈 CAM/多端口细节，按需补读；口述核心是 DRAM/SRAM/ROM 三分法。）

### 阵列 = 查找表（LUT）

存储器不只存「数据」：把地址当输入、读出当输出 → **任意组合逻辑**（FPGA 的 LUT 即此思路）。

### 对你

| 方向 | 带走 |
|------|------|
| **嵌入式** | SRAM 片上、Flash 存镜像；读手册时对照端口/刷新/擦写寿命 |
| **HFT** | 热路径优先片上 SRAM；主存 DRAM 延迟是另一量级 |
| **后章** | Ch7 用多端口 SRAM 作寄存器堆；Ch8 展开 Cache 层次 |

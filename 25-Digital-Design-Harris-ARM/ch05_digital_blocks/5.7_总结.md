## 笔记

### 本章带走

1. **算术：** 半加/全加 → CPA 族；补码减法；ALU + **NZCV**；移位；乘除很贵。  
2. **数制：** 定点 vs IEEE 浮点（含 0 / ∞ / NaN）。  
3. **时序积木：** 计数器、移位寄存器（含扫描链）。  
4. **存储：** wordline/bitline；**DRAM 刷新 vs SRAM 寄存器堆**；ROM/Flash；阵列可当 LUT。  
5. **逻辑阵列：** PLA；**FPGA = LUT + FF + 布线**（综合落地）。

### 桥梁角色

```
门/FF/Verilog →【本章黑盒】→ Ch6 ISA 语义 → Ch7 用黑盒搭微处理器
```

折中贯穿始终：行波 vs 前缀加法器、DRAM vs SRAM、硬连线 vs FPGA——同一功能不同面积/延迟/功耗点。

### 对你两条线

| 方向 | 价值 |
|------|------|
| **嵌入式** | 定时器/UART 移位、片上 SRAM、Flash；外设背后就是这些块 |
| **HFT** | 关键路径加法/移位延迟；片上 SRAM；FPGA LUT 流水是加速入口 |
| **前后** | 依赖 Ch1–4；服务 Ch7 ALU/寄存器堆/存储器端口 |

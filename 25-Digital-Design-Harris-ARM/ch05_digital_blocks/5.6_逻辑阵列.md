## 笔记

可重配置逻辑：把「写死门」换成可编程阵列。

### 5.6.1 PLA（可编程逻辑阵列）

- 以 **与–或两级** 实现组合逻辑：AND 阵列 + OR 阵列。  
- 规整、适合固定 SOP；灵活度不如现代 FPGA。

### 5.6.2 FPGA（现场可编程门阵列）

现代可重构硬件平台；**HFT 加速卡、原型验证**主战场。

| 积木 | 作用 |
|------|------|
| **LE / CLB** | 可配置逻辑元件 |
| **LUT** | 用小 SRAM 实现任意小组合函数（见 5.5 LUT 思路） |
| **触发器** | 存状态、切流水 |
| **可编程布线** | 把 LE 连成大电路 |

综合链路（接 Ch4）：

```
Verilog → 综合映射到 LUT/FF → 布局布线 → 比特流上板
```

嵌入式：部分 SoC 带 FPGA 软核/协处理器；读厂商 IP 时心里要有「这最终落在 LUT+布线」的图景。

# 第7章 微结构

> **定位：** 全书核心桥 —— 用 Ch2/3/5 的门、寄存器、ALU、存储器 + Ch3 FSM，**实现 Ch6 的 ARM ISA**。  
> **对照：** ↔ [CSAPP Ch4](../../01-CSAPP-3rd/chapter-04-processor-architecture/README.md)（Y86 SEQ/PIPE；概念同构，ISA 不同）。

**三种方案折中：**

| 微结构 | 直觉 | 代价 |
|--------|------|------|
| **单周期** | 一拍一条 | CPI=1，但 clk 被最慢指令拖死 |
| **多周期** | 一指令多拍，硬件复用 | 硬件省，clk 短；CPI>1 |
| **流水线** | 多指令重叠 | 吞吐高；**冲突**要转发/停顿/冲刷 |

**对你两条线：**

| 路线 | 作用 |
|------|------|
| **嵌入式** | 理解 MCU 流水/停顿；调试「指令已取但结果未好」 |
| **HFT** | 7.5 冲突 + 7.7 预测/超标量/乱序 = 现代 x86 延迟直觉入口 |

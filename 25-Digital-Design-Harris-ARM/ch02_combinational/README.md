# 第2章 组合逻辑设计

> **定位：** 用 Ch1 的 0/1 与逻辑门，搭出有实际功能的电路。  
> **核心：** **组合逻辑** — 输出 **只取决于当前输入**，无记忆（对照 Ch3 时序）。

**一句话：** 真值表/布尔式 → 化简 → 门级图 → MUX/译码器；还要管 **延迟与毛刺**。

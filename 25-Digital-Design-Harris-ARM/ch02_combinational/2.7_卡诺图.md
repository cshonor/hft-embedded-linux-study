## 笔记

代数化简易错；**卡诺图 (K-map)** 提供图形化化简。  
格子编号 = **最小项下标**；圈 1 / 圈 0 与 SOP / POS 关系 → [2.2](./2.2_布尔表达式.md)。  
底层仍是 **布尔代数恒等变换**（与手工套公式等价）→ [2.3](./2.3_布尔代数.md)。

### 2.7.1–2.7.2 画圈与化简

**路线 A — 圈 1（最小项 → 与或式）**

- 真值表填入方格（相邻格只差一个变量）  
- 把相邻的 **1** 圈成尽量大的、边长为 **2 的整数次幂** 的矩形  
- 每圈读出一个与项，圈与圈 **相或** → 最简 SOP  

**路线 B — 圈 0（最大项 → 或与式）**

- 圈相邻的 **0**（同样 2 的幂次大矩形）  
- 每圈读出一个 **或项**（合并最大项，消变量）  
- 圈与圈之间 **相与**（不是相或！）→ 最简 POS  

合并公式对偶、\(M_2\cdot M_3\) 例题 → [2.2.5](./2.2_布尔表达式.md)。

### 2.7.3 无关项

某些输入组合永不出现，或输出「不关心」（标 X）。  
卡诺图里可把 X 当 0 或 1，把圈画得更大 → 进一步简化。

### 2.7.4 小结

会画图、会圈、会读式即可；复杂题跟书例。

### 2.3.0 知识链：布尔代数站在哪一层？

#### 1. 布尔代数是什么

一套代数系统（课程里用 **二值** 版）：

| | |
|--|--|
| 取值 | 只有 **0、1** |
| 基本运算 | **与(·)、或(+)、非** |
| 定律 | 交换、结合、分配、重叠、互补 \(A+\overline{A}=1\)、\(A\cdot\overline{A}=0\)、德摩根…… |

> 所有逻辑式运算、变换，遵守的规则 = **布尔代数公理**。  
> 广义布尔代数可不只两值；数电 / 数字逻辑全程只用二值。

#### 2. 最小项 / 最大项的角色

它们是布尔代数里两种 **特殊标准形式**（不是另一套理论）：

| | 形式 | 布尔运算 |
|--|------|----------|
| 最小项 | 特殊与项 | \(F=\sum m_i\) = 若干最小项 **或**（布尔加法） |
| 最大项 | 特殊或项 | \(F=\prod M_i\) = 若干最大项 **与**（布尔乘法） |

细节 → [2.2](./2.2_布尔表达式.md)。

#### 3. 合并规则的根源 = 布尔代数

| 合并 | 公式 | 来源 |
|------|------|------|
| 最小项 | \(XY+X\overline{Y}=X\) | 分配律 + 互补律 \(Y+\overline{Y}=1\) |
| 最大项 | \((X+Y)(X+\overline{Y})=X\) | 分配律 + 互补律 \(Y\cdot\overline{Y}=0\) |

不是凭空规定，是公理 **直接推出**。

#### 4. 卡诺图本质

**图形化工具**，快速完成与代数化简 **等价** 的恒等变换；  
圈 1 / 圈 0 底层仍是布尔代数。→ [2.7](./2.7_卡诺图.md)

#### 5. 完整链路

```
真值表
  → 最小项之和 / 最大项之积（标准范式）
  → 布尔公式化简  或  卡诺图可视化化简（二者等价）
  → 最简布尔表达式
  → 画出逻辑门电路（SOP=与+或；POS=或+与）→ [2.4](./2.4_从逻辑到门.md)
  → 硬件实现（分立芯片 / FPGA）
```

#### 6. 三者区分（别混）

| 概念 | 是什么 |
|------|--------|
| **布尔代数** | 理论工具（公式、运算规则） |
| **最小项 / 最大项 / 范式** | 布尔代数里的 **标准表达形式** |
| **卡诺图** | 化简布尔式的 **图形手段** |

> **一句话：** 最小项、最大项互转、合并、卡诺化简，全部是在做 **布尔代数的恒等变形**。

### 2.3.1–2.3.3 公理与定理（要会用，不必全背）

**单变量：** 同一性、零元、重叠、回旋、互补等。  

**多变量：** 交换律、结合律、分配律、吸收律、合并律；以及极重要的 **德·摩根定理**（推气泡 / NAND·NOR 实现的根基）。

### 2.3.4 统一证明：完全归纳法

用真值表 **穷举** 所有变量取值，证明布尔恒等式。

### 2.3.5 等式化简

逐步应用定理，压到 **最少蕴涵项** 的最小式（例题跟书走）。  
与卡诺对照：同一函数，公式推一步 = 图上圈一步，结果应一致。

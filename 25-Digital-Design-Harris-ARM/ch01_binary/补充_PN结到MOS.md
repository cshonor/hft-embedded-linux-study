## 笔记

只讲 **NMOS**；**PMOS 全部反过来**（文末对照）。  
（「珊极」= **栅极 G**，Gate。）

### 纠正核心误区（先看这个）

| 错 | 对 |
|----|----|
| 加栅压 = **打通 / 毁掉 / 让 PN结失效** | 加栅压 = 在硅表面 **新建一条导电沟道**，**绕开** 两个 PN结 |
| 导通电流走过 PN结 | 电流路径：**源 N → 表面沟道 N → 漏 N**，**不经过衬底、不经过两个 PN结** |
| PN结正向导通才是「开」 | 正常工作时两结必须保持 **反向截止**；正向导通 = **衬底漏电**（失控、发热），要极力避免 |

> **一句话：** 加栅压 ≠ 打通 PN结；加栅压 = **旁路** 两个反向 PN结，给源漏另开一条独立通道。

---

### 前置复习（2 个关键点）

1. **PN结：** P 和 N 贴一起 → **单向导电**；反向断路；中间有耗尽层。详见 [补充_PN结](./补充_PN结.md)。  
2. **NMOS 骨架：** 整块 **P型硅衬底**；左右各掺一块 **N型** —— 左 **源极 S**、右 **漏极 D**。

---

### 第一步：先搭出 2 个背对背 PN结（不加栅极 = 天生断开）

| 结构 | 形成 |
|------|------|
| P衬底 + 左 N（S） | **源–衬底** PN结 |
| P衬底 + 右 N（D） | **漏–衬底** PN结 |

两个 PN结 **背对背**：

- 不管给 S、D 加什么电压，经衬底走总有一结 **反向截止** → 走不了  
- S 与 D 中间是 P 硅，**没有表面沟道** → **管子关断**

**大白话：** 左右两块 N（S/D），中间一堵 P 墙；墙左右各一道单向门（PN结）。这两道门 **不是** 我们用来通电流的开关——正常工作里它们要一直关着（反向）。真正的「开关」是后面栅极造出来的 **表面旁路通道**。

---

### 第二步：加绝缘栅极（MOS 核心创新）

在两块 N 区中间的 P 硅表面，从上到下三层：

| 层 | 是什么 |
|----|--------|
| 1 | 金属 / 多晶硅 = **栅极 G** |
| 2 | 极薄 **SiO₂** = **绝缘玻璃层**（关键：不导电） |
| 3 | 底层 **P型衬底** |

重点：栅极和底下硅 **完全绝缘**。  
栅压 **不会击穿、不会导通 PN结**，只产生 **垂直电场**。

---

### 第三步：栅压真正干了什么（造沟道，不是打通结）

#### 场景 1：VGS = 0（关断）

无电场 → 无表面沟道；S、D 只被 P 衬底隔开；两结反向截止 → **S–D 不通**（也没有旁路）。

#### 场景 2：VGS > 开启电压（导通）

栅极带 **正电**，向下电场：

1. **排斥** 栅下空穴 → 形成耗尽区  
2. **吸引** 电子到 SiO₂ 下方表面  

电子够多 → 表面薄层 **从 P 反转成 N** = **反型层 / N型导电沟道**（全新薄层，不是把旧 PN结「捅开」）。

---

### 第四步：电流怎么走

```
源极 N  ──►  表面 N沟道  ──►  漏极 N
              （旁路）
         ╳ 不走衬底 / 不走两个 PN结
```

- 两结此时依旧 **反向截止**，结上 **无正常工作电流**  
- 若强行让源–衬底结 **正向导通**：电子灌进 P衬底 = **漏电**，管子失控 —— **不是** MOS 正常开关

---

### 核心总结

| # | 记住 |
|---|------|
| 1 | MOS 自带源–衬底、漏–衬底两个 PN结；正常时它们 **反向截止** |
| 2 | 栅极+SiO₂ 只产电场，**不导通、不击穿 PN结** |
| 3 | 够高的栅压 → **新建表面 N沟道**，旁路掉两结，连通 S–D |
| 4 | 无沟道 = 断；有沟道 = 通；**通 ≠ PN结正向导通** |

---

### PMOS 极简对照

| NMOS | PMOS |
|------|------|
| P衬底 + N型 S/D；两结反向截止 | N衬底 + P型 S/D；两结同样反向截止 |
| 栅极 **正/高** → 吸电子 → **N沟道旁路** | 栅极 **负/低** → 吸空穴 → **P沟道旁路** |
| 电流走表面沟道，**不走 PN结** | 同样绕开 PN结；结全程不（正向）导通 |

数字电路：nMOS 爱拉 **0**，pMOS 爱拉 **1**；互补 = CMOS（见 [1.7](./1.7_CMOS晶体管.md)）。

---

### 一句话串起

两块 N + 一块 P → 两道 **反向** PN结挡着经衬底的路；  
绝缘栅隔空加电场 → 表面 **新建** N沟道；  
电流走 **源→沟道→漏**，**旁路** 两结（结仍截止）→ 源漏导通。  
**这就是 NMOS。**

### 先收起 PN结：栅极是另一套物理

前面 [补充_PN结](./补充_PN结.md) 的正偏/反偏/扩散漂移，**用来理解管子里那两个结为什么一直反向挡路**。  
**栅极怎么把开关打开，和「给 PN结加正偏」完全不是一回事。**

#### 核心一句话

> MOS 的 **栅极不参与 PN结导通 / 击穿**！  
> **栅极 + SiO₂ 氧化层 = 平板电容器**。  
> 给栅极通电 = 给电容极板充电 → 垂直电场 → **改变硅表面载流子浓度**（隔空调控）。

| | PN结二极管 | MOS 栅极 |
|--|------------|----------|
| 电压加在哪 | **直接加在半导体结两端** | 加在 **电容极板**上，**不施加在 PN结两端** |
| 干什么 | 改耗尽层厚薄；载流子**越过结面**导电 | 在硅表面**新建**一层导电沟道 |
| 有没有「结电流」 | 正偏时有大电流过结 | 栅极几乎**无直流电流**（绝缘层挡住） |
| 控制类型 | 结两端偏置 | **电压控制**（场效应）；对比 BJT 的电流控制 |

👉 这就是 MOS 和 BJT 最大区别之一：MOS 是 **电压控制型**，靠电场，不靠往结里灌电流。

---

### 结构回顾（N沟道增强型 NMOS）

- 衬底 = **P型硅**；左右两块 N = **源 S、漏 D**  
- 天然两个 PN结：S–P、D–P，**背对背** → 经衬底走不通 S–D  

---

### 四个关键错误（先纠再往下）

接法「S 接负极 / 低电位、D 接正极 / 高电位」本身 **对**；错在中间推理。

| # | 错 | 对 |
|---|----|----|
| 1 | S–D 之间电子靠 **扩散** 流动 | 是电压电场拉着跑 = **漂移**。扩散是浓度差（PN结刚贴合那种），别混。见 [补充_PN结](./补充_PN结.md) 扩散/漂移 |
| 2 | 源漏压差 **造出沟道** | **沟道只由 VGS 创造**。VDS 只管拉电子；栅极不加压，VDS 再大 Id 仍为 0 |
| 3 | 整块 **P衬底本体** 在导电 | P衬底电子极少；栅极把深处微量电子吸到 **表面薄层**（沟道），电子 **只在这层跑**，不深入大块 P衬底 |
| 4 | 两个 PN结参与导电 / **VDS 击穿、打通结** | 见下一小节：**VDS 绝不击穿结**；电流走表层天桥 |

#### 最大误区：VDS 会击穿 / 导通两个 PN结？→ **绝对不会（正常工作）**

结构：P衬底，S/D 都是 N。

| 结 | 正常偏置（衬底常与源极接一起，VS = V衬底） |
|----|---------------------------------------------|
| **源–衬底** N(S)–P | S = 衬底 → **零偏**，无电流 |
| **漏–衬底** N(D)–P | D > 衬底 → **反向偏置**，截止 |

→ 两结全程截止，**没有载流子穿过 PN结**，不存在「击穿、打通」当正常通路。

**电子走的路：完全绕开 PN结**

```
源极 N区  →  表层电子沟道（悬浮在 P 表面的薄层）  →  漏极 N区
                    ↑ 只在这层
              ╳ 不往下进 P衬底 → 碰不到两个结
```

**天桥比喻：**

- 楼下两道铁门（PN结）一直锁死  
- 楼顶天桥（沟道）连左右；人全走天桥，**不必开门、砸门**  
- **VDS** = 天桥两端高低落差，让人在桥上跑 —— **和楼下铁门无关**  
- **VGS** = 搭出天桥  

| 错 | 对 |
|----|----|
| VDS 压差会击穿 PN结 | 击穿/结正向导通 = **故障漏电**，发热损坏；不是正常开关电流 |
| 电子穿过 PN结流动 | 只走表层反型沟道，不进衬底，不经结界面 |
| 三件事搅在一起 | VGS 搭天桥；VDS 给桥两端电势差；PN结底层隔离墙，**全程关闭** |

**什么时候 PN结才会导通？**

故意把衬底电位拉得比源极还低 → S–P 结 **正向偏置** → **衬底漏电流**。  
这是**无用的损耗**，不是 MOS 想要的工作电流；电路设计会避免（体二极管误导通等场景另说）。

#### 回答最乱的两句

1. 「源漏电压差的扩散运动产生沟道？」→ **不对。** 沟道 = 栅极电场造的；VDS 只拉动沟道里**已有**电子。无栅压 → 无通道。  
2. 「衬底电子参与运动？」→ **半对半错。** 电子**原本藏在衬底深处**，被栅极吸到表面成沟道；之后 **只在表层移动**，不进下方大块 P、不穿两侧 PN结。

#### 正确流程（三步）

1. **接法：** S=低（常 0V），D=高 → 有 VDS（仅准备「拉力」）  
2. **前提：** VGS > Vth → 垂直电场推开表面空穴、吸电子到氧化层下 → **电子沟道**出现（只有栅极能完成）  
3. **VDS 干活：** 横向电场 D→S；电子受力反向 → **源 N 供电子 → 表层沟道 → 漏收集**（全程漂移）

#### 土 / 水坑 / 水膜比喻

1. P衬底 = 一块土；S、D = 左右水坑（N区，电子仓库）  
2. 栅极通电 = 在土表面刷一层 **导电水膜**（电子沟道，**只有栅极能刷出来**）  
3. S 低、D 高 = 两水坑水位差（VDS）  
4. 水位差让水膜里的水从左坑（源）流到右坑（漏）  
5. 水坑与土的隔离墙（PN结）**一直不透水** → 水只走表面水膜，不渗进土里  

#### 一句话记死

```
VGS → 造出导电通道（沟道）
VDS → 拉动通道里的电子流动（漂移），产生电流
```

### 只讲电子路径（一步步，尽量不碰 PN结）

前提：NMOS，P衬底；**S(N)=0V，D(N)=高压**；栅极已够压，表面已有 N型电子沟道。衬底与 S 同接 0V。

#### 1. 三块区域的电位

| 区域 | 电位直觉 |
|------|----------|
| 源 S | **0V（最低）** |
| 表层电子沟道 | 靠 S 处 ≈0V，靠 D 处 ≈高压（沿沟道有压降） |
| 漏 D | **高电位**（如 5V） |
| 底下整块 P衬底 | 与 S 同地 → **也是 0V** |

#### 2. 电场与电子受力

- 电位高→低：沟道内 **横向电场方向 D → S**  
- 电子带负电：受力与电场相反 → **电子受力方向 S → D**

#### 3. 电子从哪来？

源极 S = 大块 N型硅 = **电子仓库**（海量自由电子）。  
S 电位最低，没有电场把电子往衬底赶；表层沟道直接贴着 S 的 N区。  

→ 电子 **横向** 从 S 的 N区钻进表面沟道。  
⚠️ **不是往下扎进 P衬底** —— 横向进表层，碰不到下面的结界面。

#### 4. 在沟道里怎么跑？

```
源极 N区(S) → 横向进入表面薄层 N沟道 → 沿硅表面移向漏 → 进入漏极 N区(D)
```

全程只在 **硅最表层几纳米**，在衬底**上方**，不向下进 P。

#### 5. 为什么碰不到 / 不经过下方的结？（只为打消疑虑）

PN结在沟道**下方**，沟道在衬底**表面**，两层分开：

- 电子从 S **横着进表层**，不往下跨 S–P 界面  
- 到 D 后被外电路抽走，不往下钻进衬底  

衬底 0V、漏高压 → 漏–衬底结本来就是**反向**，正好帮着把电子「锁」在表层沟道里流（正常工作要的隔离）。

#### 6. 水管比喻（只盯水流）

1. 底下泥土 = P衬底；泥土与左右水池之间的隔水墙 = 一直密封  
2. 左右水池 = S、D（装满电子）；左池 0V，右池高压  
3. 栅极通电 = 泥土**最表面**铺一层连通左右的薄水管 = N沟道  
4. 右池高压拉力 → 水从左池 **横向流进表层水管** → 到右池  
5. 水全程只走表层水管，**不往下穿墙进泥土**

#### 7. 电子完整运动链条

1. 栅极加电压 → 表面拉出连通 S、D 的电子沟道  
2. S=0V，仓库供电子 → **横向**进入表层沟道  
3. D 高压形成横向电场 → 持续拉电子向漏极  
4. 电子全部在硅表层移动 → **不进衬底、不穿任何 PN结**  
5. 到漏极后流入外部电源正极 → 完成回路  

#### 两大错误再钉（路径里没有这两处）

| # | 错 | 对 |
|---|----|----|
| 1 | 电子走 **栅极那块金属板** | 栅极下有 **SiO₂ 绝缘层**，电荷过不去；栅极金属 **只产电场**，无直流电子穿过栅极 |
| 2 | 沟道是「金属通路」 | 沟道是 **硅片表面薄薄一层半导体**（反型硅），不是金属 |

#### 器件分层（从上到下，看清电子在哪一层）

| 层 | 是什么 | 通不通电子 |
|----|--------|------------|
| 最上 | 金属/多晶硅 **栅极** + **SiO₂** | 绝缘，**不通**；只造垂直电场 |
| 中间 | **硅表面反型 N沟道** | **唯一工作通路**（纯硅薄层） |
| 底下 | 整块 **P衬底** | 正常工作电流 **不走这里** |
| 左右 | N型 **S、D**，搭在表层沟道两头 | 电子仓库 / 收集端 |

#### 电子完整路径（再写一遍，避开结与栅）

S=0V，D=高压；栅压已够：

1. S 大块 N硅里无数自由电子  
2. 电子 **横向水平** 进入挨着它的表层 N沟道（只在硅表面，不往下钻 P）  
3. D 高压横向电场 → 电子沿硅表面沟道从 S 侧移向 D  
4. 进 D 的 N区 → 外部电源 → 回路  

**为什么不碰 PN结：** 结是 **垂直向下** 的结构（N区 ↓ P衬底）；电子只在 **水平表层** 走，不垂直向下进 P → 不跨 N–P 界面。下方两结反向，反而像屏障，把电子 **困在表层横向流**。

#### 浅水沟比喻（含栅极铁板）

- 底层土地 = P衬底；土地与左右水坑之间的 **地下隔墙** = PN结（向下，封死）  
- 左右水坑 = S、D（电子仓库）  
- **地面浅水沟** = 硅表面 N沟道（电子唯一通道）  
- **栅极金属** = 悬在水沟上方的铁板，隔着一层玻璃：只吸引水形成水沟，**水绝对碰不到铁板**  

水流只在地面浅沟横向流，不钻地下隔墙进泥土，也碰不到上方铁板。

> **一句话：** 电子 **不从 PN结走**，也 **不从栅极金属板走**；只走硅片表面、连接源漏的 **半导体导电沟道**。

#### 核心矛盾拆开：「沟道贴在 P 上 = PN结导通/击穿？」→ **错**

你以为：表层 N沟道贴着 P衬底 → 接触面必然是 PN结 → 等于击穿导通。  
其实要分清 **两种完全不同的 N–P 界面**。

##### ① 源/漏实体 N区 ↔ 衬底 P（真正的 PN结）

左右两块 **厚重、预先掺杂** 的 N硅，**垂直向下** 与 P衬底贴合 → **传统 PN结**：有固定离子耗尽层，正常工作反向截止，电子不能往下走。  
界面是 **整块厚 N ↔ 厚 P**，掺杂固定，天然耗尽层阻挡。

##### ② 表层反型 N沟道 ↔ 下方 P衬底（不是能导电的正向 PN结）

沟道 **不是** 一块独立、预先掺好的 N硅！  
它是 **P硅表面被强电场临时吸来的一层电子薄层** —— 临时电荷层，不是扩散平衡造出来的体 PN结。

| 关键点 | 记住 |
|------|------|
| 厚度 | 只在最表面几纳米；下方立刻仍是原本 P型 |
| 靠什么维持 | **栅极垂直电场**，不是扩散形成的平衡 PN结 |
| 算不算「导通」 | **界面存在 N–P 电荷分层 ≠ PN结导通**。导通的核心是 **载流子跨界面流动**；只贴着、不跨层移动 → 不算导通 |

##### 为什么这个表层界面不会正向导通、不会击穿？

正常：衬底与源短接，同电位（0V）。

1. 沟道电子被栅极正电 **按在表面**；往下进 P 会被空穴复合，但垂直向下运动被电场压制  
2. **没有正向偏置条件** 去驱动电子垂直向下穿界面（体结导通要 N侧相对 P侧够条件抵消内建场；这里没有「垂直过结」的工作电流）  
3. 所以：有分层，**无跨层电流** → 不叫导通、不叫击穿  

##### 沟道为何能连 S–D，却不打通下方结？

```
S（实体掺杂 N） ↔ 表层电子沟道（临时 N电荷） ↔ D（实体掺杂 N）
         ↑ 全程电子导电，全程水平
垂直向下才是实体 N/P 的 PN结 → 电子不往垂直方向走
```

**平地 + 深坑 + 水膜：**

- 平地 = P衬底；左右深坑 = S/D 实体 N（坑壁往下 = 真正 PN结，墙封死）  
- 地面薄水膜 = N沟道，左右连通两坑  
- 水只在地面横向流，不垂直钻透坑壁进土  
- 栅极电场 ≈ 盖子，把水压在地表  

##### 一句话钉死误区

> 「只要 N型电荷挨着 P硅，就一定会导通 PN结、必然击穿」→ **错。**  
> 实体 N 垂直对接 P = 稳定 PN结（有耗尽层阻挡）；  
> 电场临时吸附在 P表面的电子薄层 = 表层电荷，**无垂直跨界面载流子流** → 不算导通、不存在工作击穿。  
> 电子只水平走表层沟道。

#### 精准定位：MOS 里仅有的两个正式 PN结

**结论：大体方向对，但有一个关键例外，不能 100% 说「凡 N 贴 P 都是 PN结」。**

##### 你对的部分

源 N、漏 N = 高温离子掺杂做出的 **实体 N型半导体**。  
只要是 **实体掺杂 N硅** 与 **实体掺杂 P衬底** 直接贴合的交界面 → **全部形成 PN结**：

- 源区 **底面、侧面** 挨着 P → 全是 PN结  
- 漏区 **底面、侧面** 挨着 P → 全是 PN结  

| # | 位置 |
|---|------|
| 1 | **源极 S ↔ P衬底**（N 源区底部、侧壁） |
| 2 | **漏极 D ↔ P衬底**（N 漏区底部、侧壁） |

##### 关键例外（最易混）

栅下、源漏中间的 P 表面那层 **电子反型沟道**：

- 只是电场 **临时** 吸附的表面电荷，**不是** 提前掺杂的实体 N硅  
- 撤掉栅压 → 电子立刻消失；无永久掺杂离子  
- 与下方 P 的接触面 → **不算工艺制造出来的 PN结**

真正 PN结定义直觉：两种 **不同掺杂的半导体材料** 结合成的 **稳定界面**，有 **永久耗尽层**。  
沟道 = 临时电荷层，不满足 → 不算 PN结。

##### 一句话修正

> **实体掺杂的 N区（源、漏）和 P衬底接触的全部面 → 都会形成 PN结；**  
> **电场临时产生的表层电子沟道 ↔ P衬底 → 不算 PN结。**

##### 结合电流再串

- 实体 N–P 界面（正式结）多在 **垂直向下**，全程反向截止，电子不往下穿  
- 临时电子沟道 **横向** 连通源 N、漏 N，电子只在表层水平流，避开所有正式 PN结  

**直观分层：** 左右实心 N块扎进 P（接触面=两正式结）；中间 P 表面栅电场吸出临时电子薄层（水平连 S–D，不算结）。

#### 空间一眼看懂：电子完全不用穿过那两个 PN结

**结论：完全不用穿过那两个 PN结。** 错在默认「S→必须进 P→再沟道→再穿 P→D」。

##### 分层 + 分左右

| 位置 | 是什么 |
|------|--------|
| 左实体 N块 **S** | 埋进 P；**底面、侧壁** ↔ P = **S–P PN结（垂直朝下）** |
| 右实体 N块 **D** | 同理；**底面、侧壁** ↔ P = **D–P PN结（垂直朝下）** |
| S–D **中间顶面** | 栅极隔氧化层盖住；VGS 够 → P **最顶部表层** 吸出电子 = N沟道（**水平铺在顶面**） |
| 沟道两头 | 直接搭在 S、D 两块 N 的 **顶部边缘**（不是从底下绕） |

##### 电子完整路径（只走顶部水平面）

```
S（N区顶部） → 水平向右走表层 N沟道 → D（N区顶部）
```

- S 的 **底部/侧壁** = 朝下的 PN结（电子不去）  
- S 的 **最上边表面** = 横向连中间沟道（电子走这里）  
- 全程 **硅最表面横向移动，全程不向下**

##### 房屋类比

| | |
|--|--|
| 一楼地面 | P衬底 |
| 左右地下室 | S、D |
| 地下室墙壁 / 与地面隔墙 | PN结（垂直向下） |
| 顶层连通左右门口的走廊 | N沟道（水平） |

人（电子）只走顶层走廊：左地下室门口 → 走廊 → 右地下室门口；**不下楼、不穿地下室墙壁**。

##### 纠正错误闭环

| 错的默认 | 现实 |
|----------|------|
| S → 必须进 P衬底 → 再进沟道 → 再穿 P → D | **S 顶部直接连沟道，沟道直接连 D 顶部** |
| | **全程不进入 P衬底本体**；P 只是承载沟道的基底 |

##### 电位辅助（衬底与 S 同接 0V）

1. S–P 同电位 → 结 **零偏**，无垂直电流  
2. D 高压、P=0 → D–P **反偏**，挡电子垂直流入衬底  

两结共同作用：**锁住电子别往下跑，只能留在顶部表层横向流**。

> **一句话：** S/D–衬底 PN结是 **垂直向下** 的屏障；导电沟道是 **水平铺在顶部** 的通道；电子只横向走顶部，绝不向下穿过垂直 PN结。

#### 定稿：顶面无结，侧壁·底面才有结（你现在理解完全正确）

1. **源、漏 = 嵌入 P衬底的 N型方块（可想成圆柱体）**
   - **侧壁、底面：** N 直接贴 P → **全部形成 PN结**  
   - **顶面：** N硅上表面 **没有被 P 包围**，裸露在硅片最上层 → **不存在 PN结**

2. **N沟道** 铺在 P 最顶面；左右两端直接搭在 S、D 的 **顶面**（无 PN结的那一面）

```
通路：S顶面 → 表层 N沟道 → D顶面
```

整条路径在硅最表面，**不经过侧壁、底面那些 PN结界面**。

3. **两条完全分开的接触面**

| 方向 | 接触 | 有没有 PN结 | 作用 |
|------|------|-------------|------|
| **垂直向下** | N(S/D) ↔ P衬底 | **有** | 挡电子往下钻 |
| **水平顶面** | N(S/D 顶面) ↔ 表层电子沟道 | **无** | 同类型电子导电，无结阻隔 |

> **极简总结：** S、D 只有埋进 P 里的 **侧面、底部** 才有 PN结；**露在外面的顶部没有 PN结**，直接和中间表层沟道连通；电子只走这条顶部水平通道，完全避开下方 PN结。

#### 为什么「S顶面 ↔ 沟道」没有 PN结？（空间 + 材质一次讲透）

##### 1. 先分开两个交界，别混

| 交界 | 在哪 | 结果 |
|------|------|------|
| **① 侧面、底部** | 埋在 P硅**内部** | 实体 N + 实体 P 贴死 → **标准 PN结**（耗尽层），电子不能往下穿 |
| **② 顶面** | 芯片**最表面**，没有 P盖着 | N上头 **没有 P型材料包裹** → **不存在 N–P 贴合** → 造不出 PN结 |

栅下中间：本来是 P硅表面；栅压够 → 把电子吸到 **整片硅最表层** → 薄薄电子导电层（N沟道）；左右两头搭在 S、D **裸露的 N顶面**。

##### 2. 为什么顶面连沟道处没有 PN结？

PN结产生的唯一条件：**N型半导体 和 P型半导体两种固体直接接触**。

| 一侧 | 是什么 |
|------|--------|
| S顶面 | 实体 N硅（大量自由电子） |
| 相连的沟道 | 聚集的自由电子（导电载流子也是电子） |

二者导电粒子一样，都是电子 → **没有 N、P 两种不同材料对接** → 不满足成结条件。  
≈ 两段同种导电材料无缝对接：无隔离、无耗尽层，电子随便流通。

##### 3. 「拉到最表层」是什么物理过程

中间本来是 P硅，满是空穴。栅极带正电，隔着氧化层向下：

1. 先把表层空穴往下推开  
2. 再把硅深处微量自由电子吸附停留在 **硅最表面**  

→ 在 P表面凭空出现以电子为主的导电薄层；只停在最外表面，不往下钻进内部 P。

##### 4. 完整通路（核心困惑）

电子从源 N出发 → **不走**侧壁、底部（那里 PN结堵死）→ 走源极 **朝上裸露顶面** → 直接对接表层电子沟道（全程电子导电、无结阻隔）→ 沿最表层横向到漏极裸露顶面 → 进漏 N区。

##### 5. 一句话四点

1. PN结只长在 N区埋入 P 的 **侧面、底面**  
2. N区朝上的 **顶面没有 P**，造不出 PN结  
3. 栅极吸出的表层沟道，两端接 S/D **无结的顶面**  
4. 整条通道全部电子导电，**不存在 N–P 交界**，自然碰不到任何正式 PN结

#### N沟道有没有脱离 P衬底？→ **没有**

**核心一句话：** N沟道 **没有脱离 P衬底**，紧贴在 P 的最顶层表面；只存在极薄一层，**不往下深入** P 内部。

##### 1. 空间位置

整块基底是 **连续一体的 P型硅**：底层、中间全是 P；  
最顶上那 **几个纳米** 表层被栅极电场改造、挤满电子 = N沟道。  
沟道 **下面直接就是 P**，二者紧贴、无空隙；但电子只停在表面，不往下跑。

##### 2. 分清：「材料基底」≠「表层载流子」

| | |
|--|--|
| **基底材质** | 没变：底下依旧是 P硅，没有换成一块独立 N材料 |
| **表层载流子** | 被电场强行改：空穴推深、电子吸到表面 → 临时电子层叫 N沟道 |

它不是一块分开的 N硅片，是 **P硅表面一层临时电荷层**，扎根在 P 顶面。

##### 3. 为什么不往下钻进 P？（关键）

栅极垂直电场一直把电子 **压在最表面**；  
电子一旦往下，会与 P 里大量空穴 **复合消失**。  
→ 只能贴顶面横向移，不深入下层 P。

##### 4. 结合 S/D 再理顺

```
源极 N顶面 → P衬底表层电子沟道 → 漏极 N顶面
```

全程都在 **整片 P衬底的最上表层**：没有脱离衬底，只是不进入衬底深处。  
（侧面·底面 PN结仍朝下堵死。）

##### 通俗比喻

P衬底 = 一整块平整泥土；  
N沟道 = 只铺在泥土最表面的薄薄水膜。  
水膜 **紧贴泥土、没有飘起来脱离**，但水只在表面横向流，不渗进泥土深处。

#### 双接触模型：只有一种会形成 PN结（混淆根源）

##### 收口定稿（彻底分清）

**一半对，一半要修：** 「两块不同掺杂固体贴合」方向对，但 **手动压在一起不够**。

**1. 纠正：单纯两块 N硅、P硅贴合 → 也出不了合格 PN结**

哪怕紧紧压住表面，只是宏观接触：原子没互相扩散融合，中间还有缝隙、氧化层 → 「物理贴在一起」→ **不会**产生稳定耗尽层、内建电场 → **不算合格 PN结**。

**2. 真正 PN结的核心：两种掺杂半导体原子级融合**

N、P 区域硅原子互相渗透扩散，交界处杂质原子交错分布 → 才能形成固定带电离子的耗尽层。  
工业上靠 **高温扩散、离子注入+退火**，让杂质原子钻进硅晶格内部，原子层面融合 → 稳定生成 PN结（MOS 源漏侧壁、底面就是这种）。

**3. 回到 MOS 沟道**

栅极吸出的表层电子 = **自由电荷聚集**，没有改硅本身的掺杂原子；P衬底晶格杂质从头到尾没变。  
不存在 N型杂质原子渗透进 P硅，无原子融合 → **绝对不成 PN结**。

**4. 沟道 ↔ 源漏顶面**

源漏顶面 = 裸露固态 N；旁边 = 表层电子；两边电子导电，中间无 P隔开 → 无 N/P 固体交界 → 电子可横向直接流通。

> **极简总结：**  
> ① 只把 N、P硅物理贴合、无原子扩散融合 → 无 PN结；  
> ② 高温工艺让 N/P 杂质原子互相渗透、晶格融合 → 稳定 PN结（MOS 源漏侧壁·底面）；  
> ③ 电场临时吸附的表层电子层，不改变内部掺杂原子，无原子融合 → 无 PN结。

---

**卡点拆开（展开）：** 两种完全不同的「N–P 接触」——**只有第一种成结，第二种不成。**

##### ① 实体 N硅（源/漏块）↔ 实体 P衬底（侧面、底面）→ **标准 PN结**

源、漏 = 工厂离子掺杂、**永久改变硅原子**做成的 N型**固体**。  
永久 N固体 + 永久 P固体紧密贴合 → 扩散形成固定离子耗尽层 → **标准 PN结**。  
这是两种**掺杂材料**的交界面，一定会生成 PN结。

##### ② 表层电子沟道 ↔ 下方 P衬底（你纠结的顶面）→ **不成 PN结**

这里 **根本没有一块实体 N材料**！  
底层硅原子掺杂从头到尾仍是 P，没有改成 N固体。  
所谓 N沟道 = **漂浮在 P表面的一堆自由电子**（临时电荷），不是 N型半导体固体。

> PN结定义直觉：**两种不同掺杂的固态半导体** 的接触面。  
> 现在只有一种固态材料（P硅）+ 表面临时电子 → 不存在「N固体」→ **不满足成结条件**。  
> 电荷停在 P表面 ≠ 出现一块 N半导体 ≠ 有 PN结界面。

##### ③ 沟道 ↔ 源极 N块顶面（关键通路）→ **也无 PN结**

源极是实体 N固体；朝上顶面无 P包裹：

- 源极向下、向左右 → 贴 P → 有 PN结  
- 源极向上裸露 → 旁边是 P表面的电子沟道  

连接：`实体 N块（源顶面） ↔ 一堆自由电子（沟道）`  
两边导电载体都是电子，中间没有 P夹着 → **无 N固体–P固体贴合** → 无 PN结。

##### 砖块 / 泥土 / 积水

| | |
|--|--|
| 泥土 | P衬底（固态 P，永远不变） |
| 砖块 | 源/漏 N区（固态 N，永久掺杂） |
| 砖块埋进泥土的侧面·底部 | 砖(N固体)贴泥(P固体) → **有 PN结** |
| 泥土表面薄积水 | N沟道（只是自由电子，**不是固体**） |
| 积水挨砖块朝上顶面 | 砖(电子)↔水(电子)，中间无泥隔开 → **无 PN结** |
| 积水↔泥土 | 积水不是另一块 N固体 → **也不算 PN结** |

##### 为什么「绕不开 PN结」想错了

| 错的默认 | 事实 |
|----------|------|
| 电子一聚在 P表面 = 出现一块 N材料，一碰 P 就成结 | 只有 **永久掺杂成型的 N固体** 贴 P固体才有 PN结 |
| | 电场临时吸附的自由电子只是电荷，**不是 N半导体**，不成结 |
| | 电子横向走表层电子层，连源漏裸露顶面，**不穿任何 N固体–P固体构成的结** |

### 情况 1：栅极不加电压 VGS=0

硅表面仍是普通 P：多空穴、少电子。

```
S(N) — P硅 — D(N)
```

两结反向截止 → **S、D 不通，管子关断**。

### 情况 2：栅极接正、衬底接地（VGS > 开启电压）

栅极金属带正电，隔着 SiO₂，向下电场：

1. **排斥** 表面空穴 → 推往硅深处  
2. **吸引** 稀有自由电子 → 聚集在氧化层下方表面  

电子够多 → 表面薄层从「以空穴为主的 P」**反转**成「以电子为主的 N」= **反型层 / N型导电沟道**。

```
S(N) — 表面 N沟道 — D(N)
```

左右 N 被一条 N 通道直接连上 → 源漏可通。

### 栅压到底控制沟道的什么？（精确拆开）

**核心功能：** 栅极电压控制沟道的 **「存在与否 + 载流子浓度 / 导通能力」**。  
口语说「控制大小」太粗——要拆成两步。

#### 增强型 NMOS：相对开启电压 Vth

| 条件 | 沟道 | 结果 |
|------|------|------|
| **VGS < Vth** | **不存在**（无反型层） | S–D 仍被 P 隔开 + 两结反向；几乎无 Ids。👉 不是「沟道很小」，是 **直接没有** |
| **VGS > Vth** | 生成电子反型层 | VGS 越大 → 电场越强 → 表面自由电子**越多** → 导电能力↑、等效电阻↓ |

口语「沟道变大」的本质 = **沟道内载流子浓度提升**。  
物理上沟道几何厚度只有约几个纳米，电压几乎不改「几何厚薄」；教材说「宽窄/厚薄」多是简化说法。

精炼回答：

> 粗略：栅极控制沟道 —— **先有没有，再强不强**。  
> 严谨：VGS 经垂直电场调控硅表面反型载流子浓度，决定沟道是否形成及导通能力高低。

#### 两个工作区（有沟道之后，再看 VDS）

| 区 | 条件直觉 | 栅压干什么 |
|----|----------|------------|
| **线性区（欧姆区）** | VDS 很小 | 提高 VGS → 沟道导电能力持续增强，Ids 随 VGS 升；近似 **可调电阻** |
| **饱和区（恒流区）** | VDS 够大，靠近漏端 **沟道夹断** | 再增大 VGS 仍使 Ids 上升，但机理不再是「整条沟道均匀变粗」，而是改 **饱和电流**大小；夹断点会移动，夹断区仍在 |

> 夹断细节见下节「VGS vs VDS」末尾。

#### 再对比 PN结（别串戏）

| | PN结正向加压 | MOS 栅极加压 |
|--|--------------|--------------|
| 作用 | **削弱耗尽层壁垒**，载流子**跨越结面** | **不打通任何 PN结**；不向硅注入电荷 |
| 手段 | 结两端偏置 | 氧化层绝缘，几乎无栅流；电场改**表层载流子种类与浓度** |
| 门槛 | — | **先创造出沟道**，才能再调导电水平 |

#### 延伸：MOS vs BJT

| | BJT | MOS |
|--|-----|-----|
| 控制 | **电流控电流**（要持续基极电流） | **电压建电场调沟道**（稳态栅极几乎无电流） |

### 源极 S vs 漏极 D（NMOS，结合 VGS / VDS）

#### 源极 Source（S）：电子发射源头

| 点 | 记住 |
|----|------|
| 职能 | **载流子（电子）的来源** —— 沟道是电子通道，电子从源极进入沟道 |
| 电位 | 电路里通常**最低**，常接地 0V |
| 与衬底 | S 的 N区 + P衬底 = PN结，工作时 **反向截止** → 电子只能往表层沟道走，**不能进衬底** |

#### 漏极 Drain（D）：电子收集端

| 点 | 记住 |
|----|------|
| 职能 | **吸走沟道里的电子** —— D 接更高正压，横向电场把电子从沟道抽到漏极 |
| 电位 | **高于源极**；**VDS = 漏 − 源** 的电势差 = 电子流动的动力 |
| 与衬底 | 同样反向 PN结 → 电流只走表面沟道，不进衬底 |

#### 完整流向（NMOS 导通时）

```
电子：  源极 S  →  N型沟道（VDS 横向电场拉动）  →  漏极 D 流出
电流：  约定方向与电子相反 →  D → 沟道 → S
```

| 电极 | 核心职能 |
|------|----------|
| **源极** | **供给**导电的电子 |
| **漏极** | **吸收、抽走**电子，靠高压制造拉力 |

#### 极简比喻（串起全部）

| | 比喻 |
|--|------|
| 沟道 | 水管 |
| **源极** | 蓄水池（供水 / 供电子） |
| **漏极** | 抽水机（高压吸力，把水抽走） |
| **VGS** | 水管能不能成型 |
| **VDS** | 抽水机吸力大小 |

#### PMOS 反过来记

载流子是空穴（正电荷）：

- **源极 S**：空穴来源，电位**最高**  
- **漏极 D**：吸收空穴，电位**更低**

### VGS 造路 vs VDS 推流（两套电场，重中之重）

用词钉死：**源极 S、漏极 D**（不是「珊」）。

**铁律（增强型 NMOS）：**

```
VGS = 造出导电沟道（通路有没有）
VDS = 给沟道里电子提供横向动力 → 定向移动 → 形成电流 Id
```

| 谁 | 干什么 |
|----|--------|
| **栅极电压 VGS** | 只负责 **造出导电沟道**（硅表面铺电子通道；过 Vth 才有） |
| **源漏电压 VDS** | 负责 **推动载流子流动**，产生电流 **Id** |

#### 情景 1：有 VGS（沟道在），VDS = 0（源漏等电位）

栅极电场已吸来大量电子，N型沟道成型。  
但 S、D **电位一样** → 沟道里电子只是杂乱热运动，**没有持续定向流动**。  
宏观：**Id = 0**。

类比：铺好跑道（沟道），两头高度相同 → 运动员（电子）原地乱动，不会自发跑完全程。

#### 情景 2：有 VGS，再加 VDS（漏电位 > 源）

S 低、D 高 → 沟道内 **横向电场（水平）**。  
电子带负电 → 受力与电场相反 → **电子：源 → 沿沟道 → 漏**。  
持续定向移动 = **漏极电流 Id**。

#### 两套电场绝对不要混

| | **VGS → 垂直电场**（上→下） | **VDS → 横向电场**（沿 S–D） |
|--|------------------------------|------------------------------|
| 作用 | 改硅表面载流子，**决定能不能生成沟道** | 给电子水平拉力，实现 **S→D** |
| 能做 | 把电子「吸附到硅表层」 | 推着电子左右跑 |
| 不能 | **无法**推着电子沿 S–D 横向移动 | **无法**单独造出沟道 |

#### 最容易踩的思维坑

> ❌「只要造出沟道，电流自然产生。」  
> ❌「源漏压差靠扩散产生沟道。」  
> ❌「电子在整块 P衬底里跑。」

**对：** 要有沟道（VGS）+ 电势差（VDS 漂移驱动）；电子只在表面薄层；两结不导电。

#### 水渠对比

| | 含义 |
|--|------|
| **VGS** | 开挖水渠；电压不够 → 水渠**直接消失** |
| **VDS** | 两端水位差；没水位差 → 渠在水也不流 |
| **水流** | = 漏极电流 Id |

#### 极简总结（缺一不可）

```
VGS 负责开辟通路；
VDS 负责提供电势差，驱动载流子横向运动 → 产生电流。

只有 VGS、无 VDS  → 有路，无电流（Id=0）
只有 VDS、VGS 不够 → 根本没通道，依旧无电流
```

#### 沟道夹断（饱和区铺垫）

VDS 不断加大：漏端附近栅极有效电场被抵消 → 漏端沟道变窄 → **夹断**。  
末端即使「夹断消失」，VDS 强电场仍可把电子抽到漏极 → **电流依旧存在** → **饱和区**。

#### 再次提醒：电流不走 PN结

S–D 电流 **全程走表面沟道**；源–衬底、漏–衬底两结始终 **反向截止**。

### 最容易混淆的两点（再钉一次）

| # | 钉死 |
|---|------|
| 1 | 沟道 **不是**「打通原来那两个 PN结」。结仍 **反向截止**，无工作电流过结。电流只走表层沟道，**绕开**屏障。 |
| 2 | 栅–硅之间有绝缘体：栅上电荷 **不能流进硅**，几乎无栅极电流。不靠注入载流子，**纯粹电场隔空调控**表面导电能力。 |

电流路径：

```
源 N  ──►  表面 N沟道  ──►  漏 N
              （旁路）
         ╳ 不走衬底 / 不走两个 PN结
```

### 完整逻辑链（串联前面全部疑问）

1. MOS 原生两个背对背 PN结 → S–D **天然断开**（结锁反向）  
2. 绝缘栅 = 平板电容；**VGS 垂直电场** → 造沟道（过 Vth）  
3. **VDS 横向电场** → 推电子 S→D，产生 ID  
4. VDS 过大 → 漏端可能夹断 → 进入饱和区（仍有电流）  
5. 两结全程反向，电流只走表面沟道旁路

### PMOS 极简对照（再汇总）

| | NMOS | PMOS |
|--|------|------|
| 衬底 / S·D | P衬底 + N型 S/D | N衬底 + P型 S/D |
| 栅极造沟道 | **正/高** → N沟道（电子） | **负/低** → P沟道（空穴） |
| 源极 | 电子来源，电位常**最低** | 空穴来源，电位常**最高** |
| 漏极 | 抽走电子，电位**更高** | 吸收空穴，电位**更低** |
| 电流路径 | 表面沟道旁路；结仍反向 | 同样绕开 PN结 |

数字电路：nMOS 爱拉 **0**，pMOS 爱拉 **1** → CMOS（见 [1.7](./1.7_CMOS晶体管.md)）。

### 一句话串起

两结挡经衬底的路；  
**VGS** 垂直电场造表面沟道；**VDS** 横向电场推电流；  
电流走沟道旁路，不走过结。  
**有路无压 → 无电流；有压无路 → 也无电流。**

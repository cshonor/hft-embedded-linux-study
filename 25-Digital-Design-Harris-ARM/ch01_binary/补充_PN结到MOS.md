## 笔记

只讲 **NMOS**；**PMOS 全部反过来**（文末对照）。  
（「珊极」= **栅极 G**，Gate。）

### 前置复习（2 个关键点）

1. **PN结：** P 和 N 贴一起 → **单向导电**；反向断路；中间有耗尽层。详见 [补充_PN结](./补充_PN结.md)。  
2. **NMOS 骨架：** 整块 **P型硅衬底**；左右各掺一块 **N型** —— 左 **源极 S**、右 **漏极 D**。

---

### 第一步：先搭出 2 个背对背 PN结（不加栅极 = 天生断开）

| 结构 | 形成 |
|------|------|
| P衬底 + 左 N（S） | 第 1 个 PN结 |
| P衬底 + 右 N（D） | 第 2 个 PN结 |

两个 PN结 **背对背**：

- 不管给 S、D 加什么电压，**永远有一个结反向截止**  
- S 与 D 中间是 P 硅，没有导电通路 → **管子关断，无电流**

**大白话：** 左右两块 N「金属」（S/D），中间一堵 P 墙；墙左右各一道单向门（PN结），两道门永远关一扇，电流过不去。

---

### 第二步：加绝缘栅极（MOS 核心创新）

在两块 N 区中间的 P 硅表面，从上到下三层：

| 层 | 是什么 |
|----|--------|
| 1 | 金属 / 多晶硅 = **栅极 G** |
| 2 | 极薄 **SiO₂** = **绝缘玻璃层**（关键：不导电） |
| 3 | 底层 **P型衬底** |

重点：栅极和底下硅 **没有 PN结、完全绝缘**。  
栅极通电 **不会** 有电流灌进硅里，只产生 **垂直向下的电场**。

---

### 第三步：栅压用电场「毁掉」中间的阻断

#### 场景 1：VGS = 0（关断 = 纯 PN结状态）

无电场 → P衬底仍是空穴世界；S、D 仍被 P 隔开；两结反向截止 → **S–D 不通**。

#### 场景 2：VGS > 开启电压（导通）

栅极带 **正电**，向下电场做两件事：

1. **排斥** P 衬底里的空穴（正）往下跑 → 栅下先变成耗尽区  
2. **吸引** 深处稀少的自由电子到 SiO₂ 下方表面  

电子够多时，栅下薄薄一层硅 **从 P型反转成 N型** → 叫 **反型层（导电沟道）**。

---

### 第四步：沟道连通 S、D，PN 阻隔失效

新 N沟道左边接 N源、右边接 N漏：

```
S(N) ── 沟道(N) ── D(N)
```

中间不再有 P 隔离 → **两个背对背 PN结失去阻隔作用**。  
此时 D 加高、S 接地：电子可从 S 沿沟道到 D → **MOS 导通**。

---

### 核心总结

| # | 记住 |
|---|------|
| 1 | MOS **原生**自带 2 个 PN结 → 天生断路 |
| 2 | **栅极 + 绝缘层** 是 MOS 独有；与 PN结无关，只输出电场 |
| 3 | 够高的栅压造 **N沟道**，把隔开 S/D 的 P 层变成 N → **消除 PN 阻挡** |
| 4 | 不加栅压 = PN结起作用 = **断**；够栅压 = 沟道盖住 = PN 失效 = **通** |

> PN结是 MOS 的基础，但栅极电场能抵消 PN结的阻断作用。

---

### PMOS 极简对照（全部反过来）

| NMOS | PMOS |
|------|------|
| P衬底 + N型 S/D | **N衬底** + **P型** S/D |
| 2 个背对背 PN结（天生断） | 同样，类型对调 |
| 栅极 **高** → 吸电子 → **N沟道** → 通 | 栅极 **低（如 0）** → 吸空穴 → **P沟道** → 通 |
| 栅极低 / 0 → 无沟道 → 断 | 栅极 **高** → 无沟道 → 断 |

数字电路里：nMOS 爱拉 **0**，pMOS 爱拉 **1**；二者互补 = CMOS（见 [1.7](./1.7_CMOS晶体管.md)）。

---

### 一句话串起

硅片掺杂做出两块 N + 一块 P → 两道反向 PN结 → 电路断开；  
表面盖一层玻璃绝缘 + 金属栅极 → 隔空加电场；  
栅极通电吸来电子 → 中间造出 N型通道 → PN 阻隔失效 → 源漏导通。  
**这就是 NMOS。**

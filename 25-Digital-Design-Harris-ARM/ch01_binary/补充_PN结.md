## 笔记

不讲难词，一步一步来。芯片、MOS 原材料都是**硅**；纯硅几乎不导电，**掺杂质**才分出两种。

### 1. 两种基础半导体

#### N型硅（自由电子多）

往纯硅里掺**磷**这类元素 → 多出很多**自由电子**（带负电）。  
导电主力是电子。

> 记：N = Negative（负，电子多）

#### P型硅（空穴多）

往纯硅里掺**硼**这类元素 → 少电子，留下**空穴**。  
别的电子来填空穴，看起来像「正电荷在移动」。

> 记：P = Positive（正，空穴多）

### 2. 什么是 PN结？

一块 **P型** 和一块 **N型** 紧紧贴在一起，接触面这一层结构 = **PN结**。

![PN结示意：P区空穴 · 耗尽层 · N区电子 · 正向电流](./figures/pn-junction.png)

> 图注：P接正极、N接负极时**导通**；反接时**截止**。绿色箭头为正向电流方向。

### 3. 贴合瞬间：扩散 → 耗尽层（核心）

两边粒子往对面跑：

1. N区大量电子 → 穿过接触面进 P区，填补空穴  
2. P区大量空穴 → 穿过接触面进 N区，和电子中和  

扩散之后，接触面形成 **耗尽层**：

| 哪边 | 跑走了什么 | 剩下什么 |
|------|------------|----------|
| N区 | 电子走了 | 带**正电**的原子核 |
| P区 | 空穴被填（等于空穴走了） | 带**负电**的原子核 |

左右一正一负 → **阻挡电场**。  
这层里电子、空穴几乎耗光 → **几乎不导电**，所以叫耗尽层。

电场会阻止继续扩散，最后达到平衡，不再来回猛跑。

### 4. 单个 PN结通电：两种情况，作用完全不同

#### 一、正向通电（P接+，N接-）→ 导通

| 发生什么 | 结果 |
|----------|------|
| 内建电场被抵消，耗尽层**变薄** | 载流子持续复合 |
| | **持续导通，有稳定电流** |

**实际器件用途（单个结当二极管用）：** 整流、发光（LED）、稳压、测温等。

> 大白话：通电通路，电流顺畅流过 PN结。

#### 二、反向通电（P接-，N接+）→ 截止

| 发生什么 | 结果 |
|----------|------|
| 内建电场加强，耗尽层**变宽** | 几乎无电流（仅极小漏电流） |
| | **截止、绝缘** |

反向再分两种工况：

| 工况 | 作用 |
|------|------|
| **普通反向低压** | **隔离、阻断电流** —— MOS 里源–衬底、漏–衬底结就是靠这个阻断「经衬底走」的 S–D |
| **反向电压超过击穿电压** | 雪崩 / 齐纳击穿；**稳压二极管**专门利用击穿区稳压（普通结要避免误入失控击穿） |

> 大白话：正常反向 = 断路关死；打到击穿 = 另一套器件用法，不是日常「开关通」。

### 5. 一句话本性

单个 PN结通电核心：**正向导电，反向截止** —— 二极管单向导电的本质。

### 6. 和 MOS 怎么关联

| 点 | 记住 |
|----|------|
| MOS 里那两个结 | **永远工作在反向截止**；**绝不允许正向导通**（正向 = 衬底漏电） |
| 栅压干什么 | **不靠导通 PN结**；额外在表面生成沟道，**绕开** PN结 |
| 反向结的用处 | 正是上面「普通反向低压」：阻断经衬底的通路 |

| 管 | 衬底 + 源/漏 |
|----|--------------|
| **NMOS** | P衬底 + N源/漏 → 两个反向 PN结 |
| **PMOS** | N衬底 + P源/漏 → 同样 |

完整四步 +「旁路非打通」纠正 → **[补充_PN结到MOS](./补充_PN结到MOS.md)** · 总览 [1.7](./1.7_CMOS晶体管.md)。

### 超简背诵口诀

```
掺磷多电子 = N型，掺硼多空穴 = P型；
贴合形成 PN结，中间生出耗尽层；
P正 N负加电压 → 导通（二极管干活）；
P负 N正加电压 → 截止（MOS 里结就停在这）；
单向导电是本性；MOS 靠沟道旁路，不靠打通结。
```

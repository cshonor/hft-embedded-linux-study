## 笔记

不讲难词，一步一步来。芯片、MOS 原材料都是**硅**；纯硅几乎不导电，**掺杂质**才分出两种。

### 1. 两种基础半导体

#### N型硅（自由电子多）

往纯硅里掺**磷**这类元素 → 多出很多**自由电子**（带负电）。  
导电主力是电子。

> 记：N = Negative（负，电子多）

#### P型硅（空穴多）

往纯硅里掺**硼**这类元素 → 少电子，留下**空穴**。  
别的电子来填空穴，看起来像「正电荷在移动」。

> 记：P = Positive（正，空穴多）

### 2. 什么是 PN结？

一块 **P型** 和一块 **N型** 紧紧贴在一起，接触面这一层结构 = **PN结**。

![PN结示意：P区空穴 · 耗尽层 · N区电子 · 正向电流](./figures/pn-junction.png)

> 图注：P接正极、N接负极时**导通**；反接时**截止**。绿色箭头为正向电流方向。

### 3. 零偏（不加电压）：PN结本来什么样

P区多子 = **空穴(+)**；N区多子 = **自由电子(-)**。  
接触面：空穴往 N 跑、电子往 P 跑，复合消失。

交界处剩下：

| 侧 | 留下什么 |
|----|----------|
| P侧 | 一层**负离子**（固定、不能移动） |
| N侧 | 一层**正离子**（固定、不能移动） |

这一层 = **耗尽层**（没有可移动载流子 → 不导电）。

离子层形成 **内建电场**：方向 **N → P**，阻止载流子继续扩散。  
平衡时：扩散力 ≈ 内建电场阻力 → 耗尽层厚度固定。

（图见上：[pn-junction.png](./figures/pn-junction.png)）

### 4. 正偏 / 反偏：耗尽层变薄还是变宽？（拆两个误区）

#### 4.1 正向偏置：P接电源+，N接电源-

**电源在干嘛？**

- 正极往 P区 **强行推入空穴（+）**
- 负极往 N区 **强行推入电子（-）**

**电场方向对比：**

| 电场 | 方向 |
|------|------|
| 内建电场 | **N → P** |
| 外加电压电场 | **P → N** |

→ 两电场 **方向相反，互相抵消**（不是叠加！）。

**耗尽层怎么变？→ 变薄（不是变宽）**

1. P区新增空穴 → 中和交界处负离子  
2. N区新增电子 → 中和交界处正离子  
3. 固定离子被中和掉一大块 → 离子层变窄 → **耗尽层变薄**

**核心疑问：「内建电场被抵消，PN结是不是不存在了？」**

| 错觉 | 事实 |
|------|------|
| PN结彻底消失 | **结构还在**（P/N 掺杂分区没变） |
| 内建电场完全没了 | 只是被 **大幅削弱**，不是彻底为零 |
| 耗尽层完全没了 | 只是 **变薄**，不是消失 |

变薄之后：空穴 P→N、电子 N→P 容易穿过 → **持续电流 = 导通**。

**错误推论纠正：**「电荷变多 → 内建电场被撕开 → 耗尽层变宽」错在两点：

1. 外电场与内电场是 **反向抵消**，不是同向撕开/叠加  
2. 多出来的空穴/电子去 **中和固定离子** → 耗尽层只会 **缩小**，不会加宽  

**一句话答「P加正空穴多、N加负电子多」：**  
移动电荷跑到耗尽区中和离子 → 离子少了 → 层变薄 → 内电场削弱 → 挡不住了能穿过去；**不是 PN结消失，是阻挡变弱。**

**器件用途：** 整流、LED、稳压、测温等（当二极管用）。

#### 4.2 反向偏置：P接电源-，N接电源+（对照记）

外电场与内建电场 **同方向** → **叠加变强**：

- P区空穴被电源负极拉走，N区电子被正极拉走  
- 交界处露出 **更多** 固定离子  
- 耗尽层 **变宽** → 几乎无电流（仅极小漏电流）= **截止**

| 工况 | 作用 |
|------|------|
| **普通反向低压** | 隔离、阻断 —— MOS 源/漏–衬底结靠这个挡「经衬底」通路 |
| **超过击穿电压** | 雪崩/齐纳；稳压管利用击穿区（普通结要避免误入失控） |

#### 4.3 极简对照

| | 接线 | 外场 vs 内场 | 耗尽层 | 结果 |
|--|------|--------------|--------|------|
| **正向** | P+ N- | **抵消** | **变薄** | 导通 |
| **反向** | P- N+ | **叠加** | **变宽** | 截止 |

---

### 5. 一句话本性

单个 PN结：**正向导电，反向截止** —— 二极管单向导电的本质。

### 6. 和 MOS 怎么关联

| 点 | 记住 |
|----|------|
| MOS 里那两个结 | **永远反向截止**；**绝不允许正向导通**（正向 = 衬底漏电） |
| 栅压干什么 | **不靠导通 PN结**；表面生成沟道 **绕开** PN结 |
| 反向结的用处 | 正是「普通反向低压」：阻断经衬底的通路 |

| 管 | 衬底 + 源/漏 |
|----|--------------|
| **NMOS** | P衬底 + N源/漏 → 两个反向 PN结 |
| **PMOS** | N衬底 + P源/漏 → 同样 |

完整四步 +「旁路非打通」→ **[补充_PN结到MOS](./补充_PN结到MOS.md)** · 总览 [1.7](./1.7_CMOS晶体管.md)。

### 超简背诵口诀

```
零偏：耗尽层 + 内建电场 N→P，厚度固定；
正偏 P+N-：外场抵消内场 → 耗尽层变薄 → 导通（结还在，只是挡不住）；
反偏 P-N+：外场叠加内场 → 耗尽层变宽 → 截止；
MOS：结锁反向；通靠表面沟道旁路，不靠打通结。
```

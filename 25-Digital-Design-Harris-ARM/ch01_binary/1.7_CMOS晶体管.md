### 1. 单词拆分

| | |
|--|--|
| **C** | Complementary **互补** |
| **MOS** | Metal-Oxide-Semiconductor 金属氧化物半导体管 |
| **CMOS** | **互补型 MOS 电路** |

成对使用两种管子：

| | **NMOS**（你刚学的） | **PMOS**（反向版） |
|--|----------------------|---------------------|
| 衬底 / 源漏 | P衬底 + N源漏 | N衬底（或 N阱）+ P源漏 |
| 何时导通 | 栅加 **高** 电压 | 栅加 **低** 电压 |
| 沟道载流子 | **电子** | **空穴** |

细节机理（沟道、VGS/VDS、不穿 PN结）→ [补充_PN结到MOS](./补充_PN结到MOS.md)。

### 四图对照（看图记知识点）

学 CMOS 时手里常有四类图，各盯一件事。下面用示意标出该看什么（有实拍/课本图时按同样标注对号入座）。

#### 一、NMOS 单管剖面图（你一直在学的那根管子）

```
        G（栅极金属/多晶硅）
        ═══════
       / SiO₂  \     ← 绝缘，电子不过栅
  ────┴─────────┴──── 硅最表面
  [N+]  ~电子沟道~  [N+]
   S    （临时薄层）  D
   │ ╲             ╱ │
   │  ╲__ PN结 __╱  │   ← 侧壁·底面 ↔ P（正式结，朝下）
   └──────┬──────────┘
          P 衬底（整片）
```

| 标注 | 是什么 |
|------|--------|
| 底层粉色/整片 | **P型衬底** |
| 左右蓝 N+ 方块 | **源 S、漏 D** |
| S/D 底面·侧壁 ↔ P | **实体 PN结** |
| SiO₂ 上方 | **栅极 G**（与硅完全绝缘） |
| 栅下 P 表层 | **临时电子 N沟道**（只横向连 S/D **顶面**，不穿过下方结） |

→ 对号：[补充_PN结到MOS](./补充_PN结到MOS.md)「顶面无结 / 侧壁底面有结 / 电子只走表层」。

#### 二、CMOS 反相器电路原理图（最基础 CMOS 单元）

```
              VDD
               │
            源 │
         ┌─────┴─────┐
         │   PMOS    │  ← 上：P沟道，源接 VDD
         │   （P）   │
         └─────┬─────┘
            漏 │
    A ─────────┼────────── Y（输出）
     （两栅相连）│
         ┌─────┴─────┐
         │   NMOS    │  ← 下：N沟道，源接 GND
         │   （N）   │
         └─────┬─────┘
            源 │
              GND
```

| 点 | 记住 |
|----|------|
| 上管 | **PMOS**，源极接 **VDD** |
| 下管 | **NMOS**，源极接 **GND** |
| 输入 A | **两栅连一起** |
| 输出 Y | **两漏连一起** |
| 互补 | 输入高 → NMOS开、PMOS关；输入低 → 反过来；**永远一路通**，静态几乎不耗电 |

#### 三、CMOS 芯片硅片横截面（同一块硅上做两种管）

```
  G        G
  ║        ║
 ═╩═      ═╩═     ← 各自栅 + 氧化层
[N+][N+] [P+][P+]
  S  D     S  D
  NMOS      PMOS
────┬──────────┬────
    │  P衬底   │ N阱 │  ← 右边挖出 N阱放 PMOS
    └──────────┴────┘
```

| 工艺点 | |
|--------|--|
| 基底 | 整片 **P型硅** |
| 右边 | 挖 **N阱**，里面做 **PMOS**（PMOS 需要 N衬底环境） |
| 左边 | 直接在 P衬底做 **NMOS** |
| NMOS 的 S/D | **N+** 掺杂 |
| PMOS 的 S/D | **P+** 掺杂 |
| 结与沟道 | 各管源漏侧壁·底面与衬底/阱形成独立 PN结；表层沟道横向导电，避开结 |

#### 四、CMOS 版图 / 显微镜实拍（实物排布）

课本或显微镜图里常见：

- **上方长条区域** ≈ PMOS

### 2. 「互补」是什么意思？

同一输入下，两管永远 **一个开、一个关**，稳态不同时导通。

**最简单单元：CMOS 反相器（非门）**

```
        VDD
         │
       [PMOS]  ← 上管
         │
    ─────┼───── 输出
         │
       [NMOS]  ← 下管
         │
        GND

输入同时接两管栅极
```

| 输入 | NMOS | PMOS | 输出 |
|------|------|------|------|
| **高** | 导通 | 截止 | 拉到 **0**（地） |
| **低** | 截止 | 导通 | 拉到 **1**（VDD） |

→ nMOS 擅长拉 **0**；pMOS 擅长拉 **1** —— 和 [1.5](./1.5_逻辑门.md) 的非门对应。

#### 反相器里两个核心问题

**问题 1：VDS 在哪里？**

标准结构：

| | PMOS（上） | NMOS（下） |
|--|------------|------------|
| 源极 | **Sₚ → VDD** | **Sₙ → GND（0V）** |
| 漏极 | **Dₚ → Out** | **Dₙ → Out** |
| 栅极 | Gₚ ← 输入 A | Gₙ ← 输入 A（与 Gₚ 短接） |

> 每一只 MOS **各自有自己的 VDS**；Out = 两漏公共点。

| 管子 | VDS 怎么算 |
|------|------------|
| **NMOS** | \(V_{DS(n)} = V_{Dn} - V_{Sn} = V_{Out} - 0 = \mathbf{V_{Out}}\) |
| **PMOS** | \(V_{DS(p)} = V_{Dp} - V_{Sp} = V_{Out} - V_{DD}\)（正常工作时多为**负数**） |

易混：

1. **不要**以为两管共用一套 VDS —— S、D 电位不同，必须分算 \(V_{DS(n)}\)、\(V_{DS(p)}\)  
2. 教材有时用 \(V_{SD}\)（源减漏，正数）方便说 PMOS；**严格定义仍是** \(V_{DS}=V_D-V_S\)  

文字简图：

```
        VDD
         │
         Sₚ (PMOS 源)
    Gₚ───┼─── Dₚ
         │
         Out ──── 输出
         │
    Dₙ───┼─── Sₙ (NMOS 源)
         │
        GND

A ──── 同时接 Gₚ 与 Gₙ（两栅短接）
```

**问题 2：输入 A 同时驱动两栅，为什么天然是「非」？**

规则回顾：

| 管 | 导通直觉（增强型） | 本反相器里 |
|----|-------------------|------------|
| **NMOS** | 栅比源够高（\(V_{GS}>V_{thn}\)） | Sₙ=0；A=高 → \(V_{GSn}=A-0\) 够大 → **开** |
| **PMOS** | 源比栅够高（\(V_{SG}>\|V_{thp}\|\)） | Sₚ=VDD；A=高 → \(V_{SG}=VDD-A≈0\) → **关** |

| 输入 A | NMOS | PMOS | Out |
|--------|------|------|-----|
| **高（≈VDD）** | 导通 | 截止 | 经 NMOS 拉到 **GND → 低** |
| **低（≈0）** | 截止（\(V_{GSn}=0\)） | 导通（\(V_{SG}=VDD\)） | 经 PMOS 拉到 **VDD → 高** |

👉 输入高→输出低；输入低→输出高 = **非门**。  
不是外面又加了一级非门，是 **NMOS/PMOS 导通特性天然互补** 实现取反。

稳态永远一通一断 → 几乎没有 VDD 直通 GND 的电流 → CMOS 省电根源。  
（翻转瞬间两管可能短暂同时通 → **贯通电流**，需要时再开一小节。）

---

### 3. CMOS 最大优势：几乎无静态电流

| | 老式纯 NMOS 等 | **CMOS** |
|--|----------------|----------|
| 稳态 | 导通时可能持续耗电、发热 | 永远只有一管通、另一管断 |
| 静态电流 | 大 | **几乎为 0**（理想） |

手机、CPU、单片机几乎全是 CMOS，核心原因之一就是 **功耗极低**。  
（动态翻转时仍有充放电电流 → 见 [1.8 功耗](./1.8_功耗.md)。）

### 4. 硅片上怎么做 CMOS：N阱工艺（难点）

#### 核心矛盾

| 管子 | 需要的衬底 |
|------|------------|
| **NMOS** | **P型衬底** |
| **PMOS** | **N型衬底** |

整片原始晶圆通常只有一种类型（通用 **P型基底**）→ 不能「整片换类型」同时直接做两种管。  
**解法：离子注入制造「阱（Well）」**，局部改出第二种衬底。

#### 主流：P型硅衬底 + N阱（N-well）

**1. 原始晶圆：整片 P-Sub（P衬底）**

✅ **NMOS 直接做在 P衬底上** —— 就是你学过的结构：P衬底、N+ 源漏、栅压造 N沟道。

**2. 局部造 N阱，里面做 PMOS**

在 P衬底上 **局部深度注入磷（施主）**，再高温退火 → 一小块改成 **N型区域 = N阱（N-well）**。  
👉 **PMOS 做在 N阱内部**：阱里注入 P+ 作源漏，栅压控 P沟道（空穴）。

```
┌──────────────────────── P-Sub（整片 P衬底）────────────────────────┐
│                                                                     │
│  【NMOS】直接在 P 上造          │  ████████████ N阱(N-well) ████████ │
│  S(N+)          D(N+)          │  █ 【PMOS】在阱内              █ │
│                                 │  █  S(P+)        D(P+)        █ │
│                                 │  ████████████████████████████████ │
└─────────────────────────────────────────────────────────────────────┘
```

（对应上文「四图对照 · 三、CMOS 剖面」。）

#### 关键电位规则（防 PN结正向漏电！）

| 谁 | 接哪里 | 为什么 |
|----|--------|--------|
| **整片 P衬底（P-Sub）** | **最低电位 GND（0V）** | NMOS 的 N+/P 结保持 **反偏**，不漏电 |
| **N阱** | **最高电位 VDD** | PMOS 的 P+/N阱结保持 **反偏**，不漏电 |

> 两个「基底」电位分开：P衬底=0，孤立 N阱=VDD；中间靠耗尽层隔离，互不干扰。  
> 接反 → 结可能正向导通 → **衬底/阱漏电**（故障，不是工作电流）。

#### 一句话回答「怎么共存？」

整块硅本来只有 P衬底；**不必整片更换类型**，局部注入造出独立 **N阱**：  
- 外面原生 P → 放 **NMOS**  
- 人工 N阱 → 放 **PMOS**  
两区被 PN结隔离，各自拿到需要的衬底类型 → 同片共存。

#### 了解即可：双阱（Dual Well）

先进芯片常在 P衬底上同时做：

- **N阱** → PMOS  
- **P阱（P-well）** → NMOS  

好处：两管衬底电位可 **独立调节**，抑制闩锁、改善性能；CPU/FPGA 等高端常见。

#### 隐患：闩锁效应（Latch-up）

P衬底、N阱、N+、P+ 组合可能形成 **寄生可控硅**；电压异常时大电流可烧毁芯片。  
对策：版图保护环、正确电位接法（P-Sub=GND、N阱=VDD）等 —— CMOS 设计必学点，软件主线知名字即可。

### 5. 简单区分名词

| 词 | 指什么 |
|----|--------|
| **MOS** | 一种晶体管（NMOS 或 PMOS 单个管子） |
| **NMOS** | N沟道管，电子导电 |
| **PMOS** | P沟道管，空穴导电 |
| **CMOS** | **同时搭配** NMOS+PMOS 的成套电路体系，互补工作 |

---

### 和单个 NMOS 笔记的衔接

单个管：VGS 造沟道、VDS 推电流、结锁反向、电流走表层。  
CMOS：用 **一对** 管，同一栅信号下互补开关 → 组成非门/与非/或非 → 再堆成整个数字芯片。

软件 / CSAPP / HFT 主线：记住 **互补 + 低静态功耗 + 非门 +「P衬底放 N管 / N阱放 P管」+ 衬底接地、阱接 VDD**；双阱/闩锁知概念即可。

### 1. 单管对比：NMOS ↔ PMOS（结构对称，掺杂与极性反过来）

两者都只有 **3 个电极：G / S / D**，外形一样；衬底、载流子、导通电压、接法相反。

| | **NMOS**（你前面学的） | **PMOS** |
|--|------------------------|----------|
| 衬底 | **P型硅** | **N阱**（N型） |
| S、D | **N型**掺杂 | **P型**掺杂 |
| 载流子 | **电子** | **空穴** |
| 导通条件 | 栅比源够高（\(V_{GS}\) 正向够大） | 源比栅够高（\(V_{SG}\) 够大） |
| 常见接法 | S→地，D→高压/输出 | S→**VDD**，D→输出 |

细节机理 → [补充_PN结到MOS](./补充_PN结到MOS.md)。

### 1b. CMOS ≠ 新型单管，=「NMOS + PMOS 成对电路」

| 词 | 是什么 |
|----|--------|
| **NMOS / PMOS** | **单个晶体管**，3 脚 G/S/D |
| **CMOS** | Complementary MOS：**电路模块**，至少 1×NMOS + 1×PMOS 成对使用；**不是**单独一种管子 |

最基础单元：**CMOS 反相器（一对管子）**

```
        VDD（电源正极）
          │
          S （PMOS 源）
    G─────┼───── D（PMOS 漏）
          │
          ├──── 输出 OUT
          │
    D─────┼───── S（NMOS 源）
          │
         GND（0V）

输入 A 同时接两管栅极 G
```

| 输入 A | NMOS | PMOS | OUT |
|--------|------|------|-----|
| **高** | 栅＞源 → **通**，拉到 0 | 栅≈VDD → **断** | **低** |
| **低（0）** | 栅源无压差 → **断** | 源≫栅 → **通**，拉到 VDD | **高** |

永远一通一断 → VDD 不直通地 → **静态几乎不耗电**（CMOS 最大优势）。  
展开：各管独立 VDS、为何天然是非门 → 见下文「反相器里两个核心问题」。

#### 引脚数：单管 4 极 vs 反相器对外 4 脚

**1. 单个 MOS（NMOS / PMOS）：物理上 4 个电极**

| 极 | 名 |
|----|-----|
| **G** | 栅极 |
| **S** | 源极 |
| **D** | 漏极 |
| **B** | 衬底（Body / Bulk / 基底） |

日常符号常省略 B，看起来只有 G/S/D **3 脚**；芯片内部衬底仍是独立电极（P-Sub 或 N阱要按规则接电位）。

**2. CMOS 反相器（1×PMOS + 1×NMOS）：对外外接脚**

| 外接 | 内部怎么连 |
|------|------------|
| **输入 A** | 两管栅极短接，共用一根输入 |
| **输出 OUT** | 两管漏极相连 |
| **VDD、GND** | 供电两根 |

→ 完整外接清单：**A、OUT、VDD、GND** → **4 个外部引脚**。

内部分开看（含衬底）：

| | PMOS | NMOS |
|--|------|------|
| G | ← A | ← A（同） |
| S | **VDD** | **GND** |
| D | **OUT** | **OUT** |
| B | **N阱 → VDD** | **P衬底 → GND** |

> **极简：** 单颗 NMOS/PMOS = 4 极 G/S/D/B（图上常画 3）；CMOS 反相器逻辑门对外 = 输入、输出、VDD、GND。

### 2. 「互补」是什么意思？

同一输入下，两管永远 **一个开、一个关**，稳态不同时导通。

**最简单单元：CMOS 反相器（非门）**

```
        VDD
         │
       [PMOS]  ← 上管
         │
    ─────┼───── 输出
         │
       [NMOS]  ← 下管
         │
        GND

输入同时接两管栅极
```

| 输入 | NMOS | PMOS | 输出 |
|------|------|------|------|
| **高** | 导通 | 截止 | 拉到 **0**（地） |
| **低** | 截止 | 导通 | 拉到 **1**（VDD） |

→ nMOS 擅长拉 **0**；pMOS 擅长拉 **1** —— 和 [1.5](./1.5_逻辑门.md) 的非门对应。

#### 反相器里两个核心问题

**问题 1：VDS 在哪里？**

标准结构：

| | PMOS（上） | NMOS（下） |
|--|------------|------------|
| 源极 | **Sₚ → VDD** | **Sₙ → GND（0V）** |
| 漏极 | **Dₚ → Out** | **Dₙ → Out** |
| 栅极 | Gₚ ← 输入 A | Gₙ ← 输入 A（与 Gₚ 短接） |

> 每一只 MOS **各自有自己的 VDS**；Out = 两漏公共点。

| 管子 | VDS 怎么算 |
|------|------------|
| **NMOS** | \(V_{DS(n)} = V_{Dn} - V_{Sn} = V_{Out} - 0 = \mathbf{V_{Out}}\) |
| **PMOS** | \(V_{DS(p)} = V_{Dp} - V_{Sp} = V_{Out} - V_{DD}\)（正常工作时多为**负数**） |

易混：

1. **不要**以为两管共用一套 VDS —— S、D 电位不同，必须分算 \(V_{DS(n)}\)、\(V_{DS(p)}\)  
2. 教材有时用 \(V_{SD}\)（源减漏，正数）方便说 PMOS；**严格定义仍是** \(V_{DS}=V_D-V_S\)  

文字简图：

```
        VDD
         │
         Sₚ (PMOS 源)
    Gₚ───┼─── Dₚ
         │
         Out ──── 输出
         │
    Dₙ───┼─── Sₙ (NMOS 源)
         │
        GND

A ──── 同时接 Gₚ 与 Gₙ（两栅短接）
```

**问题 2：输入 A 同时驱动两栅，为什么天然是「非」？**

规则回顾：

| 管 | 导通直觉（增强型） | 本反相器里 |
|----|-------------------|------------|
| **NMOS** | 栅比源够高（\(V_{GS}>V_{thn}\)） | Sₙ=0；A=高 → \(V_{GSn}=A-0\) 够大 → **开** |
| **PMOS** | 源比栅够高（\(V_{SG}>\|V_{thp}\|\)） | Sₚ=VDD；A=高 → \(V_{SG}=VDD-A≈0\) → **关** |

| 输入 A | NMOS | PMOS | Out |
|--------|------|------|-----|
| **高（≈VDD）** | 导通 | 截止 | 经 NMOS 拉到 **GND → 低** |
| **低（≈0）** | 截止（\(V_{GSn}=0\)） | 导通（\(V_{SG}=VDD\)） | 经 PMOS 拉到 **VDD → 高** |

👉 输入高→输出低；输入低→输出高 = **非门**。  
不是外面又加了一级非门，是 **NMOS/PMOS 导通特性天然互补** 实现取反。

稳态永远一通一断 → 几乎没有 VDD 直通 GND 的电流 → CMOS 省电根源。  
（翻转瞬间两管可能短暂同时通 → **贯通电流**，需要时再开一小节。）

### 3. CMOS 核心作用（器件 · 电路 · 应用）

结合刚学的反相器，分三层看「CMOS 到底干什么」。

#### 一、基础功能：搭建全部数字逻辑

CMOS = **互补 NMOS+PMOS 配对**；最基础单元 = **反相器（非门）**。  
用非门可组合出：与/或/与非/或非、触发器、寄存器、加法器、存储器……  
→ CPU、单片机、FPGA、内存等芯片内部，几乎全是 CMOS 电路堆叠而成。

#### 二、核心优势（相对老式纯 NMOS 等）

| 优势 | 含义 |
|------|------|
| **1. 静态功耗极低（最大亮点）** | 输入稳定时永远一管通、一管断；VDD→GND **无直流通路**，几乎不耗电。老式纯 NMOS 静态持续耗电发热；CMOS 才能撑手机、手表等便携设备 |
| **2. 高低电平完整、驱动强** | 输入高→NMOS 开→Out 拉到 **0V**；输入低→PMOS 开→Out 拉满 **VDD**。电平不「半吊子」，逻辑识别稳 |
| **3. 噪声容限大、抗干扰** | 高低区分明显，小幅电压波动不易误判 0/1 |
| **4. 易大规模集成** | 同片做 N阱即可同时有 NMOS+PMOS，适合光刻批量、上亿晶体管处理器 |

（动态翻转仍有充放电与短暂贯通电流 → [1.8 功耗](./1.8_功耗.md)。）

#### 三、实际应用场景

| 类 | 例子 |
|----|------|
| 数字芯片 | PC CPU/GPU、手机 SoC、单片机、FPGA、Flash |
| 数字外设 | 按键/屏驱/时钟芯片、蓝牙·WiFi 数字部分 |
| 微功耗 | 电子表、蓝牙耳机、传感器、遥控器（电池长待机） |
| 模拟辅助 | CMOS 工艺也可做简单模拟（如运放）——知即可 |

#### 四、对比：单管 NMOS vs CMOS

| | 单独 NMOS | **CMOS（互补配对）** |
|--|-----------|----------------------|
| 角色 | 简单开关 | 低功耗 + 稳定逻辑 + 易集成 |
| 规模 | 极少单独大规模做整机数字芯 | **现代数字芯片主流工艺** |

#### 五、一句话

> CMOS 靠 NMOS 与 PMOS **互补开关**，以 **极低静态功耗** 构建全部数字逻辑，是现代集成电路的基础。

### 4. 硅片上怎么做 CMOS：N阱工艺全流程

> **最重要一句：** CMOS **不是一种晶体管**，是「同一块硅片同时造出 NMOS + PMOS」；核心解法 = 在整块 **P型衬底**上局部改造出一块 **N阱** 放 PMOS。

#### 收口定型（你现在理解正确）

1. 基底原材料：整块 **P-Sub（P型硅衬底）**  
2. 离子注入+退火 → **局部一小块**改造成 N型硅 = **N阱（N-well）**  
3. **PMOS 制造在这块人工 N阱里面**（N阱 = PMOS 需要的 N型衬底 / Body）

复习 PMOS 原生要求：Body=N；S/D=P+；P+↔N阱接触面 = PN结。

| 电位（防漏电） | |
|----------------|--|
| 外围原生 **P衬底** | → **GND**（给 NMOS） |
| 内部人工 **N阱** | → **VDD**（给 PMOS） |

→ 相关 PN结 **永久反偏**，不导通漏电。

```
┌────────────────────── 整片 P衬底 ──────────────────────┐
│  ← NMOS（直接长在 P 上）│ ████████ N阱（人工N硅）████ │
│  N+S        N+D         │ █ PMOS 做在阱内           █ │
│                         │ █ P+S          P+D       █ │
│                         │ ██████████████████████████ │
└────────────────────────────────────────────────────────┘
```

> **一句话：** P衬底上造独立 N阱，**N阱充当 PMOS 的 N型衬底**；外面原生 P 做 NMOS → 同片两种基底 → 才能搭 CMOS。

**易混补一句：** N阱 **只是一块 N型基底，N阱本身不是 PMOS**。  
还要在阱内注入 P+ 源漏 + 栅氧 + 多晶硅栅，才是完整 PMOS。  
同理：外面 P衬底 ≠ NMOS，必须再注入 N+ 源漏才形成 NMOS。

---

#### 核心矛盾（为何需要阱）

| 管子 | 需要的衬底 |
|------|------------|
| **NMOS** | **P型衬底** |
| **PMOS** | **N型衬底** |

初始原料：整片晶圆 = **P-Sub**。  
✅ NMOS 可直接做在此；❌ PMOS 不能直接做在原生 P 上 → 必须人工造 **N阱**。

---

#### 步骤 1：制造 N阱（最关键！解决衬底矛盾）

1. 硅片表面长一层 SiO₂  
2. 光刻+刻蚀，只打开要做 PMOS 的窗口  
3. **离子注入磷（N型杂质）**，高温退火向内扩散  
4. 窗口下形成独立 **N型区域 = N阱**

剖面：左边原生 P（放 NMOS）；右边人工 N阱（放 PMOS）。  
N阱与周围 P 形成大面积 PN结；工作时 N阱接 **VDD** → 该结 **反偏隔离**。

#### 步骤 2：器件隔离（场氧 / STI）

管与管之间隔开，防漏电、寄生管。  
不做管的区域长厚 SiO₂（场氧）或挖沟槽填介质（STI），阻断寄生沟道；  
只留 NMOS、PMOS 所在区域 = **有源区**。

#### 步骤 3：栅氧 + 多晶硅栅极 G

1. 有源区高温生长极薄高质量 SiO₂（**栅氧**，绝缘，栅与硅隔开）  
2. 淀积多晶硅，光刻刻出栅线条 → 即 **栅极 G**  

> 多晶硅栅充当 **自对准掩膜**：后面注源漏时，栅底下不被掺杂，自动留出沟道区。

#### 步骤 4：分别掺杂两组 S、D

**① NMOS 区（P衬底上）**  
遮住 N阱，只开 NMOS 有源区；注入 **砷（N型）** → **N+ 源、N+ 漏**。  
栅挡住离子 → 栅下仍是纯净 P（未来 N沟道）。

**② PMOS 区（N阱内）**  
换掩膜遮住 NMOS；注入 **硼（P型）** → **P+ 源、P+ 漏**。  
栅挡住离子 → 栅下仍是纯净 N（未来 P沟道）。

至此两管结构成型：

| | NMOS | PMOS |
|--|------|------|
| 衬底 | 原生 P | **人工 N阱** |
| S/D | N+ | P+ |
| 栅 | 栅氧 + 多晶硅 | 同左 |

#### 步骤 5：接触孔 + 金属布线（连成 CMOS 反相器）

1. 整片再覆绝缘 SiO₂  
2. 光刻接触孔，扎到 S/D、栅、衬底、N阱  
3. 淀积铝/铜，光刻出导线，按反相器接线：

| 连线 | |
|------|--|
| PMOS 源 → **VDD** | NMOS 源 → **GND** |
| 两漏短接 → **OUT** | 两栅短接 → **A** |
| **P衬底 → GND** | **N阱 → VDD**（防结正向漏电） |

#### 步骤 6：钝化保护

覆 SiO₂/氮化硅保护层，只露焊盘 → 晶圆完成，再切割封装。

#### 对应你之前三个疑问

| 问 | 答 |
|----|-----|
| 一片硅怎能同时有两种管？ | 原生 P 放 NMOS；局部注入造 **N阱** 放 PMOS |
| 阱电位（防烧毁） | **P衬底=GND，N阱=VDD** → 相关 PN结全 **反偏** |
| CMOS 是管子吗？ | **不是**；晶圆上造出独立 NMOS、PMOS，再金属 **互补连线** 才叫 CMOS 电路 |

#### 电位规则再钉（防 PN结正向漏电）

| 谁 | 接哪里 |
|----|--------|
| 整片 P-Sub | **GND（最低）** |
| N阱 | **VDD（最高）** |

#### 拓展：双阱（了解）

先进工艺同时做 **N阱（PMOS）+ P阱（NMOS）** → 衬底电位可独立调，抑闩锁；CPU/FPGA 常见。  
闩锁（Latch-up）：P/N阱与源漏寄生可控硅，异常大电流可烧毁 → 保护环 + 正确电位。

软件 / CSAPP / HFT：记住 **「P 上 N管、阱里 P管」+ 衬底接地阱接电源 + 金属连成互补**；光刻步序知概念即可。

### 5. 名词收口（再钉一次）

| 词 | 指什么 |
|----|--------|
| **MOS** | 一类晶体管的统称（含 NMOS / PMOS） |
| **NMOS** | **单管**：N沟道，电子导电 |
| **PMOS** | **单管**：P沟道，空穴导电 |
| **CMOS** | **电路体系**：NMOS+PMOS **互补配对**（反相器起），不是新型单管 |

> C = Complementary；MOS = 金属氧化物半导体管；CMOS = 互补型 MOS **电路**。

---

### 和单个 NMOS 笔记的衔接

单个管：VGS 造沟道、VDS 推电流、结锁反向、电流走表层。  
CMOS：用 **一对** 管，同一栅信号下互补开关 → 组成非门/与非/或非 → 再堆成整个数字芯片。

软件 / CSAPP / HFT 主线：记住 **互补 + 低静态功耗 + 非门 +「P衬底放 N管 / N阱放 P管」+ 衬底接地、阱接 VDD**；双阱/闩锁知概念即可。

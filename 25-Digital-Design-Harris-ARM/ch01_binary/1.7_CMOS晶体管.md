### 1. 单词拆分

| | |
|--|--|
| **C** | Complementary **互补** |
| **MOS** | Metal-Oxide-Semiconductor 金属氧化物半导体管 |
| **CMOS** | **互补型 MOS 电路** |

成对使用两种管子：

| | **NMOS**（你刚学的） | **PMOS**（反向版） |
|--|----------------------|---------------------|
| 衬底 / 源漏 | P衬底 + N源漏 | N衬底（或 N阱）+ P源漏 |
| 何时导通 | 栅加 **高** 电压 | 栅加 **低** 电压 |
| 沟道载流子 | **电子** | **空穴** |

细节机理（沟道、VGS/VDS、不穿 PN结）→ [补充_PN结到MOS](./补充_PN结到MOS.md)。

### 四图对照（看图记知识点）

学 CMOS 时手里常有四类图，各盯一件事。下面用示意标出该看什么（有实拍/课本图时按同样标注对号入座）。

#### 一、NMOS 单管剖面图（你一直在学的那根管子）

```
        G（栅极金属/多晶硅）
        ═══════
       / SiO₂  \     ← 绝缘，电子不过栅
  ────┴─────────┴──── 硅最表面
  [N+]  ~电子沟道~  [N+]
   S    （临时薄层）  D
   │ ╲             ╱ │
   │  ╲__ PN结 __╱  │   ← 侧壁·底面 ↔ P（正式结，朝下）
   └──────┬──────────┘
          P 衬底（整片）
```

| 标注 | 是什么 |
|------|--------|
| 底层粉色/整片 | **P型衬底** |
| 左右蓝 N+ 方块 | **源 S、漏 D** |
| S/D 底面·侧壁 ↔ P | **实体 PN结** |
| SiO₂ 上方 | **栅极 G**（与硅完全绝缘） |
| 栅下 P 表层 | **临时电子 N沟道**（只横向连 S/D **顶面**，不穿过下方结） |

→ 对号：[补充_PN结到MOS](./补充_PN结到MOS.md)「顶面无结 / 侧壁底面有结 / 电子只走表层」。

#### 二、CMOS 反相器电路原理图（最基础 CMOS 单元）

```
              VDD
               │
            源 │
         ┌─────┴─────┐
         │   PMOS    │  ← 上：P沟道，源接 VDD
         │   （P）   │
         └─────┬─────┘
            漏 │
    A ─────────┼────────── Y（输出）
     （两栅相连）│
         ┌─────┴─────┐
         │   NMOS    │  ← 下：N沟道，源接 GND
         │   （N）   │
         └─────┬─────┘
            源 │
              GND
```

| 点 | 记住 |
|----|------|
| 上管 | **PMOS**，源极接 **VDD** |
| 下管 | **NMOS**，源极接 **GND** |
| 输入 A | **两栅连一起** |
| 输出 Y | **两漏连一起** |
| 互补 | 输入高 → NMOS开、PMOS关；输入低 → 反过来；**永远一路通**，静态几乎不耗电 |

#### 三、CMOS 芯片硅片横截面（同一块硅上做两种管）

```
  G        G
  ║        ║
 ═╩═      ═╩═     ← 各自栅 + 氧化层
[N+][N+] [P+][P+]
  S  D     S  D
  NMOS      PMOS
────┬──────────┬────
    │  P衬底   │ N阱 │  ← 右边挖出 N阱放 PMOS
    └──────────┴────┘
```

| 工艺点 | |
|--------|--|
| 基底 | 整片 **P型硅** |
| 右边 | 挖 **N阱**，里面做 **PMOS**（PMOS 需要 N衬底环境） |
| 左边 | 直接在 P衬底做 **NMOS** |
| NMOS 的 S/D | **N+** 掺杂 |
| PMOS 的 S/D | **P+** 掺杂 |
| 结与沟道 | 各管源漏侧壁·底面与衬底/阱形成独立 PN结；表层沟道横向导电，避开结 |

#### 四、CMOS 版图 / 显微镜实拍（实物排布）

课本或显微镜图里常见：

- **上方长条区域** ≈ PMOS  
- **下方区域** ≈ NMOS  
- **同一输入** 用共用 **栅极金属/多晶走线** 串起来  

看图目的：建立「真实芯片里两种管并排、共栅」的直觉，不必背版图规则。

| ② 反相器电路 | CMOS **互补成对** 的接线逻辑 |
| ③ CMOS 剖面 | 一块硅上同时做 NMOS + PMOS（**N阱**） |

### 2. 「互补」是什么意思？

同一输入下，两管永远 **一个开、一个关**，稳态不同时导通。

**最简单单元：CMOS 反相器（非门）**

```
        VDD
         │
       [PMOS]  ← 上管
         │
    ─────┼───── 输出
         │
       [NMOS]  ← 下管
         │
        GND

输入同时接两管栅极
```

| 输入 | NMOS | PMOS | 输出 |
|------|------|------|------|
| **高** | 导通 | 截止 | 拉到 **0**（地） |
| **低** | 截止 | 导通 | 拉到 **1**（VDD） |

→ nMOS 擅长拉 **0**；pMOS 擅长拉 **1** —— 和 [1.5](./1.5_逻辑门.md) 的非门对应。

### 3. CMOS 最大优势：几乎无静态电流

| | 老式纯 NMOS 等 | **CMOS** |
|--|----------------|----------|
| 稳态 | 导通时可能持续耗电、发热 | 永远只有一管通、另一管断 |
| 静态电流 | 大 | **几乎为 0**（理想） |

手机、CPU、单片机几乎全是 CMOS，核心原因之一就是 **功耗极低**。  
（动态翻转时仍有充放电电流 → 见 [1.8 功耗](./1.8_功耗.md)。）

---

### 4. 硅片上怎么做 CMOS（直觉）

基底常见 **P型衬底**：

1. **NMOS** 直接做在 P衬底上（就是你学过的结构）  
2. **PMOS**：先在 P衬底挖一块区域，注入做成 **N型隔离阱（N阱）**，再在 N阱里做 PMOS  

一块硅片上同时有 NMOS + PMOS → **CMOS 工艺**。

### 5. 简单区分名词

| 词 | 指什么 |
|----|--------|
| **MOS** | 一种晶体管（NMOS 或 PMOS 单个管子） |
| **NMOS** | N沟道管，电子导电 |
| **PMOS** | P沟道管，空穴导电 |
| **CMOS** | **同时搭配** NMOS+PMOS 的成套电路体系，互补工作 |

### 和单个 NMOS 笔记的衔接

单个管：VGS 造沟道、VDS 推电流、结锁反向、电流走表层。  
CMOS：用 **一对** 管，同一栅信号下互补开关 → 组成非门/与非/或非 → 再堆成整个数字芯片。

软件 / CSAPP / HFT 主线：记住 **互补 + 低静态功耗 + 非门结构** 即可；阱工艺细节可不抠。

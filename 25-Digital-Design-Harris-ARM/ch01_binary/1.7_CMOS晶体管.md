### 1. 单词拆分

| | |
|--|--|
| **C** | Complementary **互补** |
| **MOS** | Metal-Oxide-Semiconductor 金属氧化物半导体管 |
| **CMOS** | **互补型 MOS 电路** |

成对使用两种管子：

| | **NMOS**（你刚学的） | **PMOS**（反向版） |
|--|----------------------|---------------------|
| 衬底 / 源漏 | P衬底 + N源漏 | N衬底（或 N阱）+ P源漏 |
| 何时导通 | 栅加 **高** 电压 | 栅加 **低** 电压 |
| 沟道载流子 | **电子** | **空穴** |

细节机理（沟道、VGS/VDS、不穿 PN结）→ [补充_PN结到MOS](./补充_PN结到MOS.md)。

### 2. 「互补」是什么意思？

同一输入下，两管永远 **一个开、一个关**，稳态不同时导通。

**最简单单元：CMOS 反相器（非门）**

```
        VDD
         │
       [PMOS]  ← 上管
         │
    ─────┼───── 输出
         │
       [NMOS]  ← 下管
         │
        GND

输入同时接两管栅极
```

| 输入 | NMOS | PMOS | 输出 |
|------|------|------|------|
| **高** | 导通 | 截止 | 拉到 **0**（地） |
| **低** | 截止 | 导通 | 拉到 **1**（VDD） |

→ nMOS 擅长拉 **0**；pMOS 擅长拉 **1** —— 和 [1.5](./1.5_逻辑门.md) 的非门对应。

---

### 3. CMOS 最大优势：几乎无静态电流

| | 老式纯 NMOS 等 | **CMOS** |
|--|----------------|----------|
| 稳态 | 导通时可能持续耗电、发热 | 永远只有一管通、另一管断 |
| 静态电流 | 大 | **几乎为 0**（理想） |

手机、CPU、单片机几乎全是 CMOS，核心原因之一就是 **功耗极低**。  
（动态翻转时仍有充放电电流 → 见 [1.8 功耗](./1.8_功耗.md)。）

---

### 4. 硅片上怎么做 CMOS（直觉）

基底常见 **P型衬底**：

1. **NMOS** 直接做在 P衬底上（就是你学过的结构）  
2. **PMOS**：先在 P衬底挖一块区域，注入做成 **N型隔离阱（N阱）**，再在 N阱里做 PMOS  

一块硅片上同时有 NMOS + PMOS → **CMOS 工艺**。

---

### 5. 简单区分名词

| 词 | 指什么 |
|----|--------|
| **MOS** | 一种晶体管（NMOS 或 PMOS 单个管子） |
| **NMOS** | N沟道管，电子导电 |
| **PMOS** | P沟道管，空穴导电 |
| **CMOS** | **同时搭配** NMOS+PMOS 的成套电路体系，互补工作 |

---

### 和单个 NMOS 笔记的衔接

单个管：VGS 造沟道、VDS 推电流、结锁反向、电流走表层。  
CMOS：用 **一对** 管，同一栅信号下互补开关 → 组成非门/与非/或非 → 再堆成整个数字芯片。

软件 / CSAPP / HFT 主线：记住 **互补 + 低静态功耗 + 非门结构** 即可；阱工艺细节可不抠。

## 4. 交易成本冰山 (Plexus Iceberg)

**Plexus Group** 隐喻：

```
        ╱ 显性：佣金、部分 impact  ╲   ← 水面之上
═══════╪═══════════════════════════╪═══════
        ╲  经理择时成本              ╱
         ╲ 交易员择时成本（交单延误）╱   ← 水面之下
          ╲  错失机会成本           ╱
```

| 水下成本 | 说明 |
|----------|------|
| **经理择时** | **决定交易** → **提交交易台** 期间不利变动 |
| **交易员择时** | **交易台** → **交给经纪人** 期间延误与变动 |
| **错失机会** | 未成交损失 |

> **水下隐形成本往往远大于水上可见成本**

| HFT 视角 |
|----------|
| **Latency** 压缩 **交易员择时** 层 — HFT 价值主张之一 |
| **PM alpha timing** 若差 → IS 巨大，与 **执行无关** 也记在执行账上 |

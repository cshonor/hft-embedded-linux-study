## 3. 反对内幕交易的理由（为何要限制？）

### 3.1 公平性与投资者信心 (Fairness)

| 论点 | 公众认为 **不公平** |
|------|-------------------|
| **效果** | 限制 → **信心增强** → 更多资金入市 → 股价上升 → **企业资本成本降低** |

→ [Ch 9 公共益处](../chapter-09-good-markets/)

### 3.2 损害流动性、增加交易成本 (Liquidity)

| 机制 | 内幕者 = **极端知情交易者** [Ch 10](../chapter-10-informed-traders-market-efficiency/) |
|------|-----------------------------------------------------------------------------|
| **后果** | 做市商 / 限价单交易者 → **拉宽 spread** 自保 [Ch 14](../chapter-14-bid-ask-spreads/) |
| **买单者** | **不知情投资者** 付 **更高交易成本** |

| HFT 视角 |
|----------|
| **Toxic flow 极端版** — 若市场 **怀疑** 内幕活跃，**spread 结构性更宽** |
| Maker 无法区分 **合法 news** vs **illegal inside** 于毫秒级 — **事前 spread 含风险溢价** |

### 3.3 公司控制与代理问题 (Corporate Control) — **最重要理由**

| 若允许内幕 | 管理者可能 **滥用职权** |
|------------|------------------------|
| **例子** | 为最大化 **信息优势** 选 **高风险项目** |
| | 回购 / 并购前 **抢先交易 (Front-run)** 股东 |
| **损害** | **直接稀释股东利益** |

→ [Ch 7 委托代理](../chapter-07-brokers/) · **公司治理** 主线

| HFT 视角 |
|----------|
| 与 **HFT 微观结构 PnL** 间接相关 — **长期市场质量 / 资本成本** |
| **Corporate action** 交易须 **合规防火墙** |

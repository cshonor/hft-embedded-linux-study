# Ch 28 泡沫、崩盘与熔断机制 · Bubbles, Crashes, and Circuit Breakers

> **Trading and Exchanges** · Larry Harris · **选读** · Part VII

本章探讨 **极端市场波动** 的成因，以及监管机构为控制剧烈波动采取的 **熔断 (Circuit breakers)** 等干预措施的 **利弊**。

> **HFT 读者：** **Flash crash、volatility halt、portfolio insurance 类比、引力效应** — 策略须含 **regime / halt 状态机**；与 [Ch 20 波动分解](./chapter-20-波动性.md)、[Ch 11 动能/预期者](./chapter-11-指令预期者.md)、[Ch 16 价值](./chapter-16-价值交易者.md)、[Ch 6 集合竞价](./chapter-06-指令驱动市场.md)、[Ch 27 系统容量](./chapter-27-场内交易与自动交易系统.md) 衔接。

---

## 0. 本章主线

```
泡沫动力学（乐观 → 动能 → 预期者）→ 崩盘加速（margin、止损）
        ↓
基本面 vol + 暂时性 vol 混合
        ↓
熔断：支持者理由 vs 引力效应反对
        ↓
监管政治经济学 — 干预≠防过度反应；价值交易者纠错才是根本
```

## 1. 泡沫与崩盘的本质与动力学

### 1.1 泡沫 (Bubbles)

| 定义 | 价格 **远高于** 基本面价值 |
|------|---------------------------|
| **形成** | 对新技术等 **过度乐观** |
| **助推** | [Ch 11 动能交易者](./chapter-11-指令预期者.md) **盲目跟风** |
| | 指令预期者 **抢在下跌前** 获利 → 进一步 **推高** |

→ [Ch 12 虚张声势](./chapter-12-虚张声势者与市场操纵.md) · momentum 受害/助推

### 1.2 崩盘 (Crashes)

| 定义 | 价格 **极速下跌** — 「市场断裂 / 熔毁 (Market breaks / meltdowns)」 |
|------|---------------------------------------------------------------------|
| **时机** | 常于 **泡沫破裂后** |

### 1.3 触发与加速

| 触发 | [Ch 16 价值交易者](./chapter-16-价值交易者.md) / [Ch 17 套利者](./chapter-17-套利者.md) 认为过高 **抛售**；或 **坏消息** |
|------|------------------------------------------------------------------------------------------------------------------|
| **加速** | **保证金追加 (Margin calls)** → 杠杆户 **被迫平仓** |
| | **止损单** 大量触发 → 价格下跌 **自我强化** |

→ [Ch 11 gunning the market](./chapter-11-指令预期者.md)

### 1.4 死猫反弹 (Dead cat bounce)

| 机制 | 恐慌 + **流动性巨大需求** → 价格 **跌破** 基本面 |
|------|-----------------------------------------------|
| **反弹** | 交易者意识到 **过度反应** → 价格回升 |

→ [Ch 19 弹性](./chapter-19-流动性.md) · [Ch 16 价值回归](./chapter-16-价值交易者.md)

| HFT 视角 |
|----------|
| **Cascade / stop run** — 微结构加速环 |
| **Liquidity vacuum** — MM **撤单** 加剧崩盘 |
| **Mean reversion** 策略在 dead cat 段 **高 variance** |

## 2. 极端波动的两种类型

[Ch 20](./chapter-20-波动性.md) 分解在极端情形下的体现：

| 类型 | 泡沫/崩盘中的角色 |
|------|-------------------|
| **基本面波动性** | 最初对 **真实好消息** 的反应（合理部分） |
| **暂时性波动性** | **伪知情** 过度推高；崩盘时 **恐慌抛售** |

> 大多数泡沫与崩盘 **两者兼有**。

| HFT 视角 |
|----------|
| **Regime filter** — fundamental shock vs panic microstructure |
| 勿把 **halt 前后** 的 vol 混为一谈 |

## 3. 历史案例（微观结构视角）

### 3.1 1987年10月股灾

| 事实 | 道指 **单日 -23%** |
|------|-------------------|
| **核心微观原因** | **投资组合保险 (Portfolio insurance)** — 下跌时 **系统性自动卖出** |
| **预期效应** | 人人知大卖将在跌时出现 → 买家 **撤退** → **流动性真空** |
| **基础设施** | 订单 **打印机瘫痪** — 不知是否成交 → **信息盲区** → 恐慌加剧 |

→ [Ch 27 系统容量](./chapter-27-场内交易与自动交易系统.md) · **feedback trading** 现代版：杠杆 ETF、vol target

| HFT 视角 |
|----------|
| **2010 Flash Crash** — 类似 **流动性真空 + 系统压力** |
| **Kill switch**、**fat finger** 防护 |

### 3.2 冷聚变与钯金泡沫 (1989)

| 事件 | 「冷聚变」宣布 → **钯金期货暴涨** → 实验无法复现 → **暴跌** |
|------|-------------------------------------------------------------|
| **分类** | 罕见 **纯基本面波动** 泡沫 — 当时信息下 **投机者预期合理** |

| HFT 视角 |
|----------|
| **Binary event risk** — 生物科技、并购 |
| 与 **meme / narrative** 泡沫对比 |

## 4. 熔断机制 (Circuit Breakers)

应对极端波动的 **限制性交易规则** 总称。

### 4.1 常见类型

| 类型 | 说明 |
|------|------|
| **交易暂停 (Trading halts)** | 停盘冷却 |
| **价格限制 / 涨跌幅 (Price limits)** | 单日波动上限 |
| **提高保证金** | 抑制杠杆 |
| **交易税** | 抑制投机频率 |
| **系统访问限制 (Collars)** | 限制程序/指数套利等 |

### 4.2 支持者观点

| 理由 | 说明 |
|------|------|
| **减缓变动** | 给交易者 **分析信息** 时间 |
| **流动性入场** | 愿提供流动性的 **知情/价值** 者有机会介入 |
| **恢复机制** | 暂停后常以 **单一价格拍卖 (Uniform pricing)** 恢复 — 保护限价单免受连续暴跌 [Ch 6](./chapter-06-指令驱动市场.md) |
| **清算系统** | 防 **破产违约** 连锁 |

### 4.3 反对者：引力效应 (Gravitational Effect)

| 机制 | 交易者怕 **即将触及熔断** → 流动性丧失 |
|------|----------------------------------------|
| **行为** | **争先恐后** 提前下单 |
| **结果** | 熔断可能 **增加暂时性波动** — 价格 **更快被拉向** 熔断点 |

| HFT 视角 |
|----------|
| **Volatility auction trigger** — 策略 **front-run the halt** |
| **LULD bands** — 价格 **磁吸** 阈值 |
| 回测须 **含 halt 规则**，否则 **虚假 alpha** |

## 5. 监管干预的政治经济学

### 5.1 监管风险 (Regulatory Risk)

| 不对称压力 | **不熔断且崩盘** → 公众指责 **不作为** |
|------------|----------------------------------------|
| | **采取熔断（即使无效）** → 无法指责 **没尝试** |
| **结果** | 监管者个人激励 → 倾向 **温和熔断** 平息舆论 — **即使经济学价值有限** |

### 5.2 监管俘获 (Regulatory Capture)

| 例子 | 1987 后 **NYSE Rule 80A** — 限制 **指数套利** |
|------|---------------------------------------------|
| **实际效果** | 打击 **跨市场搬运流动性** 的套利者 [Ch 17](./chapter-17-套利者.md) |
| **受益者** | 保护 [Ch 24 专家](./chapter-24-专家做市商系统.md) 免受套利竞争 |
| **本质** | 利益集团博弈 — **非纯公众利益** |

| HFT 视角 |
|----------|
| **Rule 611、uptick、colocation 政策** — 持续政治经济 |
| 读监管提案问：**保护谁？liquidity 还是 incumbent？** |

## 6. 根本启示（作者结论）

| 干预局限 | 事后熔断 **至多** 让极端变动 **稍显有序** — **不能防止过度反应** |
|----------|------------------------------------------------------------------|
| **防泡沫更好办法** | 降低 **价值交易者** 获取 **高质量信息** 的成本 |
| | 赋予其信心 **及时纠正错误定价** [Ch 16](./chapter-16-价值交易者.md) · [Ch 9](./chapter-09-好市场.md) |

```
熔断 = 症状管理
价值交易者 + 信息效率 = 结构纠错
```

| HFT 视角 |
|----------|
| HFT 作为 **短期 LP** 在崩盘中 **常撤离** — 不能指望熔断 **替代** 价值资本 |
| **Market making obligation** 在 crisis 的 **有限性** |

## 7. 参与者行为对照（崩盘时）

| 角色 | 典型行为 |
|------|----------|
| **动能 / 预期者** | 追涨 → 恐慌杀跌，加速崩盘 |
| **价值 / 套利** | 高估时卖；超跌时买 — **弹性来源** |
| **做市商** | 撤单或 **极宽 spread** — 流动性真空 |
| **杠杆户** | Margin call **强制卖** |
| **监管** | Halt、limit — **引力效应** 风险 |

---

## 8. 本章总结

| 要点 | 含义 |
|------|------|
| **泡沫** | 乐观 + 动能 + 预期者推高 |
| **崩盘** | 破裂 + margin + 止损 **加速**；可能 **死猫反弹** |
| **Vol 类型** | Fundamental + **transitory** 恐慌 |
| **1987** | Portfolio insurance + **流动性真空** + **系统瘫痪** |
| **熔断** | 有益论 vs **引力效应** |
| **政治** | 监管风险、**俘获** — 干预动机不纯 |
| **根本** | 强化 **价值纠错**，非仅靠 halt |

> **HFT 读者 takeaway：** 策略 **状态机** 须含：`normal` → `stressed` → `halt` → `auction reopen`。回测 **无熔断** = 过拟合。**引力效应** 下勿在阈值附近 **加杠杆**。全书微观结构收束前最后一章制度风险 — 下一章 [Ch 29 内幕交易](./chapter-29-内幕交易.md) 回到 **信息边界**。

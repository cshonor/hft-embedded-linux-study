# Ch 3 §4 交易市场的分布 · Where are the Trading Markets?

> **Trading and Exchanges** · Larry Harris · **精读**

## 4. 交易市场的分布 (Where are the Trading Markets?)

Harris 本章容易混成一锅的两个维度 —— **必须分开**：

| 维度 | 问什么 | 类比 |
|------|--------|------|
| **资产类别 (Asset class)** | 交易 **什么**？ | **「商品」** — 股票、债券、期货、期权、FX… → 见 [§3 交易工具](./section-3-3-交易工具.md) |
| **市场分布 / 层级 (Market tier & venue)** | 在 **哪条渠道、哪层市场** 成交？ | **「卖商品的渠道」** — 发行、交易所、场外大宗、暗池… |

```
        资产类别（卖什么）              市场层级（在哪卖）
        ───────────────              ─────────────────
        股票 ─────────────┬──→  第一市场：IPO 发行
        债券 ─────────────┤    第二市场：NYSE / Nasdaq
        期货 ─────────────┤    第三市场：机构 OTC 大宗
        期权 ─────────────┤    第四市场：ECN / 暗池 直接撮合
        FX   ─────────────┘         …
```

**本节 §4.1–§4.2 讲「渠道/层级」**（以 **上市股票** 为第一到第四市场的标准叙事）；**§4.3 讲「各类资产通常走哪些渠道」** — 两维 **交叉**，不是同一分类法。

---

## 4.1 第一到第四市场 — 渠道/层级（以股票为主）

美国教材里 **Primary / Secondary / Third / Fourth market** 最初描述的是 **已上市股票** 在 **不同交易场所** 的分布 —— **不是** 按「股票 vs 债券」分的。

| 层级 | 英文 | 是什么 | Harris / 笔记案例 |
|------|------|--------|-------------------|
| **第一市场** | **Primary market** | **发行 / IPO** — 企业 **首次** 把证券卖给公众；**不是二级市场买卖** | 本书 Ch 3 **不展开** IPO（Harris 边界：不讲公司融资细节） |
| **第二市场** | **Second market** | **常规股票交易所** — 我们常说的 **NYSE、Nasdaq**；公开 **报价–撮合** | [Ch 2 §1 Jennifer](../../chapter-02-trading-stories/notes/section-1-1-散户股票交易.md) |
| **第三市场** | **Third market** | **机构间大宗场外** — 上市股 **不在交易所竞价**，而在 **场外做市商 / Upstairs** 议价、大块成交 | [Ch 2 §3 Edna · Upstairs](../../chapter-02-trading-stories/notes/section-3-3-超大宗股票抛售.md) · [§2.4 OTC](./section-2-2-交易促成机构.md#24-otc场外交易--over-the-counter) |
| **第四市场** | **Fourth market** | **直接撮合、绕过传统 floor** — **ECN、ATS、暗池** 等；参与者 **直接** 或 **匿名** 交叉，不经过大厅专家 | [Ch 2 §2 Bob · POSIT 暗池](../../chapter-02-trading-stories/notes/section-2-2-机构股票交易.md) · [§2.2 ECN / §2.3 暗池](./section-2-2-交易促成机构.md) |

```
同一只上市股（资产 = 股票）可以走不同「渠道」：

  IPO 承销           →  第一市场（发行）
  散户 App 下单 NYSE  →  第二市场（交易所）
  Edna 90 万股 Upstairs →  第三市场（场外大宗）
  Bob POSIT 暗池交叉   →  第四市场（ATS/暗池）
```

**HFT：**

| 层级 | 典型关联 |
|------|----------|
| **第二 + 第四** | **主战场** — 公开 LOB、ECN、colocation、SOR |
| **第三** | 与 tick 热路径 **分离**；影响 **公开簿深度**（大单不进 LOB） |
| **第一** | 通常 **非 HFT 高频** 范畴（IPO 分配、承销） |

**Reg NMS：** 第二、第四市场并存 → **best execution** 须 **跨 venue** 找最优价 — 见 [§2.2 ECN](./section-2-2-交易促成机构.md)。

## 4.2 第一–第四市场 vs §2 促成机构（对齐表）

| §4 层级（渠道） | §2 促成机构（设施类型） | 关系 |
|-----------------|-------------------------|------|
| **第二市场** | **交易所** (NYSE/Nasdaq) | 第二市场 **核心载体** |
| **第四市场** | **ECN、暗池 (ATS)** | 第四市场 **主要组成** |
| **第三市场** | **OTC / Upstairs、经销商** | 第三市场 **偏议价**，不经集中公开簿 |
| **第一市场** | 投行 **承销 syndicate** | **发行** 渠道，非 LOB 撮合 |

> **别混：** 「第三市场 = OTC 大宗」≠ 「所有 OTC 都是第三市场」— **债券、FX** 几乎 **全 OTC**，但 **没有「上市第二市场」** 这一说；见 §4.3。

## 4.3 按资产类别：通常走哪些渠道（「商品 × 渠道」）

同一套 **第一–第四市场** 术语 **源自股票**；其他资产 **渠道分布不同** —— 用 **两维交叉** 理解：

### 股票 (Equities)

| 渠道 | 常见程度 |
|------|----------|
| 第二市场（交易所） | ✅ 零售 + 机构 **主路径** |
| 第四市场（ECN/暗池） | ✅ 机构块、HFT **并行** |
| 第三市场（Upstairs OTC） | ✅ 超大块 |
| 第一市场（IPO） | 仅 **发行阶段** |

### 期权与期货 (Options & Futures)

| 特点 | 说明 |
|------|------|
| **资产** | 衍生品 — [§3](./section-3-3-交易工具.md) |
| **渠道** | **专门交易所** — CBOE、CME/CBOT 等；**同一合约通常单 venue** |
| **与第一–四市场** | **不用** 股票那套第三/四市场标签；竞争靠 **新合约设计**、**清算所** |
| **HFT** | **单 venue 单合约** → 延迟竞争 **更集中** — [Ch 2 §4 期货](../../chapter-02-trading-stories/notes/section-4-4-期货市场-套期保值.md) |

### 债券 (Bonds)

| 特点 | 说明 |
|------|------|
| **资产** | 固定收益 |
| **渠道** | **绝大多数 OTC** — 经销商询价；[Ch 2 §6 Sam](../../chapter-02-trading-stories/notes/section-6-6-债券市场交易.md) |
| **与第一–四市场** | **无典型「第二市场集中竞价」**；不是「第四=暗池」的简单映射 |

### 外汇 (FX)

| 特点 | 说明 |
|------|------|
| **资产** | 货币对 |
| **渠道** | **现货 FX 以 OTC 为主**；**电子化平台**（EBS、Reuters 等）= **第四市场精神** 的 **直接撮合**，但业界 **少叫「第四市场」** |
| **HFT** | FX **电子化程度高**；公司债 LOB 式 HFT **niche** |

## 4.4 两维矩阵（速查）

| 资产 ↓ \ 渠道 → | 第一（发行） | 第二（交易所） | 第三（OTC 大宗） | 第四（ECN/暗池） |
|-----------------|-------------|---------------|-----------------|-----------------|
| **股票** | IPO | NYSE/Nasdaq ✅ | Upstairs ✅ | POSIT/ECN ✅ |
| **期货/期权** | 新合约上市 | CME/CBOE ✅ | 少 | 单合约单所为主 |
| **债券** | 首发 | 少数上市债 | — | **OTC 为主 ✅** |
| **FX** | — | 部分 | 银行间 | **电子 OTC ✅** |

## HFT 读者 takeaway

1. **先问资产** — 股票 / 期货 / 债 / FX？（§3）  
2. **再问渠道** — 发行、公开所、场外大宗、还是暗池/ECN？（§4.1）  
3. **go-dex 现在** = 只模拟 **第二市场精神** 的 **单一公开 LOB**；**第三 OTC + 第四 暗池** = M5+

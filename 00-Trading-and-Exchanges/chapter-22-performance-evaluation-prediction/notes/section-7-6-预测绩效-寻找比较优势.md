## 6. 预测绩效：寻找比较优势 (Comparative Advantage)

既然 **统计过去业绩不能有效预测未来** — 如何判断谁能成功？

### 6.1 唯一可靠方向

分析经理人是否具备 **真实的比较优势** — 而非 extrapolate 历史曲线。

### 6.2 入场前自问

在以下方面是否拥有 **他人不具备的绝对且相对优势**：

| 维度 | 例子 |
|------|------|
| **智力** | 建模、推理 |
| **经验** | 经历过多种 regime |
| **教育背景** | 领域专长 |
| **组织能力** | 系统、流程、风控 |
| **纪律性** | 执行一致性 |
| **信息获取渠道** | 合法、可持续的数据 edge |

> 交易是 **零和博弈** — 光是表现「好」不够，必须比 **竞争对手更好**。

| HFT 视角 |
|----------|
| **Comparative advantage** = co-lo + feed + 工程 + **合法信息** + **资本** 的组合 |
| 无优势 → 回到 Ch 14 **不知情结构性亏损** |
| **Edge 可持续性** — 技术会被复制；**关系 / 数据独占 / 监管许可** 更 durable |

# Ch 22 绩效评估与预测 · Performance Evaluation and Prediction

> **Trading and Exchanges** · Larry Harris · **选读** · Part VI

本章探讨如何衡量 **投资组合经理与交易者** 的 **历史表现**，以及为何 **仅凭过去业绩预测未来回报极其不可靠**。

> **HFT 读者：** 策略 **回测 / Sharpe / t-stat** 的 **统计学陷阱** 本章全涵盖；与 [Ch 8 零和](../chapter-08-why-people-trade/)、[Ch 21 TCA](../chapter-21-transaction-cost-measurement/)、[Ch 10 edge](../chapter-10-informed-traders-market-efficiency/) 衔接 — **「过去好看 ≠ 未来能活」**。

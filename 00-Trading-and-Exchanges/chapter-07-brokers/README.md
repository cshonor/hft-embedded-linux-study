# Ch 7 经纪人 · Brokers

> **Trading and Exchanges** · Larry Harris · **选读** · Part I

本章全面探讨 **经纪人在交易产业中的角色**、经纪公司 **内部架构**、**盈利模式**，以及委托人与经纪人之间不可避免的 **委托代理问题** 与潜在 **欺诈行为**。

> **HFT 读者定位：** 直连 (DMA) / co-location 绕过了「人工经纪人」，但 **清算结算、会员资格、路由与 best execution 义务** 仍在；PFOF、internalization 是本章经济学在现代的延续 → 见 [Ch 25](../chapter-25-internalization-preferencing-crossing/)。

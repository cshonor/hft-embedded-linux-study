# Ch 16 价值交易者 · Value Traders

> **Trading and Exchanges** · Larry Harris · **选读** · Part IV · **Ch 10 知情四类之分册**

本章深入 **价值交易者 (Value Traders)** 的行为模式 — 颠覆「投机者不供流动性」的刻板印象：他们实际上是市场中 **最重要的流动性提供者之一**，并剖析核心经济学难题 **「赢家的诅咒 (Winner's Curse)」**。

> **HFT 读者：** 与 **mean reversion / fundamental fair value** 策略相关；理解 **outside spread vs dealer spread** 解释 **为何 LOB 很窄仍有大额 mispricing**；[Ch 10](../chapter-10-informed-traders-market-efficiency/) · [Ch 12 虚张声势克星](../chapter-12-bluffers-market-manipulation/) · [Ch 13 价值帮 unload](../chapter-13-dealers/) 衔接。

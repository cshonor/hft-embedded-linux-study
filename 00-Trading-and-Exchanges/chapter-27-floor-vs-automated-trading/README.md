# Ch 27 场内交易与自动交易系统 · Floor Versus Automated Trading Systems

> **Trading and Exchanges** · Larry Harris · **精读** · Part VII

本章对比 **传统人工场内交易大厅 (Floor)** 与 **现代电子自动交易系统 (Automated)** — 技术进步虽使 **完全自动化** 可行，许多高流动交易所（如 **NYSE**）仍保留场内成分。分析两种结构的 **优劣势** 与 **为何共存**。

> **HFT 读者：** 你工作在 **自动系统极端**；本章解释 **为何 block / upstairs 仍要人声**、**audit trail 为何是监管资产**、以及 **电子化未消灭的信息交换需求**（→ dark、IOI、Ch 15）。与 [Ch 6 口头拍卖](../chapter-06-order-driven-markets/)、[Ch 15](../chapter-15-block-traders/)、[Ch 24](../chapter-24-specialists/)、[Ch 26](../chapter-26-competition-within-among-markets/) 衔接。

## 2. 交易的公共益处 (Public Benefits)

**公共益处**：**无论是否直接参与交易**，社会从 **运转良好的市场** 中获得的收益 — 经济学称 **正外部性 (Positive externalities)**。

两大来源：**信息效率价格** + **市场流动性**。

### 2.1 信息效率价格的公共益处

良好市场 **整合信息**，产生接近 **基本面价值 (Fundamental values)** 的 **信息效率价格**，影响 **生产与资源分配 (Production & Allocation)**：

| 层级 | 机制 |
|------|------|
| **一级市场 · 资本分配** | 好项目 **高估值** → 易融资；差项目 **低估值** → 难融资 → 新资本流向 **更佳投资机会** |
| **二级市场 · 管理者分配** | 股价反映经营质量 → 股东用 **期权激励** 管理者；表现差 → **低股价** → **收购 (Takeover)** 压力 → 资源交给 **更优管理者** |

| HFT 视角 |
|----------|
| **Informed trading**（Ch 10）在 **私人利润** 与 **公共信息效率** 之间 **部分一致** — 监管常 **允许** 信息融入价格，但 **限制内幕** |
| **Parasitic / manipulation**（Ch 11–12）**扭曲价格** → 损害 **公共益处**（错误资本配置、错误并购信号） |
| **Quote stuffing / spoofing** 若污染 price discovery → **负外部性** |

### 2.2 高流动性市场的公共益处

高流动性 → **社会生产效率** 提升，不限于交易者「开心」：

| 益处 | 说明 |
|------|------|
| **交换** | 资源转移到 **估值最高** 的使用者手中，减少浪费 |
| **套期保值** | 农民、制造商等以 **低成本** 转移 **专业化风险** → 专注高效生产 → **社会商品价格更低** |
| **风险分担** | 大项目风险 **分散给大众**；企业 **低成本筹资** |

| HFT 视角 |
|----------|
| HFT **narrow spread / deep book** → 直接贡献 **公共流动性益处**（若 **真实 depth** 非 flickering quotes） |
| **Flash crash / liquidity illusion** → 短暂高流动性表象、危机时 evaporates → **公共损害** |
| **Internalization**（Ch 25）提高 **某些** 参与者体验，但可能 **降低公开市场深度** — 需 **净福利** 分析 |

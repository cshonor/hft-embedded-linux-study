# Ch 9 好市场 · Good Markets

> **Trading and Exchanges** · Larry Harris · **选读** · Part II · **福利经济学评估框架**

本章探讨 **如何评判市场是否「好」**，以及如何用 **福利经济学 (Welfare Economics)** 评估 **市场结构与监管政策**。评估标准落脚于：市场为 **个体** 与 **整个经济** 带来的 **益处**。

> **HFT 读者：** 监管辩论（maker-taker、PFOF、tick size、latency fairness）背后都是本章 **三目标优先级**；与 [Ch 8](./chapter-08-为什么人们要交易.md) 交易者分类、[Ch 10](./chapter-10-知情交易者与市场效率.md) 信息效率、[Ch 19](./chapter-19-流动性.md) 流动性度量直接衔接。

---

## 0. 核心判准（一句话）

**好市场 = 高流动性 + 信息效率价格**

二者降低 **效用型交易者** 成本，并充当自由市场经济 **资本与资源配置的指南针**。

---

## 1. 交易的私人益处 (Private Benefits)

**私人益处**：交易者在交易中 **直接获得** 的利益。

| 交易者类型 | 私人益处来源 |
|------------|--------------|
| **效用型** (Utilitarian) | 投资者、借款人、套保者、资产交换者等 **自愿参与** — 期望以 **较低成本** 解决实际问题（跨期转移财富、规避风险、换资产等） |
| **利润驱动型** (Profit-motivated) | **唯一目的：获利** |

### 效用型 > 利润驱动型（存在论）

交易在 **利润维度是零和**；若 **没有效用型** 愿参与并 **承担交易成本**，利润驱动型（投机者、做市商）**无法生存**。

> **政策含义：** 评估市场时，**效用型交易者福祉** 往往比 **利润驱动型** 更重要 — 他们是市场的 **存在理由**。

| HFT 视角 |
|----------|
| **Maker rebate / tight spread** 若吸引 LP 但 **吓跑 hedger**（adverse selection 过高）→ 可能 **损害首要目标** |
| TCA 不仅看 **机构 alpha**，也看 **issuer / hedger 总成本** — 与 **best execution 社会目标** 一致 |

---

## 2. 交易的公共益处 (Public Benefits)

**公共益处**：**无论是否直接参与交易**，社会从 **运转良好的市场** 中获得的收益 — 经济学称 **正外部性 (Positive externalities)**。

两大来源：**信息效率价格** + **市场流动性**。

### 2.1 信息效率价格的公共益处

良好市场 **整合信息**，产生接近 **基本面价值 (Fundamental values)** 的 **信息效率价格**，影响 **生产与资源分配 (Production & Allocation)**：

| 层级 | 机制 |
|------|------|
| **一级市场 · 资本分配** | 好项目 **高估值** → 易融资；差项目 **低估值** → 难融资 → 新资本流向 **更佳投资机会** |
| **二级市场 · 管理者分配** | 股价反映经营质量 → 股东用 **期权激励** 管理者；表现差 → **低股价** → **收购 (Takeover)** 压力 → 资源交给 **更优管理者** |

| HFT 视角 |
|----------|
| **Informed trading**（Ch 10）在 **私人利润** 与 **公共信息效率** 之间 **部分一致** — 监管常 **允许** 信息融入价格，但 **限制内幕** |
| **Parasitic / manipulation**（Ch 11–12）**扭曲价格** → 损害 **公共益处**（错误资本配置、错误并购信号） |
| **Quote stuffing / spoofing** 若污染 price discovery → **负外部性** |

### 2.2 高流动性市场的公共益处

高流动性 → **社会生产效率** 提升，不限于交易者「开心」：

| 益处 | 说明 |
|------|------|
| **交换** | 资源转移到 **估值最高** 的使用者手中，减少浪费 |
| **套期保值** | 农民、制造商等以 **低成本** 转移 **专业化风险** → 专注高效生产 → **社会商品价格更低** |
| **风险分担** | 大项目风险 **分散给大众**；企业 **低成本筹资** |

| HFT 视角 |
|----------|
| HFT **narrow spread / deep book** → 直接贡献 **公共流动性益处**（若 **真实 depth** 非 flickering quotes） |
| **Flash crash / liquidity illusion** → 短暂高流动性表象、危机时 evaporates → **公共损害** |
| **Internalization**（Ch 25）提高 **某些** 参与者体验，但可能 **降低公开市场深度** — 需 **净福利** 分析 |

## 3. 评估市场结构的三个目标 (Objectives)

基于私人 + 公共益处，评估 **公共政策 / 市场结构变化** 时的 **优先级**：

| 优先级 | 目标 | 内容 |
|:------:|------|------|
| **1** | **效用型私人益处** | 最大化效用型从 **高流动性市场** 获得的益处 — **没有他们，市场不存在** |
| **2** | **社会公共益处** | 最大化全社会从 **高流动性 + 信息效率价格** 获得的益处 — **整体繁荣与财富** |
| **3** | **有条件支持利润驱动型** | 支持做市商 **提供流动性**、知情者 **提高效率** — **仅当** 有利于全社会；**打击** 剥削型策略（操纵、虚张声势、抢先交易等）— 它们 **长期既不供流动性，也不提高效率** |

```
政策评估
    ① 效用型还愿意来吗？（成本、信任、对冲可行性）
    ② 价格还准吗？资本还配给对吗？
    ③ 投机/做市是「建设性」还是「寄生」？
```

| HFT 视角 |
|----------|
| **Reg NMS / MiFID II / 中国注册制** 等 — 可映射到 **三目标** 而非「禁止 HFT」口号 |
| **Maker-taker**：目标 1 — retail/institutional **taker cost**；目标 3 — **rebate arbitrage** 是否 **零和转移** |
| **Latency arms race**：目标 2 — **更快 price discovery** vs **资源消耗 / 不公平 access** |
| 合规 **spoofing、layering、front-running** = 目标 3 **坚决打击** 类 |

## 4. 与 Ch 8 分类的对照

| Ch 8 类型 | Ch 9 政策立场 |
|-----------|---------------|
| **效用型** | **首要保护对象** — 私人益处 + 市场存在基础 |
| **知情投机者** | 目标 3 **有条件支持** — 因 **公共信息效率** |
| **做市商 / Dealer** | 目标 3 **有条件支持** — 因 **流动性公共品** |
| **寄生型 / 操纵** | 目标 3 **打击** — 无长期公共贡献 |
| **徒劳型** | 非政策目标； **投资者保护**（披露、适格性）减少 **社会无谓损失** |

---

## 5. 本章总结

| 维度 | Takeaway |
|------|----------|
| **好市场标准** | **高流动性** + **信息效率价格** |
| **私人 vs 公共** | 效用型私人益处 **优先**；信息价格与流动性 **正外部性** |
| **一级 / 二级** | 价格影响 **融资** 与 **公司治理** — 超出 trading PnL |
| **政策优先级** | 效用型 → 全社会 → 建设性利润驱动型； **打击寄生** |

```
高流动性 ──→ 低交易成本、有效对冲、风险分担
     │
信息效率价格 ──→ 资本配置、管理者激励、并购纪律
     │
     └── 自由市场经济的「指南针」
```

> **HFT 读者 takeaway：** 不要只问「我的策略赚不赚钱」，要问 **「我属于目标 3 的哪一侧？」** — 提供 **真实 liquidity / price discovery** 与 **寄生剥削** 在监管叙事里 **待遇相反**。设计系统（低延迟、co-lo、SOR）时，**效用型 flow 是否仍愿来** 是 **市场能否持续** 的硬约束。

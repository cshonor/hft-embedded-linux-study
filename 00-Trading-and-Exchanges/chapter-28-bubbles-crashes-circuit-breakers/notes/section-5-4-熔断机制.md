## 4. 熔断机制 (Circuit Breakers)

应对极端波动的 **限制性交易规则** 总称。

### 4.1 常见类型

| 类型 | 说明 |
|------|------|
| **交易暂停 (Trading halts)** | 停盘冷却 |
| **价格限制 / 涨跌幅 (Price limits)** | 单日波动上限 |
| **提高保证金** | 抑制杠杆 |
| **交易税** | 抑制投机频率 |
| **系统访问限制 (Collars)** | 限制程序/指数套利等 |

### 4.2 支持者观点

| 理由 | 说明 |
|------|------|
| **减缓变动** | 给交易者 **分析信息** 时间 |
| **流动性入场** | 愿提供流动性的 **知情/价值** 者有机会介入 |
| **恢复机制** | 暂停后常以 **单一价格拍卖 (Uniform pricing)** 恢复 — 保护限价单免受连续暴跌 [Ch 6](../chapter-06-order-driven-markets/) |
| **清算系统** | 防 **破产违约** 连锁 |

### 4.3 反对者：引力效应 (Gravitational Effect)

| 机制 | 交易者怕 **即将触及熔断** → 流动性丧失 |
|------|----------------------------------------|
| **行为** | **争先恐后** 提前下单 |
| **结果** | 熔断可能 **增加暂时性波动** — 价格 **更快被拉向** 熔断点 |

| HFT 视角 |
|----------|
| **Volatility auction trigger** — 策略 **front-run the halt** |
| **LULD bands** — 价格 **磁吸** 阈值 |
| 回测须 **含 halt 规则**，否则 **虚假 alpha** |

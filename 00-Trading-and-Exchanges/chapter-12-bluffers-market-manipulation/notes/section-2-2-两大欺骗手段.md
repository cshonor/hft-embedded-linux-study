## 2. 两大欺骗手段 (Bluffing Techniques)

### 2.1 散布谣言 (Rumormongering)

| 手段 | 说明 |
|------|------|
| **虚假信息** | 散布 **虚假基本面** |
| **误导性真实信息** | 在 **特定环境/方式** 发布 **真实** 公共信息，故意引导 **误解** |

| HFT 视角 |
|----------|
| **Social media pump**、假新闻、选择性披露 |
| **NLP / rumor detection** 是 **防御侧** 工具；**传播速度** 决定伤害窗口 |

### 2.2 价格操纵 (Price Manipulation)

通过 **精心安排的交易** 制造 **虚假价格与成交量**（「画线 / 做盘」）：

| 手段 | 说明 |
|------|------|
| **真实交易** | 自买自卖推价、对倒营造 **虚假活跃** |
| **对敲 / 洗盘 (Wash trades)** | 与同谋 **无经济风险转移** 的往返成交 → inflate volume / price |

| 效果 | 让他人误以为 **基本面或供需** 已变 |
|------|--------------------------------------|

| HFT 视角 |
|----------|
| **Layering / quote stuffing** — 订单簿层面的 **虚假深度** |
| **Wash trade** 检测 — 交易所 **surveillance**、关联账户图谱 |
| Crypto / 低监管 venue **操纵成本更低** — microcap 现代变体 |

## 5. 监管与法律困境 (Prosecuting Market Manipulation)

| 事实 | 说明 |
|------|------|
| **非法** | 多国（含美国）**市场操纵违法** |
| **难定罪** | **极难证明** |

### 5.1 证明意图的困难

| 挑战 | 说明 |
|------|------|
| **直接证据** | 需抓到 **造谣** 或 **对敲** 等 **铁证** |
| **辩护话术** | 「我只是 **判断失误** 的合法投机者」— 「当时真以为有投资价值」 |
| **事实认定** | 法官/陪审团难区分 **错误投机** vs **恶意操纵** |

| HFT 视角 |
|----------|
| **MAR / Dodd-Frank / 各国操纵条例** — **intent + artificial price** 标准 |
| **Surveillance**：cancel rate、order-to-trade、layering pattern、**wash 关联账户** |
| **Compliance**：保留 **策略 rationale**、避免 **无经济目的** 的挂撤模式 |
| 灰色区：**合法 liquidity provision** vs **spoofing** — 行业长期诉讼焦点 |

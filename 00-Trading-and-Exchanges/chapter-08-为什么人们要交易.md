# Ch 8 为什么人们要交易 · Why People Trade

> **Trading and Exchanges** · Larry Harris · **选读** · Part II · **交易者分类总地图**

本章是理解 **市场微观结构与交易策略** 的核心基石：系统回答——**既然交易在会计意义上是零和博弈（赢家总收益 = 输家总损失），为什么还有人愿意交易并承担亏损？**

要在交易中获利，必须知道 **利润来自何处**（谁在亏、为什么亏）。作者将所有交易者分为三大类：**效用型 · 利润驱动型 · 徒劳型**。

> **HFT 读者：** OUTLINE 标 🟡，但本章是 **Part III（投机）+ Part IV（流动性）的人物图谱**——读 Ch 10–18 前建议当 **🔴 精读** 过一遍。

---

## 0. 零和博弈与「养分」

| 概念 | 含义 |
|------|------|
| **零和 (Zero-sum)** | 交易本身不创造财富；一方所得 = 另一方所失（忽略费用前） |
| **为何仍有人交易** | 多数人交易目的 **不是**「从对手方赢钱」，而是 **效用目标** 或 **自以为能赢** |
| **利润从哪来** | 知情交易者、做市商、寄生型交易者的利润 → 来自 **效用型**（愿付成本）与 **徒劳型**（系统性亏损） |

```
效用型（愿付 spread / impact 换外部收益）
    +
徒劳型（以为能赢，平均亏）
    ↓  subsidize
利润驱动型（知情 / 做市 / 寄生）
```

---

## 1. 效用型交易者 (Utilitarian Traders)

参与交易 **不是为了赚取交易利润**，而是为了 **交易之外的其他收益** 或 **解决外部问题**。他们是 **流动性最大需求者**，通常也是 **利润驱动型交易者的利润来源**。

| 类型 | 英文 | 动机 |
|------|------|------|
| **投资者与借款人** | Investors & Borrowers | 投资者：把 **现在财富** 转移到 **未来**（如退休储蓄）；借款人：把 **未来财富** 提前到现在（卖债务凭证） |
| **资产交换者** | Asset exchangers | 用持有 A 换 **即时更有用** 的 B；现货商品、 **FX** 典型 |
| **套期保值者** | Hedgers | 用远期/期货/期权 **转移、降低** 商业或金融风险 |
| **赌徒** | Gamblers | 娱乐、刺激；对不确定结果 **下注** |
| **新手/试飞者** | Fledglings | **学习** 交易、测试能否成为盈利者；初期亏损 = **学费** |
| **交叉补贴者** | Cross-subsidizers | 机构 FM 用 **较高佣金** 换 broker **研究/数据** → **软美元 (Soft dollars)** |
| **避税者** | Tax avoiders | 利用税法：**红利捕获 (Dividend capture)**、资本损失抵扣等 |

| HFT 视角 |
|----------|
| **Hedgers / asset exchangers / investors** → **自然 taker flow**；做市商赚 spread + 有时 **adverse selection** |
| **Fledglings / gamblers** → 零售流；常与 **PFOF、internalization**（Ch 7、25）相关 |
| **Cross-subsidizers** → 机构 **commission bundling**；与 **best execution / TCA** 张力 |
| 识别 **对手方动机** → 预测 **订单 persistence、urgency、对 liquidity 的支付意愿** |

## 2. 利润驱动型交易者 (Profit-Motivated Traders)

**唯一目的：赚取交易利润（低买高卖）**。若无 **合理获利理由** 就不交易。

### 2.1 投机者 (Speculators)

通过 **预测未来价格变化** 获利。

#### A. 知情交易者 (Informed Traders)

掌握并利用 **基本面价值 (Fundamental values)** 相关信息：

| 子类 | 英文 | 行为 |
|------|------|------|
| **价值交易者** | Value traders | 用 **全部公开信息** 评估 **绝对价值** → [Ch 16](./chapter-16-价值交易者.md) |
| **新闻交易者** | News traders | 利用 **最新消息** |
| **信息导向技术交易者** | Information-oriented technicians | 识别 **系统性定价错误** |
| **套利者** | Arbitrageurs | 比较 **相关工具相对价值** → [Ch 17](./chapter-17-套利者.md) |

> **知情交易推动价格向基本面靠拢 → 提高价格信息效率** → [Ch 10](./chapter-10-知情交易者与市场效率.md)

#### B. 寄生型交易者 (Parasitic Traders)

利用 **他人交易行为** 获利；**既不提供流动性，也不提高信息效率**：

| 子类 | 英文 | 行为 |
|------|------|------|
| **指令预期者** | Order anticipators | 抢先交易者、挤空者等 → [Ch 11](./chapter-11-指令预期者.md) |
| **虚张声势者** | Bluffers | 造谣、价格操纵 → [Ch 12](./chapter-12-虚张声势者与市场操纵.md) |

| HFT 视角 |
|----------|
| **Informed / arb** → alpha 来自 **信息速度 + 建模**；latency 竞争在 **新闻、相对定价** |
| **Parasitic** → **front-run、quote fade、spoofing** 监管边界；与 Ch 6 **规则权力**、Ch 7 **front running** 衔接 |
| 现代 HFT 常 **跨类**：做市 + 短期 informed；须分清 **哪些 PnL 来自 liquidity provision vs information** |

### 2.2 做市商/交易商 (Dealers)

向 **急于交易** 的人 **提供流动性（即时性）**，赚 **买卖价差 (Bid-ask spread)** → **被动流动性提供者** → [Ch 13](./chapter-13-做市商.md) · [Ch 14](./chapter-14-买卖价差.md)

| HFT 视角 |
|----------|
| 电子 LOB 上 **limit order 做市** = dealer 功能的 **去中介化** |
| **Rebate / maker-taker** 是 spread 的 **费用表显式化** |

## 3. 徒劳型交易者 (Futile Traders)

**期望获利**，但因 **缺乏技能、不理智或被误导**，**平均而言总是亏损**。

| 特征 | 说明 |
|------|------|
| 混淆期望与结果 | 以为「应该赢」= 会赢 |
| 不可靠代理人 | 依赖 **差 broker / 差建议** |
| 过时信息 | **伪知情 (Pseudo-informed)** — 以为有 edge，实际没有 |

> **若无效用型 + 徒劳型持续提供「养分」，利润驱动型之间纯零和 → 无法长期共存。**

| HFT 视角 |
|----------|
| **Edge 自检**：你的 alpha 是 **真 informed**，还是 **从 futile flow 收税**？后者可能 **随教育/监管萎缩** |
| **Pseudo-informed** → 追涨杀跌、滞后于 public news 的 **retail 策略** |

## 4. 重要启示

### 4.1 多重身份的陷阱 (Multiple Identities)

许多交易者 **混淆身份**：

| 本意 | 实际行为 | 风险 |
|------|----------|------|
| **投资**（转移财富） | 无优势地 **投机** | 承担不必要 variance |
| **效用**（对冲） | 为 **刺激** 赌博 | 对冲成本 + 赌博损失 |
| **学习**（fledgling） | 永不止损 | 学费失控 |

> **认清真正动机** = 避免糟糕决策的前提。

### 4.2 流动性的重要性

除 **Dealers**（常从 **低流动性、宽价差** 获利）外，**几乎所有其他类型都偏好高流动性市场**——以 **更低成本** 达成效用或获利目标 → [Ch 9 好市场](./chapter-09-好市场.md) · [Ch 19 流动性](./chapter-19-流动性.md)

| HFT 视角 |
|----------|
| HFT **提供** 流动性（narrow spread）也 **消耗** 流动性；生态健康需要 **utilitarian + futile takers** |
| **Liquidity regulation**（tick size、maker obligation）影响 **谁从 whom 赚钱** |

## 5. 分类总图

```
                    为什么交易？
                         │
        ┌────────────────┼────────────────┐
        ▼                ▼                ▼
   效用型            利润驱动型          徒劳型
 (外部效用)          (要赚交易利润)      (以为能赢，平均亏)
        │                │
        │         ┌──────┴──────┐
        │         ▼             ▼
        │    投机者          做市商/Dealer
        │    ├ 知情          (赚 spread)
        │    └ 寄生
        │
   八种 utilitarian → 愿付成本 → 市场的「养分」
```

| 谁赚钱 | 典型来源 |
|--------|----------|
| **知情交易者** | 效用型 + 徒劳型 + 部分 **反应滞后的寄生者** |
| **做市商** | **急切 taker**（含 utilitarian urgency）的 spread |
| **寄生型** | **可预测的大单 / 慢反应者**（道德与合规边界内讨论） |

## 6. 本章总结

| 要点 | 含义 |
|------|------|
| **零和 ≠ 无交易** | 多数人交易为 **效用**，不是为 **对手方 PnL** |
| **Taxonomy** | Utilitarian / Profit-motivated / Futile — **全书 Part III–IV 索引** |
| **利润来源** | 想知道谁亏 → 先看 **动机**，再看 **技能** |
| **身份诚实** | 投资/对冲/赌博/投机 **勿混** |
| **流动性** | 几乎人人要；dealer 是例外（宽价差环境） |

> **HFT 读者 takeaway：** 设计策略前先问——**我在赚哪一类交易者的钱？他们为什么还在来？** 若答案只有「其他 HFT」，那是 **同质零和**，须 **更快 / 更便宜 / 更合规**；若答案是 **hedger、rebalancer、futile retail**，要评估 **flow 是否可持续、是否会被 internalization 抽走**。
